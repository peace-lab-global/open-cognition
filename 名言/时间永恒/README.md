# 时间·永恒 Time & Eternity

时间的流逝、当下的力量、变化的本体论。

## 核心命题

> 时间是人类存在最根本的境况——**不可逆、不可持、不可逃避**。对时间的三种经典态度：警醒（孔子）、接受（奥勒留）、思辨（赫拉克利特）。

## 三条时间路径

### 一、孔子：伦理的时间
临河叹逝——时间流逝不可逆，所以**自强不息**。时间是道德行动的场域。

### 二、奥勒留：接受的时间
当下已足——不必追悔过去、焦虑未来，**守住此刻**。时间是斯多葛的操练。

### 三、赫拉克利特：思辨的时间
万物皆流——同一性与变化并存，**人不能两次踏入同一条河**。时间是本体论的问题。
