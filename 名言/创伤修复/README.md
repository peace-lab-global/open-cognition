# 创伤修复 (Trauma & Healing)

> 创伤不只是记忆里的过去，更是身体里的现在；治愈也不只是“想通”，而是重新感到安全。

本主题收录三条关于创伤与修复的经典理解：

| 名言 | 作者 | 核心命题 |
|------|------|---------|
| [创伤不只是过去发生的事件，更是留在心智、大脑与身体中的印记](范德科尔克-身体从未忘记.md) | Bessel van der Kolk · 2014 | 创伤的躯体化编码 |
| [创伤事件质疑基本的人类关系](赫曼-创伤与复原.md) | Judith Herman · 1992 | 创伤的关系破裂与修复 |
| [创伤是生活的事实，但不必是终身的判决](莱文-唤醒老虎.md) | Peter A. Levine · 1997 | 完成冻结的防御反应 |

---

## 三者关系

- **范德科尔克**从**神经生物学**切入：创伤以印记形式铭刻在大脑与身体中，传统谈话疗法必须辅之以身体觉察。
- **赫曼**从**关系与社会**切入：创伤破坏依恋、自我与意义，复原需要安全关系、社群见证与正义。
- **莱文**从**躯体治疗**切入：创伤源于未完成的防御反应，身体本身具有完成与释放的先天智慧。

三者共同追问：当痛苦不只停留在故事里，我们如何与身体、关系和社会一起重建完整？
