# 梦境·无意识 (Dream & the Unconscious)

> 梦不是清醒的残余，梦是心灵的**另一种真实**——被压抑的、原型的、物化的。

---

## 核心命题

**梦的三种现代-古代解读。**

| 思想家 | 年代 | 解读 |
|--------|------|------|
| 庄子 | 前4世纪 | **梦蝶**——梦与醒的界限模糊（物化） |
| 弗洛伊德 | 1900 | 梦是**被压抑欲望**的伪装满足 |
| 荣格 | 1964 | 梦是**集体无意识原型**的诚实语言 |

三条名言横跨 2300 年、三种文明，共同质疑"醒 = 真实、梦 = 虚幻"的常识。

## 思想脉络

```

  庄子 (前4世纪) 梦与觉不可分
  亚里士多德 梦是身体刺激的残余
   ↓
弗洛伊德 (1900) 梦是伪装的欲望
   ↓
荣格 (1912起) 梦是原型的语言
   ↓

   ↓

  激活-合成假说 (Hobson, 1977)

```
