# 技术·人性 Technology & Humanity

技术是遮蔽（海德格尔）、系统（埃吕尔）、还是新主体（哈拉维）？

## 核心命题

> 技术不是中性工具——它是**本体论的开显方式**（海德格尔）、**自主运作的系统**（埃吕尔）、**人机混合的新可能**（哈拉维）。三位思想家从悲观到乐观，描绘技术的三重面孔。

## 技术的三种姿态

### 一、海德格尔（1953）：悲观 · 本体论
技术的本质是**座架** (Gestell)——一种把世界展现为"持存物"(Bestand) 的方式。森林 = 木材资源，河流 = 水电潜能，人 = 人力资源。**技术遮蔽了其他开显方式**（诗意的、神圣的、冥思的）。

### 二、埃吕尔（1954）：悲观 · 社会学
大写 **Technique** 是"所有追求效率的人类活动的总和"——不仅包括机器，也包括官僚、管理、广告、教育。技术已成**自主系统**——按自身逻辑（效率最大化）发展，不受人的意志控制。

### 三、哈拉维（1985）：乐观 · 女性主义
赛博格 (cyborg) = cybernetic organism——人/机器、自然/文化的混合体。**我们已经都是赛博格**——有心脏起搏器、智能手机、社交媒体身份。技术可以让边缘群体**摆脱生物宿命**。
