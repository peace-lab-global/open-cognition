# 信息茧房 (Filter Bubbles)

> 当算法替我们决定“世界是什么样”，我们还能否看见彼此？

本主题收录三条关于数字时代认知封闭的经典理解：

| 名言 | 作者 | 核心命题 |
|------|------|---------|
| [民主需要公民从彼此角度看世界，而我们却被困在自己的气泡里](派瑞泽-过滤气泡.md) | Eli Pariser · 2011 | 过滤气泡与民主危机 |
| [监控资本主义将人类经验单方面宣称为免费原材料](祖博夫-监控资本主义.md) | Shoshana Zuboff · 2019 | 行为数据作为新积累逻辑 |
| [网络的连接不是我们的连接](卡尔-浅薄.md) | Nicholas Carr · 2011 | 互联网对深度认知的削弱 |

---

## 三者关系

- **派瑞泽**从**公共领域**切入：个性化算法瓦解了民主所需的他者视角与共同事实基础。
- **祖博夫**从**政治经济**切入：平台以“免费”为名，将人类经验开采为可交易的行为预测能力。
- **卡尔**从**认知方式**切入：无限链接与外部记忆外包，正在削弱我们深度理解与整合经验的能力。

三者共同追问：在算法日益中介我们认知的时代，如何保卫注意力、隐私与公共对话的完整性？
