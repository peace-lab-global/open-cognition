# 微观·宏观 Micro-Macro

尺度的坍缩、感知的穿透、微观中的无限。

## 核心命题

> 真正的感知能穿透尺度——**一沙中见世界，一花中见天堂**。

---

## 本主题名言

| 名言 | 作者 | 尺度反差 |
|------|------|---------|
| [一沙一世界，一花一天堂](./布莱克-一沙一世界.md) | Blake · c. 1803 | 沙粒 ↔ 世界，野花 ↔ 天堂，无限 ↔ 掌心，永恒 ↔ 一小时 |

---

## 思想脉络

布莱克的四行诗是英语文学中最常被引用的诗句之一，其力量在于**极端的尺度反差**：

- 沙粒（最小）↔ 世界（最大）
- 野花（最寻常）↔ 天堂（最神圣）
- 掌心（有限）↔ 无限
- 一小时（短暂）↔ 永恒

这不是修辞的夸张，而是**认知方式的宣示**：真正的感知能穿透尺度，从微观直抵宏观，从瞬间直抵永恒。
