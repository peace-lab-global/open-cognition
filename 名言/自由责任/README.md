# 自由·责任 Freedom & Responsibility

自由的不可避免、难以承受、及其与责任的对价。

## 核心命题

> 自由是人的本体境况——**无法逃避，又难以承受**。三位思想家在一个世纪内（1844–1943）完成了对自由的三重深化：现象学（克尔凯郭尔）、心理学（陀思妥耶夫斯基）、本体论（萨特）。

## 自由哲学的一百年

### 一、克尔凯郭尔（1844）：现象学
"焦虑是自由的眩晕"——站在悬崖边，你不仅害怕掉下去，更害怕**自己可能会跳**。焦虑的对象不是外在，而是**自己的可能性**。

### 二、陀思妥耶夫斯基（1864）：心理学
"人最需要自由，却最不堪其负担"——《地下室手记》与《卡拉马佐夫兄弟》中"大审判官"寓言的预演：人宁可要面包、奇迹、权威，也不要自由。

### 三、萨特（1943）：本体论
"人被判处自由"——自由不是祝福，而是**刑罚**。人不能不自由：即使"不选择"，也是在选择。

## 当代应用

- **政治哲学**：极权主义为何反复出现？（弗洛姆、阿伦特）
- **心理治疗**：存在主义治疗的核心——面对自由的焦虑（亚隆）。
- **消费社会**：购物不是自由，而是自由的逃避。
- **教育**：学生选择专业的焦虑——不是信息不足，而是自由的重量。
