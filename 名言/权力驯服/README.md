# 权力·驯服 Power & Discipline

现代权力的三副面孔：制度的、弥散的、同意的。

## 核心命题

> 现代社会的权力不再只是君主的暴力——它通过**制度**（韦伯）、**监视**（福柯）、**文化**（葛兰西）运作，让人**自愿或内化**地被驯服。

## 现代权力的三重面孔

### 一、韦伯（1905）：铁笼
理性化把人关进**官僚制**的笼子——效率、计算、程序成为新神。人成为"机器上的齿轮"。

### 二、福柯（1975）：全景敞视
权力不再在某处，而在**处处**——学校、工厂、医院、监狱皆为规训机构。人**自我监视**，把权力内化。

### 三、葛兰西（1929–35）：文化霸权
统治不仅靠暴力，更靠**同意**——通过教育、媒体、宗教塑造"常识"，让被统治者**自愿接受**现存秩序。
