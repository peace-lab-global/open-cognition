# 消费·占有 (Consumption & Possession)

> 我们需要多少才够？占有带来安全，还是制造焦虑？

本主题收录三条关于消费与占有的经典反思：

| 名言 | 作者 | 核心命题 |
|------|------|---------|
| [知足不辱](老子-知足不辱.md) | Laozi · 前6世纪 | 知止才能长久 |
| [渴望越多越贫穷](塞涅卡-渴望越多越贫穷.md) | Seneca · 1 世纪 | 贫穷是欲望函数 |
| [消费社会的符号](鲍德里亚-消费社会的符号.md) | Baudrillard · 1970 | 消费是符号与社会区分 |

---

## 三者关系

- **老子**从**个人修养**出发：知足、知止，才能避免占有带来的反噬。
- **塞涅卡**从**心理机制**出发：贫穷不是拥有太少，而是渴望太多。
- **鲍德里亚**从**社会结构**出发：消费不是满足需要，而是生产意义与阶层差异。

古代智慧与现代批判相互映照，共同追问：在丰裕社会中，人如何活得轻盈？
