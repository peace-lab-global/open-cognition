# 科技伦理 (Technology Ethics)

> 当技术能重写生命、气候与记忆，谁来为不可见的后果负责？

本主题收录三条关于科技伦理的经典理解：

|------|------|---------|

| [有些事情，计算机不该被用来完成](维森鲍姆-不该让计算机做的事.md) | Joseph Weizenbaum · 1976 | 判断不可还原为计算 |
| [我们对技术期待更多，对彼此期待更少](特克尔-对技术期待更多.md) | Sherry Turkle · 2011 | 数字时代的亲密萎缩 |

---

## 三者关系

- **约纳斯**从**宏观责任**切入：技术力量已超越传统伦理的时空边界，必须以未来生命为判准。
- **维森鲍姆**从**任务边界**切入：即便计算机能做某事，也有些事情本质上不应由它来做。
- **特克尔**从**关系后果**切入：技术不仅改变工具，更改变了人对亲密、孤独与他人的期待。

三者共同追问：在能做的事越来越多的时代，人类如何守住“不该做”与“不必做”的界限？
