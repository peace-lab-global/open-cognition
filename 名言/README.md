## 文件规范

每文件包含：
- **YAML frontmatter**：id、title、type=quote、author、era、domain、tags、cross-refs（指向知识库思想家）、related_quotes（指向同模块其他名言）
- **原文**：保留原文语言（de/fr/fa/en/zh-Hant/grc/da/ru 等）+ 中文通行译本
- **出处**：作者、作品、年份、翻译者
- **解读**：多层分析（字面 / 张力 / 哲学 / 应用）
- **思想谱系**：与其他思想传统的对照表
- **延伸阅读**：原典 + 二手文献
