## 自媒体应用指南

本模块的每条名言均按「高可用语料」标准制作，可直接用于公众号、短视频、播客、朋友圈、课程讲义等场景。

### 一、单条名言的三种用法

每个文件末尾提供三类可直接取用的素材：

| 类型 | 用途 | 示例（帕斯卡） |
|------|------|---------------|
| **封面句** | 文章/视频主标题、首屏大字 | 宇宙不回答你，但你依然可以提问。 |
| **短视频 Hook** | 前 3 秒抓注意力 | 1660 年的某个夜晚，帕斯卡抬头看星空，写下了现代人最隐秘的恐惧。 |
| **朋友圈文案** | 短图文、金句卡片 | 当你觉得孤独，请记住：这种孤独从帕斯卡仰望星空时就开始了。 |

### 二、主题组合：三种内容结构

1. **单一深度型**：选一条名言，从原文、出处、解读到当代应用完整展开，适合长图文 / 播客单集。
2. **对话张力型**：选同一主题下的两条对立名言（如尼采 vs. 萨松、爱比克泰德 vs. 尼采），制造思辨冲突，适合短视频与辩论式内容。
3. **跨文化矩阵型**：从「跨文化对话矩阵」中取一行（如「生死观」），横向对比海德格尔、庄子、亚隆，适合系列内容 / 知识图谱。

### 三、主题组合包：即拿即发的系列素材

`_packs/` 目录下提供围绕具体场景/节日/情绪设计的组合包，每个包含：

- 主题定位与适用平台
- 3–7 条精选名言及排期
- 每条的核心钩子、视觉提示、发布场景
- 统一的视觉风格建议
