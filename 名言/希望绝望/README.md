# 希望·绝望 (Hope & Despair)

> 希望是生存的本能，还是一种精致的麻醉？绝望是终点，还是清醒的入口？

本主题收录三条关于希望与绝望的经典声部：

| 名言 | 作者 | 核心意象 |
|------|------|---------|
| [希望留在盒底](赫西俄德-潘多拉的盒子.md) | Hesiod · c. 700 BCE | 希望作为神赐的余存，暧昧而古老 |
| [绝望之为虚妄](鲁迅-绝望与希望.md) | Lu Xun · 1925 | 取消希望与绝望的二元，以行动抵抗虚无 |
| [希望是长着羽毛的东西](狄金森-希望是长着羽毛的东西.md) | Dickinson · c. 1861 | 希望如鸟，无词而持续地歌唱 |

---

## 三者关系

- **赫西俄德**给出**神话原型**：希望是灾难之后唯一剩下的东西，但其性质至今争论不休。
- **鲁迅**给出**现代冷峻**：希望与绝望都是简化，真正的力量在于「反抗绝望」的持续行动。
- **狄金森**给出**诗性肯定**：希望不必宏大，它是灵魂里一只轻盈、执拗、不停歌唱的鸟。

从神话到诗歌，从古代到现代，三者共同追问：当黑夜足够长，人凭什么继续相信天明？
