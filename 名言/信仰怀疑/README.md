# 信仰·怀疑 (Faith & Doubt)

> 信仰不是理性的结论，怀疑也不是理性的胜利——二者在人的存在深处**永恒交战**。

---

## 核心命题

**信仰的三种不可能性。**

| 思想家 | 年代 | 立场 |
|--------|------|------|
| 克尔凯郭尔 | 1843 | 信仰是**荒谬的跳跃**（反黑格尔） |
| 帕斯卡 | 1670 | 理性无法证明，但**必须下注**（决策论） |
| 乌纳穆诺 | 1913 | 心渴望不朽，理性证明必死——**悲剧感** |

三位思想家都承认：**理性无法抵达信仰**，但又都拒绝放弃信仰。这种张力本身成为信仰的现代形式。

## 思想脉络

```
奥古斯丁 (4世纪) 信仰寻求理解 (fides quaerens intellectum)
   ↓
安瑟伦 (11世纪) 本体论证明
   ↓
阿奎那 (13世纪) 五路证明
   ↓ 反动
帕斯卡 (1670) 理性无法证明，必须下注
   ↓
克尔凯郭尔 (1843) 信仰作为荒谬的跳跃
   ↓
尼采 (1882) 上帝之死
   ↓
乌纳穆诺 (1913) 心与理性的悲剧张力
   ↓
巴特 (1919) 启示作为垂直降临
   ↓
当代 世俗化、新无神论、灵性而非宗教 (SBNR)
```

| 传统 | 命题 |
|------|------|
| 亚伯拉罕传统 | 信仰作为盟约 |

| 克尔凯郭尔 | 信仰作为荒谬的跳跃 |
| 乌纳穆诺 | 心与理性的永恒交战 |

| 当代 | 世俗化、SBNR、新无神论 |
