# 故乡·流浪 (Home & Wandering)

> 当故乡变得可以离开、可以消费、可以退订，人还能在哪里扎根？

本主题收录三条关于故乡与流浪的经典理解：

| 名言 | 作者 | 核心命题 |
|------|------|---------|
| [诗人留下永恒](荷尔德林-诗人留下永恒.md) | Hölderlin · 1803 | 诗是故乡的保存者 |
| [地上本没有路](鲁迅-地上本没有路.md) | Lu Xun · 1921 | 故乡/路在行走中被创造 |
| [没有目的地的朝圣者](鲍曼-没有目的地的朝圣者.md) | Bauman · 2000 | 液态现代性让故乡瓦解 |

---

## 三者关系

- **荷尔德林**代表**浪漫主义的乡愁**：故乡可以被诗保存，成为永恒的意义之所。
- **鲁迅**代表**现代主义的开创**：故乡不是现成的，而是一群人用脚走出来的。
- **鲍曼**代表**后现代性的漂泊**：当一切流动，故乡作为一种稳定的归属已经难以维系。

从保存到创造，再到瓦解，三者共同追问：在流动时代，人如何安放自己的乡愁？
