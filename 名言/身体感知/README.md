# 身体·感知 (Body & Perception)

> 身体不是灵魂的容器，身体就是主体；知觉不是心灵的被动接收，知觉就是存在本身。

---

## 核心命题

**身体是认识的第一场所。**

西方自笛卡尔以降，长期将身体视为心灵的客体——"我思"是无体的、透明的、普遍的。本主题的三条名言从**现象学、谱系学、佛教**三个方向同时反驳这种二元论：

- **梅洛-庞蒂**：我就是我的身体（现象学的具身主体）
- **尼采**：没有事实，只有视角（认识的身体性）
- **佛陀**：在所见中唯有所见（知觉的离分别）

三条名言汇合于一个结论：**知觉先于概念，身体先于反思，直接认知先于对象化知识。**

   ↓ 反动

   ↓

   ↓

|------|----------|
