# 他者·伦理 (Otherness & Ethics)

> 伦理不是从"我应该如何对待他者"出发，而是从"他者的脸如何召唤我"出发。

---

## 核心命题

**伦理学是第一哲学。**

| 传统 | 起点 | 核心动作 |
|------|------|----------|
| 列维纳斯（法国） | **他者的脸** | 脸说"你不可杀人" |
| 布伯（犹太-对话） | **我-你相遇** | 一切真实的生活皆是相遇 |
| 孔子（儒学） | **恕道** | 己所不欲，勿施于人 |

三条名言横跨 2500 年、三大文明（希伯来-希腊-中国），却指向同一件事：**自我不是孤立的主体，而是被他者召唤、与他者相遇、向他者负责的存在。**

## 思想脉络

```
孔子 (前500) 恕道：己所不欲，勿施于人
   ↓
耶稣 (前4) 积极黄金法则："你们愿意人怎样待你们..."
康德 (1785) 定言命令 (Categorical Imperative)
   ↓ 反动
布伯 (1923) 我-你作为相遇
   ↓
列维纳斯 (1961) 脸作为伦理召唤
   ↓
德里达、巴特勒、南希：当代他者伦理学
```

## 跨文化对照

| 传统 | 黄金法则形式 |
|------|--------------|
| 儒学 | 消极：己所不欲，勿施于人（恕） |
| 基督教 | 积极：你们愿意人怎样待你们…（《马太福音》7:12） |
| 犹太教 | 希列尔："己所憎恶，勿施于人——这就是全部托拉" |
| 伊斯兰 | "你们中任何人，不为兄弟喜爱他为自己所喜爱的，就不是信士" |
| 印度教 | "不应以己所不悦之事加于他人"（《摩诃婆罗多》） |
| 康德 | 定言命令：只按照你愿意它成为普遍法则的准则行动 |
| 列维纳斯 | 脸先于一切法则 |
