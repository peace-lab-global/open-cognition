# 真实·表演 (Reality & Performance)

> 如果人生就是表演，那真实在哪里？面具之下，还有什么？

本主题收录三条关于真实与表演的经典理解：

| 名言 | 作者 | 核心命题 |
|------|------|---------|
| [人生如戏](戈夫曼-人生如戏.md) | Goffman · 1959 | 自我是社会互动的表演效果 |
| [全世界是一个舞台](莎士比亚-全世界是一个舞台.md) | Shakespeare · c. 1600 | 人生如戏的文学原型 |
| [人格面具](荣格-人格面具.md) | Jung · 1921 | 面具之下有阴影与自性 |

---

## 三者关系

- **莎士比亚**给出**宇宙舞台的隐喻**：人生有开场、角色、高潮与落幕。
- **戈夫曼**给出**社会学的分析**：自我如何通过前台/后台、印象管理被建构。
- **荣格**给出**心理学的纵深**：面具之下，还有阴影、自性，以及未被表演的真实。

从文学到社会学，再到深度心理学，三者共同追问：在处处需要表演的世界里，我们如何与不表演的部分保持联系？
