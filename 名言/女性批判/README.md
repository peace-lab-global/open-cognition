# 女·性批判 (Feminist Critique)

> 女人不是天生的，而是**变成的**（波伏娃）；写作需要**一间自己的房间**（伍尔夫）；但用**主人的工具**永远拆不掉主人的房子（洛尔德）。

---

## 核心命题

**女性批判的三个层次。**

| 思想家 | 年代 | 层次 |
|--------|------|------|
| 波伏娃 | 1949 | **存在论**——女人是被建构的"第二性" |
| 伍尔夫 | 1929 | **物质论**——经济与空间是写作的前提 |
| 洛尔德 | 1979 | **交叉性**——差异不是缺陷，而是变革的资源 |

三条名言横跨半个世纪，从**成为女人**到**拥有房间**再到**拆解主人的工具**，构成女性主义批判的核心路径。

## 未来可扩展

- **朱迪斯·巴特勒**（Judith Butler）：性别麻烦（*Gender Trouble*, 1990）
- **贝尔·胡克斯**（bell hooks）：女性主义是给所有人的（*Feminism is for Everybody*, 2000）
- **唐娜·哈拉维**（Donna Haraway）：赛博格宣言
- **金伯利·克伦肖**（Kimberlé Crenshaw）：交叉性
- **上野千鶴子**：结构主义的厌女（ミソジニー）
