# 自然·文明 Nature & Civilization

自由如何失落、自然/文化如何二分、现代性如何背离传统。

## 核心命题

> 自然与文明的关系是思想史上最持久的命题之一——**历史性的过渡**（卢梭）、**结构性的二分**（列维-斯特劳斯）、**形而上的背离**（盖侬）。三位思想家从三个维度切入。

## 三种"自然/文明"视角

### 一、卢梭（1762）：历史的辩证
人"生而自由"——自然状态的理想化。"无往不在枷锁中"——社会的限制。**方案**：通过社会契约，建立新的、道德的自由（自我立法）。

### 二、列维-斯特劳斯（1949）：结构的分析
自然/文化不是**历史过渡**，而是**心灵结构**。**乱伦禁忌**是自然到文化的"门槛"——既是生物规则（自然），又是社会规则（文化）。

### 三、盖侬（1927）：形而上的背离
现代世界是对"传统"（永恒的形而上学真理）的**背离**——量高于质、物质高于精神、个人高于普遍。现代是 Kali Yuga（黑暗时代）的末段。
