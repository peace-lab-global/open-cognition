# 教育·启蒙 (Education & Enlightenment)

> 教育是**加法**（敢于求知、批判意识），还是**减法**（不著一物）？两条路都通往自由，但姿态截然相反。

---

## 核心命题

**教育的两种根本姿态。**

| 思想家 | 年代 | 姿态 |
|--------|------|------|
| 康德 | 1784 | **敢于求知**——勇敢运用自己的理性（加法） |
| 弗莱雷 | 1968 | **提问式教育**——从存款式到对话式（加法，但批判性） |
| 黄檗希运 | 9世纪 | **不著一物**——破除一切见解（减法） |

三条名言横跨 1300 年、三大文明，展现"启蒙"的两种根本路径：**理性的扩张** vs **执著的消解**。

## 思想脉络

```
孔子 (前500) 学而不厌、诲人不倦
苏格拉底 (前399) 助产术 (maieutics)
   ↓
康德 (1784) 启蒙 = 敢于运用自己的理性
   ↓
黑格尔 理性的历史化
马克思 启蒙作为意识形态
   ↓
杜威 (1916) 教育即生活
   ↓
弗莱雷 (1968) 教育作为解放实践
   ↓
当代 翻转课堂 / 批判教育学
```

**东方路径：**
```
慧能 (670s) 无住生心
   ↓
黄檗 (9世纪) 不著一物即是正见
   ↓
临济 (9世纪) 无事是贵人
   ↓
当代 反教育主义 / 禅宗式"减法"
```
