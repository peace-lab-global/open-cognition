# 快慢·节奏 (Speed & Rhythm)

> 当加速成为信仰，慢是否还有价值？当信息淹没叙事，我们如何重新找回节奏？

本主题收录三条关于时间、速度与注意力的现代诊断：

| 名言 | 作者 | 核心命题 |
|------|------|---------|
| [没时间写短信](帕斯卡-没时间写短信.md) | Pascal · 1656 | 简洁是时间的产物 |
| [讲故事的艺术正在终结](本雅明-讲故事的艺术正在终结.md) | Benjamin · 1936 | 慢叙事被信息取代 |
| [倦怠社会](韩炳哲-倦怠社会.md) | Han · 2010 | 功绩主体自我剥削 |

---

## 三者关系

- **帕斯卡**揭示**时间稀缺**：不是不想简洁，是没时间为简洁付出代价。
- **本雅明**揭示**经验消亡**：故事需要时间、记忆与共同体，而现代只生产即时信息。
- **韩炳哲**揭示**自我剥削**：自由变成了一种命令，让人在积极中不断加速，直到燃尽。

从 17 世纪到 21 世纪，三者共同追问：我们能否重新学会慢下来？
