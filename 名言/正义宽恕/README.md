# 正义·宽恕 (Justice & Forgiveness)

> 正义是各得其所，还是各安其位？报复能否抵达正义？宽恕是软弱还是力量？

本主题收录三条关于正义与宽恕的经典姿态：

| 名言 | 作者 | 核心命题 |
|------|------|---------|
| [正义是各司其职](柏拉图-正义是各司其职.md) | Plato · c. 375 BCE | 正义作为秩序与和谐 |
| [以眼还眼](甘地-以眼还眼.md) | Gandhi · c. 1940s | 报复的无限循环与非暴力替代 |
| [正义的延迟](马丁路德金-正义的延迟.md) | MLK · 1963 | 时间就是正义的维度 |

---

## 三者关系

- **柏拉图**定义**古典正义**：各安其位，秩序和谐。
- **甘地**追问**实现正义的手段**：以伤害回应伤害，最终两败俱伤。
- **马丁·路德·金**追问**正义的时间性**：拖延不是耐心，而是对被压迫者的持续伤害。

从「什么是正义」到「如何实现正义」再到「何时实现正义」，三者构成完整的正义追问。
