# 美·丑 (Beauty & Ugliness)

> 美不是善的附庸，丑不是美的反面——美与丑在更深的层面上互为条件。

---

## 核心命题

**美与丑的三种现代翻转。**

| 思想家 | 年代 | 翻转 |
|--------|------|------|
| 波德莱尔 | 1857 | 恶可以开出花（炼金术） |
| 王尔德 | 1893 | 诱惑不可抵御（悖论） |
| 庄子 | 前4世纪 | 大美不言（自然美学） |

三条名言从**炼金术、悖论、自然**三个方向，解构传统"美=善"的等式。

## 思想脉络

```
柏拉图 美 = 善 = 真
   ↓ 反转
波德莱尔 (1857) 恶可以成为美的原料
   ↓
王尔德 (1893) 唯美主义：美高于道德
   ↓
尼采 (1872) 悲剧精神
   ↓
本雅明 / 阿多诺 现代性美学
   ↓
当代 丑学、怪诞、反美学
```

## 跨文化对照

| 传统 | 命题 |
|------|------|
| 柏拉图传统 | 美 = 善 = 真 |
| 波德莱尔/现代主义 | 恶之花（丑中的美） |
| 道家（庄子） | 大美不言（自然之美） |
| 日本美学 | 侘寂（wabi-sabi）——残缺中的美 |
| 印度美学 | 九种 Rasa（包括恐怖、厌恶） |
| 当代 | 丑学、怪诞美学、反美学 |
