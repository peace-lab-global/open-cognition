# 自·我超越 (Self-Transcendence)

> 人是一条**绳索**，连接动物与超人；是一次**高峰**，自我消融于整体；是一颗**尽心**，与天道贯通。

---

## 核心命题

**自我超越的三种路径。**

| 思想家 | 年代 | 路径 |
|--------|------|------|
| 尼采 | 1883 | **超人**——人是应当被克服的东西（意志的向上） |
| 孟子 | 约前 300 | **尽心知天**——尽心、知性、知天（道德的上达） |
| 马斯洛 | 1964 | **高峰体验**——自我消融于整体（心理学的超越） |

三条名言横跨 2300 年、三大文明，展现"超越"的三种方向：**意志的向上**、**道德的上达**、**心理的融解**。

## 思想脉络

```
柏拉图 灵魂向上攀升至善的理念
亚里士多德 实现 (entelecheia) / 沉思生活
   ↓
孟子 (前300) 尽心、知性、知天
宋明理学 存天理、灭人欲
   ↓
基督教神秘主义 (艾克哈特) 灵魂的火花
   ↓
尼采 (1883) 超人是大地之义
   ↓
海德格尔 本真存在
   ↓
弗兰克尔 自我超越 (Self-Transcendence)
   ↓
马斯洛 (1964) 高峰体验 / 存在心理学
   ↓
当代 超越心理学 / 冥想研究
```
