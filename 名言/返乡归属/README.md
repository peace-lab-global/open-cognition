# 返乡·归属 (Homecoming & Belonging)

> 现代人的根本困境不是贫穷，而是无家可归——不是没有房子，而是丧失了栖居的能力。

---

## 核心命题

**扎根是灵魂的需要，诗意是栖居的度量。**

| 思想家 | 年代 | 命题 |
|--------|------|------|
| 荷尔德林 | 1804 | 充满劳绩，**然而**人诗意地栖居 |
| 海德格尔 | 1951 | 诗意地栖居 = 在四重整体中**让存在** |
| 薇依 | 1943 | 扎根是灵魂最本质**又最被忽视**的需要 |

三条名言从**诗（荷尔德林）→ 哲学（海德格尔）→ 政治伦理（薇依）** 三个层面回应同一个现代病症：**拔根（Entwurzelung / déracinement）**。

## 张力

**海德格尔 vs 薇依**：
- 海德格尔的栖居是**诗性的、存在论的、几乎贵族的**——少数人能诗意栖居。
- 薇依的扎根是**政治的、伦理的、平民的**——每个工人、农民、难民都需要根。

**道家 vs 两者**：
- 庄子"乘物以游心"——既非栖居也非扎根，而是**游**。这是更彻底的解脱，还是更深的流浪？
