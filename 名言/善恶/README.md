# 善·恶 (Good & Evil)

> 善是**天赋的种子**（恻隐之心），还是**平庸之恶的缺席**？恶是**根本的**（radical），还是**思想的缺乏**（banal）？

---

## 核心命题

**善恶的三种诊断。**

| 思想家 | 年代 | 诊断 |
|--------|------|------|
| 孟子 | 约前 300 | **性善**——恻隐之心，人皆有之（善是天赋之端） |
| 陀思妥耶夫斯基 | 1880 | **若无上帝**——若无超越性根基，一切都被许可（善的根基危机） |
| 阿伦特 | 1963 | **平庸之恶**——恶不是根本的，而是思想的缺席（恶的平庸性） |

三条名言横跨 2300 年，从**善的天赋**到**善的危机**再到**恶的平庸**，构成对伦理学最尖锐的追问。

## 思想脉络

```
孟子 (前300) 性善：恻隐、羞恶、辞让、是非四端
   ↓
荀子 (前3世纪) 性恶：人之性恶，其善者伪也
   ↓
奥古斯丁 (397) 恶是善的匮乏 (privatio boni)
   ↓
康德 (1793) 根本的恶 (das radikale Böse)
   ↓
陀思妥耶夫斯基 (1880) 若无上帝，一切都被许可
   ↓
尼采 (1886) 超越善与恶
   ↓
阿伦特 (1963) 平庸之恶 (banality of evil)
   ↓
当代 道德心理学 (Philip Zimbardo / 斯坦福监狱实验)
```
