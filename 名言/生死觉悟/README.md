# 生死·觉悟 Life, Death & Awakening

面对死亡、超越生死、在荒诞中创造意义。

## 核心命题

> 死亡是人类存在最不可逃避的境况——对死亡的三种态度：**本真**（海德格尔）、**超越**（庄子）、**反抗**（加缪）。

## 三种面对死亡的态度

### 一、海德格尔（1927）：本真
"向死而在"——死亡是最本己的、不可替代的可能性。面对死亡，此在从"常人"的匿名中被唤回，进入**本真存在**。

### 二、庄子（前4世纪）：超越
"生死一如"——生与死不是两个时刻，而是**同一过程的两面**（方生方死）。死亡只是自然变化的一次转化，如四季运行。

### 三、加缪（1942）：反抗
"应当想象西西弗斯是幸福的"——拒绝超越、拒绝安慰、拒绝逃避，在**荒诞的重复**中保持清醒，**反抗**本身即是意义。
