# 爱·欲 Love & Desire

爱的三种时间性：超越的、追忆的、结构的。

## 核心命题

> 爱不是单一的现象——它是**阶梯式上升**（柏拉图）、**在失去后才被理解**（普鲁斯特）、**永远指向他者**（拉康）。三位思想家给出爱的三种时间性。

## 爱的三种时间性

### 一、柏拉图（前385）：超越
爱 (eros) 是阶梯——从一个美的身体，到所有美的身体，到美的灵魂，到美的理念。**爱的目标是超越个体、超越时间**，抵达永恒的美本身。

### 二、普鲁斯特（1927）：追忆
爱在拥有时盲目，在嫉妒中燃烧，在失去后才**被理解**。**真正的天堂是失去的天堂**——通过追忆与艺术，我们才真正拥有曾经经历的。

### 三、拉康（1959）：结构
欲望永远不在自己这里，而**指向他者**——"人的欲望是他者的欲望"。**缺乏是永恒的**，满足是短暂的。欲望是结构性的异化。
