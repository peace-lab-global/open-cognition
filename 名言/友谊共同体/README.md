# 友谊·共同体 (Friendship & Community)

> 人不是孤岛，但也不是蜂群。友谊与共同体如何既保护个体，又超越自我？

本主题收录三条关于友谊与共同体的经典理解：

| 名言 | 作者 | 核心命题 |
|------|------|---------|
| [朋友是另一个自我](亚里士多德-朋友是另一个自我.md) | Aristotle · c. 350 BCE | 德性层面的相互完成 |
| [注意是最稀有最纯粹的慷慨](薇依-注意是最稀有最纯粹的慷慨.md) | Weil · c. 1942 | 在场与给予作为关系的本质 |
| [有朋自远方来](孔子-有朋自远方来.md) | Confucius · 前5世纪 | 同道共学的喜悦 |

---

## 三者关系

- **亚里士多德**定义友谊的**最高境界**：不是利益，不是快乐，而是彼此成为更好的人。
- **薇依**指出友谊的**最小单位**：不是言语，不是礼物，而是全神贯注的注意。
- **孔子**呈现友谊的**时空维度**：远方同道来访，因共享追求而喜悦。

从德性、到注意、到共学，三者共同回答：什么是值得投入的关系？
