# 战·争和平 (War & Peace)

> 战争是**政治的延续**（克劳塞维茨），还是**对一切暴力的拒绝**（托尔斯泰）？抑或**杀一人为不义**的基本算术（墨子）？

---

## 核心命题

**战争与和平的三种逻辑。**

| 思想家 | 年代 | 逻辑 |
|--------|------|------|
| 克劳塞维茨 | 1832 | **战争是政治的延续**——以其他手段（现实主义） |
| 墨子 | 约前 400 | **非攻**——杀一人谓之不义（功利主义的反战） |
| 托尔斯泰 | 1894 | **非暴力**——不以恶抗恶（基督教和平主义） |

三条名言横跨 2300 年、三大文明，展现"战争"的三种根本态度：**现实主义的工具**、**功利主义的不义**、**基督教和平主义的绝对拒绝**。
