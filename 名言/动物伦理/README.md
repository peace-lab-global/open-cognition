# 动·物伦理 (Animal Ethics)

> 物种主义是**最后一个道德盲点**（辛格）；动物的凝视**质询着主体**（德里达）；恻隐之心**见其生不忍见其死**（孟子）。

---

## 核心命题

**动物伦理的三种起点。**

| 思想家 | 年代 | 起点 |
|--------|------|------|
| 辛格 | 1975 | **感知能力**——道德圆的扩展（功利主义） |
| 德里达 | 1997 | **动物在看**——他者性的绝对性（解构主义） |
| 孟子 | 约前 300 | **不忍之心**——恻隐的推扩（儒家） |

三条名言横跨 2300 年、三大传统，展现"动物问题"的三种进路：**利益的平等考虑**、**他者的凝视**、**恻隐的推扩**。

## 思想脉络

```
亚里士多德 动物为人类而存在
   ↓
斯多亚 动物无理性，故无道德地位
   ↓
孟子 (前300) 恻隐之心及于禽兽
   ↓
笛卡尔 (1637) 动物是自动机器 (bête-machine)
   ↓
康德 (1797) 动物是目的的手段
   ↓
边沁 (1789) 问题在于能否受苦
   ↓
辛格 (1975) 物种主义
   ↓
德里达 (1997) 动物在看
   ↓
当代 多物种研究 / 动物权利 / 关怀伦理
```
