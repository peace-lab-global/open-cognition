# 命运·偶然 (Fate & Chance)

> 人能否掌控自己的命运？偶然是世界的缺陷，还是存在的底色？

本主题收录三条面对命运与偶然的经典姿态：

| 名言 | 作者 | 核心姿态 |
|------|------|---------|
| [控制二分法](爱比克泰德-控制二分法.md) | Epictetus · c. 135 | 区分可控与不可控，守住内在自由 |
| [热爱命运](尼采-热爱命运.md) | Nietzsche · 1882 | 肯定一切发生，取消“如果当初” |
| [小径分岔的花园](博尔赫斯-小径分岔的花园.md) | Borges · 1941 | 时间是可能性的花园，而非单行道 |

---

## 三者关系

- **爱比克泰德**给出**操作手册**：哪些事值得投入情绪，哪些事应当放手。
- **尼采**给出**情感强度**：不仅要接受，还要热爱——把命运变成美学事件。
- **博尔赫斯**给出**宇宙图景**：命运不是一条锁链，而是一座分岔的花园。

从实用到诗意，从伦理到形而上学，三者构成完整的「命运观」光谱。
