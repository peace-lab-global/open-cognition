## 三者关系

- **柏拉图**把艺术放在真理的**最底层**：艺术摹仿可感事物，可感事物摹仿理型。
- **王尔德**把艺术放在生活的**上方**：不是艺术反映生活，而是生活模仿艺术。
- **本雅明**追问**技术条件**：当艺术可以被无限复制，「原作」「真实性」「灵韵」发生了什么变化？

从古代到现代，从本体论到媒介理论，三者构成艺术与真实关系的完整光谱。
