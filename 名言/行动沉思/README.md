# 行动·沉思 (Action & Contemplation)

> 知道与做到之间，隔着一条行动之河。如何跨过去？

本主题收录三条关于知行关系的经典激励：

| 名言 | 作者 | 核心命题 |
|------|------|---------|
| [知行合一](王阳明-知行合一.md) | Wang Yangming · 16 世纪 | 真知必能行，不行非真知 |
| [去做一个好人](奥勒留-去做一个好人.md) | Aurelius · 2 世纪 | 停止空谈，直接实践 |
| [知道不够必须应用](歌德-知道不够必须应用.md) | Goethe · c. 1800 | 知识与意志都必须转化为行动 |

---

## 三者关系

- **王阳明**给出**本体论答案**：知与行不是两段，而是一体。
- **奥勒留**给出**命令式劝勉**：别再讨论好人是什么样，直接成为好人。
- **歌德**给出**激昂宣言**：知道不够，要应用；愿意不够，要行动。

从东方心学到罗马斯多葛，再到德国古典主义，三者共同指向同一个现代困境：在信息过载的时代，我们更需要行动力，而非更多知识。
