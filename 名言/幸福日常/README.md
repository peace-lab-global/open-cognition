# 幸福·日常 (Happiness & the Everyday)

> 幸福不是远方的目标，而是日常生活的质地——退隐、善用、消解。

---

## 核心命题

**古典幸福的三条路径。**

| 路径 | 思想家 | 方法 |
|------|--------|------|
| 退隐 | 伊壁鸠鲁 | 默默度日 → 宁静（ataraxia） |
| 善用 | 塞涅卡 | 不在于长短，而在于善用当下 |
| 消解 | 维特根斯坦 | 让"人生有什么意义"的问题**自行消失** |

三条名言横跨 2300 年，共同反驳现代对"幸福 = 成功/财富/刺激"的误解。

## 思想脉络

```
伊壁鸠鲁 (前300) 幸福 = 宁静 + 简朴
   ↓
斯多葛 (塞涅卡、奥勒留、爱比克泰德) 幸福 = 德性 + 当下
   ↓
基督教 幸福被推迟到来世
   ↓
启蒙 (康德、边沁) 幸福被理性化/功利化
   ↓
存在主义 (克尔凯郭尔、加缪) 幸福作为反抗荒诞的方式
   ↓
维特根斯坦 (1916) 幸福作为问题的消解
   ↓
当代 慢生活 / 极简 / 正念 / 心流
```
