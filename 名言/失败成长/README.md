# 失败成长 (Failure & Growth)

> 失败不是成功的反义词，而是成长的必经之路。

本主题收录三条关于失败与成长的经典理解：

| 名言 | 作者 | 核心命题 |
|------|------|---------|
| [凡杀不死我的，必使我更强大](尼采-杀不死我的.md) | Friedrich Nietzsche · 1888 | 痛苦转化为生命力 |
| [在成长型思维中，挑战令人兴奋而非威胁](德韦克-成长型思维.md) | Carol S. Dweck · 2006 | 能力可塑的信念 |
| [再试一次，再失败一次，失败得更好](贝克特-失败得更好.md) | Samuel Beckett · 1983 | 在徒劳中继续言说 |

---

## 三者关系

- **尼采**从**生命哲学**切入：未被压垮的挫折可被整合为更强健的自我，但这不是歌颂苦难，而是肯定克服。
- **德韦克**从**认知心理学**切入：对能力的信念决定我们如何面对失败——是威胁，还是学习机会。
- **贝克特**从**荒诞文学**切入：失败可能是人类处境的常态，但“失败得更好”赋予徒劳以尊严与进益。

三者共同追问：当成功学承诺“必胜”，我们能否与失败建立更诚实、更有生产力的关系？
