# Structured Content: 楞严经全经脉络信息图

## Title
楞严经全经脉络图

## Subtitle
大佛顶首楞严神咒 · 从显见真性到圆证菩提

## Learning Objectives
- 理解楞严经十卷的整体结构
- 掌握从阿难遇魔到五十阴魔的核心脉络
- 了解七处征心、十番显见、二十五圆通等关键概念

## Modules

### MOD-1: 缘起 — 阿难遇魔
- Key: 阿难尊者乞食遭摩登伽女先梵天咒摄入淫席
- Visual: 文殊菩萨持楞严咒前往救度
- Data: 戒体将毁 → 咒力救度 → 归佛请法
- Labels: 阿难尊者, 摩登伽女, 楞严神咒, 文殊菩萨

### MOD-2: 七处征心
- Key: 佛陀七次征问阿难"心在何处"
- Visual: 七个位置标记，逐一破斥
- Data: 身内 → 身外 → 根里 → 内外间 → 随缘处 → 中间 → 无着
- Labels: 七番征心皆妄，心非在处

### MOD-3: 十番显见
- Key: 显发见性不生灭之理
- Visual: 十个见性特质
- Data: 不动、不灭、不失、不还、不杂、无碍、不分、超情、超见、离见
- Labels: 见性即是真心

### MOD-4: 世界构造
- Key: 五阴六入十二处十八界七大
- Visual: 层次结构图
- Data: 一切现象皆为如来藏妙真如性所显
- Labels: 五阴, 六入, 十二处, 十八界, 七大, 如来藏

### MOD-5: 二十五圆通
- Key: 二十五位圣者修行法门
- Visual: 25个圆形/方形排列，耳根圆通高亮
- Data: 六尘圆通6 + 六根圆通5 + 六识圆通6 + 七大圆通7 = 25
- Highlight: 观世音菩萨耳根圆通（最殊胜）
- Labels: 憍陈那(声尘), 优波尼沙陀(色尘), 香严童子(香尘), 药王药上(味尘), 跋陀婆罗(触尘), 摩诃迦叶(法尘), 阿那律陀(眼根), 周利槃特(鼻根), 憍梵钵提(舌根), 毕陵伽婆蹉(身根), 须菩提(意根), 舍利弗(眼识), 普贤菩萨(耳识), 孙陀罗难陀(鼻识), 富楼那(舌识), 优波离(身识), 大目犍连(意识), 乌刍瑟摩(火大), 持地菩萨(地大), 月光童子(水大), 琉璃光法王子(风大), 虚空藏菩萨(空大), 弥勒菩萨(识大), 大势至菩萨(根大), 观世音菩萨(耳根圆通)

### MOD-6: 四种清净明诲 & 楞严咒
- Key: 四条根本戒 + 佛顶神咒
- Visual: 盾牌/莲花与咒语符号
- Data: 不淫、不杀、不盗、不妄
- Labels: 四种清净明诲, 楞严神咒

### MOD-7: 五十种阴魔
- Key: 五阴中各十种魔境
- Visual: 五层结构，每层10个标记
- Data: 色阴10 + 受阴10 + 想阴10 + 行阴10 + 识阴10 = 50
- Labels: 色阴魔, 受阴魔, 想阴魔, 行阴魔, 识阴魔

### MOD-8: 修行要诀（底部总结）
- Key: 反闻闻自性，性成无上道
- Visual: 莲花/光芒
- Data: 耳根圆通 · 反闻自性 · 报佛深恩
- Labels: 全经以「显见真性」始，以「破魔证真」终

## Design Notes
- 风格：storybook-watercolor（柔和水彩、可见笔触、有机形状、梦幻氛围）
- 布局：dense-modules 高密度模块，一页全貌
- 配色：柔和蓝、绿、暖土色调，水彩纸质感背景
- 装饰：水彩晕染、飞溅效果、手绘线条、自然有机边框
