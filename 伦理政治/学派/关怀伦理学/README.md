# 关怀伦理学 · Care Ethics

## 一句话定义

关怀伦理学是 20 世纪末兴起的伦理学传统，强调关系、依赖、责任与具体情境中的关怀实践，反对将抽象原则视为道德判断的唯一基础。

## 历史脉络

**女性主义心理学批判（1980 年代）**
吉利根在《不同的声音》中指出，柯尔伯格的道德发展理论以男性化的正义视角为普遍标准，忽视了关怀、关系与责任在女性道德推理中的核心地位。

**教育哲学与关系伦理（1980—1990 年代）**
诺丁斯将关怀从心理学议题发展为系统伦理学，强调“关怀关系”的相互性与关怀者的专注、动机位移等特征，并应用于教育、护理与家庭伦理。

**政治哲学与社会理论扩展（1990 年代以后）**
特朗托等人将关怀伦理扩展至政治领域，分析关怀劳动的性别分工、全球照护链与福利国家制度，使关怀成为社会正义与公共政策的重要范畴。

## 核心主张

1. **关系优先于个体**
   人本质上是相互依赖的关系性存在，道德生活始于对具体他者的回应，而非原子化个体的权利主张。

2. **情境敏感与注意力**
   道德判断依赖对具体情境、情感线索与历史关系的细致关注，而非机械套用普遍规则。

3. **责任与回应**
   关怀是一种主动承担的责任，要求行为者识别他者需求并作出适当回应。

4. **相互性与情感**
   关怀关系强调关怀者与被关怀者之间的互动，情感（如同情、敏感、信任）是道德认知的重要组成部分。

## 与其他传统关系

- **德性伦理学**：关怀伦理学共享对品格与情境的重视，但更强调关系结构与性别化劳动。
- **正义理论**：与罗尔斯式正义形成对话，主张将关怀劳动与依赖关系纳入正义的分配对象。
- **动物伦理与环境伦理**：关怀视角被用来论证对动物、生态系统与未来世代的责任。

## 经典争论

- **普遍主义 vs 情境主义**：关怀伦理能否提供跨情境的规范指导，还是只能停留在对具体关系的描述？
- **性别化风险**：关怀是否可能强化对女性的传统角色期待？如何在承认关怀价值的同时避免剥削？
- **公共领域中的关怀**：关怀伦理如何从私人领域扩展至政策、经济与全球正义讨论？

## 现当代影响

关怀伦理学已渗透护理伦理、教育哲学、家庭法、劳动政策、生态伦理与国际发展研究，成为理解 21 世纪照护危机、老龄化与全球不平等的重要理论资源。
