# 社群主义 · Communitarianism

## 一句话定义

社群主义是 20 世纪 80 年代兴起的政治哲学与社会理论传统，批判自由主义对原子化个人与中立性国家的预设，强调个人认同、美德与正义原则都嵌入特定的社群传统与共享实践之中。

## 历史脉络

**对自由主义的批判兴起（1980 年代）**
麦金泰尔、桑德尔、泰勒与沃尔泽等人分别从德性伦理、正义理论、自我认同与社会分配等角度，系统批判罗尔斯《正义论》中的无负荷自我（unencumbered self）观念，主张自我总是承载着历史、语言与社群赋予的目的与归属。

**与自由主义的重叠共识（1990 年代）**
社群主义与自由主义之间的论战逐渐细化，许多社群主义者承认个人权利的重要性，同时坚持权利解释必须置于特定文化语境；自由主义者也日益重视公民社会、社会资本与共和主义传统的价值。

**当代延展**
社群主义思想影响公民共和主义、多元文化主义、宗教共同体研究以及东亚“社群式民主”讨论，并与自由主义—民族主义辩论持续对话。

## 核心主张

1. **嵌入性自我**
   个人并非优先于社会的孤立原子，而是在语言、传统与关系中形成自我理解与道德承诺。

2. **共同善优先**
   政治生活不仅关乎个人权利的分配，更关乎共同善、公共生活与公民美德的培育。

3. **传统的可争辩性**
   道德推理无法脱离特定传统展开，但传统本身可以通过内部批评与对话得到修正。

4. **社会参与的公民身份**
   好公民不仅是权利享有者，更是积极参与公共事务、承担共同责任的成员。

## 与其他传统关系

- **自由主义**：社群主义最重要的对话对象，二者在个人与社群、权利与善、中立性与完善论之间持续争论。
- **德性伦理学**：麦金泰尔等社群主义者复兴亚里士多德—阿奎那传统，强调实践、叙事与德性。
- **公民共和主义**：共享对公共善与积极参与的强调，但共和主义更突出政治自由与制度设计。

## 经典争论

- **自我是否可被社群完全定义？**：社群主义是否低估了个人的反思、批判与脱离传统的能力？
- **相对主义风险**：如果以特定传统为正义标准，是否会导致文化相对主义与对内部少数群体的压制？
- **国家角色**：社群主义是否为国家推行某种美好生活观提供了过度空间？

## 现当代影响

社群主义为公民社会、社区建设、社会资本、宗教多元、民族认同与东亚政治哲学提供了关键概念，并在数字公共领域、平台共同体与地方治理研究中继续发挥作用。
