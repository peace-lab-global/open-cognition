# 森 · Amartya Sen

> **"发展可以被看作是扩展人们享有的实质自由的过程。"**

## 基本信息
- **全名**：Amartya Kumar Sen
- **生卒年**：1933–
- **国籍**：印度/英国/美国
- **荣誉**：1998 年诺贝尔经济学奖

## 一句话定位
森是 **能力进路（capability approach）的创始人**——以"人们实际能做什么和成为什么"（能力集）替代收入/GDP作为发展的度量，将经济学与哲学深度结合，从根本上改变了发展理论与福利经济学的范式。

## 核心命题

### 1. 能力进路 · Capability Approach
平等的度量不是资源（罗尔斯的基本善）也不是效用（功利主义），而是**能力**（capability）——人们实际上能够做和成为的有价值的功能性活动（functionings）的集合。一个残疾人与一个健全人获得同样收入，实际能力截然不同。

### 2. 权利方法 · Entitlement Approach
饥荒不是由食物短缺引起的——而是由**权利关系**的崩溃引起的。人们在食物充足时也可能因失去获取食物的权利（如失业导致购买力丧失）而饿死。

### 3. 以自由看待发展 · Development as Freedom
发展 = 扩展人的实质自由。自由既是发展的**目的**也是发展的**手段**（政治自由、经济设施、社会机会、透明性保障、防护性保障互相促进）。

### 4. 社会选择理论
森发展了阿罗的社会选择理论，证明了在信息充分的条件下，人际比较是可能的，从而为福利经济学奠定了新的基础。

### 5. 身份的多元性
人有**多重身份**（宗教、性别、职业、国籍等），暴力冲突往往源于将人**简化为单一身份**（如"你是印度教徒"或"你是穆斯林"）。

## 思想坐标
- **批判**：[罗尔斯](../social-contract/rawls/README.md)的基本善度量忽视人的异质性
- **合作**：[纳斯鲍姆](nussbaum/README.md)将能力进路发展为系统的哲学理论
- **影响**：联合国人类发展指数（HDI）的哲学基础

## 常见误读
- **"能力进路就是个人能力"** → 能力（capability）是社会结构中的实质自由，不仅是个人技能
- **"森反对GDP"** → 森不反对GDP作为度量之一，但反对将其作为**唯一**度量
