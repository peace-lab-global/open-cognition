# 纳斯鲍姆 · Martha Nussbaum

> **"每个人都应享有过上有尊严的生活的基本能力。"**

## 一句话定位
纳斯鲍姆是 **能力进路的哲学系统化者**——在森的基础上提出**十项核心能力**清单，将能力进路从经济学度量工具发展为完整的正义理论，特别关注残疾人、全球正义与情感的政治角色。

## 核心命题

### 1. 十项核心能力 · Central Capabilities
每个正义的社会必须保障其公民达到以下能力的最低门槛：(1) 生命 (2) 身体健康 (3) 身体完整 (4) 感觉、想象与思维 (5) 情感 (6) 实践理性 (7) 归属 (8) 与其他物种的关系 (9) 游戏 (10) 对环境的控制。

### 2. 善的脆弱性 · Fragility of Goodness
人类的美德生活本质上是**脆弱的**——受运气、身体、社会环境的影响。真正的伦理学必须正视这种脆弱性，而非像柏拉图那样试图通过理性消除偶然性。

### 3. 情感即判断 · Emotions as Judgments
情感不是非理性的冲动，而是**价值判断**——恐惧包含对威胁的评估，愤怒包含对不公正的感知。政治制度应当重视公民的情感培育。

### 4. 前沿正义 · Frontiers of Justice
罗尔斯的正义论有三个"前沿"问题未解决：(1) 残疾人正义 (2) 国际正义 (3) 动物正义。能力进路能更好地处理这些问题。
