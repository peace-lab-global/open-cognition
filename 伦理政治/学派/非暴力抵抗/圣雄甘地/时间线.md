# 甘地生平与思想大事年表 / Gandhi Life and Intellectual Timeline

## 早年与教育（1869—1893）/ Early Years (1869-1893)

| 年份 | 事件 | 意义 |
|------|------|------|
| 1869 | 10月2日生于印度古吉拉特邦波尔班达尔 | 毗湿奴派印度教家庭 |
| 1883 | 与卡斯图尔巴·马克汉吉结婚（13岁包办婚姻） | 终身伴侣 |
| 1885 | 父亲去世 | 深刻的精神震动 |
| 1888 | 赴伦敦中殿律师学院学习法律 | 接触基督教、托尔斯泰、神智学 |
| 1891 | 获律师资格，返回印度 | 开始执业，但初期并不成功 |

## 南非岁月（1893—1914）/ South Africa Years (1893-1914)

| 年份 | 事件 | 意义 |
|------|------|------|
| 1893 | 赴南非德班为印度商人代理法律事务 | 在火车上遭受种族歧视——政治觉醒 |
| 1894 | 创立纳塔尔印度人大会（NIC） | 南非印度人政治组织化 |
| 1906 | 发起第一次"萨提亚格拉哈"运动 | 反抗《亚洲人登记法》（按指纹登记） |
| 1906 | 宣誓禁欲（brahmacharya） | 精神自律的转折点 |
| 1909 | 撰写《印度自治》（Hind Swaraj） | 核心理论文著 |
| 1910 | 建立托尔斯泰农庄（Tolstoy Farm） | Satyagraha社区实验 |
| 1913 | 大规模罢工和游行；《印度救济法》通过 | 南非运动取得重要成果 |
| 1914 | 与南非政府达成协议；离开南非 | 21年南非岁月的结束 |

## 印度独立运动（1915—1942）/ Indian Independence Movement (1915-1942)

| 年份 | 事件 | 意义 |
|------|------|------|
| 1915 | 返回印度，建立萨巴尔马蒂阿什拉姆 | 印度运动的新基地 |
| 1917 | 查帕兰运动（Champaran）——为靛蓝农民抗争 | 在印度的第一次Satyagraha实践 |
| 1919 | 反对罗拉特法案（Rowlatt Act） | 大规模公民不服从的开端 |
| 1919 | 阿姆利则惨案——英军屠杀379名印度平民 | 甘地与英国决裂的催化剂 |
| 1920 | 发起"不合作运动"（Non-Cooperation Movement） | 全国性抗争 |
| 1922 | 乔里乔拉事件——抗议者焚烧警察局 | 甘地中止不合作运动 |
| 1924 | 21天绝食以促进印度教-穆斯林团结 | 教派和平的努力 |
| 1930 | "食盐进军"——步行390公里至丹迪海边 | 公民不服从的经典实践 |
| 1931 | 与英国签署《甘地-欧文协定》；赴伦敦圆桌会议 | 短暂的政治妥协 |
| 1932 | 在狱中绝食反对"不可接触者"单独选区 | 与阿姆倍德卡尔的辩论 |
| 1933 | 创办《哈里扬》周刊 | 为"不可接触者"权利发声 |
| 1942 | 发起"退出印度"运动（Quit India） | 对英国殖民者最强硬的姿态 |

## 独立、分治与殉道（1942—1948）/ Independence, Partition and Martyrdom (1942-1948)

| 年份 | 事件 | 意义 |
|------|------|------|
| 1944 | 与真纳就印巴问题进行最后谈判 | 未能阻止分治 |
| 1946 | 赴诺阿卡利平息教派暴力 | 以个人存在制止屠杀 |
| 1947 | 8月15日印度独立；印巴分治 | 独立实现但伴随大屠杀 |
| 1947 | 9月在加尔各答绝食平息教派暴力 | "一个人的边境军" |
| 1948 | 1月在德里最后一次绝食 | 成功平息首都教派冲突 |
| 1948 | 1月30日被印度教极端分子纳图拉姆·戈德塞刺杀 | 殉道——"He Ram" |

## 逝世后重要事件 / Key Posthumous Events

| 年份 | 事件 | 意义 |
|------|------|------|
| 1951 | 《非暴力抵抗》（Satyagraha）英译出版 | 非暴力理论的系统汇编 |
| 1956 | 马丁·路德·金访问印度，深入研究甘地思想 | Satyagraha应用于美国民权运动 |
| 1960 | 《甘地全集》开始出版（至1994年完成100卷） | 完整文献基础 |
| 1982 | 理查德·阿滕伯勒电影《甘地》获奥斯卡最佳影片 | 甘地形象的全球普及 |
| 2007 | 联合国将10月2日定为"国际非暴力日" | 甘地遗产的制度化 |
| 2010s- | 全球非暴力运动的"甘地复兴" | 从"阿拉伯之春"到气候运动 |
