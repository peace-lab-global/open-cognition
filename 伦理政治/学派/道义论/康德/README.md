# 康德 · Immanuel Kant（伦理学）

> **"有两样东西，我愈经常愈持久地加以思索，它们就愈使心灵充满景仰和敬畏：在我之上的星空和居我心中的道德法则。"**

## 基本信息
- **全名**：Immanuel Kant
- **生卒年**：1724–1804
- **国籍**：普鲁士（德国柯尼斯堡，终生未离）
- **学派**：德国古典哲学 / 道义论伦理学 / 先验唯心论

## 一句话定位
康德是 **道义论伦理学的奠基者**——主张道德行为的正当性不取决于后果，而取决于行为是否出于**义务**、是否符合**定言命令**的普遍化要求。其伦理学是罗尔斯正义论、当代人权理论的哲学源头。

## 核心命题

### 1. 定言命令 · Categorical Imperative
道德法则的三种表述：
- **普遍法则公式**："仅按照你能同时意愿它成为普遍法则的那个准则去行动"
- **人性公式**："永远不要仅仅把人当作手段，而要同时当作目的"
- **自律公式**：道德行为者视自己为普遍法则的立法者

### 2. 善良意志 · Good Will
唯一无条件善的东西是**善良意志**——行为的道德价值不取决于结果、才能或幸福，而取决于**是否出于义务**。

### 3. 自律 · Autonomy
道德法则不是外在强加的，而是理性主体**为自己立法**——这就是人的尊严的来源。与他律（heteronomy）——受欲望、传统或权威支配——相对。

### 4. 义务论 · Deontology
道德行为是**出于义务**（from duty）而非**合乎义务**（in accordance with duty）。商人诚实经营如果只是为了利润，虽"合乎义务"但不具有道德价值。

### 5. 永久和平 · Perpetual Peace
通过共和制、国际联盟与世界公民法权，人类可以逐步实现永久和平——这是康德政治哲学的终极目标。

### 6. 目的王国 · Kingdom of Ends
一个理想共同体，其中每个理性存在者既是普遍法则的立法者也是服从者——所有人都互为目的而非手段。

## 思想脉络

### 前批判期（1724–1770）
- 在柯尼斯堡大学学习，受沃尔夫理性主义影响
- 1755 发表《自然通史与天体理论》

### 批判期（1781–1798）
- 1781 《纯粹理性批判》——认识论革命
- 1785 《道德形而上学奠基》——伦理学核心
- 1788 《实践理性批判》
- 1793 《单纯理性限度内的宗教》
- 1795 《永久和平论》
- 1797 《道德形而上学》

## 思想坐标
- **批判**：功利主义（后果不等于道德）、幸福论（幸福不是道德标准）
- **影响**：[罗尔斯](../social-contract/rawls/README.md)（原初状态是定言命令的程序化）
- **对立**：[亚里士多德](../virtue-ethics/aristotle/README.md)（美德 vs 义务）
- **下游**：哈贝马斯的商谈伦理、当代人权理论

## 常见误读
- **"康德是僵化的规则主义者"** → 定言命令是**程序**而非具体规则清单
- **"康德忽视情感"** → 康德区分了道德价值的基础（义务）与辅助因素（倾向）
- **"康德的反功利主义意味着不考虑后果"** → 康德承认后果相关，但否认后果是道德的**最终标准**
