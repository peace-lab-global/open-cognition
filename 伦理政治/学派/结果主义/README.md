# 结果主义 · Consequentialism

## 一句话定义

结果主义是规范伦理学的一大传统，主张行为的道德价值取决于其产生的后果；正确的行动是在相关意义上能够带来最佳总体结果的行动。

## 历史脉络

**古典功利主义（18—19 世纪）**
边沁以“最大多数人的最大幸福”奠定功利主义基础，将快乐与痛苦作为道德计算的核心单位。密尔区分快乐的质量，强调更高层次的精神快乐，使功利主义回应“猪的幸福”式批评。

**19 世纪末的修正**
西季威克在《伦理学方法》中系统比较功利主义、直觉主义与利己主义，指出功利主义与常识道德的紧张关系，为 20 世纪元伦理学讨论奠定基础。

**20 世纪的扩展与分化**
摩尔以“理想功利主义”将非快乐的价值（如美、知识）纳入后果计算。随后出现行为结果主义与规则结果主义、直接结果主义与间接结果主义、客观结果主义与主观结果主义等区分，以回应可预见性、诚信、权利等批评。

## 核心主张

1. **后果优先**
   判断行为对错的首要依据是该行为实际或预期产生的世界状态，而非行为者的动机、品格或义务本身。

2. **价值最大化**
   正确的行动应在可选方案中最大化某种内在价值（幸福、偏好满足、福祉等）的总量或平均量。

3. **不偏不倚**
   每个人的利益在道德计算中占有同等分量，不存在因身份亲近而天然加权的特权。

4. **聚合性**
   道德评价涉及对多个主体利益的整体权衡，允许个体利益在特定条件下为更大总体利益让渡。

## 与其他传统关系

- **功利主义**：结果主义最著名的分支，以幸福或福利作为后果评价标准。
- **义务论**：康德传统强调行为本身的义务与规范，对结果主义提出“个人完整性”与“权利不可侵犯”的批评。
- **德性伦理学**：亚里士多德传统关注行为者的品格与实践智慧，质疑结果主义忽视道德主体性。

## 经典争论

- **可预见性与实际后果**：应依据行动时可合理预见的后果，还是实际发生的全部后果来评价行为？
- **行为 vs 规则结果主义**：是直接评价每个行为，还是通过遵守最优规则来间接实现最佳后果？
- **要求过高（Demandingness）**：结果主义是否要求人们牺牲个人计划以持续促进总体福利？

## 现当代影响

结果主义深刻影响有效利他主义、全球正义、动物伦理、人工智能伦理与公共政策评估，成为当代应用伦理学中不可或缺的分析框架。
