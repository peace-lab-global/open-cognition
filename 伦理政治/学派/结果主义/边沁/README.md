# 边沁 · Jeremy Bentham

> **"问题不在于它们能否推理，也不在于它们能否说话，而在于它们能否受苦。"**

## 基本信息
- **全名**：Jeremy Bentham
- **生卒年**：1748–1832
- **国籍**：英国
- **学派**：功利主义创始人 / 法律实证主义 / 激进改革主义

## 一句话定位
边沁是 **功利主义伦理学的创始人**——主张行为的道德性取决于其产生的**快乐总量减去痛苦总量**，并将这一原则系统应用于法律、政治与社会改革。

## 核心命题

### 1. 功利原则 · Principle of Utility
行为的正当与否取决于它**增进还是减少**受影响者的幸福总量。"自然将人类置于两位主人——痛苦与快乐——的统治之下。"

### 2. 快乐计算 · Felicific Calculus
快乐可以用七个维度量化：强度、持续时间、确定性、接近性、繁殖性、纯粹性、范围。这使得道德判断可以像数学一样精确。

### 3. 全景监狱 · Panopticon
一种环形监狱设计——囚犯不知道自己是否被监视，从而**自我规训**。福柯将其发展为现代权力运作的隐喻。

### 4. 法律实证主义
法律不依赖自然法或道德——法律的效力来自**主权者的命令**。边沁批判自然权利为"踩在高跷上的废话"（nonsense upon stilts）。

### 5. 动物权利
边沁是最早将道德关怀扩展到动物的思想家——标准不是理性，而是**受苦的能力**。

## 思想坐标
- **影响**：[密尔](mill/README.md)的质的功利主义
- **被批判**：密尔区分高级/低级快乐，批判边沁的量化
- **下游**：福柯的全景监狱分析；当代有效利他主义

## 常见误读
- **"边沁忽视人的尊严"** → 密尔正是为此而修正
- **"功利原则鼓励不择手段"** → 边沁强调制度设计而非个体计算
- **"全景监狱是极权主义"** → 边沁视之为**人道的**监狱改革
