# 自由主义 · Liberalism

## 一句话定义

自由主义是 17 世纪以降形成于西欧的现代政治哲学传统，主张个人享有不可侵犯的基本自由与权利，政府的正当性建立在保护这些权利与经被统治者同意的基础之上。

## 历史脉络

**古典自由主义奠基（17—18 世纪）**
洛克在《政府论两篇》（1689）中论证自然权利、社会契约与有限政府，奠定古典自由主义基础。18 世纪苏格兰启蒙运动（休谟、斯密、弗格森）进一步将自由主义与商业社会、法治、分权结合。

**自由主义与革命（18 世纪末）**
美国独立宣言（1776）与法国人权宣言（1789）将自由主义原则转化为政治实践，强调生命权、自由权、财产权与反抗暴政的权利。这一时期自由主义与宪政、代议制民主紧密结合。

**19 世纪的修正与扩展**
密尔在《论自由》（1859）中系统论证个人自由、言论自由与个性发展，同时关注多数暴政与社会压迫。19 世纪末至 20 世纪初，新自由主义（New Liberalism）开始强调国家在教育、福利与市场调节中的积极作用。

**20 世纪的复兴与内在张力**
冷战时期，波普尔、哈耶克、伯林等人批判极权主义与计划经济，复兴古典自由主义。罗尔斯《正义论》（1971）以“作为公平的正义”重新奠基自由主义，引发自由至上主义（诺齐克）、社群主义（桑德尔）、自由主义女性主义等回应。

## 核心主张

1. **个人自由优先**
   个人享有一系列基本自由，如良知（conscience）、言论、结社、信仰与财产自由。这些自由构成政治权威的边界。

2. **自然权利或基本权利**
   人拥有先于国家的基本权利，政府的存在是为了保障而非创造这些权利。古典自由主义强调生命、自由与财产；当代自由主义扩展至平等、尊严与社会权利。

3. **同意与合法性**
   政治权力的正当性来自被统治者的同意，无论是通过社会契约、宪政程序还是民主选举。

4. **法治与有限政府**
   政府权力应受法律约束，避免专制与任意干预。权力分立、司法独立与宪政审查是自由主义制度的核心装置。

5. **不偏不倚（Impartiality）**
   每个人的权利与利益都应在政治考量中获得平等尊重，无人因其身份而天生优先。

6. **中立性与多元主义**
   自由主义国家不应强制推行某种特定美好生活观（conception of the good），而应在合理多元社会中保持中立，让公民自主选择和追求各自的人生计划。

## 关键概念

- **自然权利（Natural Rights）**：人生而享有的权利，不依赖政府赋予。
- **社会契约（Social Contract）**：个人通过同意建立政治共同体与政府的理论模型。
- **伤害原则（Harm Principle）**：限制个人自由的唯一正当理由是防止对他人造成伤害。
- **公共领域与私人领域**：自由主义区分国家可干预的公共领域与应受保护的私人领域。
- **积极自由与消极自由**：伯林区分“免于干涉的自由”与“实现自我主宰的自由”。

## 常见误读

**误读一：自由主义主张无政府或完全放任市场。**
纠正：自由主义强调有限政府，而非无政府；它承认国家在保护权利、提供公共品、纠正市场失灵等方面的职能。不同流派（古典自由主义、新自由主义、自由至上主义）对国家角色的理解差异显著。

**误读二：自由主义只关心个人，不关心共同体。**
纠正：自由主义关注个人权利，但并不否认共同体、家庭与公民德性的价值。其争论在于：国家是否应当强制推行某种共同体理想。社群主义批评自由主义过于原子化，但自由主义内部也有共和主义与多元主义回应。

**误读三：自由主义是西方中心主义意识形态。**
纠正：自由主义确实起源于近代西欧，但其核心问题——个人权利、政府合法性、法治——具有跨文化相关性。当代研究亦在探讨自由主义与非西方传统（如儒家、伊斯兰法）的对话与张力。
