# 罗伯特·诺齐克 · Robert Nozick

> **"对任何东西，只要不是完全属于其所有者本人，无论它有多大用处，都不应该被他人所占有。"**（反对模式化分配正义）

## 基本信息

- **全名**：Robert Nozick
- **生卒年**：1938–2002
- **国籍**：美国
- **学派**：自由至上主义、分析政治哲学

## 一句话定位

诺齐克是 20 世纪最重要的自由至上主义政治哲学家之一，他在《无政府、国家与乌托邦》中以权利为基础论证最小国家的正当性，并批判罗尔斯的模式化分配正义理论。

## 核心命题

### 1. 最小国家论
唯一正当的国家是"最小国家"，其职能仅限于保护公民免受暴力、欺诈与违约侵害。任何超出此范围的再分配或家长式职能都侵犯个人权利。

### 2. 权利作为边界
个人拥有不可侵犯的权利，这些权利构成他人与国家的道德边界。以社会整体利益为由牺牲个人权利是不正当的。

### 3. 对模式化正义的批判
诺齐克批判以特定分配模式（如按需要、按贡献、按平等）为目标的正义理论。他提出"张伯伦论证"：即使从正义分配出发，自愿交易也会打破原模式，而禁止这种交易即侵犯自由。

### 4. 资格理论（Entitlement Theory）
分配正义取决于：
- 获取的正义（对无主物的最初占有）
- 转让的正义（自愿交换、馈赠）
- 矫正的正义（对过去不正义的纠正）
若财产链条满足这三项条件，则当前分配就是正义的。

### 5. 体验机论证
在《哲学解释》中，诺齐克以"体验机"思想实验论证：人不仅追求愉快体验，还追求与真实世界发生联系、按照自己的意愿行动并成为一个真实的人。

## 常见误读

- **"诺齐克反对一切福利"**：诺齐克反对的是强制再分配，并不反对自愿慈善；他认为国家无权强制征税用于福利，但个人道德上可以援助他人。
- **"诺齐克完全否定罗尔斯"**：诺齐克与罗尔斯共享自由主义基础（个人权利优先），但分歧在于是否允许再分配。诺齐克批评的是差别原则的模式化特征。
- **"诺齐克晚年放弃了自由至上主义"**：诺齐克在《理性的性质》中对某些立场有所修正，但他从未放弃以权利为基础的最小国家核心主张。

## 跨学科关联

- **政治哲学 ↔ 自由主义**：诺齐克的权利理论 [发展] 洛克的自然权利传统；他的最小国家 [对立] 罗尔斯的自由主义再分配。本库相关：[罗尔斯](../../社会契约论/约翰-rawls.md) [洛克](../../../../哲学/学派/洛克.md)。
- **政治哲学 ↔ 经济学**：诺齐克的资格理论 [互补] 奥地利学派对自发秩序的辩护；他的权利边界 [平行] 哈耶克对法治的强调。本库相关：[哈耶克](../../古典自由主义/哈耶克.md)。
- **政治哲学 ↔ 伦理学**：体验机论证 [发展] 德性伦理学与完善论对功利主义的批判；资格理论 [对立] 功利主义的总体福利最大化。本库相关：[密尔](../../结果主义/密尔.md)。
