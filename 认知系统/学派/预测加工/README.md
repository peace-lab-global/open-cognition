# 预测加工 · Predictive Processing

## 一句话定义

预测加工是 21 世纪初兴起的跨学科研究纲领，主张大脑的核心功能是不断生成关于感觉输入的预测，并通过最小化预测误差来更新内部模型、引导感知、行动与学习。

## 历史脉络

**先驱：亥姆霍茨与知觉推断（19 世纪）**
赫尔曼·冯·亥姆霍茨（Hermann von Helmholtz）提出感知是大脑对感觉原因所作的“无意识推断”。这一观念为后来的预测性知觉理论奠定了认识论基础。

**预测编码的提出（1980—1999）**
20 世纪 80 年代，预测编码被引入信号处理与神经科学。1999 年，Rao 与 Ballard 在《自然·神经科学》发表“视觉皮层中的预测编码”，首次以层级预测编码模型解释视觉皮层的神经活动。

**自由能原理与统一框架（2005—2010）**
Karl Friston 提出自由能原理（Free Energy Principle），将预测误差最小化与变分贝叶斯推断统一起来，声称这一原理适用于从感知到行动、从学习到自我组织的广泛生命过程。2010 年发表于 *Nature Reviews Neuroscience* 的论文使该框架广为人知。

**哲学化与扩展（2010 年代至今）**
Jakob Hohwy 的《预测心智》（2013）与 Andy Clark 的“Whatever next?”（2013）分别从内部主义与延展认知视角推进预测加工的哲学阐释。预测加工已扩展至意识研究、精神病学、人工智能、美学与教育学。

## 核心主张

1. **感知即预测**
   大脑不是被动接收感觉信息，而是主动生成关于外部世界状态的预测。感知是内部模型与感觉输入之间的匹配过程。

2. **预测误差最小化**
   当预测与输入不一致时，产生预测误差。大脑通过两种方式处理误差：更新内部模型（知觉学习）或采取行动使输入符合预测（主动推理）。

3. **精度加权即注意力**
   大脑根据预测误差的可靠性（精度）动态调整其权重。精度加权解释了注意力的分配：高预测精度的信号获得更多认知资源。

4. **层级化加工**
   预测编码在大脑中层级展开：低层级预测感觉特征，高层级预测抽象因果结构。误差信号自下而上传递，预测自上而下传递。

5. **主动推理（Active Inference）**
   行动不仅是反应，也是降低预测误差的手段。通过运动，主体使感觉输入符合预期，从而实现“自我证明”（self-evidencing）。

## 关键概念

- **预测编码（Predictive Coding）**：以最小化预测误差为核心的神经计算模型。
- **自由能原理（Free Energy Principle）**：Friston 提出的统一原理，将感知、行动与学习纳入同一数学框架。
- **预测误差（Prediction Error）**：实际输入与预测之间的差异，是学习与注意力的驱动信号。
- **精度（Precision）**：对预测误差可靠性的估计，决定误差对信念更新的影响程度。
- **主动推理（Active Inference）**：通过行动改变感觉输入以降低预测误差的过程。
- **马尔可夫毯（Markov Blanket）**：系统与环境之间的统计边界概念，用于界定认知主体的边界。

## 常见误读

**误读一：预测加工认为大脑只关心自己的预测，忽视外部世界。**
纠正：预测加工恰恰强调大脑通过预测误差与世界持续互动。预测必须追踪世界的因果结构，否则误差将持续累积，模型无法维持。

**误读二：预测加工只是贝叶斯推理的另一种说法，没有新意。**
纠正：预测加工不仅包含信念更新，还包含行动选择（主动推理）、精度加权和模型选择。它将感知、行动、注意与学习整合为统一的计算—认知框架。

**误读三：预测加工已得到充分证实，是大脑的唯一工作原理。**
纠正：预测加工是一个富有影响力的研究纲领，但仍是假说。部分神经生理学证据支持预测编码机制，但其在高级认知、社会认知与意识中的普适性仍在争论中。
