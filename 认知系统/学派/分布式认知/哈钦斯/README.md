# 哈钦斯 / Edwin Hutchins

> **领域**: 分布式认知
> **时期**: 当代 (Contemporary)
> **生卒**: 1948–
> **国籍**: 美国

> "认知系统的边界不是颅骨，而是功能性的系统边界。"

## 一、思想概要

埃德温·哈钦斯（Edwin Hutchins, 1948–）是加州大学圣迭戈分校认知科学教授，分布式认知（Distributed Cognition）理论的奠基人。他的核心贡献是证明了认知不是发生在个体头脑中的孤立事件，而是分布在人、人工制品和社会组织中的系统性过程。

以海军导航为案例，《荒野中的认知》（1995）系统阐述了分布式认知理论——"导航"不是任何一个人的认知活动，而是分布在多名船员、海图、仪器、规程中的系统过程。这一洞见颠覆了"认知=个体信息处理"的传统假设。

后期他研究飞行员座舱、手术室、科学研究实验室中的认知分布，提出"认知生态"概念。在人+AI协作时代，分布式认知理论提供了理解新型认知系统的核心框架。

Edwin Hutchins founded distributed cognition, demonstrating through naval navigation studies that cognition is distributed across people, artifacts, and social organizations rather than confined within individual skulls. His work has profoundly influenced HCI, cognitive systems engineering, and our understanding of human-AI collaboration.

## 二、核心命题

1. **认知分布在人、工具、环境之间**：认知是分布在人、人工制品和社会组织中的系统性过程。
2. **文化是认知的基础设施**：文化不是认知的"背景"，而是构成认知过程本身的要素。
3. **分析单元是功能系统**：研究认知应该以"完成特定任务的功能系统"为分析单元。
4. **认知民族志**：理解认知需要深入实际工作场景。

## 五、重要概念

- **分布式认知**：认知分布在人、工具、环境之间
- **认知生态**：特定工作场景中认知要素的总体配置
- **传播表征**：认知过程 = 表征在不同媒介之间的传播和转换
- **认知民族志**：深入实际工作场景研究认知过程的方法
