# 爱德华·洛伦兹 · Edward Lorenz

爱德华·洛伦兹（Edward Lorenz，1917–2008）是美国气象学家、数学家，混沌理论奠基人之一。他在 1963 年研究大气对流简化模型时发现，确定性方程会产生非周期的混沌解；1972 年以“巴西蝴蝶翅膀”的比喻提出著名的蝴蝶效应问题。

## 核心命题 / Core Theses

### 1. 确定性混沌

系统方程完全确定，却因对初始条件的敏感依赖而表现得如同随机，从而瓦解了经典拉普拉斯式的完美预测梦想。

### 2. 预测的上限

天气等复杂系统的长期细节预测存在内在极限，这不是计算能力不足，而是系统本身的非线性结构使然。预测超过一定时间窗口将失去意义。

### 3. 吸引子与几何

洛伦兹吸引子以三维空间中的蝴蝶状轨道，展示了混沌秩序：看似混乱，却有稳定的整体结构。这为复杂系统的可视化研究打开新窗口。

## 跨学科关联 / Interdisciplinary Links

- 直接继承 [亨利·庞加莱](../../../哲学/学派/科学哲学/亨利.md) 对三体问题敏感依赖的发现。
- 与 [伯努瓦·曼德勃罗](伯努瓦.md) 的分形几何共同描绘复杂系统的尺度不变性与内在结构。
- 其方法影响了经济学、生态学、神经科学等领域对复杂动态系统的研究。
