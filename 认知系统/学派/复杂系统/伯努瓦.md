# 伯努瓦·曼德勃罗 · Benoit Mandelbrot

伯努瓦·曼德勃罗（Benoit Mandelbrot，1924–2010）是法裔美国数学家，分形几何创始人。他在 IBM 工作期间研究湍流、股市与海岸线，发现自然界的粗糙形态无法用欧几里得几何描述，于是发展出“分形”概念，并证明复杂形态背后可以存在简单的自相似规则。

## 核心命题 / Core Theses

### 1. 自然的几何

云不是球体，山不是锥体，海岸线不是圆；自然界充满分数维的粗糙结构，需要新的几何语言。这一洞见直接挑战了以理想形状为基础的传统几何。

### 2. 自相似与尺度不变性

分形的局部与整体在统计上相似，跨越多个尺度保持复杂。这为理解湍流、股市波动、地质断裂、生物形态等提供了统一工具。

### 3. 有序与混沌的交织

看似混乱的形态（如曼德勃罗集）可由极简单规则生成；混沌不是无秩序，而是另一种秩序。简单与复杂因此获得新的连接方式。

- 与 [亨利·庞加莱](../../../哲学/学派/科学哲学/亨利.md) 对几何直觉与复杂动态的重视一脉相承。
- 其方法被计算机图形学、金融风险管理、医学影像、材料科学等领域广泛采用。
