# 自动化与社会技术系统学派 · Automation & Sociotechnical Systems School

## 概述

自动化与社会技术系统学派关注人类与自动化技术之间的张力关系。该学派的核心洞见是：自动化并非简单地取代人类活动，而是从根本上改变了人机关系的结构。随着系统自动化程度的提高，人类角色从直接操作者转变为监控者与决策者，但这一转变带来了深刻的悖论——自动化越先进，人类的参与越关键，而人类保持有效参与的能力却越弱。

该学派的思想根植于 Jens Rasmussen 的认知工程传统，同时汲取了 Tavistock 学派社会技术系统理论的养分，形成了独特的理论框架，涵盖自动化层级模型、自动化偏见、认知工作分析、生态界面设计等核心概念。

## 核心关切

- **自动化悖论**：消除人为错误的努力如何产生新形式的人为错误
- **监控问题**：人类在高度自动化系统中的注意力与情境意识退化
- **信任校准**：人类对自动化系统的过度信任与信任不足
- **联合优化**：技术子系统与社会子系统的协同设计
- **工作域分析**：如何理解并支持复杂工作系统的深层结构

## 代表思想家

| 思想家 | 核心贡献 | 关键著作 |
|--------|----------|----------|
| **Thomas Sheridan** (1929-2024) | 监督控制理论、自动化十级模型 | *Telerobotics, Automation, and Human Supervisory Control* (1992) |
| **Christopher Wickens** (1943-) | 多资源模型、工程心理学 | *Engineering Psychology and Human Performance* (1984) |
| **Lisanne Bainbridge** (1941-) | 自动化悖论、监控问题 | "The Ironies of Automation" (1983) |
| **Raja Parasuraman** (1953-2015) | 自动化四阶段模型、信任校准 | "A Model for Types and Levels of Human Interaction with Automation" (2000) |
| **Kim Vicente** (1965-) | 认知工作分析、生态界面设计 | *Cognitive Work Analysis* (1999) |

## 思想源流

```
Wiener (控制论)
    ├── Sheridan (监督控制与自动化十级)
    │       └── Parasuraman (自动化四阶段模型)
    ├── Wickens (多资源模型)
    │       └── Endsley (情境意识显示设计)
    └── Rasmussen (认知工程)
        ├── Bainbridge (自动化悖论)
        └── Vicente (认知工作分析)
                └── extends Gibson (生态心理学)
```

## 跨学派关联

- **认知工程学派**：Rasmussen 的抽象层级与技能-规则-知识框架是本学派的方法论基础
- **生态心理学派**：Gibson 的可供性理论为生态界面设计提供了感知理论基础
- **分布式认知学派**：Hutchins 的分布式认知与 Vicente 的社会技术系统分析相互呼应
- **自然决策学派**：Klein 的再认启动决策模型与自动化系统中的决策分配相关
