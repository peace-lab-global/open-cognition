# 克里斯托弗·威肯斯 Christopher Wickens

> "When multiple tasks compete for the same resource pool, performance degrades — not because the operator is incompetent, but because the design ignores human limits."

## 基本信息

- **生卒**：1943年—
- **国籍**：美国
- **领域**：工程心理学（Engineering Psychology）、人因工程、注意力研究
- **机构**：伊利诺伊大学（University of Illinois）& 科罗拉多州立大学（Colorado State University）
- **地位**：多资源模型的提出者，工程心理学教科书的标杆作者，人因工程领域最具影响力的理论家之一

## 核心命题

### 多资源模型（Multiple Resource Model）

威肯斯（1984/2002/2008）提出，人类的注意力不是一个单一的容量有限的资源池，而是由多个相对独立的资源池组成。任务之间的干扰程度取决于它们是否共享相同的资源维度。模型定义了四个资源维度：

1. **加工阶段**（Processing Stage）：感知 → 认知加工 → 反应
2. **感知通道**（Modality）：视觉 vs 听觉
3. **视觉通道**（Visual Channel）：中央视觉（focal）vs 周边视觉（ambient）
4. **加工编码**（Processing Code）：空间型 vs 言语型

当两个任务占用的资源维度重叠越多，双任务干扰越严重。这一模型为显示设计、工作负荷评估和任务分配提供了可预测的理论基础。

### 工程心理学教科书体系

威肯斯的 *Engineering Psychology and Human Performance*（1984年起多版）是人因工程领域的标准教科书，系统整合了知觉、注意力、决策、动作控制等领域的人因研究，影响了几代人因研究者与设计师。

### 情境意识显示设计

与 Endsley（2003）合作的 *Designing for Situation Awareness* 将多资源模型应用于显示设计，提出应根据操作者的认知资源特性来设计界面——在正确的通道、正确的阶段提供正确的信息。

## 思想脉络

```
Simon (有限理性与有限资源)
    ↓
威肯斯 (多资源模型)
    ├── 继承 Kahneman (注意力作为有限资源)
    ├── 平行 Sweller (认知负荷理论)
    ├── 影响 Endsley (情境意识显示设计)
    └── 发展 认知负荷 (cognitive-load) 概念
```

威肯斯的第一阶段继承了 Simon 的有限理性思想，认识到人类认知资源的有限性。第二阶段发展出多资源模型，将单一的"容量"概念分解为多维度的资源池。第三阶段将理论应用于实际系统设计——从座舱显示到 AR/VR 界面，形成了理论驱动的工程设计传统。

## 重要概念

- **多资源模型**：人类注意力由多个独立资源池构成，任务干扰取决于资源重叠程度
- **资源竞争预测**：通过分析任务所占用的资源维度，可以预测双任务干扰程度
- **通道独立性**：视觉与听觉通道的资源相对独立，跨通道呈现可减少干扰
- **编码分离**：空间任务与言语任务的资源相对分离，不同类型信息应使用不同编码通道

## 当代应用

- **显示设计**：座舱、控制室显示器布局应避免同一资源通道的过载
- **座舱设计**：飞行信息的视觉-听觉分工基于多资源模型的原则
- **警报系统**：关键告警使用听觉通道以避免与视觉监控任务竞争
- **多任务分析**：评估操作者同时处理多项任务时的干扰与安全风险
- **AR/VR 界面设计**：虚拟现实中的信息呈现需要考虑空间-言语编码分离

## 常见误读

1. **误读**：多资源模型意味着人类可以"无限并行"只要使用不同通道。
   **正读**：模型说明资源是有限的且按维度分离，但各通道的容量仍然有限。跨通道并行可减少干扰，但不能消除容量上限。

2. **误读**：多资源模型与认知负荷理论是同一理论。
   **正读**：二者平行但不同。多资源模型关注资源的竞争维度（任务间如何干扰），认知负荷理论关注学习任务的内在/外在/关联负荷（如何优化学习）。

3. **误读**：四个资源维度是确定的、最终的。
   **正读**：模型持续演进，维度的具体划分仍有争论。威肯斯自己也在不同版本中进行了修订与调整。

## 跨学科关联

- **认知负荷理论**：Sweller 的认知负荷与多资源模型互补——前者关注学习设计，后者关注任务分配
- **情境意识**：Endsley 扩展了威肯斯的工作，将资源模型应用于 SA 的显示设计
- **设计心理学**：Norman 的设计原则与威肯斯的工程心理学传统相互呼应
- **认知神经科学**：多资源模型的神经基础——不同脑区的加工特化
- [卡尼曼 Kahneman](../../../心理学/学派/行为经济学/丹尼尔.md) `[继承]` 多资源模型建立在注意力作为有限资源的认知心理学传统上
- [西蒙 Simon](../控制论/西蒙.md) `[继承]` 多资源模型将有限理性从决策扩展到注意力的多维度分析
- [斯威勒 Sweller](../自然决策/sweller.md) `[互补]` 多资源模型与CLT互补——前者关注任务分配，后者关注学习设计
