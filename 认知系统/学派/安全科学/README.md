# 安全科学学派 Safety Science School

安全科学学派（Safety Science）是认知系统工程领域中专注于复杂系统安全性的思想流派。该学派从传统的"防止出错"范式出发，逐步发展出以"理解如何成功"为核心的 Safety-II 新范式，深刻改变了我们对事故、错误与组织韧性的理解。

## 核心关切

- 事故的本质是什么？是不可避免的系统特性，还是可预防的管理失败？
- 人类错误是原因还是症状？我们应当惩罚犯错者还是改进系统？
- 安全管理的目标是什么？消除失败还是增强成功的能力？

## 学派特征

1. **系统思维**：事故不是单一原因造成的，而是多个系统层次缺陷的组合结果
2. **从问责到学习**：从寻找"谁犯了错"转向理解"系统为何允许这种情况发生"
3. **从 Safety-I 到 Safety-II**：从关注"出了什么问题"转向关注"通常如何成功"
4. **组织视角**：安全不仅是技术问题，更是组织文化、决策结构和权力关系问题

## 代表思想家

| 思想家 | 核心贡献 | 关键概念 |
|--------|----------|----------|
| [埃里克·霍纳格尔 Erik Hollnagel](hollnagel.md) (1941-2024) | Safety-II 范式 | ETTO 原则、CREAM、功能共振 |
| [詹姆斯·里森 James Reason](reason.md) (1938-2025) | 瑞士奶酪模型 | 组织事故、主动失效/潜在条件 |
| [西德尼·德克尔 Sidney Dekker](dekker.md) (1971-) | 漂移失败理论 | 公正文化、安全差异化 |
| [查尔斯·佩罗 Charles Perrow](perrow.md) (1925-2019) | 常事故理论 | 交互复杂性、紧耦合、组织社会学视角 |

## 与相关学派的关系

- **认知工程学派**（Rasmussen、Woods）：安全科学学派大量继承了认知工程的系统分析方法
- **韧性工程**：霍纳格尔与 Woods、Leveson 共同创立韧性工程，是安全科学的自然延伸
- **社会学**：贝克的风险社会理论为安全科学提供了更宏观的社会理论背景
- **组织社会学**：佩罗从组织社会学视角进入安全研究，为学派提供了技术-组织交互的结构分析框架

## 学派发展脉络

```
Rasmussen (认知工程) ──┬── Reason (瑞士奶酪模型, 1990)
                       ├── Hollnagel (Safety-II, 2014)
                       │     └── 共同创立韧性工程
                       ├── Dekker (漂移失败 + 公正文化, 2011)
                       │     └── 对传统安全文化的批判性反思
                       └── Perrow (常事故理论, 1984) ← 组织社会学视角
                             └── 紧耦合 × 交互复杂性 → 事故不可避免
```
