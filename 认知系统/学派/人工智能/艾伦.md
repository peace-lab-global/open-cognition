# 艾伦·图灵 · Alan Turing

艾伦·图灵（Alan Turing，1912–1954）是英国数学家、逻辑学家、密码学家与计算机科学奠基人。他提出“图灵机”抽象模型，奠定可计算性理论；二战期间参与破译德军恩尼格玛密码；1950 年以〈计算机器与智能〉提出“模仿游戏”，把“机器能否思考”从哲学论题转化为可实验问题。

## 核心命题 / Core Theses

### 1. 机器智能的可操作性定义

图灵不追问“思考”的本质，而问机器能否在行为上无法与人类区分。这一转向使“智能”脱离内在意识，成为可测试的功能属性。

### 2. 通用计算

图灵机证明：凡可算法化的问题，皆可由通用机器执行。这构成数字计算机与后来人工智能的理论基础。

### 3. 学习即搜索

图灵预言，与其手工编写程序，不如让机器通过学习、搜索与试错获得智能——这预示了机器学习与强化学习的发展路径。

## 关键著作 / Key Works
