# 诺曼 / Donald Norman

> **领域**: 认知工程 / 设计心理学
> **时期**: 当代 (Contemporary)
> **生卒**: 1935–
> **国籍**: 美国

> "不是人犯了错，而是设计引诱了错误。"

## 一、思想概要

唐·诺曼（Don Norman, 1935–）是加州大学圣迭戈分校名誉教授、尼尔森·诺曼集团联合创始人，是设计心理学和以人为中心设计运动的奠基人。他将认知心理学的洞见引入设计领域，从根本上改变了人们对"人为错误"的理解——错误不是人的缺陷，而是糟糕设计的症状。

诺曼的核心贡献在于将认知科学的理论转化为设计师可以理解和应用的实践原则。他的"行动七阶段模型"、"可供性"（affordance）概念、"约束"（constraints）原则等成为全球UX设计师的基本工具。他的工作横跨学术研究与工业实践，使认知工效学成为设计领域的核心学科。

在AI时代，诺曼的设计原则面临新的挑战和机遇——如何让AI系统的功能"可见"，如何让用户理解AI的能力边界，如何在自动化系统中保持人的有效参与。

Don Norman transformed how we understand human error by reframing it as a design problem rather than a human failing. His action model, affordance concept, and design principles have become foundational to UX design worldwide. His work bridges cognitive psychology, engineering, and design practice.

## 二、核心命题

1. **不是人犯错，是设计犯错**：大多数"人为错误"的根因是糟糕的设计——好的设计让正确行为自然发生，让错误行为难以发生。
2. **行动七阶段模型**：人的行动可分解为七个阶段——从目标形成到评估结果。每个阶段都可能出现错误，好的设计在每个阶段提供支持。
3. **可供性（设计中的 Affordance）**：好的设计让功能"可见"——按钮"看起来"可以被按，把手"看起来"可以被拉。
4. **情感化设计**：设计不仅关乎功能，还关乎情感——美观的界面让人感觉更好，也更易用（积极情感扩展认知灵活性）。
5. **概念模型**：用户不需要理解系统内部如何工作，但需要一个正确的"概念模型"来理解如何与之交互。

## 三、思想脉络

### 早期（认知心理学）
在哈佛大学和加州大学圣迭戈分校研究记忆和注意——研究"记忆中的信息"如何被检索和组织。

### 中期（设计心理学与认知工效学）
《设计心理学》（1988，原名 *The Psychology of Everyday Things*）将认知心理学引入设计领域。《人为错误》（1990）系统分析错误的类型和设计对策。

### 晚期（情感化设计与以人为本）
《情感化设计》（2004）将情感纳入设计框架。推动"以人为中心的设计"成为全球设计实践的主流。

## 五、重要概念

- **行动七阶段**：目标→意图→行动规划→执行→感知状态→解释→评估。设计应在每个阶段提供支持。
- **可供性 Affordance**（设计语境）：设计元素应让功能"可见"——与Gibson的原义有重要区别。
- **约束 Constraints**：通过物理、逻辑、文化约束减少错误可能。
- **映射 Mapping**：控制与效果之间应有清晰的对应关系。
- **错误宽容 Error Tolerance**：设计应宽容错误——可撤销、可恢复。

## 六、思想坐标

- **承接**：吉布森（可供性）、西蒙（有限理性）、认知心理学
- **批判**：以技术为中心的设计、"用户太笨"的假设
- **影响**：UX设计、人机交互、产品设计、服务设计

## 七、当代应用

- **UX/UI设计**：诺曼的设计原则是全球UX设计师的基本功
- **AI界面**：AI系统如何让用户理解其能力边界
- **医疗设计**：医疗设备界面的设计错误可能导致致命后果
- **自动驾驶**：如何让驾驶员理解自动驾驶的状态和能力边界

## 八、常见误读

- **"Norman的affordance = Gibson的affordance"**：Norman把affordance理解为"感知到的功能提示"（perceived affordance），Gibson的原义是"环境提供的行动可能性"。
- **"以人为中心 = 用户说了算"**：以人为中心是理解用户需求，但设计需要平衡用户、技术和商业。
- **"设计心理学 = 美化界面"**：设计心理学关注认知过程——如何让界面匹配人的认知方式。

## 九、跨学科关联

- **吉布森** `[发展/改写]` 将affordance从生态心理学引入设计领域
- **西蒙** `[继承]` 有限理性是设计心理学的前提
- **卡尼曼** `[互补]` 认知偏差研究为设计提供了用户模型
- **斯威勒** `[互补]` 认知负荷理论补充了设计中的信息呈现原则
- **班布里奇** `[平行]` 自动化悖论与设计心理学——都关注设计如何影响人的行为
- **威肯斯** `[互补]` 多资源模型与行动七阶段的互补
