# 克拉克 / Andy Clark

> **领域**: 生态认知 / 延展心智
> **时期**: 当代 (Contemporary)
> **生卒**: 1957–
> **国籍**: 英国/苏格兰

> "心智不止于颅骨。"

## 一、思想概要

安迪·克拉克（Andy Clark, 1957–）是萨塞克斯大学哲学教授，当代心灵哲学与认知科学的核心人物。他的最重要贡献是"延展心智"（Extended Mind）理论——认知过程不限于大脑，笔记本、手机、计算机都是认知系统的组成部分。心智的边界不是颅骨，而是功能耦合的边界。

克拉克还发展了预测加工（Predictive Processing）理论——大脑是一台预测机器，不断生成对感官输入的预期，只处理预测与实际输入之间的"误差"。他论证人类天生就是"赛博格"——语言、数字、书写都是"认知假体"。

在AI与LLM时代，克拉克的延展心智理论提供了理解人+AI协作的核心框架——LLM作为认知假体参与推理和记忆。

Andy Clark proposed the Extended Mind thesis — cognition extends beyond the skull to tools and environments that form functional coupling with the brain. His predictive processing framework and "natural-born cyborg" thesis position humans as fundamentally technology-augmented cognizers, deeply relevant to understanding human-AI collaboration.

## 二、核心命题

1. **延展心智**：认知过程可以延展到颅外的工具和环境。
2. **预测加工**：大脑通过生成预测并最小化预测误差来认知世界。
3. **自然天生的赛博格**：人类天生就是认知增强者——技术构成人的认知本质。
4. **认知卸载**：人主动将认知任务卸载到环境中——这是智能的表现。

## 五、重要概念

- **延展心智**：认知过程可以延展到颅外的工具和环境
- **预测加工**：大脑通过生成预测并最小化预测误差来认知世界
- **主动推理**：有机体主动行动以验证自己的预测
- **认知脚手架**：环境中的结构支撑和增强认知能力
- **007原则**："知道如何做到 = 知道在哪里找到"
