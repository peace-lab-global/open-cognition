# 西蒙思想年表 / Herbert Simon Timeline

### 1916 · 出生于美国威斯康星州
### 1947 · 《Administrative Behavior》— 有限理性概念
### 1956 · Logic Theorist — 与Newell开发第一个AI程序
### 1957 · 达特茅斯AI夏季研究项目
### 1969 · 《The Sciences of the Artificial》
### 1972 · 《Human Problem Solving》（与Newell）
### 1975 · 图灵奖（与Newell共同获得）
### 1978 · 诺贝尔经济学奖
### 2001 · 逝世

## 思想演变关键词
```
组织决策 (1947) → AI创建 (1956) → 人工科学 (1969) → 问题解决理论 (1972)
```
