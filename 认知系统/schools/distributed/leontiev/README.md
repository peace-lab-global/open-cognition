# 列昂捷夫 / A.N. Leontiev

> **领域**: 分布式认知 / 活动理论
> **时期**: 现代 (Modern)
> **生卒**: 1903–1979
> **国籍**: 苏联/俄罗斯

> "意识不是先于活动存在的，而是在活动中产生的。"

## 一、思想概要

阿列克谢·列昂捷夫（A.N. Leontiev, 1903–1979）是莫斯科大学心理学教授、活动理论的创始人。他在维果茨基文化-历史理论基础上，独立发展了以"活动"为核心范畴的心理学框架，提出动机-行动-操作的三层次分析模型。

列昂捷夫的核心洞见是：意识不是预先存在的，而是在主体与客体的交互活动中历史地形成的。同一个行动在不同活动中具有不同意义——这一区分对理解工作中的认知行为具有重要意义。

A.N. Leontiev founded activity theory, establishing activity as the central category of psychology. His three-level analysis (Activity-Action-Operation, corresponding to Motive-Goal-Conditions) provides a framework for understanding how consciousness emerges from practical engagement with the world.

## 二、核心命题

1. **活动是心理学的核心范畴**：心理不是对刺激的被动反应，而是在活动中产生的。
2. **三层次分析**：活动→行动→操作，分别对应动机→目标→条件。
3. **行为的意义不等于行为本身**：同一个行动在不同活动中有不同意义。
4. **意识在活动中形成**：意识是在主体与客体的交互活动中历史地形成的。

## 五、重要概念

- **活动 Activity**：由动机驱动的主体-客体交互过程
- **行动 Action**：有意识的、指向目标的过程
- **操作 Operation**：自动化的、受条件制约的执行过程
- **动机 Motive**：活动的驱动力
- **内化 Internalization**：外部活动转化为内部心理过程
