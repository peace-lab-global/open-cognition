# 苏格拉底诘问

## 一句话定义

苏格拉底诘问是一种通过系统性追问来暴露论证中隐含矛盾、未检验假设和逻辑缺陷的探究方法——不是教人"正确答案"，而是通过提问帮助人发现自己思维中的盲区与矛盾，"我唯一知道的是我什么都不知道"。

## 提出者与背景

- **首要提出者**：Socrates（苏格拉底），雅典哲学家，一生不著述，只在广场与人对话
- **记录者**：Plato（柏拉图）的对话录是苏格拉底方法的主要文本来源
- **历史语境**：雅典智者派（Sophists）以修辞术取胜，追求说服而非真理；苏格拉底反对这种方式，追求通过追问抵达真知
- **针对的问题**：人们往往"自以为知道"而实际并不知道——如何让一个人意识到自己的无知，从而开启真正的求知？

## 核心要义

### 诘问的六个层次（Richard Paul）

| 层次 | 追问方向 | 典型问题 |
|------|----------|----------|
| 1 | **澄清概念** | "你说的 X 具体是什么意思？" |
| 2 | **追问假设** | "你为什么认为这是理所当然的？" |
| 3 | **追问证据** | "你如何知道这是真的？有什么证据？" |
| 4 | **追问视角** | "如果有人从 Y 的角度看，会怎么说？" |
| 5 | **追问后果** | "如果接受你的结论，会产生什么影响？" |
| 6 | **追问问题本身** | "为什么这个问题重要？我们真正该问的是什么？" |

### 诘问的核心结构（Elenchus）

```
Step 1:  对方陈述信念 B
Step 2:  追问 → 引出对方的隐含前提 P1, P2
Step 3:  从 P1 + P2 推导出结论 C
Step 4:  C 与 B 矛盾
Step 5:  Aporia（困惑）→ 意识到自己的无知
Step 6:  重新审视信念 B → 更接近真理
```

### 关键论点

1. **知识即回忆**（柏拉图诠释）：真知已在灵魂中，诘问只是"产婆术"——帮助人"生出"已有但未被意识到的知识
2. **承认无知是智慧的起点**：苏格拉底之优于他人，在于他知道自己不知道（"我比这个人更有智慧：他以为他知道什么，其实不知道；我至少知道自己不知道"）
3. **对话优于独白**：思维在对话中被检验和精炼，独白容易陷入自我确认的循环
4. **追问而非说服**：苏格拉底方法的目标不是赢得辩论，而是共同接近真理
5. **不适感是信号**：当你感到被追问"刺痛"了，往往意味着触碰到了未经审视的核心信念

## 通俗用法 vs 学术原义

| 维度 | 通俗用法 | 学术原义 |
|------|----------|----------|
| "苏格拉底式提问" | "多问问题" | 有结构、有层次地系统性追问，目标是暴露逻辑矛盾 |
| "产婆术" | "引导对方自己想出来" | 一种认识论立场：知识不可被灌输，只能被引导发现 |
| "反问" | "用问句表达不同意见" | Elenchus：从对方前提推导出矛盾，迫使重新审视 |

## 与相关概念的关系

- **哲学根基**：[批判性思维](./critical-thinking.md) — 苏格拉底诘问是批判性思维的古代原型
- **同源**：[辩证法](../../philosophy/concepts/dialectics.md) — 诘问法 → 柏拉图辩证法 → 黑格尔辩证法 → 马克思辩证法
- **互补**：[逆向思维](./inversion-thinking.md) — 苏格拉底追问"什么是错的"来逼近"什么是对的"
- **关联**：[心智模型](./mental-model.md) — 诘问的目标之一就是使隐性心智模型显性化
- **关联**：[奥卡姆剃刀](./occams-razor.md) — 追问中自然应用简约原则：剔除多余假设

## 代表思想家

- **Socrates（苏格拉底）** — 诘问法的创始实践者，"未经审视的生活不值得过"
- **Plato（柏拉图）** — 将苏格拉底方法记录并发展为辩证法
- **Richard Paul** — 将苏格拉底诘问系统化为六层次框架，纳入批判性思维教育
- **Peter Boghossian** — "街头认识论"——将苏格拉底方法应用于日常对话中的信念审视
- **Karl Popper** — "批判理性主义"延续了苏格拉底"知识来自批判"的精神

## 应用场景

- **教育**：代替讲授法，用追问引导学生自主发现——法学、医学、MBA 的核心教学法
- **心理咨询**：认知行为疗法（CBT）中的"苏格拉底式提问"用于帮助患者审视不合理信念
- **团队讨论**：用结构化追问代替"谁的声音大谁赢"的讨论模式
- **产品需求分析**："用户真的需要这个功能吗？""我们假设了什么？""什么证据支持这个假设？"
- **自我反思**：定期对自己进行苏格拉底式追问——"我为什么相信这件事？"

## 常见误读

- **误读 1**："苏格拉底方法就是不断问为什么" — 它有结构和层次，不是随机发问
- **误读 2**："苏格拉底方法用于攻击他人" — 苏格拉底的目标是共同求真，不是赢得辩论（尽管他的对手常感到被冒犯）
- **误读 3**："苏格拉底方法只适合哲学讨论" — 它适用于任何需要审视假设和检验论证的场景
- **误读 4**："苏格拉底不教任何东西" — 他教的是"如何思考"，而非"思考什么"——这是更根本的教育
