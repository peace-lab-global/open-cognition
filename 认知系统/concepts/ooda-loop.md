# OODA 循环

## 一句话定义

OODA 循环是描述快速决策与适应性竞争的认知模型——通过观察（Observe）、判断（Orient）、决策（Decide）、行动（Act）的持续循环，在对手完成一轮认知之前先行行动，从而夺取主动权。

## 提出者与背景

- **首要提出者**：John Boyd（约翰·博伊德），美国空军上校、军事战略家
- **历史语境**：Boyd 在朝鲜战争空战研究中提出——F-86 对 MiG-15 的胜率并非来自飞机性能优势，而来自座舱设计使飞行员能更快观察与判断
- **针对的问题**：在时间紧迫、信息不完备、对手不断变化的竞争环境中，如何做出比对手更快的决策？

## 核心要义

### 四阶段循环

```
    ┌─────── Observe（观察）◄──────────────────────┐
    │         收集环境信息、感知变化                  │
    ▼                                               │
 Orient（判断）──── 最关键阶段 ────┐
    │  综合信息形成对局势的理解     │
    │  受基因、文化、经验、新信息影响│
    ▼                              │
 Decide（决策）                    │
    │  基于判断选择行动方案          │
    ▼                              │
    Act（行动）────────────────────┘
       执行并观察结果 → 反馈回 Observe
```

### 关键论点

1. **Orient 是灵魂**：四个阶段中 Orient 最为关键——它不只是"分析信息"，而是将信息与既有心智模型碰撞，生成新的理解
2. **速度是武器**：不是"绝对速度"，而是"相对速度"——你的循环比对手快，就能让对手的每一步决策都建立在过时的信息上
3. **进入对手循环**：通过快速行动改变环境，迫使对手不断重新 Orient，使其陷入混乱与被动
4. **多模型竞争**：Boyd 强调 Orient 阶段应同时维持多个心智模型，让它们在竞争中筛选出最适应的一个——这与 [自然选择](../concepts/predictive-processing.md) 思想相通
5. **不仅是循环，更是有机体**：OODA 不是机械流水线，而是一个自组织的适应系统；反馈回路存在于每个阶段之间

### OODA 的深层结构

Boyd 后期将 OODA 从四步循环深化为一个更复杂的系统：

| 阶段 | 深层内涵 |
|------|----------|
| **Observe** | 不仅是感官输入，还包括对"未发生之事"的觉察 |
| **Orient** | 解构旧模型 + 建构新模型的辩证过程（分析与综合） |
| **Decide** | 选择即假设——决策是对世界运作方式的一个赌注 |
| **Act** | 行动即实验——每一次行动都在测试假设 |

## 通俗用法 vs 学术原义

| 维度 | 通俗用法 | 学术原义 |
|------|----------|----------|
| "快速决策" | "赶紧做决定" | 在竞争对抗中，通过更快的认知循环夺取主动权 |
| "观察-判断-决策-行动" | "四步流程" | 一个非线性、多反馈回路的适应系统 |
| "比对手快" | "动作快" | 让对手的 Orient 阶段持续失效，使其决策建立在过时信息上 |

## 与相关概念的关系

- **同源**：[系统思维](./systems-thinking.md) — OODA 本质是一个包含反馈回路的认知系统
- **互补**：[麦肯锡方法](./mckinsey-method.md) — 麦肯锡适合深度分析，OODA 适合时间压力下的快速迭代
- **互补**：[贝叶斯思维](./bayesian-thinking.md) — Orient 阶段的信息更新本质上是贝叶斯式的
- **关联**：[意义建构](./sensemaking.md) — Orient 阶段的核心过程就是意义建构
- **关联**：[心智模型](./mental-model.md) — Boyd 强调多模型竞争，避免单一模型的锁定效应

## 代表思想家

- **John Boyd** — 美国空军上校，OODA 循环创始人，"20世纪最伟大的战略家之一"
- **Frans Osinga** — 荷兰军事学者，系统阐释 Boyd 战略理论的学术价值
- **Robert Coram** — Boyd 传记作者，使其思想为公众所知
- **John Richards** — 将 OODA 引入商业竞争与创业领域

## 应用场景

- **军事指挥**：Boyd 的原始场景——在空战中通过更快循环夺取制空权
- **商业竞争**：敏捷企业通过快速迭代（MVP → 反馈 → 调整）击败大公司的缓慢决策
- **创业**：精益创业的 Build-Measure-Learn 循环本质上是 OODA 的产品化版本
- **危机管理**：突发事件中需要快速感知-判断-行动-反馈的闭环
- **竞技体育**：实时观察对手动作 → 判断意图 → 决策应对 → 执行
- **软件开发**：敏捷开发中的 Sprint 回顾即是一种 OODA 循环

## 常见误读

- **误读 1**："OODA 就是快" — 速度只是表面，核心是 Orient 阶段的认知质量；快速做出错误决策比不做更糟
- **误读 2**："OODA 是线性四步" — Boyd 后期强调它是非线性系统，每个阶段都有到其他阶段的直接反馈
- **误读 3**："OODA 只适用于竞争场景" — 任何需要在不确定性中快速迭代决策的场景都适用
- **误读 4**："OODA 与计划矛盾" — OODA 不反对计划，而是强调计划必须能被快速调整
