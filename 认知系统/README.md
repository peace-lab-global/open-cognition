# 认知系统工程 · Cognitive Systems Engineering

> 认知不是发生在颅腔内的孤立过程，而是**人—技术—环境**联合系统中的涌现。

---

## 领域定位

本领域收录**认知系统工程（CSE）** 及其相关传统的思想家、概念与实践方法。CSE 诞生于 1980 年代（Rasmussen、Woods、Hollnagel），核心主张：

1. **认知是系统性的**：不能仅从个体心智理解，必须在人—技术—环境的耦合中分析
2. **认知是可设计的**：通过理解认知机制，可以设计更好的工作系统
3. **安全来自韧性**：不是防止错误，而是建设系统持续适应的能力

本领域与已有领域的关系：

| 已有领域 | 互补关系 |
|---|---|
| 心理学（认知） | CSE 超越个体认知，分析认知在系统中的分布 |
| 社会学 | CSE 关注微观工作系统中的认知，社会学关注宏观结构 |
| 佛教认知理论 | 八识体系分析个体意识结构，CSE 分析系统认知结构 |
| 哲学（现象学） | 梅洛-庞蒂的具身性是 CSE 感知观的哲学源头 |

## 设计原则

1. **系统观优先**：不把人当作孤立的信息处理器，而是分析人-技术-环境的联合认知系统
2. **工程导向**：不止于理解认知，更关注如何设计系统以优化认知表现
3. **韧性而非防错**：承认错误不可避免，关注系统如何在意外中保持功能
4. **跨传统对话**：与心理学、哲学（现象学）、佛教认知理论建立显式关联
