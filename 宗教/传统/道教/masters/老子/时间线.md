# 老子年表 / Timeline of Laozi

## 老子其人 / The Historical Laozi

关于老子的历史身份，自司马迁以来即有争议。以下列出传统记载与学术考证两条线索。

The historical identity of Laozi has been debated since Sima Qian. Below are two threads: traditional accounts and scholarly考证.

### 传统记载（司马迁《史记》）/ Traditional Account (Sima Qian)

| 时间 / Time | 事件 / Event |
|------|------|
| 约公元前601年 | 生于楚国苦县厉乡曲仁里（今河南鹿邑），姓李名耳，字聃 |
| 春秋末期 | 任周守藏室之史（国家图书馆馆长） |
| 约公元前518年 | 孔子问礼于老子（传说）——老子批评孔子的骄气和多欲 |
| 约公元前6世纪末 | 见周之衰，骑青牛西出函谷关 |
| — | 关令尹喜请老子著书，乃著《道德经》五千言 |
| — | 出关后不知所终 |

司马迁另列两位候选人：老莱子（楚人，与孔子同时）、太史儋（周太史，孔子后百余年）。

Sima Qian lists two additional candidates: Lao Laizi (a man of Chu, contemporary of Confucius) and Tai Shi Dan (Grand Historian of Zhou, over a century after Confucius).

### 学术考证 / Scholarly Perspectives

| 观点 / View | 代表学者 / Scholars | 论据 / Evidence |
|------|----------|------|
| 春秋末期人，与孔子同时或略早 | 传统说法 | 《史记》记载、先秦文献引用 |
| 战国中期人 | 梁启超、冯友兰 | 文本中的战国用语、反对儒家仁义的历史背景 |
| 文本渐成说（非一人一时之作） | 当代主流 | 郭店楚简证明文本是逐渐编纂成书的 |

## 《老子》文本年代 / Timeline of the Text

| 年代 / Date | 事件 / Event |
|------|------|
| 约公元前6世纪 | 传统认为老子著《道德经》（但学界存疑） |
| 约公元前4世纪 | 郭店楚简——最早已知文本，约2000字，无分章 |
| 约公元前3世纪 | 文本继续扩充，可能加入反儒内容 |
| 约公元前200年 | 马王堆帛书——接近今本，德经在前道经在后 |
| 约公元前2世纪 | 河上公注本——最早的完整注疏之一 |
| 约249年 | 王弼注本——玄学化阐释，影响最大的通行本基础 |
| 1973年 | 马王堆帛书甲乙本出土——德经在前道经在后 |
| 1993年 | 郭店楚简甲乙丙出土——最早文本，揭示文本层累过程 |
| 2009年 | 北大汉简入藏——77章，填补帛书与通行本之间的空白 |

## 老子在道教中的神化 / Deification in Religious Daoism

| 时代 / Era | 事件 / Event |
|------|-----------|
| 东汉（约公元2世纪） | 道教兴起，老子开始被神化；《老子想尔注》将老子宗教化解读 |
| 东汉桓帝（146–168） | 桓帝亲祠老子，将其视为"道"的化身 |
| 东晋 | 老子被尊为"太上老君"（最崇高的道教神祇之一），位列"三清"之一 |
| 唐代 | 李唐皇室尊老子为祖先，追封"太上玄元皇帝"；道教鼎盛 |
| 宋代 | 宋真宗加封"太上老君混元上德皇帝" |

## 老子影响史 / History of Influence

| 时代 / Era | 事件/发展 / Development |
|------|-----------|
| 战国中期（约前4世纪） | 庄子继承和发展老子思想，将"道"从宇宙论推向生存论 |
| 战国末期（约前3世纪） | 《吕氏春秋》吸收老子思想；法家韩非著《解老》《喻老》 |
| 西汉初（约前2世纪） | 黄老之学成为治国理念——文景之治以"无为"治国 |
| 东汉（1–2世纪） | 道教兴起，老子被神化为太上老君 |
| 魏晋（3–4世纪） | 王弼、何晏以老子为核心建立玄学——"贵无论" |
| 唐代（7–10世纪） | 李唐皇室尊老子为祖先；道教鼎盛；《道德经》成为科举考试科目 |
| 宋明（10–17世纪） | 理学吸收老子本体论思维；道教内丹学发展 |
| 近现代（19–20世纪） | 出土文献（马王堆、郭店）革命性地改变了老子研究 |
| 当代（21世纪） | 老子成为世界哲学的重要组成部分；生态哲学、管理学、心理治疗等领域广泛应用 |
