# 马祖道一年表 / Timeline of Mazu Daoyi

## 历史语境 / Historical Context

### 时代背景

- **唐代中期** (709-788): 安史之乱后的社会重建
- **禅宗发展**: 六祖慧能后禅宗分化为南北多系
- **洪州宗**: 马祖在江西建立的禅修中心

### 传承背景

- **六祖慧能** (638-713): 南宗禅创始人
- **南岳怀让** (677-744): 慧能弟子，马祖的老师
- **石头希迁** (700-790): 与马祖同时代，青原系

## 重要弟子 / Key Disciples

- **百丈怀海** (720-814): 建立禅宗丛林制度
- **南泉普愿** (748-835): 洪州宗重要传人
- **大珠慧海**: 《顿悟入道要门论》作者
- **庞蕴**: 在家禅修典范

## 历史影响 / Historical Impact

| 时期 | 影响 |
|------|------|
| 8 世纪 | 洪州宗成为禅宗最有影响力的支派 |
| 9 世纪 | 百丈怀海建立丛林制度，奠定禅宗组织基础 |
| 10-12 世纪 | 马祖语录成为公案禅核心文献 |
| 13 世纪 | 传入日本、朝鲜 |
| 当代 | "平常心是道"成为全球禅修最广引用的教导之一 |
