# 马祖道一著作目录 / Works of Mazu Daoyi

## 核心文献 / Core Documents

马祖道一不以著述闻名，其教导主要通过语录、公案和机缘对话传承。

### 《马祖道一禅师广录》

- **地位**: 马祖教导的最主要文献汇编
- **内容**: 收录马祖的上堂法语、机缘对话、书信等
- **现存**: 大正藏 T1981（《古尊宿语录》卷一）

### 《马祖语录》

- **内容**: 马祖的核心教导集
- **特点**: 简洁、直截、富有冲击力

## 核心教导 / Core Teachings

### 三阶段教学

1. **"即心是佛"** — 直指心性：你的心就是佛
2. **"非心非佛"** — 破除对"心"和"佛"的概念执取
3. **"平常心是道"** — 回归日常：不刻意造作，自然即是道

### 著名公案

- **"日面佛月面佛"** — 马祖临终教导
- **"野鸭子"** — 与百丈怀海的对话
- **"磨砖作镜"** — 对南岳怀让的点化回应
