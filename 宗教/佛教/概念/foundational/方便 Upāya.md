# 方便 Upāya

## 一句话定义

方便（upāya / upāya-kauśalya，善巧方便）是大乘佛教的核心方法论——佛陀根据众生的根器、时机、因缘，以**不同的方式**传达同一实相。方便不是"不真实"，而是**通向真实的有效路径**。《法华经》的"火宅三车喻"是最经典的方便论：三乘（声闻/缘觉/菩萨）是方便，一佛乘是真实。

## 核心要义

### 法华经七喻——方便的经典展示

| 喻 | 内容 | 方便与真实 |
|---|------|-----------|
| **火宅三车** | 长者以三车诱子女出火宅 | 三车（三乘）是方便，大白牛车（一佛乘）是真实 |
| **穷子喻** | 穷子不知父富，父以渐进方式引导 | 小乘是方便，大乘是真实 |
| **药草喻** | 同一雨水，草木各取所需 | 一雨（一乘）随根器而有三乘之别 |
| **化城喻** | 导师化出城市供旅人休息 | 涅槃（小乘）是化城，佛果是目的地 |
| **衣珠喻** | 贫人不知衣中有宝珠 | 众生不知自身有佛性 |
| **髻珠喻** | 国王将最珍贵的宝珠赐予最勇猛的战士 | 一佛乘是最珍贵的教法 |
| **医师喻** | 医师留药后假装死去，以激发子女服药 | 佛入涅槃是方便，佛实不灭 |

### 权实之辨

| 概念 | 含义 |
|------|------|
| **权**（方便） | 适应特定根器的暂时教法 |
| **实**（真实） | 一切众生皆可成佛的究竟教法 |
| **开权显实** | 《法华经》的方法——揭示权教中的实义 |

### 方便与般若的关系

《般若经》强调**方便与般若不可分离**：
- 无般若的方便 = 盲目的善行（不导向解脱）
- 无方便的般若 = 空洞的智慧（无法利益众生）
- **般若为母，方便为父**——二者共同成就佛果

## 常见误读

- ❌ "方便 = 不究竟 = 不真实" → 方便是通向真实的有效路径，不是"假的"。火宅中的三车确实把孩子救出来了。
- ❌ "方便 = 随便" → 方便有严格的前提——必须基于般若智慧和对众生根器的准确判断。
