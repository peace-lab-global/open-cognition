# 判教 Doctrinal Classification

## 一句话定义

**判教**（教相判释）是中国佛教各宗派为**整合庞杂的印度佛教经典**而发明的**学科分类体系**——判断一部经/一部论在整个佛教体系中的**位置与层次**，是天台、华严、唯识各宗立宗的理论基础。

## 核心要义

### 1. 天台五时八教

智顗将佛陀一代教法判为五时八教：

**五时**（时间次第）：

| 时 | 内容 | 经典 |
|---|------|------|
| 1. 华严时 | 佛陀成道后直说大乘 | 《华严经》 |
| 2. 鹿苑时 | 适应声闻根机 | 《阿含经》 |
| 3. 方等时 | 引导小乘入大乘 | 《维摩经》等 |
| 4. 般若时 | 说空性 | 《般若经》 |
| 5. 法华涅槃时 | 开权显实 | 《法华经》《涅槃经》 |

**八教**（教化方式）：
- **化仪四教**：顿/渐/秘密/不定
- **化法四教**：藏/通/别/圆

### 2. 华严五教十宗

法藏将佛教判为五教：

| 教 | 内容 | 代表 |
|---|------|------|
| 1. 小乘教 | 声闻缘觉 | 《阿含经》 |
| 2. 大乘始教 | 唯识/中观 | 《解深密经》《中论》 |
| 3. 大乘终教 | 如来藏 | 《楞伽经》《胜鬘经》 |
| 4. 顿教 | 直指人心 | 禅宗 |
| 5. 圆教 | 事事无碍 | 《华严经》 |

### 3. 唯识三时教

窥基将佛陀教法判为三时：

| 时 | 内容 | 代表 |
|---|------|------|
| 1. 有教 | 说一切法有 | 《阿含经》 |
| 2. 空教 | 说一切法空 | 《般若经》 |
| 3. 中道教 | 非有非空 | 《解深密经》 |

### 4. 判教的意义

- **整合**：将庞杂经典系统化
- **定位**：判断经论在体系中的位置
- **立宗**：各宗通过判教确立自宗经典为最高

### 5. 现代判教尝试

| 尝试 | 内容 |
|------|------|
| 印顺法师三系判教 | 性空唯名/虚妄唯识/真常唯心 |
| 太虚大师三系 | 法性空慧/法相唯识/法界圆觉 |

## 通俗 vs 学术

| 通俗化版本 | 学术原意 |
|---|---|
| "判教 = 分等级" | 是定位，非优劣判断 |
| "圆教最高" | 是各宗自家立场，非客观结论 |
| "判教过时了" | 其精神 = 系统化分类思维 |

- **教育学**：判教 = 学科分类体系的古代版本

- **信息管理**：判教精神 = 元数据分类

## 常见误读

- ❌ "判教 = 分等级" → 是定位，非优劣。
- ❌ "圆教最高 = 客观事实" → 是各宗自家立场。
- ❌ "判教 = 古代的事" → 其精神（系统化分类）仍有现代价值。
