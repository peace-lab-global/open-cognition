# 五明 Pañcavidyā

## 一句话定义

**五明**（Pañcavidyā）是印度佛学院的**完整学科体系**——菩萨必须通达的五种学问：**内明**（佛学）、**因明**（逻辑）、**声明**（语言）、**医方明**（医学）、**工巧明**（技艺），代表佛教"博学多闻"的教育理想。

## 核心要义

### 1. 五明体系

| 明 | 梵文 | 内容 | 现代对应 |
|---|------|------|---------|
| **内明** | Adhyātmavidyā | 佛教教理（经律论三藏） | 哲学/宗教学 |
| **因明** | Hetuvidyā | 逻辑学、辩论术 | 逻辑学/批判思维 |
| **声明** | Śabdavidyā | 语言学、语法、修辞 | 语言学/文学 |
| **医方明** | Cikitsāvidyā | 医学、药学、养生 | 医学 |
| **工巧明** | Śilpavidyā | 工艺、建筑、天文、数学 | 工程/科学 |

### 2. 内明（核心学科）

内明是五明的核心——佛教教理的系统学习：
- **经藏**：佛陀言教
- **律藏**：行为规范
- **论藏**：系统化教理

### 3. 因明（佛教逻辑）

| 要素 | 内容 |
|------|------|
| **量** | 认知工具（现量/比量/圣言量） |
| **宗** | 论点 |
| **因** | 论据 |
| **喻** | 例证 |
| **合** | 合论 |
| **结** | 结论 |

代表人物：陈那（Dignāga）、法称（Dharmakīrti）。

### 4. 声明（语言学）

- 梵文语法学
- 修辞学与诗歌
- 音韵学与诵经
- 翻译学

### 5. 医方明与工巧明

- **医方明**：阿育吠陀、草药学、针灸
- **工巧明**：建筑（塔/寺）、造像、天文历法、数学

## 通俗 vs 学术

| 通俗化版本 | 学术原意 |
|---|---|
| "五明 = 五种技术" | 是完整的学科体系，非单纯技术 |
| "内明最重要" | 五明相互支撑，非优劣之分 |
| "五明过时了" | 其精神 = 跨学科通识教育 |

## 应用场景

- **教育学**：五明 = 古代跨学科通识教育
- **佛学院**：五明仍是课程设置的基础框架
- **比较教育**：五明 vs 中世纪七艺（三艺+四艺）
- **跨学科研究**：五明精神 = 打破学科壁垒

## 常见误读

- ❌ "五明 = 五种技术" → 是完整的学科体系。
- ❌ "内明 = 只学佛" → 内明是佛学，但需要其他四明支撑。
- ❌ "五明过时了" → 其精神（跨学科）正与现代教育趋势一致。
