# 八正道 Noble Eightfold Path

## 一句话定义

**八正道**是佛陀在初转法轮时提出的**道谛**——从苦中解脱的八项实践：**正见、正思维、正语、正业、正命、正精进、正念、正定**。这是佛教的中道，离纵欲与苦行二边。

## 核心要义

### 三学归类

| 三学 | 八正道 |
|---|---|
| **慧 Prajñā** | 正见、正思维 |
| **戒 Śīla** | 正语、正业、正命 |
| **定 Samādhi** | 正精进、正念、正定 |

### 逐项详解

1. **正见 Sammā-diṭṭhi**：理解四圣谛、因果、缘起。
2. **正思维 Sammā-saṅkappa**：出离、无嗔、无害的意向。
3. **正语 Sammā-vācā**：不妄语、不两舌、不恶口、不绮语。
4. **正业 Sammā-kammanta**：不杀生、不偷盗、不邪淫。
5. **正命 Sammā-ājīva**：以正当职业生活。
6. **正精进 Sammā-vāyāma**：断恶、修善的四正勤。
7. **正念 Sammā-sati**：四念处（身受心法）。
8. **正定 Sammā-samādhi**：四禅（初禅至四禅）。

## 通俗 vs 学术

| 通俗化版本 | 学术原意 |
|---|---|
| "八正道 = 八条戒" | 是八个层次的实践，含智慧与定 |
| "正见 = 相信佛教" | 是对缘起因果的直观理解 |
| "正定 = 冥想" | 是四禅次第，非一般冥想 |

## 常见误读

- ❌ "八正道 = 八条戒" → 含智慧与定。
- ❌ "正念 = 注意力" → 是觉知 + 四念处。
- ❌ "正定 = 发呆" → 是高度专注的禅定。
