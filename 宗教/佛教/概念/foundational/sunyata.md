# 空性 Śūnyatā

## 一句话定义

空性是佛教最核心的哲学概念——**一切事物都没有独立不变的自性（svabhāva）**，皆由因缘和合而生。但"空"不是"什么都没有"，而是"有而无自性"。各宗对空性的理解有显著差异，形成佛教哲学最深层的论争。

## 核心要义

### 各宗空观对比

| 宗派 | 空观 | 核心表达 |
|------|------|----------|
| **有部** | 人空法有 | "我"空，但五蕴等法实有 |
| **经量部** | 现在实有过未无体 | 只有当下一刹那实有 |
| **中观** | 一切法空 | 法我俱空，缘起即空性 |
| **唯识** | 遍计空/依他有/圆成实有 | 妄想空，识的因缘有，真如实有 |
| **天台** | 即空即假即中 | 三谛圆融，空不是独立的真理 |
| **华严** | 真空妙有 | 空即有，有即空，理事无碍 |

### 空性的三个层次

1. **人无我**（pudgala-nairātmya）：没有独立不变的"自我"
2. **法无我**（dharma-nairātmya）：没有独立不变的"法"（事物/概念）
3. **空空**（śūnyatā-śūnyatā）：连"空"本身也不可执着

### 空 ≠ 虚无

| 误解 | 正解 |
|------|------|
| 空 = 什么都没有 | 空 = 没有独立自性，但因缘和合的现象存在 |
| 空 = 消极 | 正因为空，一切才有可能（"以有空义故，一切法得成"——龙树） |
| 空 = 否定因果 | 空恰恰是因果（缘起）的另一面 |

## 常见误读

- ❌ "空 = 虚无主义" → 龙树明确说"空亦复空"，空不是另一种"有"也不是"无"。
- ❌ "所有宗派理解的空都一样" → 中观、唯识、天台的空观有根本差异。
- ❌ "空性可以用语言说清楚" → 空性最终指向超越概念的实相体验。
