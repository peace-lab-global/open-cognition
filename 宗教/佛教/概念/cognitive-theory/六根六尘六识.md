# 六根六尘六识 · Six Sense Constituents

## 一句话定义

六根（眼耳鼻舌身意）+ 六尘（色声香味触法）+ 六识（眼识耳识鼻识舌识身识意识）= 十八界，构成了人类认知的完整结构——根触尘，识生分别，三者和合方成认知。

## 提出者与背景

- **奠基**：《阿毗达摩俱舍论》（世亲）——说一切有部到瑜伽行派的核心结构
- **背景**：部派佛教对认知机制的精细分析
- **针对的问题**：认知是如何发生的？感知、思维、意识之间的关系是什么？

## 核心要义

### 六根（感觉器官）

| 根 | 净色根 | 对应识 |
|---|---|---|
| 眼根 | 眼球及视神经 | 眼识 |
| 耳根 | 耳蜗及听神经 | 耳识 |
| 鼻根 | 鼻腔及嗅觉神经 | 鼻识 |
| 舌根 | 舌面及味觉神经 | 舌识 |
| 身根 | 全身触觉神经 | 身识 |
| 意根 | 意（末那识 + 第六意识） | 意识 |

### 六尘（认知对象）

| 尘 | 认知内容 |
|---|---------|
| 色尘 | 颜色、形状（眼） |
| 声尘 | 声音（耳） |
| 香尘 | 气味（鼻） |
| 味尘 | 味道（舌） |
| 触尘 | 冷热、软硬、涩滑（身） |
| 法尘 | 概念、法相（意） |

### 六识（认知功能）

| 识 | 功能 | 特点 |
|---|---|---|
| 眼识 | 色尘了别 | 瞬间直观，无名言 |
| 耳识 | 声尘了别 | 随有间断（熟睡无梦时断） |
| 鼻识 | 香尘了别 | 刹那即过，不续 |
| 舌识 | 味尘了别 | 同上 |
| 身识 | 触尘了别 | 触时即生，不待意 |
| 意识 | 法尘了别 | 最胜，能思量，有漏/无漏 |

### 根尘识三和合

认知的发生条件：
1. **根**不坏（感官正常）
2. **尘**在（对象当前）
3. **识**生（心识的了别功能激活）
4. **作意**（注意力的投入）

> "根尘识三和合，故有触。触故有受。"

### 十八界

六根 × 六尘 × 六识 = 十八界，构成了佛教完整的认识论框架。

## 通俗用法 vs 学术原义

| 维度 | 通俗用法 | 学术原义 |
|---|---|---|
| 眼根 | 眼睛 | 净色根，不只是肉眼，是视神经+视觉皮层等整个系统 |
| 法尘 | 想法 | 意识的认识对象，不是"正确的法" |
| 根尘识和合 | 三者合在一起 | 三者同时俱起，缺一认知不生 |
| 意根 | 第六意识 | 意根包括第七末那识和第六意识，两者协同 |

## 与相关概念的关系

- **同源**：[八识体系](eight-consciousness.md) — 前六识+意根（含第七、第六识）+第八识
- **前置**：[八识体系](eight-consciousness.md) — 六识是八识的前六
- **发展**：[意识转化](consciousness-transformation.md) — 转六识为"成所作智"、"妙观察智"
- **平行**：[卡尼曼·双系统](../../../../心理学/学派/行为经济学/丹尼尔.md) — 系统1（快速、直观）≈前五识；系统2（慢速、分析）≈意识

## 应用场景

- 认知科学：六识可对应视觉/听觉/嗅觉/味觉/触觉/跨模态整合
- 心理咨询：感知扭曲的诊断（根、尘、识哪一层出问题）
- 人工智能：传感器（根）—信号（尘）—感知算法（识）
- 修行实践：根尘识三者和合时，观"识生处"——识如何生起？

## 常见误读

- **误读 1**：把"根"当成纯粹的器官——根是"净色根"，不只是肉体器官
- **误读 2**：把"尘"当成"尘埃"——尘是"境"，是认识的物件
- **误读 3**：把"意识"等同于"第六识"——意根还包括第七末那识，意识不独立于末那识
