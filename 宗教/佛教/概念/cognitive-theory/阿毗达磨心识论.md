# 阿毗达磨心识论 · Abhidharma Psychology of Mind

## 一句话定义

阿毗达磨（abhidharma / abhidhamma）是佛教对心智现象做的最早、最细的分类心理学：将任何一刹那的"心"拆解为**心**（citta，了别）与**心所**（cetasika，心理伴随因素）的组合，并用**心路过程**（citta-vīthi）描述认知如何按时间序列展开。它不承认有统一的"自我"，只承认**89 心**（或 121 心）与**52 心所**在因缘条件下的瞬时组合。

## 提出者与背景

- **起源**：佛陀灭度后 2–3 世纪内由部派（说一切有部、上座部等）系统化
- **南传代表**：《法集论》、觉音《清净道论》、阿耨楼陀《摄阿毗达磨义论》
- **北传代表**：世亲《俱舍论》（有部 + 经部综合）
- **针对的问题**：佛陀教法的"名色"、"五蕴"、"十二处"如何被精确定义与彼此配合？修行各阶段的心智状态具体是什么？

## 核心要义

### 心（citta）的分类

南传上座部分 89 心（或按超世间细分到 121 心）：

| 类别 | 数量 | 内容 |
|------|------|------|
| 欲界心 | 54 | 善、不善、无记（果报、唯作） |
| 色界心 | 15 | 与色界禅那相关 |
| 无色界心 | 12 | 与四无色定相关 |
| 出世间心 | 8 / 40 | 四道四果（按禅支细分达 40） |

> 北传《俱舍论》简化为**八识**（六识 + 意根 + 阿赖耶的雏形）或**六识**（仅眼耳鼻舌身意）。

### 五十二心所（cetasika）

| 类别 | 数量 | 代表心所 |
|------|------|----------|
| 通一切心所 | 13 | 触（phassa）、受（vedanā）、想（saññā）、思（cetanā）、一境性、名命根、作意 |
| 不善心所 | 14 | 贪、嗔、痴、慢、见、疑、昏沉、掉举、无惭、无愧、嫉、悭、恶作、懈怠 |
| 美心所 | 25 | 信、念、惭、愧、无贪、无嗔、中捨、悲、喜、慧、轻安、定等 |

**核心原则**：任何一刹那的"心" = 一个心（citta）+ 若干心所（cetasika）的集合。**没有"纯粹的心"**，心所决定了心的质地。

### 心路过程（citta-vīthi）

南传描述的认知时间序列（以眼识为例）：

```
有分（bhavaṅga） → 有分波动 → 有分断 → 五门转向 → 眼识 → 领受 → 推度 → 确定 → 速行（7刹那）→ 彼所缘（2刹那）→ 有分
```

| 阶段 | 功能 | 认知意义 |
|------|------|----------|
| 有分 | 生命流的基底意识 | 无对象、持续的"背景流" |
| 五门转向 | 注意力转向刺激 | 注意机制 |
| 眼识 | 纯粹感知 | 现量（pratyakṣa） |
| 领受 | 接受对象 | 初步加工 |
| 推度 | 初步判断 | 识别 |
| 确定 | 对象定性 | 善恶判定 |
| 速行 | 7 刹那的善/不善反应 | 业的主要生成点 |
| 彼所缘 | 对象再处理 | 记忆形成 |

### 有分识（bhavaṅga）

- 介于两次心路之间的"基底意识"
- 是阿赖耶识（ālayavijñāna）思想的部派前身
- 解释：为什么我们睡觉或无对象时，生命之流仍在继续

### 心所与修行的对应

| 修行目标 | 需培养的心所 | 需对治的心所 |
|----------|--------------|--------------|
| 正念 | 念（sati） | 掉举、昏沉 |
| 慈悲 | 悲（karuṇā）、无嗔 | 嗔、嫉 |
| 智慧 | 慧（paññā） | 痴、见、疑 |
| 禅定 | 一境性、轻安 | 昏沉、掉举 |

## 通俗用法 vs 学术原义

| 维度 | 通俗用法 | 学术原义 |
|------|----------|----------|
| 心 | "思维/情绪" | citta，单纯的了别 |
| 心所 | "心理特质" | cetasika，与心同生同灭的伴随因素 |
| 有分 | "潜意识" | 生命基底流，非弗洛伊德式潜意识 |
| 速行 | "快速反应" | 7 刹那的业生成点 |

## 应用场景

- **当代心理学对话**：心所分类可作为情绪粒度（emotional granularity）的古典资源
- **正念训练**：识别"受、想、行"心所 = 内观（vipassanā）的标注对象
- **认知科学**：心路过程 ≈ 认知架构的"处理阶段模型"
- **AI 心智模型**：用 citta + cetasika 组合而非单一状态机建模智能体
- **道德心理学**：速行 7 刹那 ≈ 道德判断的快速直觉过程

## 常见误读

- **误读 1**：把 89 心看作 89 个独立"心理状态"——它们是分类矩阵，任一刹那只起一心
- **误读 2**：把心所看作"独立心灵模块"——心所必须与心同起同灭，不可独存
- **误读 3**：把心路过程看作"物理时间"——它是认知逻辑时间，一刹那 ≈ 一个处理单位
- **误读 4**：把有分识等同"永恒自我"——有分也是刹那生灭、无常无我
