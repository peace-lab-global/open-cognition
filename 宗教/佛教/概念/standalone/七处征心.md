# 七处征心 · Seven Investigations of the Heart-Mind

## 一句话定义

七处征心是《楞严经》中阿难尊者七次提出"心在何处"，而佛陀逐一破斥——证明凡夫所谓的"心"既不在身内、身外、根间、明暗之中，也不在生死、中位、一切处，最终指向"心不在任何固定处所"，但心不是虚无，而是"能见之性"本身。

## 提出者与背景

- **提出者**：释迦牟尼佛，对阿难尊者
- **背景**：阿难被摩登伽女以幻术诱惑，几乎破戒，请求佛陀开示"定"——佛以此七处征心开始
- **历史语境**：针对当时外道的"心在身内/身外/中间"等常见执见
- **针对的问题**：凡夫把"心"当作一个可以找到的实体对象——这是认知错乱的根源

## 核心要义

### 七处次第

| 征问 | 阿难所答 | 佛陀破斥 |
|------|---------|---------|
| 第一征 | 心在身内 | 心不在身内——盲人守株 |
| 第二征 | 心在身外 | 心不在身外——灯必有待 |
| 第三征 | 心含于根（眼根） | 根不是心所——相夺相倾 |
| 第四征 | 心在明暗之中 | 明暗代谢，心不在中 |
| 第五征 | 心在生死（随有） | 生死本空，心不随有 |
| 第六征 | 心在中间（两楹） | 中无定所，兼二不真 |
| 第七征 | 心在一切处 | 心非一切处，心无所在亦无不在 |

### 核心命题

七处征心的结论：**心不是空间中的任何一处**

- 心不在内/外/中间——不在空间里
- 心不在明暗/生死——不在时间/变化里
- 心不是"东西"——心无形无相，不可对象化

### 七趣沉沦

七处征心之后，《楞严经》接着讲述"七趣"——众生因不同的执著（临济义玄的"有事"/无事；米拉日巴的"裸见觉性"；道元的"佛性遍一切处"），而坠入不同的生命形态：

| 趣 | 执著类型 |
|---|---------|
| 地狱 | 十习因 + 六交报 |
| 饿鬼 | 贪物 |
| 畜生 | 慢 |
| 人 | 善恶夹杂 |
| 仙 | 坚固想 |
| 天 | 贪恋心 |
| 修罗 | 怨想 |

## 通俗用法 vs 学术原义

| 维度 | 通俗用法 | 学术原义 |
|---|---|---|
| 七处征心 | 禅宗公案 | 对"心是什么"的系统性解构 |
| 心在身内 | 常识 | 佛陀指出：这是盲人守株——能见之性不在身内 |
| 心无所在 | 虚无主义 | 心不是"东西"，不是对象，但心不是无——心是功能 |
| 七趣 | 因果报应 | 不同认知模式导致不同的"存在方式" |

## 应用场景

- 心理咨询：破除当事人"我的心在某个问题/某个人身上"的实体化认知
- 认知科学：心不是离身的处理器——七处征心预演了具身认知的哲学批判
- 灵性修行：破"心有所住"的根本执著

## 常见误读

- **误读 1**：把"心不在一切处"当成"心不存在"——这是断灭见，不是佛陀的意思
- **误读 2**：把七处征心当成纯粹的否定——它是"遮"（破执），正面是"能见之性"本身
- **误读 3**：把七趣当成道德惩罚——七趣是认知模式的自然后果，不是奖惩机制
