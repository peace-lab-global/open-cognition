# 禅宗 Chan/Zen

## 一句话定义

**禅宗**是中国佛教最具原创性的宗派，主张**不立文字，直指人心，见性成佛**——以禅修（坐禅）与顿悟为核心方法，绕过繁琐教理，直探心性本源。8 世纪后传入日本、朝鲜、越南，成为东亚文化的精神主轴之一。

## 历史脉络

- **6 世纪**：传说**菩提达摩**自印度来华，为初祖。
- **7 世纪**：五祖**弘忍**东山法门；六祖**慧能**（638–713）以《六祖坛经》确立南宗"顿悟"路线。
- **8–10 世纪**：马祖、百丈创丛林制度；临济、曹洞、沩仰、云门、法眼五家分化。
- **12–13 世纪**：**荣西、道元**传曹洞入日本；**圆尔、兰溪**传临济入日本。
- **20 世纪**：铃木大拙将禅传播欧美，影响 Beat Generation 与正念运动。

## 核心要义

### 1. 教外别传，不立文字
禅宗自认佛陀正法眼藏以心传心，不依赖经典文字。

### 2. 直指人心，见性成佛
通过公案、棒喝、坐禅，直接指向"能见之性"。

### 3. 顿悟 vs 渐修
- **慧能（南宗）**：顿悟见性，渐修保任
- **神秀（北宗）**：渐修成佛
- 南宗最终胜出

### 4. 公案 Kōan
无法用逻辑思维解答的问题（"单手拍掌声"、"狗子有无佛性"），用以逼迫理性崩解，直入悟境。

### 5. 棒喝
临济宗的教学方式，以身体冲击（棒）或突然喊叫（喝）切断概念思维。

### 6. 日常即道
"运水搬柴，无非妙道"——修行不局限于打坐。

## 通俗 vs 学术

| 通俗化版本 | 学术原意 |
|---|---|
| "禅 = 打坐放松" | 坐禅是方法，见性才是目标 |
| "禅宗反经典" | 是不立文字，不是不读经典 |
| "顿悟 = 瞬间开悟" | 顿悟是见性，但仍需长期修行保任 |
| "公案无解" | 是有特定功能的**反逻辑工具** |

## 常见误读

- ❌ "禅 = 静坐放松" → 禅的终点是见性，不是放松。
- ❌ "禅反智" → 是不立文字，非反智。
- ❌ "顿悟就是全部" → 悟后仍需修行。
- ❌ "公案是谜语" → 是反逻辑的修行工具。

## Cognitive Architecture

禅宗的认知架构以"直接性"为特征，是佛教中最激进的反概念化认知路径：

| 认知模块 | 术语 | 功能描述 |
|---|---|---|
| 顿悟/渐修 | Sudden/Gradual | 两种认知风格 |
| 公案 | Kōan | 语义饱和与框架崩解 |
| 默照 | Silent Illumination | 无拣择觉知 |
| 不立文字 | 教外别传 | 反概念化认知 |

- **顿悟与渐修作为认知风格**：顿悟（sudden awakening）对应格式塔转换（Gestalt switch）——认知框架的瞬间重组；渐修（gradual cultivation）对应渐进式学习（incremental learning）。二者不是对立，而是同一认知过程的不同侧面："顿悟见性，渐修保任"。
- **公案的认知机制**：公案通过语义饱和（semantic satiation）——对同一问题的反复追问导致语义系统崩溃——以及框架崩解（frame collapse），迫使认知系统跳出概念化处理的惯性模式，进入前概念的直觉状态。
- **默照禅与无拣择觉知**：曹洞宗的"默照"（silent illumination / 只管打坐）训练一种不拣择的觉知（non-selective awareness），不同于专注式冥想（focused attention）或开放式监控（open monitoring），而是超越主客二分的直接体验——认知科学称之为"非二元觉知"（non-dual awareness）。
- **"不立文字"的认知含义**：禅宗对语言文字的怀疑不是反智，而是对概念化认知局限性的深刻洞察——概念总是对直接经验的"二次加工"，而禅要求回到"第一序"的前概念认知。

**跨领域参照**：公案的认知机制可与 [Kuhn 的范式转换](../../../../science/philosophy/kuhn.md) 比较（框架崩解与科学革命的结构同构），默照禅与 [正念认知疗法 MBCT](../../../心理学/概念/正念.md) 的"去中心化"（decentering）技术平行。详见 [七处征心](../concepts/cognitive-theory/qichu-zhengxin.md)、[百还辨见](../concepts/cognitive-theory/baihuan-bianjian.md)。
