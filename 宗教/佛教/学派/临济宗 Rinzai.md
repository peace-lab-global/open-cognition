# 临济宗 Rinzai

## 一句话定义

临济宗是禅宗两大主流之一——以**看话禅**（参公案）为核心修行方法，经中国临济义玄→大慧宗杲→日本荣西→白隐慧鹤的传承，形成了系统化的公案修行体系，强调**"大疑大悟、小疑小悟、不疑不悟"**。

## 历史脉络

- **9c**：临济义玄创立临济宗（中国）
- **12c**：大慧宗杲提倡看话禅，编《碧岩录》
- **1191**：**荣西**将临济宗传入日本
- **13–14c**：五山文学时期，临济宗成为日本文化主流
- **18c**：**白隐慧鹤**中兴——系统化公案体系，今天日本临济宗法脉皆出白隐
- **20c**：铃木大拙将临济禅推向西方

## 核心要义

### 1. 看话禅（Kanhua / Kanna Zen）

| 要素 | 说明 |
|------|------|
| **话头** | 从公案中提取的核心词/句（如"无""只手之声"） |
| **疑情** | 全身心投入的"不明白"——参究的引擎 |
| **破参** | 疑团的崩解——不是"得到答案" |

### 2. 公案体系（白隐系统）

| 层级 | 类型 | 功能 | 示例 |
|------|------|------|------|
| 初 | 理屈公案 | 破除知见 | 狗子佛性 |
| 中 | 机关公案 | 截断思维 | 只手之声 |
| 深 | 研锥公案 | 深入细参 | 万法归一 |
| 极 | 狮子啮人公案 | 大机大用 | 赵州柏树子 |
| 终 | 透法身公案 | 最终透过 | 五祖"倩女离魂" |

### 3. 与曹洞宗的对比

| 维度 | 临济宗 | 曹洞宗 |
|------|--------|--------|
| 方法 | 看话禅（参公案） | 默照禅（只管打坐） |
| 核心 | 疑情→破参 | 觉照→即心即佛 |
| 祖师 | 临济义玄→白隐 | 道元→莹山 |
| 气质 | 激烈、动态 | 沉静、稳当 |
| 社会基础 | 武士/知识分子 | 农民/町人 |

### 4. 悟后修行

白隐特别强调：开悟不是终点——"悟后十牛"（悟后的保任与深化）才是真正的修行。

## 通俗 vs 学术

| 通俗化版本 | 学术原意 |
|---|---|
| "公案 = 禅宗谜语" | 公案不是用来"解答"的，而是截断概念思维的装置 |
| "临济 = 暴力禅" | "棒喝"是教学手段，核心是公案参究的理性过程 |
| "悟 = 一次性事件" | 白隐强调"大悟多次、小悟无数次" |

## 常见误读

- ❌ "临济禅 = 日本武士道" → 临济禅影响了武士文化，但其核心是公案参究，不是武道训练。
- ❌ "公案越多越好" → 白隐的体系强调质而非量——每则公案需深参而非速过。
