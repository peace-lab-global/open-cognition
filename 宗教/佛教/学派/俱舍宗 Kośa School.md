# 俱舍宗 Kośa School

## 一句话定义

**俱舍宗**以世亲《阿毗达磨俱舍论》（Abhidharmakośa）为根本，系统整理**说一切有部**（Sarvāstivāda）哲学，提出**五位七十五法**的万法分类体系，被誉为佛教哲学的"聪明论"（入门教科书），在中国与日本都曾作为佛学院必修课程。

## 历史脉络

- **4 世纪**：世亲先学有部，后转向经量部，著《俱舍论》总结并批判有部。
- **5 世纪**：真谛首次汉译《俱舍释论》。
- **7 世纪**：**玄奘**重新翻译《阿毗达磨俱舍论》，普光、法宝、神泰三大弟子撰疏，形成**中国俱舍宗**。
- **8 世纪**：传入日本，成为**南都六宗**之一。
- **近现代**：俱舍论仍是佛学院阿毗达磨课程的核心教材。

## 核心要义

### 1. 五位七十五法

将一切法（存在的一切现象）归纳为五位七十五种：

| 位 | 数量 | 内容 | 性质 |
|---|------|------|------|
| **色法** | 11 | 五根+五境+无表色 | 物质 |
| **心法** | 1 | 心（识） | 精神主体 |
| **心所法** | 46 | 与心相应而起的心理功能 | 精神功能 |
| **心不相应行法** | 14 | 既不属色也不属心的抽象法 | 抽象存在 |
| **无为法** | 3 | 虚空/择灭/非择灭 | 不生不灭 |
| **合计** | **75** | | |

### 2. 三世实有 vs 现在有体

| 主张 | 代表 | 核心 |
|------|------|------|
| **三世实有** | 说一切有部 | 过去、现在、未来都存在 |
| **现在有体** | 经量部 | 只有现在是实有，过未是假名 |

世亲在《俱舍论》中**先述有部、后附经量部批判**——这是《俱舍论》最精彩的部分。

### 3. 四缘六因

| 类型 | 内容 |
|------|------|
| **四缘** | 因缘、等无间缘、所缘缘、增上缘 |
| **六因** | 能作因、俱有因、同类因、相应因、遍行因、异熟因 |

这是佛教最精密的**因果分析体系**。

### 4. 业与果

俱舍论对"业"的分类：

| 业的类型 | 说明 |
|---------|------|
| **身业** | 身体的行为 |
| **口业** | 语言的行为 |
| **意业** | 心理的行为 |
| **表业** | 可见的行为 |
| **无表业** | 不可见的业力（戒体/恶习） |

### 5. 修道次第

| 阶段 | 名称 | 内容 |
|------|------|------|
| 1 | 资粮道 | 积累福德智慧 |
| 2 | 加行道 | 四善根（暖/顶/忍/世第一法） |
| 3 | 见道 | 初证真如 |
| 4 | 修道 | 断除烦恼 |
| 5 | 无学道 | 阿罗汉果 |

## 应用场景

- **哲学**：五位七十五法 = 古代最精密的本体论分类
- **认知科学**：心所法 = 古代最详尽的心理功能分类
- **佛学院教育**：俱舍论 = 佛学院必修"聪明论"
- **心理学**：46 心所 vs 现代心理学的情绪/认知分类

## 常见误读

- ❌ "俱舍 = 有部 = 实有论" → 世亲在论中批判有部，倾向经量部。
- ❌ "俱舍 = 小乘" → 是佛教哲学的基础训练，大小乘共学。
- ❌ "七十五法 = 固定实体" → 是分析框架，非本体论承诺。
