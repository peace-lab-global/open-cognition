# 中观 Madhyamaka

## 一句话定义

**中观派**由 2–3 世纪的**龙树**创立，主张一切法**缘起性空**，以**二谛**（世俗谛与胜义谛）和**四句破**（四句否定）为方法论，离"常见"与"断见"二边，是大乘佛教的哲学基石。

## 历史脉络

- **2–3 世纪**：龙树著《中论》，系统化空性哲学。
- **3–4 世纪**：提婆《四百论》扩展破斥。
- **5–7 世纪**：分化为**自续派**（Svātantrika，清辩）与**应成派**（Prāsaṅgika，佛护、月称）。
- **8 世纪后**：传入西藏，成为藏传佛教哲学主干。
- **中国**：鸠摩罗什译出"三论"（《中论》《十二门论》《百论》），形成三论宗。

## 核心要义

### 1. 缘起性空（Śūnyatā）
> "众因缘生法，我说即是空，亦为是假名，亦是中道义。"——《中论》

一切现象无自性（niḥsvabhāva），故空；空不是虚无，而是**依存缘起**。

### 2. 二谛（Dve Satye）
- **世俗谛 Saṃvṛti-satya**：日常语言与现象层面（可说"有"）
- **胜义谛 Paramārtha-satya**：终极实相（一切法空）
- **二者不可分**：离世俗无胜义

### 3. 四句破（Catuṣkoṭi）
对任何命题 P：
- 是 P？非
- 非 P？非
- 是 P 且非 P？非
- 非 P 且非非 P？非

四句全破，超越一切执著。

### 4. 八不中道
不生不灭、不常不断、不一不异、不来不去——对存在论两极的全面否定。

## 通俗 vs 学术

| 通俗化版本 | 学术原意 |
|---|---|
| "空 = 什么都没有" | 空是无自性，不是虚无 |
| "中观是虚无主义" | 中观明确批判虚无主义（断见） |
| "龙树否定一切" | 否定的是**自性**，不是**缘起有** |
| "二谛是真假两世界" | 二谛是**认知方式**，不是本体分层 |

- **提婆 Āryadeva**（3 c.）：《四百论》

## 应用场景

- **哲学**：与维特根斯坦、海德格尔、德里达的跨文化对话
- **认知科学**：破自性 = 破本质主义
- **心理治疗**：对"我"的执着消解（ACT、正念）
- **伦理学**：空性如何承担伦理责任

## 常见误读

- ❌ "空 = 虚无" → 空是无自性，不是无物。
- ❌ "中观反理性" → 中观是高度理性的论证。
- ❌ "一切皆空所以可以为所欲为" → 中观明确主张缘起因果不虚。
- ❌ "二谛是两套真理" → 是同一实相的两种言说方式。

## Cognitive Architecture

中观学派的认知架构不以建构模型为目标，而以**解构一切认知固化**为方法论，其核心贡献在于认知批判：

| 认知工具 | 梵文 | 功能 |
|---|---|---|
| 二谛 | Dve Satye | 认知层次的双重框架 |
| 四句破 | Catuṣkoṭi | 穷尽并超越一切逻辑可能 |
| 八不 | Aṣṭa-niṣedha | 对存在论两极的系统否定 |
| 空性 | Śūnyatā | 反本质主义的认知立场 |

- **[二谛](../concepts/cognitive-theory/two-truths.md)作为认知双框架**：世俗谛（saṃvṛti-satya）是日常概念化认知，胜义谛（paramārtha-satya）是对认知本身无自性的洞察。二谛不是两个世界，而是同一认知过程的两个视角——类似于认知科学中"对象层次"与"元层次"的区分。
- **四句破的逻辑穷尽**：对命题 P 的四种逻辑可能（是、非、亦是亦非、非是非非）全部否定，迫使认知系统放弃对任何固定立场的执取，进入概念的"悬置"状态（epoché）。
- **应成派（Prāsaṅgika）vs 自续派（Svātantrika）**：应成派只破不立，以归谬法瓦解对方自性见；自续派允许在世俗层面建立正面论证。这一分歧反映了认知批判的彻底性程度之争。
- **与维特根斯坦的对话**：中观对"自性"（svabhāva）的破斥与维特根斯坦对"本质"的消解高度平行——二者都主张意义/存在不来自内在本质，而来自关系网络（缘起/语言游戏）。
- **反本质主义的认知遗产**：中观的核心贡献不是提供新的认知内容，而是训练一种认知姿态——对所有概念化（包括"空"本身）保持非执取的觉知。

**跨领域参照**：中观的反本质主义与 [维特根斯坦的语言批判](../../../philosophy/schools/analytic/wittgenstein.md) 深度呼应，对"自性"的解构可与 [德里达的延异](../../../philosophy/schools/post-structuralism/derrida.md) 比较。详见 [中观](../concepts/cognitive-theory/madhyamaka.md)、[二谛](../concepts/cognitive-theory/two-truths.md)。
