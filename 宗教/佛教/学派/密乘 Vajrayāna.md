# 密乘 Vajrayāna

## 一句话定义

**密乘/金刚乘**是大乘佛教的"快速道"，主张通过**身口意三密相应**（手印、真言、观想）与**本尊瑜伽**，于**即身成佛**，不必经三大阿僧祇劫。主要形态包括唐密（空海）、藏传四派（宁玛、噶举、萨迦、格鲁）。

## 历史脉络

- **6–7 世纪**：密教在印度兴起（《大日经》《金刚顶经》）。
- **8 世纪**：**莲花生大士**入藏传密；**善无畏、金刚智、不空**传唐密（开元三大士）。
- **9 世纪**：**空海**（774–835）入唐求法，回日本创立**真言宗**（东密）。
- **11–14 世纪**：藏传佛教四派形成（宁玛、噶举、萨迦、格鲁）。
- **14 世纪**：**宗喀巴**（1357–1419）改革创立格鲁派，成为达赖、班禅系统基础。

## 核心要义

### 1. 三密相应
- **身密**：手印（mudrā）
- **口密**：真言（mantra）
- **意密**：观想（visualization）
三密与本尊相应，即身成佛。

### 2. 本尊瑜伽（Deity Yoga）
修行者观想自身为本尊，融空性于形象。

### 3. 灌顶 Abhiṣeka
入门需经上师灌顶授权，方可修习密法。

### 4. 四部密续
事部（Kriyā）、行部（Caryā）、瑜伽部（Yoga）、无上瑜伽部（Anuttarayoga）——层次递进。

### 5. 即身成佛 Sokushin-jōbutsu
不需三大阿僧祇劫，即身可成佛（密乘最激进的承诺）。

### 6. 无上瑜伽的"双运"
最高层次包含气脉明点（藏传）或阴阳双修（部分唐密）——极具争议的象征实践。

## 通俗 vs 学术

| 通俗化版本 | 学术原意 |
|---|---|
| "密宗 = 神秘仪式" | 是高度系统化的修行体系 |
| "即身成佛 = 魔法" | 是通过三密相应实现的转化 |
| "双修 = 性" | 是象征性的阴阳调和，非一般理解的性行为 |
| "灌顶 = 祝福" | 是传承授权，非普通加持 |

- **空海**（774–835）：日本真言宗
- **阿底峡**（982–1054）：藏传后弘期

- **萨迦班智达**（1182–1251）：萨迦派
- **宗喀巴**（1357–1419）：格鲁派创立者
- **第十四世达赖喇嘛**（1935–）：当代藏传精神领袖

## 应用场景

- **艺术**：唐卡、曼荼罗、坛城
- **心理学**：荣格原型与本尊形象
- **政治**：藏传与地缘政治

## 常见误读

- ❌ "密宗 = 淫秽" → 双修是象征性修法，非世俗性。
- ❌ "藏传 = 西藏文化" → 藏传是普世宗教，跨民族。
- ❌ "密宗高于显教" → 是不同路径，非高低。
- ❌ "灌顶是迷信" → 是传承系统，类似学术师承。

## Cognitive Architecture

密乘的认知架构以"转化"为方法论，将身口意三业全部纳入修行认知系统：

| 认知模块 | 梵/藏文 | 功能描述 |
|---|---|---|
| 三密相应 | Tri-guhya | 身口意同步瑜伽 |
| 本尊观 | Devatā-yoga | 身份重构 |
| 灌顶 | Abhiṣeka | 认知启动/授权 |
| 大圆满明觉 | Rigpa（藏） | 非二元觉知 |

- **三密（身口意）作为具身认知瑜伽**：身密（手印 mudrā）、口密（真言 mantra）、意密（观想 visualization）三者同步运作，将身体行为、语言系统与心理意象整合为统一的认知操作——这与当代具身认知（embodied cognition）和多感觉整合（multisensory integration）理论高度一致。
- **本尊瑜伽（Deity Yoga）与身份重构**：观想自身为本尊不是幻想，而是系统性的身份重构（identity reconfiguration）——通过解构凡夫自我模型（self-model），以本尊形象建构新的认知-身份框架，最终再解构此框架，证入空性。这是一种"先建构后解构"的两阶段认知转化。
- **灌顶（Abhiṣeka）作为认知启动**：灌顶不只是仪式，而是一种认知启动（cognitive initiation）——通过上师-弟子的直接传承，将特定修法权限与认知框架传递给修行者。在认知科学视角下，这是一种制度化的"认知脚手架"（cognitive scaffolding）搭建。
- **大圆满（Dzogchen）明觉（Rigpa）**：宁玛派最高修法所指向的"明觉"（rigpa），是超越主体-客体二分的非二元觉知（non-dual awareness）。不同于普通意识（rnam-shes / vijñāna）的分别认知，明觉是认知本身的直接自显——无需对象、无需主体的纯粹觉知。

**跨领域参照**：本尊瑜伽的身份重构可与 [荣格的原型激活](../../../心理学/学派/分析心理学/卡尔.md) 比较（主动想象 active imagination 的技术平行），具身认知与 [梅洛-庞蒂的身体现象学](../../../哲学/学派/现象学/梅洛庞蒂.md) 深度呼应。详见 [八识体系](../concepts/cognitive-theory/eight-consciousness.md)、[转识成智](../concepts/cognitive-theory/consciousness-transformation.md)。
