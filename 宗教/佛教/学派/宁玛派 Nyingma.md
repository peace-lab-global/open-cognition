# 宁玛派 Nyingma

## 一句话定义

宁玛派（"旧译派"）是藏传佛教最古老的宗派——以**莲花生大士**（8c）为传承祖师，以**大圆满**（Dzogchen）为最究竟教法，以**九乘判教**统摄一切佛法，保有独特的**伏藏**（terma）传承系统，是藏传四大教派中与"前弘期"直接相连的唯一传承。

## 历史脉络

- **8c**：**莲花生大士**入藏，建立桑耶寺，开创"前弘期"
- **9c**：朗达玛灭佛（842），前弘期终结
- **10–11c**："后弘期"兴起，新译派（噶举/萨迦/格鲁）成立
- **14c**：**隆钦巴**系统整理大圆满，宁玛派教义成形
- **17–18c**：吉美林巴发掘"龙钦心髓"伏藏，大圆满传承更新
- **19c**：蒋扬钦哲旺波发起"利美"（不分派）运动
- **当代**：宁玛派是藏传四大教派之一，全球传播

## 核心要义

### 1. 九乘判教

宁玛派将一切佛法判为九乘：

| 乘 | 名称 | 内容 |
|---|------|------|
| 1 | 声闻乘 | 四谛、阿罗汉果 |
| 2 | 缘觉乘 | 十二因缘、辟支佛果 |
| 3 | 菩萨乘 | 六度、十地、佛果 |
| 4 | 事续（Kriyā） | 外在净行、沐浴 |
| 5 | 行续（Caryā） | 内外兼修 |
| 6 | 瑜伽续（Yoga） | 内在观想 |
| 7 | 玛哈瑜伽 | 生起次第、本尊观 |
| 8 | 阿努瑜伽 | 圆满次第、气脉明点 |
| 9 | **阿底瑜伽（大圆满）** | 本初清净、任运自成 |

> 前 3 = 显教（外三乘）；4–6 = 密教下部（内三乘）；7–9 = 密教上部（密三乘）

### 2. 大圆满（Dzogchen / Atiyoga）

| 概念 | 含义 |
|------|------|
| **本初清净**（ka dag） | 心的空性面——本来无染 |
| **任运自成**（lhun grub） | 心的光明面——本具功德 |
| **彻却**（khregs chod） | 立断——直接安住于觉性（rig pa） |
| **脱嘎**（thod rgal） | 顿超——在觉性中自然显现与超越 |

大圆满与其他密法的根本区别：不依气脉明点的造作修法，而是直接指向心的本性。

### 3. 伏藏传统（Terma）

| 类型 | 说明 |
|------|------|
| **岩伏藏** | 莲花生埋藏于岩石/湖水中的经文，由伏藏师（terton）发掘 |
| **意伏藏** | 伏藏师在禅定中直接从莲花生处获得的教授 |

伏藏传统使宁玛派具有持续的"启示更新"能力——与格鲁派的经院传统形成对比。

### 4. 与其他藏传教派的对比

| 维度 | 宁玛派 | 格鲁派 |
|------|--------|--------|
| 传承 | 前弘期（莲花生） | 后弘期（阿底峡→宗喀巴） |
| 最究竟法 | 大圆满 | 应成中观 |
| 修行风格 | 瑜伽士传统 | 经院+闭关 |
| 组织 | 较松散 | 高度组织化 |
| 僧侣 | 有在家瑜伽士传统 | 严格出家 |

## 通俗 vs 学术

| 通俗化版本 | 学术原意 |
|---|---|
| "宁玛 = 红教" | "红教"是清代汉人对宁玛派的俗称，不够准确 |
| "大圆满 = 什么都不修" | 大圆满需要大量前行修行（五加行/上师瑜伽），"不修"是指最高阶段 |
| "伏藏 = 伪造" | 伏藏是藏传佛教内部的合法传承方式，与"伪造"的判准不同 |

## 常见误读

- ❌ "宁玛派只修大圆满" → 宁玛派有完整的修道次第（九乘），大圆满是最高阶段。
- ❌ "大圆满 = 禅宗" → 虽表面相似（都讲"直指人心"），但大圆满有完整的气脉理论和脱嘎修法，与禅宗路径不同。
