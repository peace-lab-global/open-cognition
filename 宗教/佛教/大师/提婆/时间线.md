# 提婆年表 / Timeline of Āryadeva

## 历史语境 / Historical Context

### 时代背景

- **南印度**: 萨陀伐诃那王朝时期，婆罗门教与佛教并存
- **中观学派**: 龙树创立中观后，需要有力的继承者和辩护者
- **外道论争**: 数论派、胜论派等与佛教的激烈辩论

### 传承背景

- **龙树 Nāgārjuna**: 提婆的老师和中观创始人
- **斯里兰卡**: 提婆可能出身斯里兰卡王室

## 生平年表 / Biographical Timeline

| 年代 (约) | 事件 | 备注 |
|-----------|------|------|
| ~170 CE | 生于南印度或斯里兰卡 | 传说为南印度王子或斯里兰卡王子 |
| ~185 | 放弃王位出家 | 传说以一只眼睛布施 |
| ~190 | 从龙树学习中观哲学 | 成为龙树首席弟子 |
| ~200 | 撰写《四百论》Catuḥśataka | 核心著作 |
| ~210 | 撰写《百论》Śata-śāstra | 逻辑论辩 |
| ~215-230 | 在各地与外道辩论 | 捍卫中观学说 |
| ~230 | 圆寂 | 传说被外道刺杀（一说自然圆寂） |

## 学术争议 / Scholarly Debates

### 身份问题

- "提婆"可能是一个称号（圣天），与多位历史人物有关
- 与"迦那提婆"(Kāṇadeva, 一只眼的提婆)是否为同一人

### 著作问题

- 《四百论》和《百论》是否为同一原著的不同版本

## 历史影响 / Historical Impact

| 时期 | 影响 |
|------|------|
| 3 世纪 | 与龙树并称"龙提"，巩固中观学派 |
| 5-7 世纪 | 月称为《四百论》撰注，确立其经典地位 |
| 5 世纪 | 鸠摩罗什将《百论》传入中国 |
| 8-12 世纪 | 在西藏中观传统中占有核心地位 |
| 当代 | 西方学术界开始研究提婆的独立贡献 |
