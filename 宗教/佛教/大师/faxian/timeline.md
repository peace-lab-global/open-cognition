# 法显年表 / Timeline of Faxian

## 历史语境 / Historical Context

### 时代背景

- **东晋十六国时期** (317-420): 中国南北分裂，北方多胡族政权
- **佛教传入中国**: 佛教经中亚陆路和海路持续传入
- **律藏缺失**: 中国佛教缺乏完整的戒律体系，成为法显西行的直接动因
- **海上丝绸之路**: 从印度经东南亚到中国的海路贸易活跃

### 佛教环境

- **后秦**: 鸠摩罗什在长安主持译经 (401-413)
- **印度笈多王朝**: 佛教在印度持续发展，那烂陀大学尚未建立
- **斯里兰卡**: 上座部佛教在斯里兰卡稳固发展

## 旅行路线 / Travel Route

**陆路西行**: 长安 → 河西走廊 → 敦煌 → 鄯善 → 于阗 → 疏勒 → 葱岭 → 犍陀罗 → 中印度

**海路东归**: 巴连弗邑 → 耽摩利底 → 斯里兰卡 → 耶婆提 (爪哇) → 青州 (山东)

## 历史影响 / Historical Impact

| 时期 | 影响 |
|------|------|
| 5 世纪 | 律藏补阙，完善中国佛教戒律体系 |
| 5-7 世纪 | 《佛国记》成为了解 5 世纪印度佛教的核心文献 |
| 7 世纪 | 为玄奘、义净西行提供重要参考 |
| 19 世纪 | James Legge 英译，成为西方印度学研究基础 |
| 当代 | 中亚和南亚考古学、历史学的重要参考 |
