# 善导年表 / Timeline of Shàndǎo

## 历史语境 / Historical Context

### 时代背景

- **唐代初期** (613-681): 中国佛教的黄金时代
- **净土信仰**: 弥陀净土信仰在民间广泛流传
- **道绰** (562-645): 善导的老师，净土宗先驱

## 生平年表 / Biographical Timeline

| 年代 | 事件 | 备注 |
|------|------|------|
| 613 | 生于临淄（今山东淄博） | 或说泗州人 |
| ~630 | 少年出家 | 读《观无量寿佛经》发愿 |
| ~640 | 师从道绰 | 在并州玄中寺 |
| ~645 | 道绰圆寂后独立弘法 | 移居长安 |
| ~650 | 在长安光明寺弘法 | 倡导称名念佛 |
| ~655 | 教化长安城 | 据说"满城念佛" |
| ~670 | 完成《观经四帖疏》 | 核心著作 |
| ~675 | 参与洛阳龙门奉先寺大佛建造 | 监造卢舍那大佛 |
| 681 | 圆寂于长安 | 享年 69 岁 |

## 历史影响 / Historical Impact

| 时期 | 影响 |
|------|------|
| 7 世纪 | 确立净土宗的教理体系和修行方法 |
| 8 世纪 | 净土宗成为中国最大宗派之一 |
| 9 世纪 | 善导著作传入日本 |
| 12 世纪 | 法然以善导为宗祖创立日本净土宗 |
| 13 世纪 | 亲鸾创立净土真宗，以善导为精神祖师 |
| 当代 | 善导被视为中国净土宗实际创立者 |
