# 善导著作目录 / Works of Shàndǎo

## 核心著作 / Core Works

### 《观无量寿佛经疏》(观经四帖疏)

- **地位**: 善导最重要的著作，净土宗立宗之本
- **结构**: 4 卷 (玄义分、序分义、定善义、散善义)
- **内容**: 系统诠释《观无量寿佛经》，建立净土宗的教理基础
- **核心**: "凡夫入报"——凡夫凭信心即可往生阿弥陀佛报土
- **现存**: 大正藏 T1753

### 《往生礼赞》

- **地位**: 净土礼仪文献
- **内容**: 往生净土的礼赞仪轨
- **特点**: 实用性修行指导

### 《观念法门》

- **地位**: 念佛方法的系统阐述
- **内容**: 称名念佛、观想念佛等方法论

### 《法事赞》

- **地位**: 净土法事仪轨
- **内容**: 净土法会的仪轨和赞文

## 其他作品

- 《般舟赞》— 般舟三昧的赞文
- 《转经行道愿往生净土法事赞》— 净土法事
