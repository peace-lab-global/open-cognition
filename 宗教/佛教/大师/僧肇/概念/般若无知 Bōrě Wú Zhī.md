# 般若无知 Prajñā as Non-Knowing

> 关联思想家：[僧肇 Sēngzhào](../README.md)

## 概念概述 Concept Overview

"般若无知"是僧肇对般若智慧（prajñā）的独特阐释。般若之知不是通常的"知"——它没有认识对象（无所知），但却能通达一切（无不知）。"无知"不是缺乏知识，而是超越主客对立的认知模式——没有"能知"（主体）与"所知"（客体）的二元分裂。

"Prajñā as non-knowing" is Seng-chao's unique interpretation of wisdom: prajñā-knowing has no object (knows nothing) yet comprehends everything (knows all). "Non-knowing" is not ignorance but a mode of cognition transcending subject-object duality.

## 哲学分析 Philosophical Analysis

僧肇的论证：
1. 通常的"知"是主体对客体的认识——有"能知"必有"所知"
2. 般若之知以空性为"对象"——但空性不是客体，不可被"认识"
3. 因此般若之知是"无知"——不以概念方式把握对象
4. 但正因"无知"，般若能通达一切——"无不知"

这与海德格尔的"泰然任之"（Gelassenheit）——一种非概念性的、开放的"思"——形成有趣的平行。

## 历史源流 Historical Development

"般若无知"是僧肇对鸠摩罗什所传般若思想的中国化表达。它回应了当时中国学者对般若经中"无所得"概念的困惑——如果般若无知，如何修行？僧肇的答案是：正因为般若不执着于任何知见，它才是最究竟的智慧。

## 修行应用 Practice Applications

在禅修中，"般若无知"意味着不追求特定的禅修体验或认知状态——不执着于"空"的体验，不执着于"光明"的境界，让认知自然流动而不固着。
