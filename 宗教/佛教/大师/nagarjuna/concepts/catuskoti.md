# 四句破 Catuskoti / Tetralemma

## 定义 / Definition

**四句破 (catuskoti)** 是龙树的核心逻辑方法：对任何形而上学命题，穷尽其四种逻辑可能性——有、无、亦有亦无、非有非无——然后全部否定。这不是虚无主义或反逻辑，而是以严格逻辑展示概念固化的内在矛盾。

The **tetralemma** is a four-cornered negation method: for any metaphysical proposition, all four logical possibilities are systematically examined and negated, driving cognition beyond conceptual fixation.

## 四句结构 / Four-Cornered Structure

对于命题 P：

| 句 Koti | 逻辑形式 | 含义 |
|---------|---------|------|
| 第一句 | P (有) | 肯定命题 |
| 第二句 | ~P (无) | 否定命题 |
| 第三句 | P & ~P (亦有亦无) | 同时肯定与否定 |
| 第四句 | ~(P & ~P) (非有非无) | 既非肯定亦非否定 |

龙树的论证：所有四句都不成立。这穷尽了经典逻辑的所有可能。

## 逻辑地位 / Logical Status

### 不是反逻辑

四句破运用的是严格的形式逻辑——否定是基于对每个选项内在矛盾的精确分析。这与直觉主义或神秘主义不同，龙树不诉诸"超越逻辑"的直觉。

### 是逻辑的自我消解

当逻辑穷尽所有可能后全部否定，这不是逻辑的失败，而是逻辑展示了自身的极限。类似：
- **哥德尔不完备定理**: 形式系统的内在限制
- **罗素悖论**: 集合论的自我指涉问题
- **维特根斯坦的梯子**: 用完即弃的逻辑工具

### 与希腊怀疑论的区别

皮浪式怀疑论 (Pyrrhonism) 悬搁判断 (epoche)。龙树不是悬搁判断，而是论证所有判断都不成立——这是更激进的立场。但目的不同：龙树的目标是解脱，不是纯粹的理论。

## 认知意义 / Cognitive Significance

四句破的认知功能是**认知穷尽后的超越**：

1. **概念穷尽**: 系统穷尽所有概念框架
2. **框架瓦解**: 展示每个框架的内在矛盾
3. **认知开放**: 当所有框架都被否定，认知被迫进入非概念化状态
4. **自由涌现**: 从概念固化中解脱的认知自由

这是一种**以逻辑为工具的认知解构技术**——类似于认知疗法中的认知重构，但更为彻底。

## 后世发展 / Later Developments

- **月称**: 将四句破发展为应成法 (prasanga)——不立自宗，只破他宗
- **清辨**: 主张中观也需要建立自己的逻辑论证 (自续)
- **宗喀巴**: 在应成立场上发展出更精细的四句破分析
- **禅宗**: 以非逻辑的方式继承了四句破的精神——公案
