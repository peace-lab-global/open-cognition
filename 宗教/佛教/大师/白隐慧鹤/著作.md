# 白隐慧鹤著作目录 / Works of Hakuin Ekaku

## 核心著作 / Core Works

### 《槐安国语》

- **地位**: 白隐最重要的日文著作
- **内容**: 以通俗日文阐述禅的修行和见地
- **特点**: 语言生动，面向大众

### 《远罗天釜》(Orategama)

- **地位**: 白隐禅修指导的核心文献
- **内容**: 书信形式，阐述禅修方法和见地
- **特点**: 针对在家修行者的实用指导

### 《夜船闲话》

- **地位**: 白隐关于"内观"修行的重要文献
- **内容**: "软酥法"等身体修行法门的描述
- **特点**: 禅修与身心健康的关系

### 《毒语注》

- **地位**: 对《碧岩录》等公案的注释
- **内容**: 公案禅的系统注释

### 《坐禅和赞》

- **地位**: 广为流传的坐禅赞歌
- **内容**: 以诗歌形式阐述坐禅的意义和方法

## 公案贡献 / Koan Contributions

- **"只手音声"公案** (隻手の音声) — 白隐创作的著名公案
  - "两手合拍有声，只手之音如何？"
  - 成为全球最广为人知的禅修入门公案

## 书画作品

- 白隐留下了大量禅画和书法作品，以粗犷有力的笔法著称
- 禅画主题包括达摩、布袋和尚等
