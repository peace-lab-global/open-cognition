# 即心是佛 Jixin Shifo / Mind Is Buddha

## 定义 / Definition

**即心是佛**是达摩禅的根本见地：众生本具佛性，不假外求。此心即是佛心，只因妄想执着而不能证得。达摩的革命性在于将此如来藏思想直接落实为禅修实践——不通过渐次修学，直接指向心性。

**Mind is Buddha** is the fundamental view of Bodhidharma's Chan: all sentient beings inherently possess Buddha-nature; Buddhahood is not something to be acquired but something to be recognized.

## 如来藏思想基础 / Tathagatagarbha Foundation

达摩的即心是佛建立在如来藏思想之上：
- **《楞伽经》**: "如来藏自性清净……为客尘所染"
- **如来藏 = 佛性**: 一切众生心中本具成佛的可能性
- **客尘**: 妄想执着是"客"（暂时的），不是"主"（本质的）

## 与印度佛性论的区别 / Difference from Indian Buddha-Nature Theory

印度如来藏思想偏理论探讨；达摩将其转化为**直接的禅修指示**——"直指人心"。这标志着佛性论从学术讨论到修行实践的关键转变。

## 在禅宗传承中的发展 / Development in Chan Lineage

- **慧可**: 继承即心是佛，强调"觅心了不可得"
- **慧能**: "菩提本无树"——将即心是佛推向极致
- **马祖道一**: "即心即佛"→"非心非佛"→"平常心是道"

## 认知意义 / Cognitive Significance

即心是佛的认知含义：修行不是"获取"新知识，而是"认出"本有的认知能力——从"获取模式" (acquisition mode) 转为"发现模式" (recognition mode)。这与当代认知科学中"内隐知识" (tacit knowledge) 的概念形成对话。
