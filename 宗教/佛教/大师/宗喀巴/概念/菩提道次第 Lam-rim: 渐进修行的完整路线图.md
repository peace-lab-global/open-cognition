# 菩提道次第 Lam-rim / The Graduated Path

## 定义 / Definition

**菩提道次第**是宗喀巴在《菩提道次第广论》中系统阐述的修行路线图，将所有佛法组织为从初学到成佛的渐进阶梯。

**Lam-rim** is Tsongkhapa's systematic organization of all Buddhist teachings into a graduated path from beginner to full enlightenment.

## 三士道 / Three Scopes

### 下士道 (Hinayana scope)
- 念死无常
- 恶趣苦
- 皈依三宝
- 深信因果

### 中士道 (Individual liberation)
- 苦谛 (轮回之苦)
- 集谛 (烦恼与业)
- 灭谛 (解脱的可能性)
- 道谛 (戒定慧三学)

### 上士道 (Bodhisattva path)
- 发菩提心
- 六度 (布施/持戒/忍辱/精进/禅定/智慧)
- 止观双运
- 空性见

## 认知意义 / Cognitive Significance

三士道不是三种不同的人，而是同一认知主体的三个发展阶段——从动机管理 (下士) 到认知出离 (中士) 到认知圆满 (上士)，构成渐进式认知发展的完整课程。
