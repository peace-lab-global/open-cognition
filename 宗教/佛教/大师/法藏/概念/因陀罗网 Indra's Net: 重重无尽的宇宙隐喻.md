# 因陀罗网 / Indra's Net

帝释天宝网中每颗珠子映照一切珠子，重重无尽——认知不是孤立节点的操作，而是网络中每个位置对所有其他位置的响应。

**Indra's Net**: An infinite web of jewels, each reflecting all others — a metaphor for the holographic interpenetration of all phenomena.

## 当代应用
互联网拓扑、万维网、生态系统网络
