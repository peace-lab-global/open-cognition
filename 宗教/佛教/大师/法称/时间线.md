# 法称年表 / Timeline of Dharmakīrti

## 历史语境 / Historical Context

### 时代背景

- **戒日王 Harṣavardhana** (约 606-647 CE): 印度北部最后的佛教护法大帝，统一北印度
- **那烂陀寺 Nālandā**: 佛教最高学府，法称可能在此学习和教学
- **超戒寺 Vikramaśīla**: 尚未建立（8 世纪才建立），但后来的量论传统以此为中心
- **南印度**: 法称的出生地，婆罗门学术传统活跃

### 学术环境

- **佛教量论的兴起**: 陈那 (约 480-540) 创立量论传统，自在军继承
- **印度教复兴**: 弥曼差派 (Kumārila Bhaṭṭa)、正理派的学术活跃
- **瑜伽行派的成熟**: 世亲和无著的本体论框架已经被广泛接受
- **因明 (hetu-vidyā) 传统**: 佛教与婆罗门学派之间的逻辑辩论传统日益精密

## 生平年表 / Biographical Timeline

| 年代 (约) | 事件 | 备注 |
|-----------|------|------|
| ~600 CE | 生于南印度（或苏门答腊）| 婆罗门种姓，自幼学习吠陀和婆罗门哲学 |
| ~615 | 少年时期精通吠陀、正理派逻辑 | 传说在与佛教学者辩论后皈依佛教 |
| ~620 | 皈依佛教，深入学习阿毗达磨与瑜伽行派 | 可能前往中印度或那烂陀 |
| ~625 | 从自在军 Īśvarasena 学习量论 | 陈那的再传弟子；传说对自在军的解释不满 |
| ~630 | 开始撰写《量抉择论》Pramāṇavārttika | 核心著作，历时多年完成 |
| ~635 | 完成《量抉择论》七品 | 传说在辩论中多次验证其论证 |
| ~635-640 | 撰写《正理一滴》Nyāyabindu | 量论入门纲要 |
| ~640 | 撰写《量决定论》Pramāṇaviniścaya | 对《量抉择论》的深化补充 |
| ~640-645 | 撰写《因一滴》《成他相续论》等支分论 | 逻辑学与心灵哲学专题 |
| ~645 | 与弥曼差派学者（可能包括 Kumārila）进行辩论 | 传说法称在辩论中多次获胜 |
| ~650 | 继续教学与修订，培养弟子 | 主要弟子可能包括 Devendrabuddhi |
| ~655 | 晚年可能在南印度或中印度教学 | 影响扩展至印度各地 |
| ~660 | 圆寂 | 具体地点不详 |

> **注**: 法称的精确生卒年和著作年代在学界仍有争议。上述年表是基于 Erich Frauwallner 和 Ernst Steinkellner 等学者的研究成果构建的大致框架。

## 学术争议 / Scholarly Debates

### 年代问题

- **传统观点**: 藏传传统将法称定位于 7 世纪
- **Frauwallner 说**: 约 600-660 CE（目前学界主流）
- **少数观点**: 有学者主张更早 (~550-620) 或与 Kumārila 完全同时
- **与玄奘的关系**: 玄奘 (602-664) 在印度时 (约 630-645) 未提及法称，可能因为法称当时尚未成名或不在玄奘的主要活动区域

### 出生地问题

- **南印度说**: 多罗那他 Tāranātha 等藏传史家主张
- **苏门答腊说**: 部分学者基于东南亚佛教传播路线提出
- **学界主流**: 倾向于南印度本土

### 与陈那的关系

- **再传弟子说**: 传统观点认为法称师从自在军，自在军师从陈那
- **直接弟子说**: 部分学者质疑中间是否存在自在军这一环节
- **"批判性继承"说**: 法称对陈那体系多有批评，可能表明他并非陈那学派的内部成员，而是外部的批判性继承者

### 学派归属问题

- **唯识派说**: 传统观点认为法称在本体论上是唯识论者
- **经量部说**: Dunne 等学者认为法称在方法论上更接近经量部
- **不可确定说**: 部分学者认为法称有意模糊学派界限，其思想超越了简单的学派标签

## 思想传承 / Intellectual Lineage

### 前驱传承

```
世亲 Vasubandhu (~320-400)
  ↓
陈那 Dignāga (~480-540) — 量论创始人，《集量论》
  ↓
自在军 Īśvarasena — 陈那弟子
  ↓
法称 Dharmakīrti (~600-660) — 量论集大成者
```

### 后继传承

```
法称 Dharmakīrti (~600-660)
  ↓
天慧 Devendrabuddhi (~630-690) — 早期注释者
  ↓
释迦慧 Śākyabuddhi (~700) — 翻译与注释
  ↓
法上 Dharmottara (~9 世纪) — 最重要的注释者
  ↓
智藏智 Jñānaśrīmitra (~975-1025) — 晚期量论发展
  ↓
宝称 Ratnakīrti (~11 世纪) — 最后的印度量论大师
  ↓
──────────── 传入西藏 ────────────
  ↓
寂护 Śāntarakṣita (~725-788) — 将量论引入西藏
  ↓
莲华戒 Kamalaśīla (~740-795) — 桑耶寺辩论
  ↓
萨迦班智达 Sa-skya Paṇḍita (1182-1251) — 《量理宝藏论》
  ↓
宗喀巴 Tsongkhapa (1357-1419) — 格鲁派量论教育
```

## 历史影响 / Historical Impact

| 时期 | 影响 |
|------|------|
| 7-8 世纪 | 法称弟子和早期注释者开始传播量论传统 |
| 8 世纪 | 寂护和莲华戒将量论带入西藏，桑耶寺辩论 (792-794) 运用量论 |
| 9-11 世纪 | 法上、智藏智等在印度继续发展量论；正理派和弥曼差派对法称做出系统回应 |
| 12-13 世纪 | 萨迦班智达在西藏建立系统的量论教育；伊斯兰入侵导致印度本土量论传统中断 |
| 14-17 世纪 | 宗喀巴和格鲁派将量论纳入寺院教育核心；量论成为藏传佛教学僧必修课 |
| 19 世纪 | 西方学者开始关注佛教逻辑 |
| 20 世纪初 | Stcherbatsky 的 *Buddhist Logic* (1930-32) 开启现代学术研究 |
| 20 世纪中叶 | 梵文写本的发现推动文本学研究 |
| 20 世纪末至今 | Dreyfus、Dunne、Kellner 等学者推动法称哲学与当代分析哲学的对话 |

## 重要历史事件 / Key Historical Events

- **桑耶寺辩论 (792-794)**: 莲华戒运用法称的量论方法在辩论中胜出，确立了印度佛教传统（而非中国禅宗）在西藏的主导地位
- **量论七部的确立**: 藏传传统将法称的七部著作确立为量论教育的根本文本
- **梵文写本的再发现 (1930s)**: Sankrtyayana 在西藏寺院发现梵文写本，改写了法称研究的文本基础
- **当代哲学对话 (2000s-)**: 法称的认识论和语义学开始被纳入当代分析哲学和认知科学的讨论框架
