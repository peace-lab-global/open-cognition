# 荣西年表 / Timeline of Yōsai

## 历史语境 / Historical Context

### 时代背景

- **镰仓时代初期** (1185-1333): 日本政治权力从京都贵族（公家）转移到武士阶层（武家），源赖朝建立镰仓幕府
- **末法思想盛行**: 佛教末世论广泛流行，净土宗、日莲宗等"新佛教"运动兴起
- **宋日贸易活跃**: 商船往来于明州（宁波）与博多之间，文化交流频繁
- **旧佛教危机**: 比叡山天台宗和南都佛教面临贵族化、形式化、军事化等问题

### 学术环境

- **中国禅宗黄金时代**: 临济宗大慧宗杲的看话禅和曹洞宗宏智正觉的默照禅并盛
- **天台教学**: 宋代天台宗正经历"山家山外"之争
- **茶文化**: 宋代"点茶法"（抹茶）高度发展，禅林饮茶传统成熟

## 学术争议 / Scholarly Debates

### 荣西的历史定位

- **传统观点**: 荣西是"日本禅宗之祖"，将纯粹的临济禅传入日本
- **修正主义观点**: 荣西实际上是"兼修禅"的代表，他传入的并非"纯禅"，而是融合了天台和密教的综合体
- **当前共识**: 荣西的历史角色是"传播者"和"综合者"——他的兼修立场是为禅宗在日本立足所采取的必要策略

### 《吃茶养生记》的成书年代

- **传统说法**: 1214年（荣西晚年）
- **部分学者**: 认为上卷和下卷可能成书于不同时期
- **下卷（论桑）**: 有学者认为下卷可能参考了中国本草学文献，而非完全出自荣西之手

### 荣西与道元的关系

- 道元年轻时曾在建仁寺修行，可能受到荣西思想的影响
- 但道元后来强烈批判"兼修禅"，转向"纯粹禅"
- 学界对道元是否直接受教于荣西（或其弟子）仍有争议

## 历史影响 / Historical Impact

| 领域 | 影响 |
|------|------|
| **日本禅宗** | 创立临济宗，奠定日本禅宗法脉基础 |
| **日本茶文化** | 引入茶种与饮茶习俗，被誉为"日本茶祖" |
| **日本佛教制度** | 建仁寺"三宗兼学"成为禅寺制度的先驱 |
| **中日文化交流** | 两度入宋，是宋日文化交流的重要桥梁 |
| **日本美学** | 茶禅思想为后来的茶道、枯山水等美学传统奠定精神基础 |
