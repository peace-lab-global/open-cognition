# 月称年表 / Timeline of Candrakirti

## 编年

| 年代 | 事件 |
|------|------|
| 约 600 CE | 生于南印度萨曼塔 (Samanta)，今奥里萨邦 (Odisha) 一带，婆罗门种姓 |
| 约 610s | 在那烂陀寺 (Nalanda) 出家为僧，学习中观与唯识 |
| 约 620s | 精通三藏，开始著述；任那烂陀寺长老 (sthavira) |
| 约 625-630 | 著《明句论》(Prasannapada)，注释龙树《中论》 |
| 约 630-640 | 著《入中论》(Madhyamakavatara) 及其他注释著作 |
| 约 630s-640s | 与瑜伽行中观派学者月官 (Candragomin) 在那烂陀寺展开辩论（传说持续七年） |
| 约 650 CE | 圆寂 |

## 思想发展分期

| 时期 | 核心工作 | 代表成果 |
|------|----------|----------|
| 早期 (610s-620s) | 在那烂陀寺学习与教学 | 精通中观与唯识两大学派 |
| 中期 (620s-640s) | 著述与辩论 | 《明句论》《入中论》及其他注释 |
| 晚期 (640s-650) | 教学与传承 | 建立应成中观的学派身份 |

## 历史语境 / Historical Context

### 那烂陀寺的学术环境

月称活跃时期的那烂陀寺是印度最大的佛教学术中心，容纳了来自各地的数千僧侣学者。寺中同时存在中观与唯识两大学派的教学传统，这为月称与月官的辩论提供了制度背景。

### 戒日王朝的佛教支持

戒日王 (Harsa, 606-647) 统治下的北印度是佛教繁荣时期。那烂陀寺获得了大量王室赞助，成为国际性的学术中心。中国僧人玄奘于 630s-640s 在此学习。

### 与唯识学派的交锋

月称所处的时代是佛教内部中观-唯识争论的高峰期。月称不仅批判自续派的方法论，也对唯识学派的核心主张（依他起有、阿赖耶识）进行了系统批判——《入中论》第六品中有大量对唯识的反驳。

## 历史影响

| 时代 | 发展 |
|------|------|
| 7 世纪 | 月称确立应成中观的独立身份 |
| 8 世纪 | 寂护将中观传入西藏，但以瑜伽行中观为主 |
| 11 世纪 | 阿底峡入藏，带来月称的中观传统 |
| 12 世纪 | 印度佛教因伊斯兰入侵而衰亡；中观传统在西藏延续 |
| 14-15 世纪 | 宗喀巴复兴并深化月称的应成中观 |
| 15-20 世纪 | 格鲁派以月称-宗喀巴的中观为正统哲学 |
| 20 世纪 | 月称著作开始被翻译为西方语言 |
| 当代 | 月称哲学在西方学术界获得越来越多的关注 |
