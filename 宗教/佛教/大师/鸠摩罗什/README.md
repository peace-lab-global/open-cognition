# 鸠摩罗什 Kumārajīva

> "翻译佛经的人，就像嚼饭喂人——虽失原味，却是众生得度的因缘。"
> — 鸠摩罗什 (据《高僧传》)

> "The translator of sutras is like one who chews rice to feed others — though the original flavor is lost, it becomes the cause of beings' liberation."
> — Kumārajīva (from *Gaoseng Zhuan*)

## 基本信息 / Basic Information

**鸠摩罗什 Kumārajīva**（344–413），龟兹国 (Kucha，今新疆库车) 人，被公认为**汉传佛教最伟大的翻译家**，与玄奘 (Xuanzang)、真谛 (Paramārtha)、不空 (Amoghavajra) 并称"四大译经家"。其译经以**文学性**和**流畅度**著称，所译经典——《法华经》《维摩诘经》《金刚经》《阿弥陀经》——成为中国佛教各宗派的根本经典，深刻塑造了汉传佛教的语言、思想与文化面貌。

- **出生地 / Birthplace**: 龟兹 Kucha（今新疆库车 Kuqa），丝绸之路北道重镇
- **年代 / Era**: 344–413 CE，东晋十六国时期
- **身份 / Role**: 三藏法师 (Tripitaka Master)、翻译家、中观学派传播者
- **传承 / Lineage**: 初学小乘，后从须利耶苏摩 (Sūryasoma) 学大乘中观；入华后成为汉传中观学派奠基人
- **核心贡献 / Core Contribution**: 12 年间译出经论 35 部 294 卷（一说 74 部 384 卷），系统引入大乘般若—中观思想
- **门下 / Disciples**: "什门四哲"——僧肇 (Sengzhao)、僧叡 (Sengrui)、道生 (Daosheng)、道融 (Daorong)

## 核心命题 / Core Propositions

### 1. 意译优于直译 / Free Translation over Literal Translation

罗什的翻译理念是"达意"而非"逐字"。他批评前人译本"虽得大意，殊隔文体"，主张以优美流畅的汉文传达佛法精髓，而非拘泥于梵文的语法结构。其译文"如行云流水"，成为中国文学的典范。

**Kumārajīva championed sense-for-sense translation over word-for-word rendering. His goal was to convey the essence of Buddhist teachings in elegant, natural Chinese rather than preserve the grammatical structure of Sanskrit.**

详见 [翻译方法](concepts/translation-method.md)。

### 2. 中观思想的汉传化 / Sinification of Madhyamaka

首次系统翻译龙树 (Nāgārjuna) 中观三论——《中论》《百论》《十二门论》——为三论宗 (Sanlun) 奠基。罗什不仅翻译文本，更通过口授讲解将中观思想的核心——缘起性空、八不中道、二谛——传授给中国弟子。

**He was the first to systematically translate the core Madhyamaka treatises into Chinese, laying the foundation for the Sanlun (Three Treatises) school and introducing Nāgārjuna's 哲学 to East Asia.**

详见 [汉传中观](concepts/madhyamaka-chinese.md)。

### 3. 般若与方便的统一 / Unity of Prajñā and Upāya

通过翻译《维摩诘经》，罗什展示了大乘菩萨道"烦恼即菩提"的核心精神——智慧 (prajñā) 与方便 (upāya) 不可分割，出世与入世不二。这一思想深刻影响了禅宗和中国文人佛教。

**Through his translation of the Vimalakīrti Sūtra, Kumārajīva demonstrated the Mahāyāna ideal that wisdom and skillful means are inseparable — samsara and nirvana are not two.**

详见 [空义传授](concepts/emptiness-teaching.md)。

### 4. 经典翻译的文化奠基 / Cultural Foundation through Sutra Translation

罗什的翻译不仅是宗教活动，更是文明级别的文化工程。他创造的佛教汉语——一套融合梵文概念与中国哲学语汇的新语言——成为东亚佛教文明的底层代码。

**His translations constituted a civilization-level cultural project, creating a Buddhist Chinese lexicon that became the foundational code of East Asian Buddhist civilization.**

详见 [经典翻译](concepts/sutra-translation.md) 和 [大乘弘传](concepts/mahayana-propagation.md)。

## 思想脉络 / Intellectual Context

### 前驱 / Predecessors

- **支谦 Zhi Qian** (3c): 早期译经者，译文"尚文"但有时偏离原义
- **竺法护 Dharmarakṣa** (230–316): 翻译大量般若经典，但"虽得大意，殊隔文体"
- **道安 Dao'an** (312–385): 提出"五失本三不易"翻译理论，为罗什奠基
- **说一切有部 Sarvāstivāda**: 龟兹地区的主要佛教学派，罗什最初学习的传统

### 同时代 / Contemporaries

- **慧远 Huiyuan** (334–416): 庐山白莲社创始人，与罗什通信讨论法义（《大乘大义章》）
- **僧肇 Sengzhao** (374–414): 嫡传弟子，"解空第一"，著《肇论》
- **僧叡 Sengrui** (352–436): 罗什译场核心参与者，为多部译经作序
- **姚兴 Yao Xing** (366–416): 后秦皇帝，罗什的护法国王

### 后继 / Successors

- **道生 Daosheng** (355–434): 提出"阐提成佛"说，被视为涅槃学派先驱
- **昙影 Tanying**: 著《中论疏》，早期汉传中观注释
- **僧导 Sengdao**: 传承《成实论》学
- **吉藏 Jizang** (549–623): 三论宗集大成者，远承罗什中观传统
- **智顗 Zhiyi** (538–597): 天台宗创始人，以罗什译《法华经》为根本经典
- **玄奘 Xuanzang** (602–664): 第二位翻译巨匠，开创"新译"传统，与罗什"旧译"形成对照

## 翻译理念详析 / Translation Philosophy in Detail

罗什的翻译方法论包含以下核心要素：

### 意译原则 / Sense-for-Sense Principle

罗什明确反对"胡言不尽""拘泥原典"的翻译方式。据《高僧传》记载，罗什曾感叹："天竺国俗，甚重文制。其宫商体韵，以入弦为善。凡觐国王，必有赞德。见佛之仪，以歌叹为贵。经中偈颂，皆其式也。但改梵为秦，失其藻蔚。虽得大意，殊隔文体。有似嚼饭与人，非徒失味，乃令呕哕也。"

### 译场制度 / Translation Assembly System

罗什在长安的翻译活动是中国历史上首次大规模、有组织的译经工程。其译场通常有数百人参与：
- **主译 / Chief Translator**: 罗什本人手持梵本，口宣汉语
- **笔受 / Scribes**: 记录罗什口译的汉语
- **证义 / Verifiers**: 核对义理准确性
- **润色 / Editors**: 文学润色
- **校勘 / Proofreaders**: 最终校对

### 创造佛教汉语 / Creating Buddhist Chinese

罗什在翻译过程中大量创造或赋予新义的汉语词汇，如"世界""因果""境界""实际""平等""相对""绝对"等，这些词汇后来成为中国日常用语的一部分，深刻影响了汉语的发展。

## 思想坐标 / Intellectual Positioning

### 哲学定位

- **翻译哲学 Philosophy of Translation**: 罗什的"意译"理念与奈达 (Eugene Nida) 的"动态对等"理论、本雅明 (Walter Benjamin) 的"纯语言"思想形成跨时代对话
- **文化传播学 Cultural Transmission**: 罗什是文明间思想传播的典范案例——如何将一个完整的哲学体系从一种语言/文化移植到另一种
- **语言哲学 Philosophy of Language**: 罗什的翻译实践本身就是对"意义是否可以跨语言传达"这一哲学问题的回答
- **比较哲学 Comparative Philosophy**: 罗什将印度中观哲学与中国老庄传统对接，开创跨文化哲学对话

### 在佛教史上的位置

- **旧译 vs 新译**: 罗什代表"旧译"传统的最高成就；玄奘代表"新译"。旧译重文学性与可读性，新译重精确性与系统性
- **义学 vs 行门**: 罗什偏重义理翻译（"义学"），但所译《坐禅三昧经》等也涉及修行实践
- **北方 vs 南方**: 罗什在长安（北方后秦），同时代慧远在庐山（南方东晋）；两大佛教中心各有侧重

## 当代应用 / Contemporary Applications

### 翻译学 / Translation Studies

- **动态对等 Dynamic Equivalence**: 罗什的"意译"与奈达的翻译理论形成跨时代对话
- **翻译伦理 Translation Ethics**: 罗什的"嚼饭喂人"之喻涉及翻译者的责任与限度
- **不可译性 Untranslatability**: 罗什对"失其藻蔚"的感叹预示了现代翻译学的核心困境

### 比较文学 / Comparative Literature

- **叙事开场**: "如是我闻" (evaṃ mayā śrutam) 的叙事框架成为中国文学的叙事原型
- **偈颂韵文美**: 罗什翻译的偈颂兼具文学性与哲学深度，影响中国诗歌
- **维摩诘居士**: 成为中国文人"在世出世"理想的原型

### 文化史 / Cultural History

- **语言底层**: 罗什创造的佛教词汇渗透入汉语日常用语
- **东亚文明**: 罗什译经传至朝鲜、日本、越南，成为东亚佛教文明的共同基础
- **丝绸之路**: 罗什的生平是丝路文化交流的缩影

### 认知科学 / Cognitive Science

- **概念迁移 Concept Transfer**: 如何将一套哲学概念从一种语言完整移植到另一种
- **认知框架重组 Cognitive Reframing**: 罗什的翻译如何重组了中国人的认知框架
- **双语认知 Bilingual Cognition**: 罗什精通梵、汉、龟兹语，其翻译涉及深层双语认知

## 常见误读 / Common Misreadings

### 1. "罗什只是翻译匠，没有自己的思想"

**误读**: 罗什仅是忠实的文本翻译者，没有原创性哲学贡献。
**正解**: 罗什选择翻译哪些经典本身就是深刻的思想判断——他以中观三论为核心、以般若经为系统、以法华经为归趣，构建了一套完整的大乘佛学体系。他对空义的理解通过译文的遣词造句传达。僧肇的"解空"直接源自罗什的口授。

### 2. "罗什的译本最准确"

**误读**: 罗什的译本在梵汉对照中最为精确。
**正解**: 罗什是"最优美"和"最有影响力"的，但现代梵文对照研究显示某些段落与原文有显著差异。罗什会为了汉文流畅度或义理清晰而调整原文结构、省略重复段落、甚至添加解释性文字。玄奘的"新译"在精确度上超越了罗什。

### 3. "罗什的翻译是个人行为"

**误读**: 译经是罗什一人的成就。
**正解**: 罗什的译经是集体工作——译场中有数百人参与，包括一流的汉地学者如僧叡、僧肇。许多序文和注释显示，汉地弟子对译文有重要的修改和补充。

### 4. "罗什只翻译不创作"

**误读**: 罗什没有自己的著作。
**正解**: 罗什虽以翻译为主，但《大乘大义章》（与慧远通信）包含其本人的思想表述。此外，他在口授中对经论的注释被弟子记录流传。
