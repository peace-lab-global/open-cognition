# 鸠摩罗什年表 / Timeline of Kumārajīva

## 历史语境 / Historical Context

### 时代背景

- **4 世纪中亚**: 龟兹 (Kucha) 为丝绸之路重镇，佛教繁荣
- **五胡十六国**: 北方多政权并立，各政权支持佛教
- **后秦**: 姚兴统治时期 (394-416)，崇佛重法

### 传承背景

- **犍陀罗佛教**: 中亚佛教的学术中心
- **说一切有部**: 罗什早年的学术基础
- **大乘中观**: 须利耶苏摩传授的中观思想

## 历史影响 / Historical Impact

| 时期 | 影响 |
|------|------|
| 5 世纪 | 开创中国佛教翻译新纪元 |
| 5-6 世纪 | 三论宗以其翻译为基础建立 |
| 6 世纪 | 天台宗以罗什译《法华经》为根本 |
| 7 世纪 | 禅宗以罗什译《金刚经》为入门 |
| 至今 | 罗什译本仍是中国佛教最广泛使用的版本 |
