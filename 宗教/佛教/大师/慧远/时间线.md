# 慧远年表 / Timeline of Huiyuan

## 历史语境 / Historical Context

### 时代背景

- **东晋** (317-420): 西晋灭亡后，晋室南渡建康（今南京），偏安江南。北方为十六国纷争
- **门阀政治**: 东晋朝廷受士族门阀操控，皇权式微
- **玄学清谈**: 士大夫阶层盛行玄学清谈，佛学与玄学深度融合
- **佛教传播**: 佛教在中国进入快速发展期，般若学与玄学合流

### 佛教环境

- **般若学六家七宗**: 道安、支遁等般若学者活跃，佛教义学达到高峰
- **鸠摩罗什入华**: 401 年鸠摩罗什抵达长安，开启中国佛教翻译的新纪元
- **南北分裂**: 南方以义学为主（建康、庐山），北方以禅修为主

## 学术争议 / Scholarly Debates

### 出生年份

- **传统记载**: 334 年（多数文献采用）
- **争议**: 部分学者根据慧远圆寂时的年龄推算，认为可能是 333 或 335 年

### 出家时间

- **传统说法**: 约 21 岁 (约 355 年) 听道安讲经而出家
- **争议**: 具体年份不确定，因为早期文献记载模糊

### 白莲社人数

- **传统记载**: 123 人
- **争议**: "十八贤"是否为后世建构？实际参与的核心成员人数不确定

### 与陶渊明的交往

- **传说**: "虎溪三笑"——慧远送客不过虎溪，唯送陶渊明与陆修静时不觉过溪，虎辄鸣，三人相视大笑
- **历史分析**: 陆修静（406-477）在慧远圆寂时仅 10 岁，三人不可能同时出现。此传说为后世附会

## 历史影响 / Historical Impact

| 时期 | 影响 |
|------|------|
| 5 世纪 | 庐山成为中国南方最重要的佛教中心 |
| 5-6 世纪 | 僧祐编纂《弘明集》，收录慧远多篇论著 |
| 6 世纪 | 昙鸾发展称名念佛，净土思想演变 |
| 7 世纪 | 善导创立净土宗，追溯慧远为初祖 |
| 宋元 | 白莲社模式被反复模仿；白莲教借用"白莲"之名 |
| 明清 | 慧远作为净土初祖的地位进一步巩固 |
| 20 世纪 | 汤用彤、许理和等学者的现代学术研究 |
| 当代 | 印光大师等继承慧远净土精神 |
