# 慧能年表 / Timeline of Huineng

## 历史语境 / Historical Context

- **唐朝 (618-907)**: 中国佛教黄金时代
- **禅宗分化**: 南北二宗的宗派竞争
- **文化繁荣**: 诗歌、书法、绘画与佛教交融

## 生平年表 / Biographical Timeline

| 年代 | 事件 | 备注 |
|------|------|------|
| 638 | 生于岭南新州 (今广东新兴) | 父亲早逝 |
| ~650s | 少年砍柴养母 | 出身贫寒 |
| ~658 | 听《金刚经》"应无所住而生其心"有感 | 决心求法 |
| ~660 | 北上黄梅参五祖弘忍 | "獦獠亦有佛性" |
| ~661 | 作偈"菩提本无树" | 弘忍密传衣钵 |
| ~661 | 南归避难 | 惠明追赶，"不思善不思恶" |
| ~661-676 | 隐居岭南约 15 年 | 混迹猎人群 |
| 676 | 在广州法性寺遇印宗法师 | "风动幡动"公案 |
| 677 | 回曹溪宝林寺弘法 | 此后 37 年 |
| 677-713 | 说法教化，法嗣众多 | 南岳怀让、青原行思等 |
| 713 | 圆寂于新州国恩寺 | 肉身不坏，现存南华寺 |

## 禅宗分化 / Chan Schism

- **北宗 (神秀系)**: "渐修"——时时勤拂拭；受朝廷支持
- **南宗 (慧能系)**: "顿悟"——本来无一物；民间传播
- **后世**: 南宗胜出，成为中国禅宗主流

## 法嗣传承 / Dharma Successors

| 弟子 | 法系 | 后发展 |
|------|------|--------|

| 永嘉玄觉 | 独立 | 证道歌 |
