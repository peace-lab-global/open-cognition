# 寂天年表 / Timeline of Śāntideva

## 历史语境 / Historical Context

### 时代背景

- **印度佛教中期** (7-8 世纪): 那烂陀大学的鼎盛时期
- **中观-瑜伽行综合**: 寂护 (Śāntarakṣita) 创建瑜伽行中观派
- **大乘伦理学**: 菩提心与菩萨行成为核心关注

### 传承背景

- **中观学派**: 龙树-月称传统
- **瑜伽行派**: 无著-世亲传统
- **那烂陀学术**: 多元学派共存的学术环境

## 生平年表 / Biographical Timeline

| 年代 (约) | 事件 | 备注 |
|-----------|------|------|
| ~685 | 生于南印度 Saurashtra (今古吉拉特邦) | 王子出身 |
| ~700 | 放弃王位出家 | 传说在加冕典礼前夜出家 |
| ~710 | 入那烂陀大学 | 学习佛教哲学和修行 |
| ~715-720 | 撰写《学处集要》Śikṣāsamuccaya | 修行纲要 |
| ~720-725 | **撰写《入菩萨行论》Bodhicaryāvatāra** | 核心著作 |
| ~725 | 传说在辩论中宣说第九品 | 身体升空的传奇 |
| ~730 | 离开那烂陀 | 游历南印度 |
| ~750 | 圆寂 | 具体地点不详 |

## 学术争议 / Scholarly Debates

### 年代问题

- 学界共识为 8 世纪，但具体年代有 ±25 年的浮动
- 与寂护 Śāntarakṣita 的先后关系仍有争议

## 历史影响 / Historical Impact

| 时期 | 影响 |
|------|------|
| 8 世纪 | 成为那烂陀大学重要论师 |
| 9-12 世纪 | 《入菩萨行论》成为印度佛教修行核心文本 |
| 10 世纪 | 传入西藏，成为藏传佛教最重要伦理文本 |
| 14 世纪 | 宗喀巴在《菩提道次第广论》中大量引用 |
| 20 世纪 | 达赖喇嘛多次讲授，成为全球性文本 |
| 当代 | 与西方伦理学、心理学、有效利他主义的对话 |
