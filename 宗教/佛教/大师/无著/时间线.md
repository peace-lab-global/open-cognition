# 无著世亲年表 / Timeline of Asanga and Vasubandhu

## 编年

| 年代 | 事件 |
|------|------|
| 约310-315 CE | 无著生于犍陀罗（今巴基斯坦白沙瓦一带），婆罗门种姓 |
| 约320 CE | 世亲生于犍陀罗，无著之弟 |
| 约330s-340s | 无著出家，先修小乘，不满足于空观，传说升兜率天从弥勒菩萨受法 |
| 约340s-360s | 世亲在迦湿弥罗（Kashmir）学习有部阿毗达磨，著《俱舍论》 |
| 约360s | 世亲在无著影响下转入大乘（传说无著派弟子诵《十地经》感化世亲） |
| 约360s-380s | 兄弟在犍陀罗和中印度弘扬唯识学说 |
| 约370s-390s | 世亲著《唯识三十颂》《唯识二十论》等大乘著作 |
| 约390 CE | 无著去世 |
| 约400 CE | 世亲去世（一说约380 CE） |

## 思想发展分期

### 无著
| 时期 | 核心工作 |
|------|----------|
| 早期 | 修小乘禅法，不满足 |
| 中期 | 从弥勒受法，组织瑜伽行传统 |
| 晚期 | 建立完整的唯识体系，著《瑜伽师地论》《摄大乘论》 |

### 世亲
| 时期 | 核心工作 | 代表作 |
|------|----------|--------|
| 小乘期 | 系统整理有部阿毗达磨 | 《俱舍论》 |
| 转折期 | 在无著影响下转入大乘 | — |
| 大乘早期 | 注释弥勒-无著著作 | 《辨中边论》释 |
| 大乘晚期 | 建立唯识论证体系 | 《唯识三十颂》《唯识二十论》 |

## 同时代人物

| 龙树（Nāgārjuna） | 约150-250 CE | 前辈，中观学派创始人 |

## 唯识学派传承

| 世代 | 人物 | 年代 | 贡献 |
|------|------|------|------|
| 奠基 | 弥勒/无著 | 约4世纪 | 瑜伽行传统的建立 |
| 体系 | 世亲 | 约4-5世纪 | 唯识论证体系 |
| 分化 | 安慧（Sthiramati） | 约470-550 | 无相唯识派 |
| 分化 | 陈那（Dignāga） | 约480-540 | 唯识因明学 |
| 集大成 | 护法（Dharmapāla） | 约530-561 | 有相唯识派 |
| 传播 | 玄奘 | 602-664 | 唯识传入中国 |
| 中国化 | 窥基 | 632-682 | 法相宗建立 |

## 历史影响

| 时代 | 发展 |
|------|------|
| 5-6世纪 | 那烂陀寺成为唯识学中心 |
| 7世纪 | 玄奘将唯识学传入中国，建立法相宗 |
| 8-9世纪 | 法相宗在中国逐渐衰微，但影响华严、天台 |
| 8世纪 | 唯识学传入日本（南都六宗之一） |
| 20世纪 | 唯识学复兴（欧阳渐、太虚、吕澂） |
| 当代 | 唯识学与认知科学、心理学的对话 |
