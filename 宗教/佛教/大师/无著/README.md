# 无著与世亲 Asanga-Vasubandhu

## 基本信息

**无著 Asanga** (约 315-390 CE)
- 出生地:犍陀罗 Gandhara (今巴基斯坦白沙瓦一带)
- 身份:瑜伽行派 Yogacara 创始人，大乘唯识学派奠基者
- 传说:从弥勒菩萨 Maitreya 受法，得《瑜伽师地论》

**世亲 Vasubandhu** (约 4-5 世纪 CE)
- 无著之弟，初习说一切有部 Sarvastivada
- 著《俱舍论》Abhidharmakosa，为小乘阿毗达磨巅峰
- 后转入大乘，著《唯识三十颂》Trimsika、《唯识二十论》Vimsatika

**兄弟合作**
- 并称"瑜伽行二圣"
- 无著:组织体系、弥勒传法、宏观架构
- 世亲:逻辑精炼、从小乘转唯识、论证辩难

---

## 一句话定位

**无著与世亲建立了佛教史上第一个系统化的心识心理学，将认知分析从零散教义提升为完整的八识理论体系，开创了大乘瑜伽行派，深刻影响了东亚佛教、心理学与认知科学。**

---

## 兄弟合作与分工

### 无著:体系建构者

无著是瑜伽行派的总设计师。传说他在兜率天 Tushita 从弥勒菩萨受法，获得《瑜伽师地论》Yogacarabhumi-sastra 等根本经典。这一"天启"叙事赋予其学说权威性，但现代学者多认为弥勒可能是历史人物或无著的精神导师。

无著的核心贡献:
- **组织弥勒学说**:将零散的瑜伽行传统系统化为完整理论
- **建立阿赖耶识缘起**:提出根本识作为一切心理活动的基底
- **整合部派佛教**:吸收有部、经量部的阿毗达磨方法，转向大乘立场
- **修道次第**:在《瑜伽师地论》中建立从凡夫到成佛的完整修行阶梯

### 世亲:逻辑精炼者

世亲的思想经历了从小乘到大乘的重大转变。早期作为说一切有部学者，他撰写了《俱舍论》Abhidharmakosa，这是部派佛教阿毗达磨的巅峰之作。在无著影响下转入大乘后，他以严密的逻辑和辩论技巧捍卫唯识学说。

世亲的核心贡献:
- **从经量部到唯识**:将经量部的认识论资源转化为唯识学的论证工具
- **逻辑论证**:在《唯识二十论》中以辩论形式系统论证"唯识无境"
- **精炼体系**:在《唯识三十颂》中以偈颂形式凝练唯识要义
- **心所分类**:在《大乘百法明门论》中建立五位百法的心识分析框架

## 六个核心命题

### 1. 唯识无境 Vijnapti-matrata

一切现象皆是心识的显现 (vijnapti)，没有独立于认知之外的客观实在。这不是否认外界存在，而是说我们所经验的一切都是心识构造的表象。

- 梵文:vijnapti-matrata (唯识、唯表)
- 汉译:唯识无境、三界唯心
- 关键论证:梦喻、镜像喻、幻觉喻

详见 [三性说](concepts/三性.md) 对认知层次的深入分析。

### 2. 八识体系 Asta-vijnana

心识分为八个层次:
1. 眼识 caksur-vijnana (视觉)
2. 耳识 srotra-vijnana (听觉)
3. 鼻识 ghrana-vijnana (嗅觉)
4. 舌识 jihva-vijnana (味觉)
5. 身识 kaya-vijnana (触觉)
6. 意识 mano-vijnana (概念思维)
7. 末那识 manas (自我执取)
8. 阿赖耶识 alaya-vijnana (潜意识储藏)

详见 [八识体系](concepts/八识体系.md)。

### 3. 三性说 Trisvabhava

认知对象的三种存在样态:
- **遍计所执性** parikalpita-svabhava:虚妄分别，如绳误认为蛇
- **依他起性** paratantra-svabhava:缘起依存，如绳的因缘和合
- **圆成实性** parinispanna-svabhava:真实本性，如绳的麻质实相

详见 [三性说](concepts/三性.md)。

### 4. 种子熏习 Bija-Vasana

经验以"种子" bija 形式储存于阿赖耶识，遇缘则"现行"显现为新的经验。这一过程称为"熏习" vasana。

- 种子:潜在的心理习气
- 熏习:经验对心识的印刻作用
- 现行:种子遇缘显现为具体经验

详见 [种子与熏习](concepts/种子与熏习.md)。

### 5. 转识成智 Asraya-paravrtti

修行的终极目标是实现"转依"，将八识转化为四智:
- 前五识 → 成所作智 krtyanusthana-jnana
- 第六意识 → 妙观察智 pratyaveksana-jnana
- 第七末那识 → 平等性智 samata-jnana
- 第八阿赖耶识 → 大圆镜智 adarsa-jnana

详见 [转识成智](concepts/转识成智.md)。

### 6. 瑜伽行修道次第

《瑜伽师地论》建立完整的修行阶梯:
- **十七地**:从五识身相应地到无余依地
- **五位**:资粮位、加行位、通达位、修习位、究竟位
- **四寻思、四如实智**:观照三性的具体方法

## 当代应用

### 心理学

**与弗洛伊德对话**
- 阿赖耶识 ≈ 潜意识 (unconscious)
- 种子 ≈ 被压抑的记忆痕迹
- 熏习 ≈ 心理印刻机制
- 差异:阿赖耶识是无我的流动系统，弗洛伊德的潜意识以自我为核心

**与荣格对话**
- 阿赖耶识 ≈ 集体无意识 (collective unconscious)
- 种子 ≈ 原型 (archetypes)
- 差异:唯识强调种子的缘起性，荣格的原型更具本体论色彩

### 认知科学

**记忆研究**
- 种子理论预示了现代记忆的三阶段模型:编码 (encoding)、储存 (storage)、提取 (retrieval)
- 熏习机制类似于长期增强 (long-term potentiation, LTP)

**认知架构**
- 八识体系是早期的多层认知模型
- 末那识的自我监控功能类似于元认知 (metacognition)
- 阿赖耶识类似于认知科学中的"认知无意识" (cognitive unconscious)

**具身认知**
- 唯识学强调心识与身体的互动，与具身认知 (embodied cognition) 有共鸣

### 心灵哲学

**意向性 Intentionality**
- 唯识学的"识"具有意向性结构，与胡塞尔现象学对话
- 三性说类似于现象学还原的不同阶段

**意识研究**
- 八识体系为"意识的困难问题" (hard problem of consciousness) 提供东方视角
- 转识成智描述了意识转化的完整路径

## 常见误读

### 误读 1:"唯识 = 唯心论"

**误读**:唯识学否认外界存在，是主观唯心主义。

**澄清**:唯识学否认的是"独立自性的外界"，而非经验世界本身。这是认识论立场，不是本体论唯心。唯识学承认缘起的相对真实性 (依他起性)，只否定我们对事物的虚妄执着 (遍计所执性)。

### 误读 2:"阿赖耶识 = 灵魂"

**误读**:阿赖耶识是永恒不变的灵魂实体。

**澄清**:阿赖耶识是无我的、流动的、缘起的系统。它像河流一样持续变化，种子不断熏习、现行、再熏习。它不是固定实体，而是动态过程。

### 误读 3:"唯识否认外界"

**误读**:唯识学认为外界完全不存在，一切都是幻觉。

**澄清**:唯识学区分三种存在样态 (三性)。它否认的是遍计所执性 (虚妄分别)，但承认依他起性 (缘起依存) 的相对真实，并追求圆成实性 (真实本性)。

### 误读 4:"种子是固定实体"

**误读**:种子是储存在阿赖耶识中的固定心理单元。

**澄清**:种子是动态的、缘起的、无自性的。它们在熏习中不断变化，在现行中重新组合。种子不是原子式的心理粒子，而是流动的习气倾向。

### 误读 5:"转识成智是消灭心识"

**误读**:修行目标是消灭八识，进入空无。

**澄清**:转识成智是转化而非消灭。八识转化为四智，认知功能依然存在，但去除了虚妄执着。这是认知的净化与提升，不是认知的否定。

## 跨学科关联

### 胡塞尔 Edmund Husserl (1859-1938)

**平行点**:
- 意向性 (Intentionality):胡塞尔的"意识总是关于某物的意识"与唯识的"识必有境"
- 现象学还原:悬置自然态度，类似于从遍计所执转向依他起
- 先验自我:胡塞尔的先验自我与末那识的自我执取功能

**差异**:
- 胡塞尔追求先验主体性，唯识学追求无我
- 胡塞尔的现象学还原是方法论，唯识的三性是存在论

### 弗洛伊德 Sigmund Freud (1856-1939)

**平行点**:
- 潜意识:阿赖耶识与弗洛伊德的潜意识系统
- 压抑机制:种子熏习与被压抑记忆的储存
- 心理动力:心识的流动与力比多 (libido) 的能量模型

**差异**:
- 弗洛伊德以性驱力为核心，唯识以认知为核心
- 弗洛伊德的潜意识主要是被压抑的内容，阿赖耶识包含一切经验种子

### 皮亚杰 Jean Piaget (1896-1980)

**平行点**:
- 图式 (schema):种子类似于认知图式
- 同化与顺应:熏习机制与图式的调整过程
- 认知发展阶段:八识体系描述认知的层次结构

**差异**:
- 皮亚杰关注儿童认知发展，唯识关注普遍的认知结构
- 皮亚杰的图式是认知结构，种子包含情感、意志等全部心理内容

### 现代认知科学

**平行点**:
- 认知无意识 (cognitive unconscious):阿赖耶识的自动处理功能
- 工作记忆 (working memory):意识的焦点与第六意识的功能
- 元认知 (metacognition):末那识的自我监控功能
- 神经可塑性 (neuroplasticity):熏习机制与大脑的可塑性

**差异**:
- 认知科学以神经科学为基础，唯识以现象学描述为基础
- 认知科学追求客观解释，唯识追求主观转化

## 东西方对话:无著/世亲 × 胡塞尔/萨特/波伏娃

### 八识 vs 意识地形

**无著/世亲的八识体系**
- 前五识:感官认知
- 第六意识:概念思维
- 第七末那识:自我执取
- 第八阿赖耶识:潜意识储藏

**胡塞尔的意识结构**
- 感性直观 (sinnliche Anschauung):感官层面
- 范畴直观 (kategoriale Anschauung):概念层面
- 先验自我 (transzendentales Ego):自我极
- 生活世界 (Lebenswelt):前反思的经验基底

**对话**:
- 前五识与感性直观都描述感官层面
- 第六意识与范畴直观都涉及概念构造
- 末那识与先验自我都涉及自我功能，但末那识是虚妄执取，先验自我是认识论条件
- 阿赖耶识与生活世界都是前反思的经验基底，但阿赖耶识是个体潜意识，生活世界是主体间性的

### 三性 vs 现象学还原

**无著/世亲的三性**
- 遍计所执性:虚妄分别
- 依他起性:缘起依存
- 圆成实性:真实本性

**胡塞尔的现象学还原**
- 自然态度 (naturliche Einstellung):日常经验的预设
- 现象学悬置 (epoche):悬置存在信念
- 先验还原 (transzendentale Reduktion):回到纯粹意识

**对话**:
- 遍计所执性类似于自然态度中的虚妄预设
- 依他起性类似于现象学还原后揭示的意向性结构
- 圆成实性类似于先验还原达到的纯粹意识，但圆成实性超越主客二分，纯粹意识仍在主体性框架内

**萨特的意识与虚无**
- 萨特:意识是虚无，存在先于本质
- 唯识:识是缘起，无自性但有功能
- 对话:两者都否定固定实体，但萨特强调存在的偶然性，唯识强调缘起的规律性

### 转识成智 vs 本真承担

**无著/世亲的转识成智**
- 转八识为四智
- 去除虚妄执着
- 实现觉悟智慧

**萨特的本真性 (authenticity)**
- 承认存在的偶然性
- 承担自由选择的责任
- 拒绝自欺 (mauvaise foi)

**波伏娃的模糊性伦理学**
- 承认存在的模糊性
- 在自由中承担他者
- 实现本真的伦理关系

**对话**:
- 转识成智是认知转化，本真承担是存在选择
- 两者都要求去除虚妄 (执着/自欺)
- 两者都追求更高的意识状态 (智慧/本真)
- 差异:转识成智追求普遍的觉悟，本真性强调个体的承担

### 种子 vs 被抛/情境

**唯识的种子**
- 过去经验的储存
- 决定当下的认知倾向
- 通过熏习可以改变

**萨特的被抛 (Geworfenheit)**
- 存在的偶然给予
- 情境的限制与可能
- 通过选择可以超越

**波伏娃的情境 (situation)**
- 自由与限制的辩证
- 身体、历史、社会的交织
- 在情境中实现自由

**对话**:
- 种子与被抛都是过去的给予性
- 两者都影响当下但不决定当下
- 种子通过熏习改变，被抛通过选择超越
- 差异:种子是心理习气，被抛是存在论条件

### 无我 vs 存在先于本质

**唯识的无我 anatman**
- 没有固定不变的自我实体
- 自我是五蕴的缘起和合
- 末那识的虚妄执取产生自我幻觉

**萨特的存在先于本质**
- 人没有预定的本质
- 存在是虚无，通过选择创造本质
- 自我是自由的产物

**对话**:
- 两者都否定固定本质
- 无我否定实体自我，萨特否定预定本质
- 无我追求觉悟，萨特追求本真
- 差异:无我是缘起论，存在先于本质是存在论
