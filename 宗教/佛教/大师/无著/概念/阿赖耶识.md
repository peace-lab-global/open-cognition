# 阿赖耶识 / Ālayavijñāna (Storehouse Consciousness)

## 定义

阿赖耶识（ālaya-vijñāna，藏识、第八识）是唯识学最核心的概念——一切心识活动的最深层基底。"阿赖耶"意为"藏"（storehouse），此识含藏一切经验的"种子"（bīja），种子遇缘则现行（manifest），现行又熏习新种子——形成一个动态的循环系统。阿赖耶识不是灵魂或实体，而是一个不断流转的功能性过程。

## 核心功能

### 1. 持种（种子仓库）
阿赖耶识含藏一切善恶无记种子。种子不是物质性的存在，而是功能性潜能——类似现代认知科学中的"长期记忆"或"图式"（schema），但更根本——它包含了一切认知、情感和业力的潜能。

### 2. 根身器界（身体与世界的基础）
阿赖耶识不仅储存心理种子，还是"根身"（感官身体）和"器界"（外在世界）的共同基础。我们感知的身体和世界都是阿赖耶识种子的现行——这是唯识学"唯识无境"的核心论证。

### 3. 执持（相续不断）
阿赖耶识是生死流转中唯一相续不断的识——前六识在熟睡、闷绝、无想定中会中断，但阿赖耶识从不间断，它是生命相续的底层载体。

## 阿赖耶识的特征

| 特征 | 说明 |
|------|------|
| 无记（avyākṛta） | 阿赖耶识本身非善非恶，是中性的载体 |
| 恒转如瀑流 | 如同瀑布——看似恒常，实际是不断变化的流 |
| 不可知 | 非前六识所能直接观察（需要瑜伽禅定） |
| 异熟（vipāka） | 阿赖耶识是过去业的异熟果报 |

## 种子（bīja）与现行

种子与现行构成阿赖耶识的核心循环：
1. **种子**（bīja）：潜藏在阿赖耶识中的功能性潜能
2. **现行**（pravṛtti）：种子遇缘而生起的心识活动和外在现象
3. **熏习**（vāsanā）：现行的经验反过来在阿赖耶识中留下新种子

这三者构成"种子生现行、现行熏种子"的动态循环——唯识学称之为"三法二重因果"。

## 阿赖耶识 vs 灵魂

| 维度 | 阿赖耶识 | 灵魂（ātman/soul） |
|------|----------|-------------------|
| 本体性质 | 功能性过程（非实体） | 实体性的自我 |
| 变化性 | "恒转如瀑流"——不断变化 | 永恒不变 |
| 无我 | 阿赖耶识不是"我" | 灵魂是"我" |
| 可转性 | 可以"转识成智" | 不可转化 |

## 与弗洛伊德潜意识的比较

| 维度 | 阿赖耶识 | 弗洛伊德潜意识 |
|------|----------|---------------|
| 位置 | 八识中最深层 | 意识地形模型的最底层 |
| 功能 | 种子储存与现行 | 被压抑的欲望和记忆 |
| 运作方式 | 种子→现行→熏习 | 压抑→症状→移情 |
| 可访问性 | 需要瑜伽禅定 | 需要精神分析 |
| 价值性 | 无记（非善非恶） | 主要是本能和欲望 |

## 与海德格尔/梅洛-庞蒂的比较

- **海德格尔的"被抛"（Geworfenheit）**：此在总是已经被抛入一个世界——阿赖耶识的"器界"也是被抛入的，不是被选择的
- **梅洛-庞蒂的"身体图式"**：身体对空间的前反思把握——阿赖耶识的"根身"也是前反思的身体基础
