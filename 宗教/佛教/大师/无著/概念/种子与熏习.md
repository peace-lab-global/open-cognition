# 种子与熏习 / Bīja and Vāsanā

## 定义

种子（bīja）和熏习（vāsanā，又译"习气"）是唯识学描述心识运作机制的核心概念。种子是潜藏在阿赖耶识中的功能性潜能；熏习是现行经验在阿赖耶识中留下印记的过程。两者构成"种子生现行、现行熏种子"的动态循环——这是唯识学对"经验如何影响未来认知"的最精密描述。

## 种子（Bīja）

### 定义
种子不是物质性的"微粒"，而是功能性的潜能——类似现代认知科学中的"长期记忆痕迹"或"认知图式"，但涵盖范围更广（包括情感、业力、认知模式）。

### 种子的六种特征
世亲在《成唯识论》中提出种子的六种特征：
1. **刹那灭**：种子是刹那生灭的，不是恒常存在
2. **果俱有**：种子与它所生的现行同时存在
3. **恒随转**：种子相续不断地随阿赖耶识流转
4. **性决定**：善种子生善果，恶种子生恶果
5. **待众缘**：种子需要外缘才能现行
6. **引自果**：种子只引生与自己同类的果

### 种子的分类
- **本有种子**（无始以来就有）
- **新熏种子**（由后天经验熏习而成）
- **名言种子**（概念性的认知潜能）
- **业种子**（善恶业力的潜能）

## 熏习（Vāsanā）

### 定义
熏习（vāsanā，又译"习气"）——字面义为"香气"——是现行经验在阿赖耶识中"染香"的过程。如同衣服被香熏过后留下香味，每一次心识活动都在阿赖耶识中留下印记。

### 熏习的条件
熏习需要四个条件（"能熏四义"）：
1. **有生灭**：能熏者必须是有生灭的（非无为法）
2. **有胜用**：能熏者必须有强盛的作用力
3. **有增减**：能熏者的力量可以增减
4. **与所熏和合**：能熏者必须与所熏（阿赖耶识）同时同处

## 种子与现行的循环

```
种子（bīja）→ 遇缘 → 现行（pravṛtti）→ 熏习（vāsanā）→ 新种子 → ...
```

这个循环被称为"三法二重因果"：
- **第一重因果**：种子（因）→ 现行（果）
- **第二重因果**：现行（因）→ 新种子（果）

## 与现代认知科学的比较

| 唯识 | 认知科学 | 对应 |
|------|----------|------|
| 种子（bīja） | 长期记忆痕迹 | 经验的编码储存 |
| 现行（pravṛtti） | 记忆提取/认知激活 | 潜能的显性化 |
| 熏习（vāsanā） | 记忆巩固/学习 | 新经验的储存 |
| 名言种子 | 概念图式（schema） | 认知分类框架 |
| 业种子 | 情感记忆/条件反射 | 情绪性行为模式 |

## 与皮亚杰"图式"的比较

- **皮亚杰的图式（schema）**：认知结构通过同化（assimilation）和顺应（accommodation）不断发展
- **唯识的种子**：认知潜能通过熏习和现行不断更新
- **共同点**：都描述了经验的累积性结构
- **差异**：皮亚杰关注认知发展，唯识关注解脱转化

## 与梅洛-庞蒂"身体图式"的比较

- **身体图式**：身体对空间的前反思把握——是通过重复行动形成的动态结构
- **种子-习气**：通过重复经验形成的认知潜能——也是动态的、可转化的
- **共同点**：都描述了身体/心识层面的"积淀"
- **差异**：梅洛-庞蒂强调身体性，唯识涵盖身-心-业力全维度
