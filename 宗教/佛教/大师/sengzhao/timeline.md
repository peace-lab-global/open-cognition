# 僧肇生平年表 · Sengzhao Biography Timeline

## 早年与玄学修养（384–401）

| 年份 | 事件 |
|------|------|
| 384 | 生于京兆（今陕西西安），家境贫寒 |
| c. 395 | 少年时以抄书为业，精通文字 |
| c. 396–399 | 好老庄玄学，深通《老子》《庄子》，以玄学闻名 |
| c. 400 | 读《维摩诘经》而发心向佛，始知玄学不及佛法深妙 |
| 401 | 鸠摩罗什入长安，僧肇即往投其门下 |

## 从学鸠摩罗什（401–413）

| 年份 | 事件 |
|------|------|
| 401 | 投鸠摩罗什门下，开始系统学习中观般若学 |
| 401–405 | 参与罗什译场，协助翻译和注释中观经典（《中论》《百论》《十二门论》） |
| c. 404 | 撰写《物不迁论》——论证事物的刹那性与不迁 |
| c. 406 | 撰写《不真空论》——论证因缘假有、体性本空 |
| c. 406 | 参与《维摩诘经》翻译，撰写注释 |
| c. 406 | 为提婆《百论》作序（《百论序》） |
| c. 408 | 撰写《般若无知论》——论证般若超越概念认知 |
| c. 410 | 撰写《涅槃无名论》——论证涅槃超越名言概念 |

## 英年早逝（413–414）

| 年份 | 事件 |
|------|------|
| 413 | 鸠摩罗什去世——僧肇失去最重要的师承庇护 |
| 413–414 | 后秦政治动荡，僧团内部纷争加剧 |
| 414 | 被后秦主姚兴下令处死（具体原因不明，有政治牵连、僧团纷争、与姚兴论争等多种说法），年仅31岁 |

## 身后影响

| 年份 | 事件 |
|------|------|
| 5世纪中 | 《肇论》在僧肇去世后结集流传，成为中国佛教哲学的经典文本 |
| 6世纪 | 三论宗创始人吉藏大量引用《肇论》，尊僧肇为中观正统 |
| 7世纪 | 唐代慧达、元康为《肇论》作疏 |
| 11世纪 | 天台宗、华严宗将僧肇视为重要先贤 |
| 1940s | 汤用彤《汉魏两晋南北朝佛教史》高度评价僧肇 |
| 1968 | Walter Liebenthal出版 *Chao Lun* 英译 |
| 2000s | 当代中国哲学界重新重视僧肇的哲学地位 |

## 思想分期总结

| 阶段 | 年代 | 关键词 |
|------|------|--------|
| 玄学少年 | 384–401 | 老庄、抄书、向佛 |
| 从学罗什 | 401–413 | 中观、般若、四论 |
| 英年早逝 | 413–414 | 处死、31岁 |
