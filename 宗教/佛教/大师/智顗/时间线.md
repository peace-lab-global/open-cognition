# 智顗年表 / Timeline of Zhìyǐ

## 历史语境 / Historical Context

### 时代背景

- **南北朝至隋** (538-597): 中国从分裂走向统一
- **佛教中国化**: 天台宗是中国第一个本土化的佛教宗派
- **中观传统**: 鸠摩罗什-僧肇-吉藏的中观学在中国传播

### 传承背景

- **慧思** (515-577): 智顗的老师，止观修行的先驱
- **法华经传统**: 法华经在中国的传播和诠释
- **北方禅修与南方义学**: 南北佛教传统的融合

## 历史影响 / Historical Impact

| 时期 | 影响 |
|------|------|
| 6 世纪 | 创立天台宗，中国佛教第一个宗派 |
| 7-8 世纪 | 灌顶、智威等继承天台法脉 |
| 8 世纪 | 天台宗传入日本（最澄） |
| 10-11 世纪 | 宋代天台宗复兴（知礼等） |
| 12 世纪 | 日本天台宗发展为日本佛教母体 |
| 当代 | 天台教观研究成为全球佛教学术热点 |
