# 吉藏 Jizang

> "破邪即是显正，无所得即是中道。"
> — 吉藏三论宗核心纲领

## 基本信息 / Basic Information

**吉藏 Jizang**（549-623），金陵（今南京）人，安息（中亚帕提亚）后裔，**三论宗 Sanlun School 集大成者**。师从法朗，继承僧肇、鸠摩罗什以来的汉传中观传统，以《中论》《百论》《十二门论》三论为根本，建立了完整的**汉传中观学派**（三论宗），是隋唐佛教八大宗派之一。

- **出生地 / Birthplace**: 金陵（今南京），祖籍安息（Parthia，中亚）
- **年代 / Era**: 549-623 CE，历经南朝陈、隋、唐三代
- **身份 / Role**: 三论宗实际创始人、佛学论辩家、注释家
- **传承 / Lineage**: 龙树→提婆→鸠摩罗什→僧肇→僧诠→法朗→吉藏
- **核心经典 / Core Texts**: 《中论》《百论》《十二门论》
- **别号 / Honorific**: 嘉祥大师（因居会稽嘉祥寺而得名）

## 核心命题 / Core Propositions

### 1. 言教二谛 Pedagogical Two Truths

吉藏对二谛论做出了革命性的重新诠释：世俗谛与胜义谛不是两种本体论层次的真实，而是**两种言说方式**（言教）。一切佛说皆为方便施设，二谛本身也是方便，最终必须超越二谛的对立。这一"言教二谛"立场是吉藏对汉传中观最独特的贡献。

**The two truths are not two levels of ontological reality but two pedagogical modes of discourse. All Buddhist teachings are expedient means (upaya); even the two truths themselves must ultimately be transcended.**

详见 [言教二谛](concepts/two-truths-pedagogical.md)。

### 2. 八不中道 Eight Negations and the Middle Way

不生不灭、不常不断、不一不异、不来不去——继承龙树《中论》开篇的八不偈，吉藏将八不视为超越一切边见（antagraha-drshti）的系统方法。八不是对八种极端概念的双重否定，构成中道的否定性表达。

**Eight negations — no arising, no ceasing, no permanence, no discontinuity, no identity, no difference, no coming, no going — constitute the systematic deconstruction of all conceptual extremes.**

详见 [八不中道](concepts/eight-negations.md)。

### 3. 破邪显正 Refuting the False Reveals the True

吉藏的方法论核心：不需要建立任何正面主张（pratijna），破除一切邪见即是中道。"破邪即是显正"——这不是消极的否定，而是一种独特的哲学方法：当所有偏执都被消解，中道自然显现。

**One does not need to establish any positive thesis. The systematic refutation of all wrong views is itself the revelation of the Middle Way — negation as affirmation.**

详见 [破邪显正](concepts/refute-reveal.md)。

### 4. 无所得 Non-attainment

一切法无所得——包括"无所得"本身也不可得。吉藏将"无所得"从修行论提升为方法论原则：不执着于任何概念立场，包括"不执着"本身。这构成三论宗最彻底的反本质主义表达。

**Nothing is to be attained — including the state of non-attainment itself. This recursive negation ensures that Madhyamaka does not degenerate into yet another essentialist position.**

详见 [无所得](concepts/non-attainment.md)。

### 5. 重空 Emptiness of Emptiness

空亦复空——"空"本身也是空的。吉藏继承龙树"空空"（sunyata-sunyata）的思想，强调空性不是一个替代性的终极实体，而是对一切实体化思维的彻底瓦解。"重空"（double emptiness）确保中观不会退化为另一种本体论。

**Emptiness itself is empty. This recursive negation — the 'emptiness of emptiness' — prevents sunyata from becoming a substitute absolute.**

详见 [重空](concepts/emptying-emptiness.md)。

## 思想脉络 / Intellectual Context

### 前驱 / Predecessors

- **龙树 Nagarjuna** (约 150-250): 中观学派创始人，《中论》作者
- **提婆 Aryadeva** (约 3 世纪): 龙树弟子，《百论》作者
- **鸠摩罗什 Kumarajiva** (344-413): 将三论译入中文，奠定汉传中观文本基础
- **僧肇 Sengzhao** (384-414): 汉传中观先驱，"解空第一"
- **僧诠 Sengchuan**: 三论学传承的关键环节
- **法朗 Falang** (507-581): 吉藏之师，兴皇寺高僧，将三论学从山林引入都市

### 同时代 / Contemporaries

- **智顗 Zhiyi** (538-597): 天台宗创始人，以中观为基础发展出三谛圆融
- **真谛 Paramartha** (499-569): 唯识学传入中国的关键人物
- **慧远 Huiyuan** (523-592): 净土宗先驱
- **吉藏与天台宗的辩论**: 吉藏曾与天台学派就二谛、中道等核心问题展开深入论辩

### 后继 / Successors

- **慧均 Huijun**: 吉藏弟子，传承三论宗
- **智拔 Zhiba**: 三论宗后继学者
- **日本三论宗**: 三论宗经朝鲜传入日本，成为"南都六宗"之一
- **法藏 Fazang** (643-712): 华严宗集大成者，吸收三论宗方法论

## 思想发展分期 / Intellectual Development

### 早年：金陵时期 (549-581)

吉藏生于金陵（南京），祖籍安息（中亚帕提亚）。少年出家，七岁即投法朗门下学习三论。在法朗指导下，系统研习《中论》《百论》《十二门论》，展现出卓越的佛学天赋。南朝陈代，三论学在建康（南京）一带已有深厚基础，吉藏得以继承和发展这一传统。

### 中期：会稽嘉祥寺时期 (581-606)

隋统一南北后，吉藏移居会稽（今绍兴）嘉祥寺，在此讲学十五年，号"嘉祥大师"。这一时期是他思想成熟的关键阶段：广泛讲说三论，与天台宗、成实师等各派展开论辩，逐步形成自己独特的"言教二谛"和"破邪显正"理论体系。

### 晚期：长安时期 (606-623)

隋炀帝延请吉藏入长安，住大禅定寺（后改称日严寺）。在长安期间，吉藏完成了最重要的注疏和论著，包括《中观论疏》《三论玄义》《二谛义》《大乘玄论》等，将三论宗的理论体系推向完成。入唐后，唐高祖授以"十大德"之一，吉藏继续著述直至圆寂。

## 思想坐标 / Intellectual Positioning

### 哲学定位

- **汉传中观的完成者 Completer of Chinese Madhyamaka**: 将僧肇以来的汉传中观传统系统化为完整学派
- **方法论革新者 Methodological innovator**: "言教二谛"是对龙树二谛论的创造性发展
- **辩论家 Polemicist**: 与天台、唯识、成实等各宗展开深入论辩
- **注释家 Commentator**: 其《中观论疏》是《中论》最详尽的汉文注释

### 与佛教内部的关系

- **vs 天台宗**: 吉藏批评天台的"性具"说有本质主义倾向，天台则批评三论宗过于偏空
- **vs 唯识宗**: 吉藏以"无所得"批判唯识的"识有"立场，认为唯识学仍有执着
- **vs 成实师**: 吉藏批评成实师将"空"理解为某种实有的状态
- **vs 涅槃师**: 吉藏以中观立场重新诠释佛性概念，反对将佛性实体化

### 汉传中观 vs 印度中观

吉藏的三论宗虽然继承龙树、提婆的印度中观传统，但具有鲜明的中国特色：

| 维度 | 印度中观 Indian Madhyamaka | 汉传中观 Sanlun |
|------|--------------------------|-----------------|
| 文本基础 | 梵文原典 + 藏译 | 鸠摩罗什汉译 |
| 方法侧重 | 四句破 (catuskoti) 逻辑分析 | 破邪显正 + 言教二谛 |
| 对话对象 | 有部、经量部 | 天台、唯识、成实 |
| 文化语境 | 印度哲学辩论传统 | 中国玄学与格义佛教 |
| 独创贡献 | 应成/自续的分化 | 言教二谛、重空 |

## 重要概念 / Key Concepts

1. **言教二谛 Pedagogical Two Truths** — 二谛是言说方便，非本体区分
2. **八不中道 Eight Negations** — 超越一切边见的否定性方法
3. **破邪显正 Refuting- Revealing** — 不立正面主张的方法论
4. **无所得 Non-attainment** — 一切法无所得，包括"无所得"本身
5. **重空 Emptiness of Emptiness** — 空亦复空，防止空性实体化

详见各概念子条目。

## 当代应用 / Contemporary Applications

### 哲学 / Philosophy

- **德里达 Derrida**: "破邪显正"与解构主义 (deconstruction) 的方法论平行——二者都通过消解二元对立来揭示被遮蔽的意义
- **维特根斯坦 Wittgenstein**: 言教二谛与"梯子"隐喻的结构性相似——用完即弃的认知工具
- **实用主义 Pragmatism**: 吉藏的言教二谛将一切教说视为方便施设，与罗蒂 Rorty 的反基础主义形成对话

### 语言学 / Linguistics

- **言教理论 Pedagogical theory**: 二谛作为"言说方式"而非"本体层次"，预示了语言哲学中言语行为理论 (speech act theory) 的洞见
- **元语言 Meta-language**: 吉藏关于"教"与"理"的区分，类似于对象语言与元语言的层次分析

### 逻辑学 / Logic

- **多值逻辑 Multi-valued logic**: 八不中道对一切二元对立的否定，可视为多值逻辑或次协调逻辑 (paraconsistent logic) 的古代先声
- **否定逻辑 Logic of negation**: "破邪显正"方法论提供了一种非命题性的逻辑操作模式

### 跨文化对话 / Cross-cultural Dialogue

- **比较哲学**: 吉藏的"无所得"与海德格尔"Gelassenheit"（任其自然）的比较
- **宗教对话**: 三论宗与基督教否定神学 (apophatic theology) 的方法论比较
- **科学哲学**: "破邪显正"与波普尔 Popper 的证伪主义 (falsificationism) 的结构性平行

## 常见误读 / Common Misreadings

### 误读 1: "吉藏 = 否定一切的虚无主义者"

**误读**: 吉藏否定一切立场，所以他是虚无主义者。
**正解**: 吉藏的"破邪"是消解一切概念固化 (conceptual fixation)，不是否定现象世界本身。"破邪即是显正"——消解偏执恰恰是为了让中道自然显现。这与龙树"以有空义故，一切法得成"的精神一脉相承。

### 误读 2: "三论宗 = 印度中观的翻版"

**误读**: 三论宗只是印度中观学派的简单移植。
**正解**: 吉藏的三论宗是在中国思想语境中独立发展的汉传中观。"言教二谛"是吉藏的独创，在印度中观传统中没有对应概念。三论宗融合了玄学、格义佛教等中国思想资源，形成了独特的理论面貌。

### 误读 3: "无所得 = 不需要修行"

**误读**: 既然一切无所得，那就不需要修行了。
**正解**: "无所得"是不执着于修行成果，不是放弃修行。吉藏强调"无所得"恰恰是最精进的修行——因为不执着，所以能真正无碍地修行。这是般若波罗蜜多"无住生心"精神的贯彻。

### 误读 4: "重空 = 什么都没有"

**误读**: "空空"意味着连空都否定了，所以什么都没有。
**正解**: "空空"是否定对"空"的实体化理解，不是否定空性本身。重空的目的是防止空性成为另一种执着——"空"是药不是病，病愈药亦应舍。

### 误读 5: "言教二谛 = 相对主义"

**误读**: 既然二谛都是言教方便，那所有观点都一样有效。
**正解**: 吉藏的言教二谛不是相对主义。虽然一切教说都是方便，但方便有善巧与不善巧之分。二谛的价值在于引导众生从执着走向解脱，不同层次的教说有明确的导向性。
