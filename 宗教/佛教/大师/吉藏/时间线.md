# 吉藏年表 / Timeline of Jizang

## 编年 / Chronology

| 年代 | 事件 |
|------|------|
| 549 | 吉藏生于金陵（今南京），祖籍安息（中亚帕提亚），俗姓安；同年侯景之乱，梁武帝死，南朝动荡 |
| 约556 | 七岁投兴皇寺法朗门下出家，开始系统学习三论 |
| 556-581 | 在法朗指导下深入研习《中论》《百论》《十二门论》，展现卓越佛学天赋 |
| 581 | 隋文帝统一中国，建立隋朝；法朗圆寂（581年），吉藏离开建康 |
| 约581-596 | 移居会稽（今绍兴）嘉祥寺，在此讲学约十五年，号"嘉祥大师" |
| 587 | 智顗在金陵讲《法华经》，吉藏可能参与听法或与之论辩 |
| 589 | 隋灭陈，南北统一；吉藏在嘉祥寺广泛讲说三论，声名远播 |
| 约590s | 与天台宗、成实师等各派展开深入论辩，逐步形成"言教二谛"和"破邪显正"理论体系 |
| 597 | 智顗圆寂；天台宗由灌顶继续发展 |
| 606 | 隋炀帝延请吉藏入长安，住大禅定寺（后改日严寺） |
| 606-618 | 长安时期：完成最重要的注疏和论著 |
| 约608 | 完成《中观论疏》（10卷）——三论宗核心注疏 |
| 约610 | 完成《三论玄义》——三论宗总纲 |
| 约612 | 完成《二谛义》——言教二谛的系统阐述 |
| 约615 | 完成《大乘玄论》（5卷）——大乘教理综论 |
| 618 | 隋朝灭亡，唐朝建立；吉藏继续在长安弘法 |
| 619 | 唐高祖授吉藏"十大德"之一 |
| 约620s | 继续著述，完成《法华玄论》《胜鬘宝窟》等经论注释 |
| 623 | 吉藏圆寂于长安，享年75岁 |

## 思想发展分期 / Intellectual Development

### 第一阶段：学习期 (约556-581)

| 时期 | 核心活动 | 学术特征 |
|------|----------|----------|
| 少年出家 | 投法朗门下，学习三论 | 继承僧诠-法朗的三论学传统 |
| 深入研习 | 系统学习《中论》《百论》《十二门论》 | 奠定坚实的中观学基础 |
| 南朝佛学 | 浸润于南朝佛学的义理传统 | 接触般若、法华、涅槃等各类经典 |

### 第二阶段：成熟期 (约581-606)

| 时期 | 核心活动 | 学术特征 |
|------|----------|----------|
| 嘉祥寺讲学 | 在会稽嘉祥寺讲说三论约十五年 | 思想成熟，形成独特理论体系 |
| 宗派论辩 | 与天台、成实、地论等各派论辩 | 在论辩中发展"言教二谛"和"破邪显正" |
| 学派建立 | 三论宗的理论框架初步形成 | 号"嘉祥大师"，声名远播 |

### 第三阶段：完成期 (606-623)

| 时期 | 核心活动 | 学术特征 |
|------|----------|----------|
| 长安著述 | 完成全部重要注疏和论著 | 三论宗理论体系推向完成 |
| 国家认可 | 隋炀帝延请、唐高祖授"十大德" | 三论宗获得国家层面的认可 |
| 经论融通 | 以三论宗立场诠释法华、涅槃、华严等经典 | 展现三论宗与各宗派的教理融通 |

## 历史语境 / Historical Context

### 南朝陈 (557-589)
- 南朝佛学以义理研讨为主，重经论讲说
- 成实师、地论师、涅槃师等各派活跃
- 三论学在摄山（栖霞山）一带有深厚传统

### 隋朝 (581-618)
- 隋文帝、隋炀帝大力复兴佛教
- 南北佛学传统开始融合
- 天台宗、三论宗等宗派在此时期形成

### 唐朝初期 (618-623)
- 唐初佛教继续繁荣
- 玄奘西行取经（629年出发）尚在吉藏圆寂之后
- 三论宗在吉藏圆寂后逐渐让位于新兴的法相宗和华严宗

## 历史影响 / Historical Impact

| 时代 | 发展 |
|------|------|
| 7世纪初 | 三论宗在吉藏努力下达到鼎盛 |
| 7世纪中 | 吉藏圆寂后，三论宗逐渐被天台、华严、法相等宗派超越 |
| 7世纪末 | 三论宗传入日本（慧灌），成为"南都六宗"之一 |
| 8世纪后 | 三论宗在中国逐渐衰微，但其方法论影响了华严宗和禅宗 |
| 9世纪后 | 日本三论宗继续传承，至今仍有法脉 |
| 20世纪 | 学术界重新关注三论宗，吉藏研究成为佛学研究的重要领域 |
| 当代 | 吉藏的"言教二谛"和"破邪显正"与当代哲学（解构主义、实用主义）的对话日益深入 |
