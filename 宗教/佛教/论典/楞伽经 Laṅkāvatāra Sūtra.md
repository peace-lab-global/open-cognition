# 楞伽经 Laṅkāvatāra Sūtra

## 一句话定义

《楞伽经》是大乘佛教中**融合如来藏与唯识**两大系统的核心经典——它以**五法、三自性、八识、二无我**四大框架统摄佛法全体，既是禅宗初祖达摩"印心"的传法经典（"吾观震旦所有经教，惟《楞伽》四卷可以印心"），也是唯识学的重要理论依据。

## 历史脉络

- **约 3–4 世纪**：印度成立，如来藏与唯识思想交汇期的产物
- **443**：**求那跋陀罗**（Guṇabhadra）译出 4 卷本（最早汉译，禅宗所依）
- **513**：菩提流支译 10 卷本（北魏）
- **704**：实叉难陀（Śikṣānanda）译 7 卷本（唐译，最完整）
- **5–6 世纪**：**菩提达摩**以此经传法慧可，为禅宗印心经典
- **7 世纪后**：随着《金刚经》地位上升，楞伽经在禅宗内的核心地位有所转移

## 核心要义

### 1. 四大框架

| 框架 | 内容 |
|------|------|
| **五法** | 名（nāman）/ 相（nimitta）/ 分别（vikalpa）/ 正智（samyagjñāna）/ 如如（tathatā） |
| **三自性** | 妄想自性（遍计所执）/ 缘起自性（依他起）/ 成实自性（圆成实） |
| **八识** | 眼耳鼻舌身意 / 末那 / 阿赖耶（如来藏识） |
| **二无我** | 人无我 / 法无我 |

### 2. 如来藏识 = 阿赖耶识

《楞伽经》的独特贡献：将如来藏与阿赖耶识**等同**：

- **如来藏**（tathāgatagarbha）= 自性清净、具足如来功德
- **阿赖耶识**（ālayavijñāna）= 含藏染净种子、为七识之根本
- **如来藏识** = 同一心体，从净面名如来藏，从染面名阿赖耶

> "如来藏识藏，虽自性清净，客尘所覆故，犹见不净。"

这一融合解决了"杂染如何从清净中生起"的难题——如来藏本净，但受客尘烦恼覆蔽，呈现为阿赖耶识的染净和合。

### 3. 自觉圣智

楞伽经以**自觉圣智**（pratyātmāryajñāna）为修行的究竟：

| 层次 | 说明 |
|------|------|
| 离心意意识 | 超越八识的分别作用 |
| 离五法三自性 | 超越一切概念建构 |
| 自觉圣智 | 不从他悟，自心现量直证真如 |

这与禅宗"不立文字，直指人心"的精神高度一致。

### 4. 禅宗印心传统

| 祖师 | 与楞伽经的关系 |
|------|--------------|
| 达摩 | 以四卷楞伽传法慧可，"可以印心" |
| 慧可 | 以楞伽经教授弟子 |
| 僧璨–道信 | 渐转向般若/金刚经 |
| 弘仁–慧能 | 以金刚经印心，楞伽地位下降 |

> 早期禅宗因此亦称"**楞伽宗**"（胡适考证）。

## 常见误读

- ❌ "楞伽经 = 纯粹唯识经典" → 楞伽经是如来藏与唯识的融合，不能仅归一方。
- ❌ "如来藏 = 阿赖耶识完全等同" → 是同体异面——净面为如来藏，染面为阿赖耶，并非简单等同。
- ❌ "禅宗从来不用经典" → 早期禅宗（达摩–慧可）以楞伽经印心，"不立文字"不等于"不用经典"。
