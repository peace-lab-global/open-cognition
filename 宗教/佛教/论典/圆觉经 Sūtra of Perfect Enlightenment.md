# 圆觉经 · Sūtra of Perfect Enlightenment

> "知幻即离，不作方便；离幻即觉，亦无渐次。"——圆觉经

## 一句话定义

《圆觉经》是大乘佛教最具**修行路线图精度**的经典之一——以**十二位圆觉菩萨**依次向佛陀提问为结构，系统阐述**如来藏**（一切众生本具圆满觉性）与**三观**（奢摩他/三摩钵提/禅那）的修行方法，提出"**知幻即离，不作方便；离幻即觉，亦无渐次**"的顿悟思想，是中国禅宗与天台宗共同推崇的修行指南。

## 历史脉络

- **约 7–8 世纪**：由**佛陀多罗**（Buddhatrāta）在洛阳白马寺译出。
- **真伪争议**：部分学者认为是中国撰述（伪经），但其思想体系与如来藏系经典完全一致。
- **唐以降**：宗密（华严五祖）作《圆觉经大疏》，成为最重要的注释。
- **宋明以降**：与《楞严经》《起信论》并称"楞严、圆觉、起信"三大家。
- **现代**：南怀瑾《圆觉经略说》使此经重新流行。

## 核心要义

### 1. 圆觉（Perfect Enlightenment）

"圆觉"即**圆满觉性**——一切众生本具的如来藏，不增不减、不垢不净。修行不是"获得"圆觉，而是**发现**自己本来就是圆觉的。

### 2. 十二圆觉菩萨

十二位菩萨依次向佛陀提问，每对问答构成一个修行主题：

| 菩萨 | 提问焦点 | 佛陀核心回答 |
|------|---------|-------------|
| **文殊** | 如何认识本觉？ | 知幻即离，离幻即觉 |
| **普贤** | 既知是幻，如何修行？ | 以幻修幻，幻尽觉满 |
| **普眼** | 如何观照？ | 先静后观，由定发慧 |
| **金刚藏** | 若众生本佛，为何有迷？ | 如矿中金，金在矿中 |
| **弥勒** | 如何断轮回根本？ | 断除贪爱，即断轮回 |
| **清净慧** | 修行有几等？ | 凡夫、菩萨、如来三层次 |
| **威德自在** | 修行有几门？ | 奢摩他、三摩钵提、禅那 |
| **辩音** | 三观如何组合？ | 二十五种清净定轮 |
| **净诸业障** | 修行障碍何在？ | 我相、人相、众生相、寿者相 |
| **普觉** | 如何择师？ | 四病（作、任、止、灭） |
| **圆觉** | 末法时代如何修行？ | 安居、忏悔、发愿 |
| **贤善首** | 此经功德如何？ | 经名、护持、流通 |

### 3. 三观（三种修行方法）

| 观法 | 名称 | 核心方法 | 类比 |
|------|------|---------|------|
| **奢摩他** Śamatha | 止/寂静 | 先静后观，以静力照见诸幻 | 如镜照物 |
| **三摩钵提** Samāpatti | 观/等持 | 先观后静，以幻修幻 | 如苗渐长 |
| **禅那** Dhyāna | 止观双运 | 不取幻化及诸静相 | 如钟发声 |

三观可以单独修、交替修、同时修——组合出**二十五种清净定轮**。

### 4. 四病（修行陷阱）

| 病 | 表现 | 佛陀告诫 |
|---|------|---------|
| **作病** | 努力造作，以为修行能"做出"觉悟 | 圆觉非造作可得 |
| **任病** | 放任自然，以为不需要修行 | 圆觉非放任可证 |
| **止病** | 强止一切念头，以为静就是觉 | 圆觉非止念可入 |
| **灭病** | 追求断灭，以为什么都没有就是空 | 圆觉非断灭可证 |

### 5. 知幻即离

圆觉经最精炼的一句话——"**知幻即离，不作方便；离幻即觉，亦无渐次**"：

- **知幻**：认识到一切身心世界都是幻化
- **即离**：不需要另做功夫去"离开"幻化
- **即觉**：离开幻化的那个，就是觉性本身
- **亦无渐次**：没有阶梯，是顿悟

## 通俗 vs 学术

| 通俗化版本 | 学术原意 |
|---|---|
| "知幻即离 = 知道是梦就醒了" | 是对认知本体的直接指认，非简单的认知转变 |
| "圆觉经是伪经" | 真伪有争议，但思想体系完整且影响深远 |
| "三观 = 三种冥想" | 是三种认识论立场，非单纯技术 |
| "四病 = 四种错误" | 是修行过程最微细的执着模式 |

## 应用场景

- **禅修指导**：三观 = 三种冥想路线的选择
- **心理治疗**：四病 = 修行过程中的心理陷阱识别
- **教育学**：十二菩萨的问答结构 = 循序渐进的教学法
- **哲学**：知幻即离 = 现象学还原的东方版本
- **管理学**：四病 = 领导者"过度干预/放任/停滞/放弃"的四种陷阱

## 常见误读

- ❌ "知幻即离 = 知道是梦就醒了" → 是对认知本体的直接指认，需要实证，非概念理解。
- ❌ "圆觉经是伪经，不值得读" → 真伪有争议，但思想深度不可否认。
- ❌ "三观 = 三种冥想技术" → 是三种认识论立场与修行态度。
- ❌ "四病 = 只有初学者才有" → 四病贯穿修行全程，越深越微细。
