# 中论 Mūlamadhyamakakārikā

## 一句话定义

《中论》是**龙树**（Nāgārjuna，约 2–3 世纪）所造的**中观学派根本论典**——全书 27 品，以**八不中道**开篇，以**四句破**为核心逻辑工具，系统论证了一切法自性空（niḥsvabhāva），奠定大乘佛教**空性哲学**的基础，是汉传三论宗与藏传中观学的共同底本。

## 历史脉络

- **约 150–250**：龙树造《中论》（449 颂，27 品），同时造《六十颂如理论》《七十空性论》等
- **3–4 世纪**：青目（Piṅgala）作注
- **401**：**鸠摩罗什**汉译《中论》（含青目释）4 卷
- **5–6 世纪**：佛护（Buddhapālita）以应成派（prāsaṅgika）方法注释
- **7 世纪**：月称（Candrakīrti）造《明句论》（Prasannapadā），确立应成中观正统
- **8 世纪**：寂护（Śāntarakṣita）融合中观与唯识，开瑜伽行中观派

## 核心要义

### 1. 八不中道

开篇偈："不生亦不灭，不常亦不断，不一亦不异，不来亦不出。"

| 对 | 所破 |
|---|------|
| 不生不灭 | 破本体论的有无二见 |
| 不常不断 | 破时间论的常断二见 |
| 不一不异 | 破同一性/差异性二见 |
| 不来不出 | 破运动/因果的实有见 |

> 八不是四对极端（四句）的全面否定，不是主张某种"中间立场"——**中道 = 离一切边见**。

### 2. 四句破（Catuṣkoṭi）

《中论》的核心逻辑工具，穷尽一命题的所有可能：

| 句 | 形式 | 举例（因果） |
|---|------|-------------|
| 有（是） | A | 因中有果 |
| 无（非） | ¬A | 因中无果 |
| 亦有亦无 | A ∧ ¬A | 因中亦有亦无果 |
| 非有非无 | ¬A ∧ ¬¬A | 因中非有非无果 |

四句全部遮破，不立自宗——**破而不立**（prasaṅga）。

### 3. 空性论证核心

| 品 | 核心论证 |
|---|---------|
| 观因缘品（第1品） | 诸法不自生/不他生/不共生/不无因生 → 无生 |
| 观去来品（第2品） | 运动分析：已去无去/未去无去/去时无去 |
| 观六情品（第3品） | 六根不能自见/不能见他 → 认识论空 |
| 观五阴品（第4品） | 五蕴无自性 |
| 观法品（第18品） | 业烦恼灭故解脱，业烦恼从分别生 |
| 观四谛品（第24品） | 空不坏缘起——"以有空义故，一切法得成" |

### 4. 二谛论

"诸佛依二谛为众生说法：一以世俗谛，二第一义谛。"（24:8）

- **世俗谛**（saṃvṛti-satya）：世间约定俗成的真理，因果/伦理/修道依之成立
- **胜义谛**（paramārtha-satya）：诸法实相——空性
- "若不依俗谛，不得第一义"——二谛不可偏废

### 5. 汉传 vs 藏传中观

| 维度 | 汉传（三论宗） | 藏传（格鲁派） |
|------|--------------|--------------|
| 底本 | 鸠摩罗什译《中论》（含青目释） | 月称《入中论》为主 |
| 代表 | 吉藏 | 宗喀巴 |
| 侧重 | "破邪显正"，无所得 | 应成派细破所破事 |
| 判教 | 三论并列为通教 | 中观为显教最究竟 |

## 常见误读

- ❌ "空 = 什么都没有" → 《中论》第 24 品明确说"以有空义故，一切法得成"，空不破坏缘起有。
- ❌ "四句破 = 怀疑论" → 四句破是逻辑遮遣，不是否认一切认识的可能，而是去除对自性的执着。
- ❌ "中观否定因果" → 恰恰相反，"以有空义故，一切法得成"——正因为无自性，因果才可能。
