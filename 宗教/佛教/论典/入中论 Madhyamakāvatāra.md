# 入中论 Madhyamakāvatāra

## 一句话定义

《入中论》是**月称**（Candrakīrti，约 600–650）所造的**应成中观派代表作**——以菩萨十地（daśabhūmi）为骨架，以六度波罗蜜为主线，在每一地中破斥唯识学派与自续中观派的核心主张，确立"名言中亦无自性"的应成见，是藏传佛教格鲁派中观学的根本论典。

## 历史脉络

- **约 600–650**：月称造《入中论》（Madhyamakāvatāra），含自释（bhāṣya）
- **8 世纪**：传入西藏，成为中观教学核心文本
- **14 世纪**：宗喀巴以《入中论善显密意疏》重新诠释，确立为格鲁派中观正统教材
- **至今**：为藏传佛教各大寺院中观课程的必修论典

## 核心要义

### 1. 十地结构

全书以菩萨十地为骨架，每一地对应一度波罗蜜：

| 地 | 度 | 月称的核心论述 |
|----|-----|---------------|
| 初地（极喜地） | 布施 | 证悟空性，生大欢喜 |
| 二地（离垢地） | 持戒 | 破唯识"自证分"说 |
| 三地（发光地） | 忍辱 | 详破唯识"三性说" |
| 四地（焰慧地） | 精进 | 破"我"的实有 |
| 五地（极难胜地） | 禅定 | 止观双运 |
| **六地（现前地）** | **智慧** | **全书核心：详破唯识八识说、自续派自性有，确立应成空见** |
| 七–十地 | 方便/愿/力/智 | 佛果功德渐显 |

### 2. 第六地核心论证（智慧度）

- **破唯识**：否定阿赖耶识实有、否定自证分、否定三性中的依他起有自性
- **破自续派**：清辨等主张"名言中有自性"，月称认为名言中亦全无自性
- **应成法**（prasaṅga）：不立自宗，只指出对方主张的内在矛盾
- **所破事精确化**：明确区分"名言有"（世俗中可安立）与"自性有"（应破）

### 3. 二谛观

- **世俗谛**：名言中一切法可以安立（因果、伦理、修道皆成立）
- **胜义谛**：以胜义观察时，一切法皆无自性
- **关键**：不坏世俗而说空性——"若不依俗谛，不得第一义"

## 重要概念

- [应成法](../concepts/foundational/emptiness.md) [发展] — 月称将佛护的应成方法系统化为完整的中观体系
- [唯识](../concepts/foundational/vijnaptimatrata.md) [批判] — 入中论对唯识八识说与三性说做了最详尽的破斥
- [二谛](../concepts/foundational/two-truths.md) [继承] — 继承龙树二谛说并精确化"名言有"与"自性有"的区分

## 与其他论典的关系

- [中论](mulamadhyamakakarika.md) [继承] — 入中论是《中论》思想的系统性发展与教学化重构
- [入菩萨行论](bodhicaryavatara.md) [互补] — 入中论侧重哲学论证，入菩萨行论侧重修行实践
- [大智度论](mahaprajnaparamita-sastra.md) [平行] — 二者皆以般若思想为核心，但入中论更具论辩性

## 常见误读

- **误读**：应成中观否定一切因果伦理，落入虚无主义
- **正解**：月称明确区分"自性有"（应破）与"名言有"（不坏），因果在名言中完全成立，所破仅是"自性"的附加执著
