# 成唯识论 Chéng Wéishí Lùn

## 一句话定义

《成唯识论》是 7 世纪由**玄奘**编译、**窥基**作疏的唯识学集大成巨著——它以世亲《唯识三十颂》为底本，**合糅**印度十大论师的注释，建立了**八识、三性、转识成智**的完整唯识哲学体系，是法相唯识宗的根本论典。

## 历史脉络

- **4–5 世纪**：世亲造《唯识三十颂》（Triṃśikā）
- **5–6 世纪**：十大论师各自注释——护法（Dharmapāla）、安慧（Sthiramati）等
- **648–651**：玄奘在长安**合糅**十师注释译为 10 卷，以护法义为主
- **652–682**：窥基作《成唯识论述记》（20 卷），系统阐释
- **7 世纪**：法相唯识宗以此论为根本成立

## 核心要义

### 1. 八识体系

| 识 | 功能 | 转智 |
|---|---|---|
| 眼耳鼻舌身（前五识） | 感性知觉 | 成所作智 |
| 第六意识 | 概念思维 | 妙观察智 |
| 第七末那识 | 自我执着 | 平等性智 |
| 第八阿赖耶识 | 种子储藏 | 大圆镜智 |

### 2. 三性

| 性 | 含义 |
|---|---|
| 遍计所执性 | 概念虚妄分别（如绳上之蛇） |
| 依他起性 | 因缘和合的现象（如绳本身） |
| 圆成实性 | 真实本性（如麻——绳的本质） |

### 3. 五位修道

| 位 | 内容 |
|---|---|
| 资粮位 | 积累福慧资粮 |
| 加行位 | 暖/顶/忍/世第一法 |
| 通达位（见道） | 现证真如 |
| 修习位 | 十地菩萨修行 |
| 究竟位 | 佛果 |

### 4. "合糅"方法论

玄奘的翻译策略：不是逐一翻译十位论师的注释，而是**选择性地合糅**——以护法（Dharmapāla）的注释为主，间采其他论师，形成统一论述。这一方法本身就是创造性诠释。

## 常见误读

- ❌ "成唯识论 = 玄奘原创" → 是合糅翻译，主要反映护法系唯识，非玄奘自创。
- ❌ "唯识 = 唯心主义" → 唯识不是否认外境存在，而是说"认识不离识"。
