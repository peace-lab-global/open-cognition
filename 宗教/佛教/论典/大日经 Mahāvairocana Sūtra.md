# 大日经 Mahāvairocana Sūtra

## 一句话定义

《大日经》是**密教根本经典**之一——以**大日如来**（Mahāvairocana）为法身佛，宣说**胎藏界曼荼罗**的构建与**三密相应**（身密/口密/意密）的修行法门，主张"即身成佛"，是汉传密宗（唐密）与日本真言宗（东密）的根本经典。

## 历史脉络

- **约 7 世纪中叶**：《大日经》成书于印度（梵文本 Mahāvairocanābhisaṃbodhivikurvitādhiṣṭhānavaipulyasūtra）
- **716**：善无畏（Śubhakarasiṃha）携梵本入唐
- **724–725**：善无畏与一行合译《大毗卢遮那成佛神变加持经》7 卷（前 6 卷为经，第 7 卷为供养法）
- **8 世纪**：一行撰《大日经疏》20 卷，成为最权威注释
- **804**：空海入唐学习，携回日本，创立真言宗（东密）
- **至今**：为日本真言宗与台密的根本经典之一

## 核心要义

### 1. 大日如来与法身说法

- **大日如来**（Mahāvairocana）即毗卢遮那佛，为密教法身佛——不同于显教的应身释迦牟尼
- **法身说法**：密教主张法身佛亦能说法（不同于显教认为法身无言说），《大日经》即法身大日如来的直接宣说
- **心即是佛**：经文核心命题——"菩提心为因，大悲为根本，方便为究竟"

### 2. 胎藏界曼荼罗

| 层次 | 内容 |
|------|------|
| 中台八叶院 | 大日如来居中，四佛四菩萨围绕 |
| 遍知院 | 上方，代表佛的智慧 |
| 观音院 | 西方，代表大悲 |
| 金刚手院 | 东方，代表力量 |
| 释迦院 | 下方，代表应化身 |
| 文殊院/除盖障院/地藏院/虚空藏院/苏悉地院 | 围绕中台的各功德院 |

- **曼荼罗**（maṇḍala）：象征大日如来法身遍满一切处的宇宙图景
- **胎藏界**（garbhadhātu）：一切众生本具的佛性如胎儿藏于母胎

### 3. 三密相应

| 密 | 修行方式 | 对应 |
|----|---------|------|
| 身密 | 结手印（mudrā） | 佛的身业 |
| 口密 | 持真言（mantra） | 佛的语业 |
| 意密 | 观想本尊与曼荼罗 | 佛的意业 |

- **三密相应**：修行者的身口意三业与佛的身口意三密相应合一
- **即身成佛**：不须经三大阿僧祇劫，于此身即证佛果——这是密教的核心主张
- **加持**（adhiṣṭhāna）：三密相应时获得佛力加持

### 4. 与金刚界的关系

- **胎藏界**（大日经）：侧重理（本具佛性、大悲）
- **金刚界**（金刚顶经）：侧重智（智慧破障、证悟）
- **两界合一**：真言宗以两界曼荼罗为完整体系，合称"金胎两界"

## 重要概念

- [三身](../concepts/foundational/trikaya.md) [发展] — 大日经将法身佛推至说法者地位，发展了三身理论
- [菩提心](../concepts/foundational/bodhicitta.md) [密教化] — "菩提心为因"是大日经的修行起点
- [如来藏](../concepts/foundational/tathagatagarbha.md) [密教化] — 胎藏界即如来藏思想的密教表达
- [方便](../concepts/foundational/upaya.md) [极致] — "方便为究竟"将善巧方便提升至最高原则

## 与其他论典的关系

- [大日经](mahavairocana.md) — 本条目
- [华严经](avatamsaka.md) [平行] — 大日如来与华严毗卢遮那佛同源，但密教赋予不同的修行体系
- [楞严经](surangama.md) [平行] — 二者皆涉及"即身"成佛的理念，但路径不同

## 常见误读

- **误读**：密教就是迷信咒术，缺乏哲学基础
- **正解**：密教有完整的哲学体系（如来藏思想/法身说法论/三密相应理论），仪轨与真言是其哲学思想的外在表达形式，一行《大日经疏》包含深邃的哲学论述
