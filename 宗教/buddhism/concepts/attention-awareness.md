# 注意与觉知 / Attention and Awareness

## 一句话定义

注意与觉知（sati-sampajañña，念-正知）是佛教禅修的核心操作——"念"把心锚定于所选对象，"正知"对当下身心状态保持清明觉察；二者合一构成从散乱到专注、从专注到洞察的完整注意机制。

## 历史脉络

```
早期佛教：四念处（身、受、心、法）——念的四个所缘
    ↓
《念处经》：satipaṭṭhāna 系统化为21天证果的直道
    ↓
上座部内观：念+正知=对身心现象的如实观察
    ↓
当代：与认知科学的注意网络理论对话
```

## 核心要义

### 1. 念（Sati）
- 不是"记住"，而是"忆持"——把心持续带回到当下所缘
- 功能：防止心随烦恼流失，维持觉察的连续性
- 在[八识体系](cognitive-theory/eight-consciousness.md)中对应第六识"意识"的运作

### 2. 正知（Sampajañña）
- 对当下行为的清晰明白：知道自己在做什么、怎么做的
- 四种正知：目的正知、适宜正知、行处正知、无痴正知
- 念提供"注意的稳定"，正知提供"觉察的清明"——缺一不可

### 3. 止观的双轨
- **止**（samatha）：以念专注一境，发展定力——注意的聚焦
- **观**（vipassanā）：以正知如实观察三法印——觉察的穿透
- 止是手段，观是目的；念-正知贯穿始终

### 4. 与现代注意理论的接口
- "念" ≈ 注意网络的**执行控制**（dorsolateral PFC）
- "正知" ≈ **元觉察 / 监测**（anterior insula、ACC）
- 默认模式网络（DMN）的安静——减少自我参照的反刍

## 通俗 vs 学术

| 层次 | 含义 |
|------|------|
| 日常 | "专心""觉察当下"的笼统说法 |
| 禅修 | 有明确技术定义的双重操作：念（锚定）+ 正知（清明） |
| 认知科学 | 可映射到注意网络三元模型（警觉、定向、执行控制）的经验范式 |

## 跨学科关联

- **冥想总论**：注意与觉知是[冥想与默观](../meditation.md)在佛教语境的核心机制
- **八识体系**：念-正知运作于第六识（意识），其深层净化涉及[八识体系](cognitive-theory/eight-consciousness.md)的转识
- **正念心理学**：当代[正念](../../../psychology/concepts/mindfulness.md)疗法（MBSR/MBCT）正是把 sati 操作化为临床训练
- **认知科学**：与[卡尼曼](../../../psychology/schools/behavioral-economics/kahneman.md)的注意资源理论、系统1/2切换形成对照——念是主动调用系统2的持续训练

## 常见误读

1. ❌ "念就是集中注意力" → ✅ 念是"忆持所缘"，含稳定与回归两层；集中只是其一
2. ❌ "正知=自我观察" → ✅ 正知是清明觉察而非内省分析，不卷入概念评判
3. ❌ "止和观是两种不同的冥想" → ✅ 止观是同一禅修的两面，念-正知同时运作
