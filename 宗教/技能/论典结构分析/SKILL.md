# 论典结构分析 · Śāstra Structure Analysis

## 一句话功能

分析佛教论典（śāstra）的论证结构——识别宗（命题）、因（理由）、喻（例证）、合（综合）、结（结论），以及论主如何与对手（外人/异师）展开辩证，从而穿透论典的层层论证直达核心主张。

## 何时使用

- 阅读佛教论典时感到论证繁复，需要理清结构
- 分析论主的论证策略（正面建立 vs 破邪显正 vs 问答辩证）
- 比较不同论师对同一问题的论证方式
- 佛学院因明/论典课程的结构性阅读训练
- 写学术论文时需要提炼论典中的论证逻辑

## 何时不使用

- 纯粹阅读经（sūtra）——经的文体是宣说而非论证，需要圣典诠释学而非结构分析
- 禅宗语录——禅宗刻意打破论证结构，用此框架会扭曲其旨趣
- 密教仪轨文本——仪轨是操作性的，非论证性的
- 用户对论典内容完全陌生（应先读概要再分析结构）

## 理论基础

- **因明五支**：宗（pratijñā）、因（hetu）、喻（dṛṣṭānta）、合（upanaya）、结（nigamana）——印度逻辑论证的标准结构 → [五明](../../buddhism/concepts/foundational/pancavidya.md)
- **破邪显正**：三论宗吉藏的方法论——不立自宗，只破他执 → [三论宗](../../buddhism/schools/sanlun.md)
- **经-论-疏**三层：经是佛说，论是菩萨/论师释经，疏是后人释论
- **四依四不依**：论典阅读的元方法论 → [四依](../../buddhism/concepts/foundational/four-reliances.md)
- **世亲的论证风格**：《俱舍论》的"以释破颂"——在注释中暗暗批判被注释文本 → [俱舍论](../../buddhism/treatises/abhidharmakosa.md)

## 操作流程

### Step 1：识别论典类型

首先判断这部论属于哪种类型：

| 类型 | 特征 | 示例 |
|---|---|---|
| **释经论** | 逐句解释某部经 | 《大智度论》（释般若经）、《成唯识论》（释唯识三十颂） |
| **宗义论** | 建立某宗派的核心主张 | 《中论》、《百论》、《大乘起信论》 |
| **阿毗达磨论** | 系统分析法相 | 《俱舍论》、《大毗婆沙论》 |
| **因明论** | 专论逻辑与认识论 | 《因明正理门论》、《集量论》 |
| **问答论** | 以问答形式展开 | 《那先比丘经》、《清净道论》 |

**提问范式**：
- "这部论是在释经、立宗、还是破他？"
- "它的对话对象（论敌）是谁？"

### Step 2：绘制宏观结构

在深入论证之前，先把握全论的大框架：

1. **序分/归敬颂**：论主顶礼的对象（暗示本论归属的传承）
2. **论题/发端**：为什么造此论？（造论因缘）
3. **正论/论体**：主体论证（通常分品/章）
4. **结颂/回向**：论末总结与功德回向

**提问范式**：
- "全论分几品/几章？各品之间的逻辑递进关系是什么？"
- "论主在序分中顶礼谁？这暗示了什么宗派归属？"

### Step 3：识别论证单元（宗因喻合结）

对正论中的每个论证段落，识别五支结构：

| 支 | 功能 | 识别标记 |
|---|---|---|
| **宗**（pratijñā） | 主张/论题 | "XX是XX"、"应知…"、"今当说…" |
| **因**（hetu） | 理由 | "何以故？"、"以…故"、"因为…" |
| **喻**（dṛṣṭānta） | 例证/类比 | "譬如…"、"如…亦如是" |
| **合**（upanaya） | 将喻与宗结合 | "如是…"、"此亦如是" |
| **结**（nigamana） | 重申结论 | "是故…"、"故知…" |

**注意**：很多论典省略合与结，只保留宗因喻三支。

**提问范式**：
- "这段论证的核心主张（宗）是什么？"
- "用什么理由（因）支撑？用了什么类比（喻）？"

### Step 4：识别辩证结构（论主 vs 外人）

佛教论典常以**虚拟对话**展开——"外人"（对手/质疑者）提出问题，论主回应：

- **外人问**："若如汝说，则应有X过"——对手用归谬法挑战
- **论主答**："不然，以Y故"——论主化解归谬
- **引经据论**：引用佛经或前代论师支撑论点
- **自设问**：论主自己提出问题再回答（ pedagogical 策略）

**提问范式**：
- "哪些段落是'外人'的观点？哪些是论主的回应？"
- "论主是用正面论证还是用反证/归谬来回应？"

### Step 5：分析论证策略

不同论典有不同的论证策略：

| 策略 | 特征 | 代表 |
|---|---|---|
| **正面建立** | 直接陈述自己的体系 | 《俱舍论》的五位七十五法 |
| **破邪显正** | 不立自宗，只破他执 | 《中论》《百论》 |
| **以释破颂** | 表面注释，暗中批判 | 《俱舍论》世亲对迦湿弥罗有部的批判 |
| **问答递进** | 通过层层问答深入 | 《大乘起信论》 |
| **科判分析** | 把文本分成层层嵌套的小节 | 窥基《成唯识论述记》 |

**提问范式**：
- "论主是在建立自己的体系，还是在破斥他人？"
- "如果是'释经论'，论主对被释文本是忠实注释还是有创造性诠释？"

### Step 6：标记关键论争点

在结构分析中标记：

- **核心论争**：不同学派在此处分歧最大
- **隐含批判**：论主没有点名但在暗中批判谁
- **创新点**：论主在何处超越了前人
- **弱点/争议**：后世论师对论主此处的批评

**提问范式**：
- "后世哪些论师在此处提出了不同看法？"
- "论主的这个论证有没有逻辑漏洞？"

### Step 7：输出结构分析报告

```
【论典结构分析报告】
├── 论典：《___》
├── 类型：释经论 / 宗义论 / 阿毗达磨论 / 因明论 / 问答论
├── 宏观结构：
│   ├── 序分：___
│   ├── 正论分___品：品1=___, 品2=___, ...
│   └── 结颂：___
├── 核心论证（选段）：
│   ├── 宗：___
│   ├── 因：___
│   ├── 喻：___
│   └── 结：___
├── 论证策略：___
├── 辩证结构：外人主张___ → 论主回应___
├── 关键论争点：___
└── 后世评价：___
```

## 完整示例

**输入场景**：
> 用户："《中论》第一品第一颂'不生亦不灭，不常亦不断，不一亦不异，不来亦不出'的论证结构是什么？"

**Skill 应用过程**：
1. 类型：宗义论（破邪显正型）
2. 宏观结构：全论 27 品，第 1 品"观因缘品"破四缘实有
3. 论证单元分析：
   - 宗：诸法不从自生、不从他生、不从共生、不无因生（八不 = 四不生）
   - 因：自生则无穷过、他生则一切生一切、共生兼二过、无因生则善恶无别
   - 喻：如种芽关系——种不自生芽（种子灭故）、不从他生（种芽同类故）
4. 辩证结构：外人主张"万物有生"→ 龙树以四不生穷尽一切可能的"生"，一一归谬
5. 论证策略：**破邪显正**——龙树不立"无生"为正面主张，而是通过穷尽一切"生"的可能性并逐一破斥，让读者自悟"无生"
6. 关键论争：此处的"无生"是**本体论的无生**（一切法无自性故无生）还是**认识论的遮遣**（我们无法认知"生"）——后世中观应成派 vs 自续派的核心分歧

**输出**：
> 《中论》"八不"采用**穷举归谬**策略：将"生"穷尽为四种可能（自生/他生/共生/无因生），逐一证明每种都有不可接受的过失。这不是在正面建立"无生"的学说，而是通过让一切"生"的理论自行崩溃，使读者在概念的穷尽处直接体悟缘起性空。

## 反例（误用）

**误用场景**：
> 用户试图用"宗因喻合结"五支分析《六祖坛经》中慧能的"不是风动，不是幡动，仁者心动"。

**问题**：慧能此语是禅宗的**直指**而非论证——它刻意打破论证结构，用悖论式的回答截断思维。用因明五支去分析会错失其禅机。

**正确做法**：对禅宗语录、密教仪轨、诗歌偈颂等非论证性文本，应使用文学分析或禅宗诠释方法，而非论典结构分析。
