# 公案参究法 · Koan Practice Method

## 一句话功能

将禅宗公案参究**操作化**——从阅读公案文本，到提取话头（hwadu）、起疑情、参究破关，提供五步结构化方法，使"不立文字"的禅宗修持获得可追踪的认知-体验框架。

## 何时使用

- 研读禅宗公案时，不满足于"看懂"文本，想进入参究实践
- 佛学院禅宗课程需要理解看话禅（kanhwa sŏn）的操作方法
- 比较禅宗与逻辑/分析方法的认知差异（公案 vs 因明）
- 跨学科研究：公案如何打破概念思维（与认知科学/心理治疗对话）
- 指导禅修者进行看话禅练习

## 何时不使用

- 学术研究公案的文学/历史面向 → 用文本分析或历史批评方法
- 默照禅（silent illumination）修习 → 默照不依话头，需要不同的指导
- 用户完全不了解禅宗基本背景（应先了解禅宗宗义）
- 追求"正确答案" → 公案没有标准答案，此 Skill 是参究方法而非解题工具

## 理论基础

- **看话禅**（kanhwa sŏn）：大慧宗杲（1089–1163）提倡——不参整个公案，只抓住其中的**话头**（核心词/句）反复参究 → [禅宗](../../buddhism/schools/chan.md)
- **疑情**（great doubt / 大疑）：参究的关键引擎——不是怀疑论的"不信"，而是全身心投入的"不明白"
- **公案的本质**：不是谜语或哲学命题，而是**截断概念思维的装置**——迫使心从"思量"跳入"不思量"
- **禅宗"不立文字"**：不立文字不是否定文字，而是指出文字指向的体验在文字之外 → [菩提达摩](../../buddhism/masters/bodhidharma.md)
- **壁观**：达摩的"外息诸缘，内心无喘"——参究的定力基础 → [止观](../../buddhism/concepts/foundational/zhi-guan.md)

## 操作流程

### Step 1：选择与阅读公案

从适合参究的公案中选择（非所有公案都适合参话头）：

| 类型 | 适合参究 | 示例 |
|------|---------|------|
| **本参公案** | 最常用 | "狗子无佛性""万法归一，一归何处""父母未生前的本来面目" |
| **机锋对话** | 可提取话头 | "庭前柏树子""吃茶去""干屎橛" |
| **纯叙事公案** | 不适合 | 传记性公案（如达摩见梁武帝） |

**阅读原则**：
- 读公案**不是为了理解意思**，而是为了找到让你"不明白"的那个点
- 快速阅读全案，标记哪个词/句让你"卡住"——那就是话头

**提问范式**：
- "这个公案中，哪个词/句让我感到最深的'不明白'？"
- "如果必须用一个词/一句话来参，我会选哪个？"

### Step 2：提取话头（Hwadu）

从公案中提炼**话头**——参究的核心焦点：

| 公案 | 话头 |
|------|------|
| 赵州"狗子无佛性" | "无"（Wú / Mu） |
| "万法归一，一归何处" | "一归何处" |
| "父母未生前的本来面目" | "本来面目" |
| 赵州"庭前柏树子" | "柏树子" |

**话头的特征**：
- 不是一个问题（不需要回答）
- 不是概念（不引导思考）
- 是一个**让思维"卡住"的装置**——心在话头上反复碰壁

**操作**：
- 将话头缩短为 1–4 个字
- 不要试图解释它
- 让它成为心中的一个"疑团"

### Step 3：起疑情（Great Doubt）

疑情是参究的引擎——没有疑情，参话头只是念咒：

| 层次 | 状态 | 特征 |
|------|------|------|
| **浅层** | 提起话头 | 心中有话头的存在，但容易被杂念打断 |
| **中层** | 疑情成片 | 话头持续不断，杂念自然减少 |
| **深层** | 大疑 | 全身心的"不明白"——行住坐卧只有一个疑团 |

**操作要领**：
- **提撕**：反复提起话头，不让心散落到其他对象
- **不要用力**：疑情不是咬牙切齿，而是自然地"不明白"
- **不要追求答案**：一旦想"答案"，疑情就散了

**提问范式**：
- "我的疑情是真实的'不明白'，还是在假装参究？"
- "有没有落入'等待答案'的陷阱？"

### Step 4：参究持续（Maintaining Inquiry）

将参究贯穿日常：

| 时段 | 方法 |
|------|------|
| **坐禅时** | 安住于话头，不起第二念 |
| **行住坐卧** | 话头在心中持续（如猫捕鼠） |
| **遇到困境** | 将困境本身融入疑情——"这个困境是什么？" |

**常见障碍与对治**：

| 障碍 | 表现 | 对治 |
|------|------|------|
| **昏沉** | 话头模糊、想睡 | 睁眼、挺直、出声提话头 |
| **掉举** | 杂念纷飞 | 回到话头，不随杂念走 |
| **理路化** | 试图用逻辑分析话头 | 放下分析，回到"不明白" |
| **等待开悟** | 期待某种特殊体验 | 放下期待，只参话头 |

> 大慧宗杲："不得将心等悟"——等待开悟本身就是最大的障碍。

### Step 5：破参（Breakthrough）与印证

破参不是"得到答案"，而是**疑团的彻底崩解**：

| 特征 | 说明 |
|------|------|
| **非概念性** | 不是"想通了"，而是概念思维本身的穷尽 |
| **非体验性** | 不是"看到光/听到声音"，而是认知结构的根本转变 |
| **不可说** | "如人饮水，冷暖自知"——无法用语言传达 |

**破参之后**：
- 需要**善知识**（禅师）印证——防止"认贼作父"（把某种体验当成开悟）
- 破本参后继续参究（透重关、牢关）——禅宗的"保任"

**提问范式**：
- "我是否真的'破'了，还是只是获得了某种宁静的体验？"
- "需要禅师印证的环节不能跳过"

## 完整示例

**输入场景**：
> 用户："我想参究'万法归一，一归何处'这个公案。"

**Skill 应用过程**：

1. **选择与阅读**：此公案适合参究——"万法归一"可理解，但"一归何处"让人"卡住"
2. **提取话头**："一归何处"——不是问"一"去了哪里，而是让"归何处"成为心中的一个疑团
3. **起疑情**：
   - 反复提起"一归何处"
   - 不试图回答（不是归于空、不是归于心、不是无归处）
   - 让"不明白"本身充满整个意识
4. **持续参究**：
   - 坐禅时安住于"一归何处"
   - 日常生活中保持疑情（如猫捕鼠——全身心的注意）
   - 遇到"逻辑上想回答"的冲动时，回到"不明白"
5. **破参**：当疑情到达极致时——不是"得到答案"，而是"问本身"崩解了。需要禅师印证

**输出**：
> "一归何处"的话头设计精妙——"万法归一"是理性可以理解的（一切现象归于统一的本源），但"一归何处"将理性推向其极限：如果一切归一，那个"一"又归于何处？这不是一个需要回答的问题，而是一个**让问题本身穷尽**的装置。参究此话头的核心是保持"不明白"而不跳向任何答案。

## 反例（误用）

**误用场景**：
> 用户："帮我分析'狗子无佛性'公案的哲学含义——赵州为什么说'无'？"

**问题**：这是在**分析**公案，不是在**参究**公案。公案参究法要求你不去理解赵州为什么说"无"，而是让"无"成为你心中的一个疑团。分析公案的哲学含义属于学术研究，不是禅修。

**正确做法**：如果想理解公案的哲学/历史背景，用学术方法；如果想参究，放下分析，提起话头。
