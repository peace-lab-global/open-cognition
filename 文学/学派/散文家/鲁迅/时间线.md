# 鲁迅 Lu Xun 生平年表 / Biography Timeline

## 历史背景
晚清至民国，中国社会剧烈转型期。西方思潮涌入，传统文化面临挑战。

| 年份 | 事件 |
|------|------|
| 1881 | 出生于浙江绍兴，原名周樟寿，后改名周树人 |
| 1893 | 祖父因科场案入狱，家道中落 |
| 1898 | 入南京江南水师学堂，后转入矿路学堂 |
| 1902 | 赴日本留学，入仙台医学专门学校 |
| 1906 | "幻灯片事件"后弃医从文，决心以文学改变国民精神 |
| 1909 | 回国，任杭州浙江两级师范学堂教员 |
| 1912 | 任教育部佥事，随部迁北京 |
| 1918 | 发表《狂人日记》，中国现代文学第一篇白话小说，首次使用"鲁迅"笔名 |
| 1921 | 发表《阿Q正传》，批判国民劣根性的经典之作 |
| 1923 | 出版小说集《呐喊》 |
| 1924 | 出版小说集《彷徨》 |
| 1925 | 参与"女师大风潮"，支持学生运动 |
| 1926 | 因"三一八惨案"被通缉，赴厦门大学任教 |
| 1927 | 赴广州中山大学任教；"四一二"后辞职赴上海 |
| 1927 | 出版散文诗集《野草》、散文集《朝花夕拾》 |
| 1928 | 与许广平同居；参与"革命文学"论争 |
| 1930 | 参与发起中国左翼作家联盟（"左联"） |
| 1932 | 与瞿秋白建立深厚友谊，共同推动左翼文学 |
| 1936 | "两个口号"论争；10月19日病逝于上海 |

## 思想发展阶段

1. **启蒙期（1902–1918）**：从医学到文学的转向，确立"立人"思想
2. **创作期（1918–1926）**：小说与杂文并重的黄金时期
3. **战斗期（1927–1936）**：以杂文为主要武器的社会批判时期
