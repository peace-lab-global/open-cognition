# C·赖特·米尔斯 · C. Wright Mills

## 基本信息

- **生卒年**：1916年8月28日—1962年3月20日（年仅45岁）
- **国籍/文化背景**：美国社会学家，德克萨斯州人
- **学术身份**：公共知识分子，社会学想象力的提出者，权力精英理论家
- **学派归属**：古典/批判社会学传统
- **主要职位**：哥伦比亚大学社会学教授

## 核心命题

1. **社会学想象力（Sociological Imagination）**：将个人经验与更广泛的社会历史结构联系起来的能力——"个人困扰"（personal troubles）与"公共议题"（public issues）之间的联系，是社会学想象力的核心。
2. **权力精英（Power Elite）**：美国社会由军事、企业、政治三大领域的精英组成的相互交织的权力网络所支配——他们做出影响数百万人的重大决定，却不受民主监督。
3. **白领阶层（White Collar）**：美国中产阶级的"新小人物"——表面上拥有体面生活，实际上是科层体制中的异化工作者，丧失了独立性和创造性。
4. **宏大理论与抽象经验主义的批判**：米尔斯同时批判帕森斯的宏大理论（过于抽象，脱离经验）和拉扎斯菲尔德的抽象经验主义（技术化、去政治化）——主张回到古典社会学传统。

## 思想脉络

### 早期（1940年代）
- 在威斯康星大学和马里兰大学接受学术训练
- 早期研究劳工运动和工会权力
- 1948年《权力新贵》：分析美国劳工领袖的权力和局限
- 与汉斯·格斯合译韦伯著作，深受韦伯影响

### 中期（1950年代）
- 1951年《白领》：美国新中产阶级的社会学分析
- 1953年《性格与社会结构》：社会心理学与宏观社会学的整合
- 1956年《权力精英》：引发巨大争议的美国权力结构分析
- 1959年《社会学想象力》：社会学方法论的纲领性宣言

### 晚期（1960年代初）
- 积极介入公共政治辩论，批评美国的冷战政策和古巴政策
- 1960年《听吧，扬基》：批评美国对拉丁美洲的帝国主义政策
- 1962年因心脏病发作去世，年仅45岁

## 重要概念

### 1. 社会学想象力
- **定义**：在个人生活与更广泛的社会历史之间建立联系的能力
- **核心区分**：个人困扰（troubles）vs 公共议题（issues）
  - 失业是个人困扰，但当一个社会有数百万人失业时，它就是公共议题
- **方法论意义**：社会学想象力是一种心智品质，帮助人们理解自身生活与历史之间的关系

### 2. 权力精英
- **三大领域**：军事（五角大楼）、企业（大公司）、政治（白宫行政机构）
- **精英循环**：三大领域的精英通过人事旋转门相互流动
- **大众社会**：权力精英之下是大众社会——原子化的个人缺乏有效的政治参与渠道
- **与多元主义的对立**：米尔斯批评达尔等多元主义者的乐观假设

### 3. 白领阶层
- **新小人物**：美国中产阶级不再是独立的小企业主，而是科层体制中的雇员
- **异化**：白领工作者对自己的工作缺乏控制权和认同感
- **消费主义**：白领通过消费来建构身份——"快乐机器人"的形象

### 4. 宏大理论与抽象经验主义
- **宏大理论（Grand Theory）**：帕森斯的社会系统理论——概念过度抽象，脱离经验现实
- **抽象经验主义（Abstracted Empiricism）**：拉扎斯菲尔德式的量化研究——技术化、去政治化、服务于既得利益
- **古典传统的回归**：米尔斯主张回到马克思、韦伯、涂尔干的古典传统

## 思想坐标

```
马克思阶级分析 ←—— 米尔斯 ——→ 韦伯科层制理论
（权力/异化）    （权力精英/白领）   （理性化/理性化批判）
         ↕                    ↕
    帕森斯功能主义          拉扎斯菲尔德经验主义
   （宏大理论批判）       （抽象经验主义批判）
```

- **发展**：马克思——权力精英理论继承了马克思对资本主义权力结构的分析
- **发展**：韦伯——白领分析继承了韦伯对科层制理性化的批判
- **平行**：阿伦特——对权力与公共领域的分析有相似关切
- **平行**：米尔斯——米尔斯本人同时批判功能主义和经验主义

## 当代应用

1. **媒体研究**：米尔斯对大众媒体塑造"快乐机器人"的分析，预示了后来的媒体批评理论
2. **不平等研究**：权力精英理论为理解当代的收入不平等、企业游说、政治旋转门提供了分析框架
3. **公共社会学**：米尔斯是"公共知识分子"的典范——社会学应介入公共辩论
4. **科层制研究**：白领分析对理解当代平台经济中的"数字白领"仍有启示
5. **方法论辩论**：宏大理论与抽象经验主义的批判，至今仍影响社会学方法论讨论

## 常见误读

1. ❌ "米尔斯是阴谋论者" → 权力精英理论不是阴谋论——精英的权力来自结构性位置，而非秘密计划。
2. ❌ "米尔斯反对一切量化研究" → 米尔斯反对的是脱离理论关怀的量化技术主义，而非量化方法本身。
3. ❌ "社会学想象力只是方法论" → 社会学想象力既是方法论，也是政治立场——它要求社会学家介入公共事务。
4. ❌ "米尔斯只关注精英" → 米尔斯同时关注普通人的生活处境——白领阶层、劳工、大众社会。
