# 社会学 · Sociology

本目录覆盖社会学经典与当代主要理论范式，按学派组织思想家，按概念组织跨学派观念。

## 学派树

```
社会学
├── 古典三大家 (classical)
│   ├── 涂尔干 · 社会事实
│   ├── 韦伯 · 理性化
│   └── 马克思 · 异化
├── 结构主义 / 后结构 (structuralism)
│   ├── 布迪厄 · 场域与资本
│   └── 福柯 · 规训与权力
├── 互动论 (interactionism)
│   └── 戈夫曼 · 拟剧论
├── 批判理论 (critical)
│   └── 哈贝马斯 · 交往理性
├── 现代性理论 (modernity)
│   ├── 鲍曼 · 流动现代性
│   └── 贝克 · 风险社会
└── 功能主义 (functionalism)
    └── 帕森斯 · 结构功能
```
