# 皮亚杰 Jean Piaget

> "每次过早地教孩子他自己能发现的东西，就是阻止他去发明它。"

## 基本信息

瑞士心理学家、哲学家，**发生认识论**（Genetic Epistemology）创立者。从生物学起步，毕生研究"知识如何在儿童身上发生"，建立了**认知发展四阶段**理论与建构主义学习观，深刻影响 20 世纪教育学、皮下发展心理学。

## 核心命题

1. **儿童不是小型成人**：每个发展阶段有质上不同的思维结构。
2. **知识是建构的**：通过主体与环境的互动建构，而非被动接受。
3. **同化与顺应的平衡**：同化（Assimilation）= 把新信息纳入旧图式；顺应（Accommodation）= 修改图式以适应新信息；平衡（Equilibration）是认知发展的动力。
4. **发展是阶段性的**：每个阶段有不可逆的质变。

## 思想脉络

- **早期（生物学背景）**：博士论文研究瓦莱州的软体动物，关注"适应"。
- **临床方法（1920s）**：在比奈实验室，注意儿童**错误答案**的系统性，从此转向认知发展。
- **四阶段理论（中期）**：感知运动期 → 前运算期 → 具体运算期 → 形式运算期。
- **晚期（认识论）**：日内瓦学派，将研究扩展到逻辑、数学、物理概念的发生学。

## 重要概念

- **图式 Schema**：行动或思维的可重复结构
- **同化 Assimilation**：用旧图式应对新刺激
- **顺应 Accommodation**：修改图式以适应新刺激
- **平衡 Equilibration**：同化与顺应间的动态调节
- **客体永久性 Object Permanence**：感知运动期末获得（约 8 个月）
- **守恒 Conservation**：具体运算期获得（约 7 岁，水量、数量、长度守恒）
- **去中心化 Decentration**：摆脱单一视角

### 四阶段
| 阶段 | 年龄 | 特征 |
|---|---|---|
| 感知运动期 | 0–2 | 通过感觉与动作认识世界，获得客体永久性 |
| 前运算期 | 2–7 | 符号思维兴起，但有自我中心、不能守恒 |
| 具体运算期 | 7–11 | 能对具体事物进行逻辑运算，掌握守恒 |
| 形式运算期 | 11+ | 能进行抽象、假设演绎推理 |

## 思想坐标

- **建构主义**：知识从主客互动中建构。
- **结构主义**：每阶段有整体的逻辑结构。
- **生物学类比**：认知发展类比生物适应。

## 当代应用

- **教育**：建构主义课程、发现学习、按发展阶段设计教学
- **测评**：守恒任务等任务被广泛沿用
- **AI**：发展机器人学（developmental robotics）
- **认识论**：知识社会学的"建构"概念前驱

## 常见误读

- ❌ "皮亚杰的阶段年龄是固定的。" → 是**顺序**固定，年龄因文化、经验而异。
- ❌ "形式运算期所有人都达到。" → 跨文化研究显示并非普遍。
- ❌ "建构主义 = 不要教。" → 皮亚杰强调结构化环境，不是放任。
- ❌ "皮亚杰只研究儿童。" → 他追问的是"知识本身的发生学"。
