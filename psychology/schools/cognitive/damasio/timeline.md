# 达玛西奥年表 / Timeline of Damásio

## 编年 / Chronology

| 年份 | 事件 |
|------|------|
| 1944 | 2月25日生于葡萄牙里斯本（Lisbon, Portugal） |
| 1960s | 在里斯本大学医学院（Faculty of Medicine, University of Lisbon）学习医学 |
| 1969 | 获里斯本大学医学学位，专攻神经病学 |
| 1970s | 在里斯本圣玛丽亚医院（Hospital de Santa Maria）任神经科住院医师 |
| 1974 | 与汉娜·达玛西奥（Hanna Damásio）合作，开始脑损伤患者的系统研究 |
| 1975 | 赴美国爱荷华大学（University of Iowa）进行访问研究 |
| 1976 | 正式加入爱荷华大学，任神经病学助理教授 |
| 1980s | 积累大量前额叶损伤患者案例，包括"艾略特"（Elliot） |
| 1982 | 任爱荷华大学神经病学系主任 |
| 1988 | 获 M. W. Van Allen 讲席神经病学教授称号 |
| 1990s | 与 Bechara 等人设计爱荷华博弈任务（Iowa Gambling Task） |
| 1994 | 出版《笛卡尔的错误》（Descartes' Error），提出躯体标记假说 |
| 1997 | Bechara, Damásio 等人在 *Science* 发表关键论文，证明躯体标记在意识之前运作 |
| 1999 | 出版《感受发生的一切》（The Feeling of What Happens），探索意识的神经基础 |
| 2001 | 转任南加州大学（University of Southern California） |
| 2003 | 出版《寻找斯宾诺莎》（Looking for Spinoza） |
| 2005 | 获阿斯图里亚斯亲王奖（Prince of Asturias Award for Scientific Research） |
| 2006 | 任南加州大学脑与创造力研究所（Brain and Creativity Institute）所长 |
| 2010 | 出版《自我的建构》（Self Comes to Mind），提出意识三阶段模型 |
| 2010 | 获高尔吉亚国际奖（Gairdner Foundation International Award） |
| 2014 | 《笛卡尔的错误》出版20周年，发表多篇回顾与展望文章 |
| 2018 | 出版《万物的奇异秩序》（The Strange Order of Things），将理论扩展到文化领域 |
| 2021 | 出版《感觉与认知》（Feeling & Knowing），意识理论的最新综合 |

## 同时代人物 / Contemporaries

- **卡尼曼**（Daniel Kahneman, 1934–2024）：行为经济学家，双系统理论创始人。躯体标记与快系统的互补关系
- **勒杜**（Joseph LeDoux, b. 1949）：神经科学家，恐惧条件反射研究先驱
- **弗里斯顿**（Karl Friston, b. 1959）：计算神经科学家，自由能原理创始人
- **潘克塞普**（Jaak Panksepp, 1943–2017）：情感神经科学先驱
- **巴雷特**（Lisa Feldman Barrett, b. 1963）：情绪建构论创始人，与达玛西奥有学术对话
- **德莱福斯**（Hubert Dreyfus, 1929–2017）：哲学家，用梅洛-庞蒂论证具身AI
- **瓦雷拉**（Francisco Varela, 1946–2001）：认知科学家，生成认知创始人

## 思想分期 / Intellectual Periods

| 时期 | 年代 | 核心主题 | 代表著作/事件 |
|------|------|----------|--------------|
| 临床期 | 1970s–1990 | 脑损伤研究、案例积累 | 艾略特案例；爱荷华大学神经科 |
| 突破期 | 1994–1999 | 躯体标记假说；身心二元论批判 | 《笛卡尔的错误》(1994)；《感受发生的一切》(1999) |
| 理论建构期 | 2003–2010 | 情感理论；意识层级模型 | 《寻找斯宾诺莎》(2003)；《自我的建构》(2010) |
| 文化拓展期 | 2018–至今 | 感受与文化创造；理论综合 | 《万物的奇异秩序》(2018)；《感觉与认知》(2021) |

## 学术影响里程碑 / Milestones of Academic Impact

- **1994**: 《笛卡尔的错误》出版后成为国际畅销书，将"躯体标记"引入公共话语
- **1997**: *Science* 论文被引用超过5000次，成为决策神经科学的经典文献
- **2005**: 阿斯图里亚斯亲王奖标志着其工作获得国际学术界最高认可
- **2010s**: 躯体标记假说被广泛引入经济学、人工智能、教育学等领域
- **2018**: 《万物的奇异秩序》将理论从个体神经科学扩展到文化生物学
