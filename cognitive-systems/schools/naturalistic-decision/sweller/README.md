# 约翰·斯威勒 / John Sweller

> **领域**: 教育心理学 / 认知科学 / 教学设计
> **时期**: 当代（1946–）
> **生卒**: 1946年生于南非，主要在澳大利亚从事学术研究
> **国籍**: 澳大利亚（南非出生）

> "工作记忆的容量是有限的——教学设计必须尊重这个限制。"

## 一、思想概要

约翰·斯威勒（John Sweller）是认知负荷理论（Cognitive Load Theory, CLT）的创始人，也是20世纪末至21世纪初最具影响力的教育心理学家之一。他于1988年发表的里程碑论文"Cognitive Load During Problem Solving: Effects on Learning"彻底改变了人们对教学设计的理解——证明了传统的"问题解决"教学法对新手学习者来说恰恰是认知过载的，反而阻碍了学习。

斯威勒的核心洞见在于将工作记忆的有限容量作为教学设计的根本约束条件。他发展了一套完整的理论框架，区分了三种类型的认知负荷：内在负荷（材料本身的复杂性）、外在负荷（不良呈现方式造成的不必要负荷）和关联负荷（用于建构图式的"好的"负荷）。这一框架为教学设计者提供了系统化的分析工具，帮助他们优化学习材料的呈现方式。

斯威勒的理论不仅在学术界产生了深远影响，更直接推动了全球范围内的教学设计改革。从在线教育（MOOC）、医疗培训到多媒体学习设计，CLT的原则被广泛应用于各种学习环境中。近年来，他将CLT与进化认识论相结合，从人类认知进化的角度解释了为什么某些学习方式更符合人类的认知架构。

John Sweller's Cognitive Load Theory has become one of the most cited and empirically supported frameworks in educational psychology. His demonstration that working memory limitations fundamentally constrain learning has reshaped instructional design across disciplines. The theory's three types of cognitive load (intrinsic, extraneous, germane) and associated effects (expertise reversal, redundancy, split attention) provide actionable principles for educators, curriculum designers, and researchers worldwide.

## 二、核心命题

1. **认知负荷理论（CLT）**：工作记忆容量极其有限（约4个组块）——教学设计必须管理三种认知负荷：内在负荷（材料本身的复杂性）、外在负荷（不良呈现方式）、关联负荷（建构图式的努力）。

2. **外在负荷应最小化**：很多教学失败不是因为内容太难，而是因为呈现方式制造了不必要的外在认知负荷——分散注意、冗余信息、分裂注意等效应都会消耗宝贵的认知资源。

3. **图式获取是学习的本质**：学习的目标是在长时记忆中建构图式（schemas）——图式一旦形成并自动化，就不再占用工作记忆资源。长时记忆的巨大容量与组织化结构是克服工作记忆限制的关键。

4. **专家反转效应（Expertise Reversal Effect）**：对新手有效的教学方法对专家可能无效（反之亦然）——教学设计必须考虑学习者的先前知识水平，同样的教学干预对不同水平的学习者可能产生截然相反的效果。

5. **样例学习优于问题解决**：对于初学者，学习已解决的问题样例（worked examples）比自行解决问题更有效——因为后者会消耗大量工作记忆资源在搜索解题路径上，而非建构图式。

6. **进化认识论基础**：人类认知架构是进化的产物——长时记忆存储的图式相当于"知识库"，替代了生物进化在物种层面完成的信息积累功能。教学设计必须尊重这一进化形成的认知架构。

## 三、创作/思想脉络

### 早期：问题解决与认知负荷的发现（1982-1990）

1988年，斯威勒在研究数学问题解决时发现了一个违反直觉的现象：要求学生解决复杂问题的教学方法，反而不如让学生学习已解决问题的样例。这一发现促使他提出认知负荷的概念——问题解决过程消耗了太多工作记忆资源，使得学习者无法将认知资源用于建构长时记忆中的图式。

### 中期：CLT的系统化与效应研究（1990-2010）

这一时期斯威勒致力于将CLT发展为完整的理论框架。他与Van Merrienboer、Paas等合作者共同识别了一系列重要的教学效应：冗余效应（同时呈现文字和语音的相同内容会增加负荷）、分裂注意效应（文字和图示分开呈现会导致注意力分散）、模态效应（视觉和听觉通道的合理利用可以扩展有效工作记忆）等。1999年与Van Merrienboer合著的论文成为教学设计领域被引用最多的文献之一。

### 晚期：进化认识论与CLT的生物学基础（2010-至今）

2011年出版的专著 *Cognitive Load Theory* 将CLT置于更宏大的理论框架中。斯威勒提出"进化认识论"视角——人类认知架构是进化选择的产物，工作记忆的有限性和长时记忆的无限性都有深层的进化逻辑。他与David Geary合作，区分了"生物初级知识"（进化形成的认知能力，如口语习得）和"生物次级知识"（文化传承的知识，如读写算），后者需要通过教学设计来传递。

## 五、重要概念

- **认知负荷（Cognitive Load）**：工作记忆中信息处理的负担，由内在、外在和关联三种类型构成
- **内在负荷（Intrinsic Load）**：材料本身的复杂性决定的负荷——由元素交互性（element interactivity）决定
- **外在负荷（Extraneous Load）**：不良呈现方式造成的不必要负荷——应该通过好的教学设计消除
- **关联负荷（Germane Load）**：用于建构图式的"好的"负荷——应该被鼓励和优化
- **元素交互性（Element Interactivity）**：学习材料中元素之间的相互依赖程度——高交互性 = 高内在负荷
- **专家反转效应（Expertise Reversal Effect）**：同一教学干预对不同水平学习者效果相反的效应

## 六、思想坐标

- **承接**: Simon的有限理性和工作记忆限制；Baddeley的工作记忆模型；Miller的"魔法数字7"
- **批判**: 发现学习（对新手认知过载）；多媒体学习中的"越多越好"假设；建构主义教学法对初学者的不适用
- **影响**: 教学设计；多媒体学习（Mayer）；医学教育；在线学习与MOOC设计

## 七、当代影响

- **在线教育**: MOOC和在线课程设计必须管理认知负荷——视频长度、信息密度和交互设计都受CLT指导
- **AI辅助学习**: AI应根据学习者水平动态调整信息呈现，避免认知过载——个性化学习路径的CLT基础
- **医疗培训**: 复杂的医学知识如何有效传授——CLT指导解剖学、诊断学和手术培训的教学设计
- **信息可视化**: 图表设计应减少外在负荷，增加关联负荷——数据可视化的CLT原则
- **多媒体学习**: Mayer的多媒体学习理论直接建立在CLT基础之上

## 八、常见误读

- **"认知负荷越低越好"**: 关联负荷是"好的负荷"——学习者需要投入认知努力来建构图式。目标是优化而非最小化总负荷。
- **"CLT反对问题解决"**: CLT反对的是在图式未建立前就让学生解决复杂问题——应该先通过样例建立图式，再逐步过渡到问题解决。
- **"专家反转效应意味着专家不需要学习"**: 它说明教学应个性化——同一方法对不同水平的学习者效果相反。专家需要不同的学习策略。
- **"CLT只适用于数学和科学"**: CLT的原则适用于所有需要学习复杂知识的领域，包括语言学习、艺术培训和职业技能。

## 九、跨学科关联

- **[继承] Simon的有限理性**: 工作记忆限制是有限理性的认知基础——Sweller将Simon的洞见转化为教学设计原则
- **[对话] Piaget的图式概念**: 图式概念被CLT重新定义为长时记忆中的知识结构，与Piaget的发展性图式有联系但更精确
- **[互补] Kahneman的双系统理论**: 系统2的高认知负荷导致系统1主导——CLT和双系统理论互为补充
- **[互补] Norman的设计心理学**: 设计心理学中的信息呈现原则与CLT的外在负荷管理一致
- **[互补] Wickens的多资源模型**: 前者关注任务分配中的资源管理，后者关注学习设计中的负荷管理
- **[平行] Geary的进化教育心理学**: 生物初级/次级知识的区分为CLT提供了进化论基础
