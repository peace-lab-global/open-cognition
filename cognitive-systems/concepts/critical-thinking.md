# 批判性思维

## 一句话定义

批判性思维是一种对信息、论证和信念进行主动、系统、有纪律的审视与评估的认知能力——不是"批评他人"，而是"检验思想本身"，包括检验自己的思维过程。

## 提出者与背景

- **哲学起源**：苏格拉底（Socrates）的"诘问法"（Elenchus）——通过追问暴露论证中的矛盾
- **现代系统化**：John Dewey 提出"反思性思维"（Reflective Thinking）；Richard Paul 建立批判性思维的要素-标准-障碍框架
- **认知科学补充**：Daniel Kahneman 的双系统理论揭示人类思维的自动化偏差（System 1）与刻意审查（System 2）
- **针对的问题**：人类天然倾向于接受符合既有信念的信息，批判性思维是对这种认知惰性的系统纠偏

## 核心要义

### Paul-Elder 框架

```
批判性思维 = 思维要素 × 智力标准 × 智力特质

要素（Elements）：     标准（Standards）：     特质（Traits）：
  目的                 清晰性                 智识谦逊
  问题                 准确性                 智识勇气
  信息                 精确性                 智识同理心
  推理                 相关性                 智识正直
  概念                 深度                   智识毅力
  假设                 广度                   对推理的信心
  含义/后果            逻辑性                 智识自主性
  视角                 重要性
                       公正性
```

### 关键论点

1. **所有思维都有结构**：任何论证都由目的、假设、信息、推理、概念、含义、视角组成
2. **所有思维都可被评估**：用清晰性、准确性、逻辑性等标准检验每个要素
3. **最大的障碍是自我中心**：人们倾向于认为自己已经是理性的——批判性思维首先要求审视自己
4. **System 2 需要刻意激活**：批判性思维是费力的认知活动，需要意志力来对抗自动化反应

### 论证评估的核心问题

| 要素 | 核心问题 |
|------|----------|
| **目的** | 这个论证试图达到什么？ |
| **问题** | 试图回答的核心问题是什么？ |
| **信息** | 使用了什么数据/证据？是否充分、可靠？ |
| **推理** | 从信息到结论的逻辑是否有效？ |
| **概念** | 关键术语的定义是否清晰、一致？ |
| **假设** | 什么被当作了不证自明的前提？这些前提可靠吗？ |
| **含义** | 如果接受这个结论，会引发什么后果？ |
| **视角** | 还有哪些替代视角被忽略了？ |

## 通俗用法 vs 学术原义

| 维度 | 通俗用法 | 学术原义 |
|------|----------|----------|
| "批判性思维" | "挑毛病"、"批评" | 对思维过程本身的系统性审查，包括审查自己的思维 |
| "独立思考" | "不听别人的" | 基于证据与逻辑的自主判断，而非盲从权威或多数意见 |
| "质疑一切" | "什么都不信" | 对论证结构进行有纪律的评估，同时承认合理共识 |

## 常见认知偏差与纠偏

| 偏差 | 描述 | 批判性思维的纠偏 |
|------|------|------------------|
| **确认偏误** | 只关注支持已有信念的证据 | 主动寻找反面证据（钢人论证） |
| **可得性偏差** | 根据"容易想到的例子"做判断 | 追问基础概率与样本代表性 |
| **诉诸权威** | "专家说的就一定对" | 检验专家的论证结构与证据链 |
| **滑坡谬误** | "如果 A 发生，必然导致 Z" | 检查每一步因果链的证据强度 |
| **稻草人** | 曲解对方立场再攻击 | 钢人论证（Steel Man）：先把对方立场强化到最佳版本 |

- **互补**：[麦肯锡方法](./mckinsey-method.md) — MECE 分解 + 批判性评估 = 既结构化又严谨的分析
- **互补**：[贝叶斯思维](./bayesian-thinking.md) — 贝叶斯提供概率更新框架，批判性思维提供证据评估标准

- **关联**：[有限理性](./bounded-rationality.md) — 批判性思维是对有限理性的自觉补偿
- **关联**：苏格拉底诘问法（[philosophy/socratic-questioning](../../philosophy/concepts/dialectics.md)）

## 代表思想家

- **Socrates（苏格拉底）** — 诘问法：通过持续追问暴露论证的隐含矛盾
- **John Dewey** — "反思性思维"：在经验与思考之间建立循环验证
- **Richard Paul & Linda Elder** — 建立批判性思维的要素-标准-特质三维框架
- **Daniel Kahneman** — 双系统理论揭示人类思维的自动化偏差
- **Karl Popper** — 可证伪性原则：科学思维的核心是"可以被证明是错的"

## 应用场景

- **信息素养**：在假新闻泛滥的时代评估信息来源与论证质量
- **学术写作**：构建经得起审查的论证结构
- **商业决策**：评估提案中的隐含假设与逻辑链
- **团队讨论**：用批判性思维框架结构化讨论，避免情绪化争论
- **自我反思**：定期审视自己的核心信念是否有充分的证据支撑

## 常见误读

- **误读 1**："批判性思维就是批评别人" — 它的核心是审视思维过程本身，包括自己的思维
- **误读 2**："批判性思维会让人变得消极" — 它不是怀疑一切，而是在合理怀疑的基础上做出更好的判断
- **误读 3**："聪明人不需要批判性思维" — 恰恰相反，聪明人更容易用智力为错误信念构建精巧的辩护（动机推理）
- **误读 4**："批判性思维可以教" — 研究表明，通用批判性思维课程的效果有限；它更需要在具体领域中长期实践
