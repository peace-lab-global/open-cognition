# 杜波依斯生平与思想大事年表 / Du Bois Life and Intellectual Timeline

## 早年与学术起步（1868—1895）/ Early Years (1868-1895)

| 年份 | 事件 | 意义 |
|------|------|------|
| 1868 | 2月23日生于马萨诸塞州大巴灵顿 | 混血家庭，母亲为黑人，父亲为法裔黑人 |
| 1885 | 入菲斯克大学（田纳西州纳什维尔） | 首次进入南方种族隔离环境 |
| 1888 | 获菲斯克大学学士学位 | 暑期在田纳西乡村教书，直面种族贫困 |
| 1888 | 入哈佛大学（从大三读起） | 师从威廉·詹姆斯 |
| 1890 | 获哈佛大学第二个学士学位 | 毕业典礼上作为学生代表发言 |
| 1891 | 获哈佛大学硕士学位 | 研究方向：哲学与历史 |
| 1892-94 | 在柏林大学学习 | 师从施莫勒，接受德国历史学派方法论 |
| 1895 | 获哈佛大学博士学位 | 首位获哈佛博士的非裔美国人 |

## 学术奠基（1896—1909）/ Academic Foundations (1896-1909)

| 年份 | 事件 | 意义 |
|------|------|------|
| 1896 | 出版博士论文《奴隶贸易的压制》 | 学术生涯正式起步 |
| 1896-97 | 在宾夕法尼亚大学任社会学助理教授 | 进行费城黑人社区调查 |
| 1897 | 任亚特兰大大学经济学与历史学教授 | 在南方黑人大学建立学术基地 |
| 1899 | 出版《费城黑人》 | 美国首部系统城市社会学研究 |
| 1900 | 出席第一次泛非会议（伦敦） | 提出"肤色线是二十世纪的问题" |
| 1900 | 在巴黎世博会展示非裔美国人数据图表 | 数据可视化与社会正义的先驱 |
| 1903 | 出版《黑人的灵魂》 | 双重意识；批判华盛顿路线 |
| 1905 | 发起尼亚加拉运动 | 激进民权运动的组织基础 |
| 1909 | 参与创建全国有色人种协进会（NAACP） | 美国最重要的民权组织 |

## 社会运动与政治介入（1910—1934）/ Activism (1910-1934)

| 年份 | 事件 | 意义 |
|------|------|------|
| 1910 | 创办并主编《危机》杂志 | NAACP官方刊物，成为种族正义舆论阵地 |
| 1911 | 出席第一次世界种族大会 | 国际反种族主义网络 |
| 1919 | 组织第一次泛非大会（巴黎） | 与凡尔赛和会平行 |
| 1920 | 出版《暗水》 | 种族、性别与帝国主义批判 |
| 1927 | 首次访问苏联 | 对社会主义产生兴趣 |
| 1934 | 辞去《危机》主编和NAACP职务 | 与NAACP主流路线分歧加深 |

## 马克思主义转向与晚年（1935—1963）/ Marxist Turn and Final Years (1935-1963)

| 年份 | 事件 | 意义 |
|------|------|------|
| 1935 | 出版《美国黑人重建》 | 颠覆重建史学；马克思主义分析 |
| 1940 | 出版《黎明的黄昏》 | 种族概念的自传式分析 |
| 1944 | 重返NAACP任特别研究部主任 | 晚年回归组织工作 |
| 1947 | 向联合国提交《向世界吁请》 | 种族问题国际化 |
| 1950 | 竞选纽约州参议员（美国劳工党） | 直接政治参与 |
| 1951 | 被联邦政府以"外国代理人"罪名起诉 | 冷战政治迫害；后被无罪释放 |
| 1958 | 护照恢复后环游世界 | 访问苏联、中国、加纳 |
| 1961 | 加入美国共产党 | 最终政治立场的选择 |
| 1961 | 移居加纳 | 接受恩克鲁玛邀请参与《非洲百科全书》 |
| 1963 | 8月27日在加纳去世 | 恰在"我有一个梦想"演讲同日 |

## 学术遗产 / Academic Legacy

| 年份 | 事件 | 意义 |
|------|------|------|
| 1968 | 《杜波依斯全集》开始系统编辑 | 学术遗产的保存和整理 |
| 1990s | 杜波依斯学术地位重新确立 | 社会学史重新承认其方法论先驱地位 |
| 2018 | 《美国社会学评论》杜波依斯特刊 | 学科正典化 |
