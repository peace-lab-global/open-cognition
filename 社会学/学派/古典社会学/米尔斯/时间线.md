# 米尔斯生平与思想大事年表 / Mills Life and Intellectual Timeline

| 年份 | 事件 | 意义 |
|------|------|------|
| 1916 | 8月28日生于德克萨斯州韦科 | 普通家庭出身 |
| 1935 | 入德克萨斯农工大学 | 后转学至德克萨斯大学 |
| 1939 | 获德克萨斯大学学士学位 | 哲学与社会学双修 |
| 1941 | 获威斯康星大学博士学位 | 受实用主义传统影响 |
| 1946 | 任哥伦比亚大学助理教授 | 进入美国社会学核心阵地 |
| 1948 | 出版《新的人物》 | 劳工领袖研究 |
| 1951 | 出版《白领》 | 新中产阶级分析 |
| 1953 | 与格特合编韦伯选集 | 韦伯思想引入美国 |
| 1956 | 出版《权力精英》 | 最具争议性的著作 |
| 1959 | 出版《社会学的想象力》 | 方法论批判与重构 |
| 1960 | 出版《听着，美国佬》和《第三次世界大战的起因》 | 介入政治争论 |
| 1962 | 3月20日因心脏病去世 | 年仅46岁 |
