# 帕森斯 · Talcott Parsons

## 基本信息

- **国别**：美国
- **学派**：结构功能主义
- **历史定位**：20 世纪中叶美国社会学的中心人物，建立大综合理论

## 核心命题

1. **社会行动的志愿主义理论**：行动 = 行动者 + 目的 + 情境 + 规范取向
2. **AGIL 模型**：任何社会系统都需四种功能子系统（适应/达标/整合/潜模式维持）
3. **模式变量**：行动取向的五对二分（特殊主义-普遍主义、归属-成就 等）
4. **系统层级**：行为有机体 → 人格系统 → 社会系统 → 文化系统
5. **现代化即分化**：社会系统通过功能分化进化

## 思想脉络

- **早期综合**：1937《社会行动的结构》— 涂尔干、韦伯、帕累托综合
- **系统转向**：1951《社会系统》— 从行动到系统
- **AGIL 整合**：50–60 年代成熟
- **晚期**：进化理论与现代化理论

## 关键著作

- 《社会行动的结构》(1937) — 综合古典理论
- 《社会系统》(1951) — 系统理论奠基
- 《经济与社会》(1956)
- 《社会：进化与比较视角》(1966)

## 重要概念

- **行动**：志愿、规范取向的有目的行为
- **AGIL**：
  - **A** Adaptation 适应（经济）
  - **G** Goal-attainment 目标达成（政治）
  - **I** Integration 整合（共同体 / 法律）
  - **L** Latency 潜模式维持（家庭 / 教育 / 宗教）
- **模式变量（pattern variables）**：行动取向的二分维度
- **角色（role）**：地位+期望
- **社会化**：内化共享价值的过程
- **均衡（equilibrium）**：系统的相对稳态

## 思想坐标

- **综合**：涂尔干（结构）+ 韦伯（行动）+ 弗洛伊德（人格）+ 经济学
- **批评**：默顿（中层理论）；C. W. Mills（社会学的想象力对宏大理论批判）；冲突论（Dahrendorf）
- **后继**：哈贝马斯（系统-生活世界）、亚历山大（新功能主义）、卢曼（系统论的极致化）

## 当代应用

- 系统论研究的祖型
- 比较现代化研究
- 组织理论（功能整合模型）
- 制度分析（家庭/教育/政治/经济功能区分）

## 常见误读

- **误读 1**：把帕森斯等同保守——他承认变迁，但偏向均衡分析
- **误读 2**：把 AGIL 当万能模板——它是分析工具而非现实图谱
- **误读 3**：把功能主义等同辩护现状——理论上中立，争议在使用方式
