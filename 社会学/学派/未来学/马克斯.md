# 马克斯·莫尔 · Max More

马克斯·莫尔（Max More，1964–）是英国出生的美国哲学家、未来学家，超人类主义运动的重要组织者。他 1990 年发表〈超人类主义：迈向未来主义哲学〉，提出以“Extropy”（熵的对立面）为核心的技术乐观主义纲领，强调通过科学与技术扩展生命、智力、自由、知识与幸福。

## 核心命题 / Core Theses

### 1. 积极演化的伦理

超人类主义不是被动等待演化，而是主动利用技术扩展生命、智力、自由、知识与幸福。技术被视为延续并加速智能生命演化的手段。

### 2. 人性作为半成品

人类现状只是演化路径上的一个阶段，没有理由把它视为终点或完美形态。疾病、衰老与认知局限都是可以也应当被克服的问题。

### 3. 动态乐观主义

莫尔以“Extropy”对抗熵增：通过开放社会、分散知识与技术创新，推动生命与智能的无限扩展。悲观不是谨慎，而是对生命潜能的放弃。

## 关键著作 / Key Works

## 跨学科关联 / Interdisciplinary Links

- 与 [朱利安·赫胥黎](朱利安.md) 共享“人类自我超越”的愿景，但更强调技术手段与社会机制。
- 与 [唐娜·哈拉维](../性别研究/唐娜.md) 的赛博格政治形成对照：同样谈论人-技混合，但价值预设与政治关怀不同。
- 其 Extropian 思想与库兹韦尔的奇点预测、博斯特罗姆的存在风险研究构成超人类主义内部的不同分支。
