# 尼古拉斯·卡尔 · Nicholas Carr

尼古拉斯·卡尔（Nicholas Carr，1959—）是美国记者、技术批评家。他 2008 年在《大西洋月刊》发表〈谷歌让我们变笨了吗？〉，2010 年扩展为《浅薄：互联网如何毒化了我们的大脑》，引发关于数字媒介对认知影响的广泛讨论。

## 核心命题 / Core Theses

### 1. 媒介塑造心智 · Media Shape the Mind
援引麦克卢汉“媒介即讯息”，卡尔认为互联网作为一种媒介，正在重塑我们的感知、记忆与思维方式。

### 2. 深度阅读的消逝 · The Decline of Deep Reading
屏幕阅读鼓励扫描、跳转与多任务，削弱了纸质书培养的线性、专注与沉思能力。

### 3. 记忆外包的代价 · The Cost of Outsourced Memory
把记忆交给机器，不仅损失信息，更削弱了整合经验、形成自我认同的认知过程。

## 关键著作 / Key Works
