# 萨拉·艾哈迈德 · Sara Ahmed

萨拉·艾哈迈德（Sara Ahmed，1969– ）是英国女性主义学者与作家，曾任伦敦大学金匠学院教授。她以“情感的文化政治”研究闻名，分析情感如何在话语与制度中流通并建构“他者”。

## 核心命题 / Core Theses

### 1. 情感的经济

情感不是私人内在状态，而是在社会中被生产、流通与归属的“经济物”。恐惧、厌恶、爱可以被引导向特定群体。

### 2. 幸福的霸权

“幸福”常被用作规范工具：偏离主流规范的人被描绘为不快乐，从而合理化对他们的排斥。

### 3. 批判性女性主义的生存

提出“ killy joy”概念：女性主义者因拒绝配合压迫性的快乐脚本而被视为扫兴者，但这正是批判工作的必要代价。
