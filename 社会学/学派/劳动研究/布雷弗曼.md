# 哈里·布雷弗曼 · Harry Braverman

哈里·布雷弗曼（Harry Braverman，1920–1976）是美国马克思主义社会学家与工人运动活动家。他在《劳动与垄断资本》中系统分析了资本主义劳动过程中的“去技能化”趋势，成为劳动过程理论的奠基之作。

## 核心命题 / Core Theses

### 1. 劳动过程的去技能化

资本家为了控制劳动过程，倾向于将工人的知识转移到管理方与机器中，使劳动碎片化、标准化，从而削弱工人的议价能力。

### 2. 概念与执行的分离

当设计、规划与实际操作被拆分时，工人失去对工作整体的理解与控制权；知识被垄断，劳动变得单调。

### 3. 垄断资本与劳动转型

在垄断资本主义阶段，科学技术与管理技术成为控制劳动的新工具，去技能化从手工劳动扩展到白领与服务业。

## 关键著作 / Key Works

- *Labor and Monopoly Capital: The Degradation of Work in the Twentieth Century* (1974)

## 跨学科关联 / Interdisciplinary Links

- 与 [卡尔·马克思](../古典社会学/马克思/README.md) 的剩余价值理论直接相承，布雷弗曼将其具体化为劳动过程分析。
- 与 [大卫·格雷伯](格雷伯.md) 后来的“狗屁工作”论述形成对话：去技能化与无意义劳动共同构成当代工作的批判维度。
