# 大卫·格雷伯 · David Graeber

大卫·格雷伯（David Graeber，1961–2020）是美国人类学家与社会活动家，以“占领华尔街”运动与《狗屁工作》《债：第一个5000年》闻名。他将无政府主义实践与人类学视角结合，重新审视劳动、债务与社会制度。

## 核心命题 / Core Theses

### 1. 狗屁工作的蔓延

大量现代工作缺乏社会价值，却被包装成有意义。人们从事这些工作不是因为社会需要，而是因为制度性的支配与雇佣关系。

### 2. 护理劳动的贬值

真正创造关怀、教育、养育价值的工作往往报酬最低、最不被尊重；这与资本主义对“生产力”的狭隘定义有关。

### 3. 债务作为社会权力

债务不仅是经济关系，更是道德与政治控制工具。它让负债者感到羞耻与义务，从而维系不平等结构。
