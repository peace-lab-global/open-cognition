# 凯西·奥尼尔 · Cathy O'Neil

凯西·奥尼尔（Cathy O'Neil，1972– ）是美国数学家、数据科学家与作家。她在《数学杀伤性武器》中揭示大数据模型如何以“客观”之名固化偏见、放大不平等。

## 核心命题 / Core Theses

### 1. 数学模型的道德政治性

算法与统计模型并非中立工具，它们嵌入设计者的目标、数据的历史偏见与权力的运作逻辑。

### 2. 伤害的可扩展性

当模型被大规模应用于招聘、信贷、警务与教学时，即使微小的偏见也会被系统性放大，形成“数学杀伤性武器”。

### 3. 可解释性与反馈循环

不透明的模型会自我强化：对贫困社区的预测性警务增加逮捕记录，又“证明”社区更危险，形成闭环歧视。
