# 斯宾诺莎思想年表 · Spinoza Intellectual Timeline

## 早年与教育 · Early Years and Education (1632–1656)

| 年份 | 事件 |
|------|------|
| 1632 | 11月24日生于阿姆斯特丹的塞法迪犹太移民家庭 |
| 1638 | 母亲汉娜·德博拉去世，斯宾诺莎年仅6岁 |
| 1640s | 在阿姆斯特丹犹太社区的学校接受传统犹太教育：希伯来文、圣经、塔木德 |
| 1649 | 兄长以撒去世后，斯宾诺莎进入家族进出口商行工作 |
| 1654 | 商行因债务问题破产，斯宾诺莎获得了一定的经济独立 |
| 1650s | 开始接触笛卡尔哲学、新科学思想，师从弗朗西斯·范·登·恩德 |

| Year | Event |
|------|-------|
| 1632 | Born 24 November in Amsterdam to a Sephardic Jewish immigrant family |
| 1638 | Mother Hannah Debora dies; Spinoza is only 6 |
| 1640s | Receives traditional Jewish education in Amsterdam's community school: Hebrew, Bible, Talmud |
| 1649 | After brother Isaac's death, Spinoza enters the family import-export business |
| 1654 | Business fails due to debt; Spinoza gains some economic independence |
| 1650s | Encounters Cartesian philosophy and new scientific thought; studies under Franciscus van den Enden |

## 革除教门与流亡 · Excommunication and Exile (1656–1663)

| 年份 | 事件 |
|------|------|
| 1656 | 7月27日，阿姆斯特丹犹太社区对斯宾诺莎颁布最严厉的革除教门令（cherem），永不撤销 |
| 1656–1660 | 离开犹太社区，将名字改为拉丁语"本尼迪克特"（Benedictus） |
| 1660 | 撰写《简论上帝、人及其幸福》（生前未出版，1862年发现） |
| 1661 | 移居莱茵斯堡（Rijnsburg），以磨镜片为生 |
| 1662 | 撰写《知性改进论》（未完成，死后出版） |
| 1663 | 完成《笛卡尔哲学原理》（唯一署名的生前出版物）；移居伏尔堡（Voorburg） |

| Year | Event |
|------|-------|
| 1656 | 27 July: Amsterdam's Jewish community issues the harshest cherem (excommunication) against Spinoza, never revoked |
| 1656–1660 | Leaves the Jewish community; Latinizes name to "Benedictus" |
| 1660 | Writes *Short Treatise on God, Man, and His Well-Being* (unpublished until discovered in 1862) |
| 1661 | Moves to Rijnsburg, earns living grinding lenses |
| 1662 | Writes *Treatise on the Improvement of the Understanding* (unfinished, posthumous) |
| 1663 | Completes *Principles of Cartesian Philosophy* (only work published under his name in his lifetime); moves to Voorburg |

## 哲学成熟期 · Philosophical Maturity (1663–1677)

| 年份 | 事件 |
|------|------|
| 1663 | 移居伏尔堡，开始与梅森圈子的学者交流 |
| 1664 | 开始与亨利·奥尔登堡（Henry Oldenburg，英国皇家学会秘书）通信 |
| 1665 | 开始撰写《神学政治论》 |
| 1669 | 移居海牙，住在斯皮克运河旁的房子里 |
| 1670 | 《神学政治论》匿名出版于阿姆斯特丹，引起巨大争议 |
| 1673 | 拒绝海德堡大学哲学教授职位——"因为这会限制我的思想和写作自由" |
| 1674 | 《神学政治论》被荷兰议会正式禁止 |
| 1675 | 完成《伦理学》手稿，但因争议太大不敢出版 |
| 1676 | 莱布尼茨访问海牙，与斯宾诺莎进行了数日深入交谈 |
| 1677 | 2月21日因肺部疾病（可能是硅肺病）去世，年仅44岁 |
| 1677 | 遗著集（*Opera Posthuma*）出版，包含《伦理学》《政治论》等 |

| Year | Event |
|------|-------|
| 1663 | Moves to Voorburg; begins exchanging ideas with scholars in Mersenne's circle |
| 1664 | Begins correspondence with Henry Oldenburg (Secretary of the Royal Society) |
| 1665 | Begins writing the *Theological-Political Treatise* |
| 1669 | Moves to The Hague, to a house by the Spui canal |
| 1670 | *Theological-Political Treatise* published anonymously in Amsterdam; provokes enormous controversy |
| 1673 | Declines professorship at Heidelberg — "because it would restrict my freedom of thought and writing" |
| 1674 | *Theological-Political Treatise* officially banned by the States of Holland |
| 1675 | Completes *Ethics* manuscript but dares not publish due to controversy |
| 1676 | Leibniz visits The Hague; spends several days in deep conversation with Spinoza |
| 1677 | Dies 21 February of lung disease (likely silicosis from lens-grinding), aged only 44 |
| 1677 | *Opera Posthuma* published, containing *Ethics*, *Political Treatise*, etc. |

## 身后影响 · Posthumous Legacy

| 年份 | 事件 |
|------|------|
| 1678 | 《伦理学》荷兰文译本出版 |
| 1785 | 德国"泛神论争论"（Pantheismusstreit）——雅可比和门德尔松争论莱辛是否为斯宾诺莎主义者 |
| 1799 | 歌德承认斯宾诺莎对他的深刻影响 |
| 1800s | 黑格尔称"不做斯宾诺莎主义者就没有哲学" |
| 1880s | 尼采发现斯宾诺莎是他的"先驱" |
| 1968 | 德勒兹出版《斯宾诺莎与表达问题》，引发斯宾诺莎的当代复兴 |
| 2003 | 达马西奥出版《寻找斯宾诺莎》，将情感神经科学与斯宾诺莎哲学联系 |
| 2012 | 斯宾诺莎的肖像出现在荷兰10欧元纸币上（2002年版） |

| Year | Event |
|------|-------|
| 1678 | Dutch translation of *Ethics* published |
| 1785 | German "Pantheism Controversy" (Pantheismusstreit) — Jacobi and Mendelssohn debate whether Lessing was a Spinozist |
| 1799 | Goethe acknowledges Spinoza's profound influence |
| 1800s | Hegel: "To be a philosopher, one must first have been a Spinozist" |
| 1880s | Nietzsche discovers Spinoza as his "precursor" |
| 1968 | Deleuze publishes *Spinoza and the Problem of Expression*, sparking contemporary Spinoza revival |
| 2003 | Damasio publishes *Looking for Spinoza*, connecting affect neuroscience with Spinoza's philosophy |
| 2012 | Spinoza's portrait on the Dutch 10-euro banknote (2002 edition) |
