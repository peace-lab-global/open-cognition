# 梅洛-庞蒂年表 / Timeline of Merleau-Ponty

## 编年

| 年份 | 事件 |
|------|------|
| 1908 | 3月14日生于法国滨海罗什福尔（Rochefort-sur-Mer） |
| 1915 | 父亲在第一次世界大战中阵亡 |
| 1919 | 随母迁居巴黎 |
| 1926 | 入巴黎路易大帝中学（Lycée Louis-le-Grand） |
| 1929 | 入巴黎高等师范学院（École Normale Supérieure）；与萨特结识 |
| 1930 | 哲学教师资格考试第一名（agrégation） |
| 1931-33 | 服兵役 |
| 1933-34 | 在博韦中学任教 |
| 1934 | 开始研读胡塞尔著作，赴卢汶胡塞尔档案馆 |
| 1935-39 | 在巴黎沙尔勒曼中学、索邦大学任教；研究格式塔心理学 |
| 1939 | 第二次世界大战爆发，应征入伍 |
| 1940 | 法国沦陷，复员 |
| 1942 | 出版《行为的结构》（La Structure du comportement） |
| 1945 | 出版《知觉现象学》（Phénoménologie de la perception）；与萨特共同编辑《现代》杂志 |
| 1947 | 出版《人道主义与恐怖》 |
| 1948 | 任里昂大学教授 |
| 1949 | 任索邦大学教授 |
| 1952 | 当选法兰西学院哲学教授（年仅44岁） |
| 1953 | 因朝鲜战争立场与萨特决裂；离开《现代》杂志 |
| 1955 | 出版《辩证法的历险》 |
| 1960 | 出版《符号》《眼与心》 |
| 1961 | 5月3日因心脏病突发去世，年仅53岁 |
| 1964 | 遗著《可见的与不可见的》出版（Claude Lefort 编辑） |

## 同时代人物

- **萨特**（Jean-Paul Sartre, 1905-1980）：同代、同事、后决裂
- **波伏娃**（Simone de Beauvoir, 1908-1986）：同代，《现代》杂志同事
- **列维-斯特劳斯**（Claude Lévi-Strauss, 1908-2009）：同代，结构主义人类学家
- **拉康**（Jacques Lacan, 1901-1981）：精神分析学家，巴黎知识圈
- **胡塞尔**（Edmund Husserl, 1859-1938）：现象学创始人，梅洛-庞蒂的核心影响源
- **海德格尔**（Martin Heidegger, 1889-1976）：存在哲学，梅洛-庞蒂借鉴其"在世存在"

## 思想分期

| 时期 | 年代 | 核心主题 | 代表著作 |
|------|------|----------|----------|
| 早期 | 1930s-1942 | 格式塔心理学、行为结构 | 《行为的结构》 |
| 成熟期 | 1945-1952 | 身体现象学、知觉首要性 | 《知觉现象学》 |
| 政治期 | 1947-1955 | 马克思主义、暴力与历史 | 《人道主义与恐怖》《辩证法的历险》 |
| 晚期 | 1952-1961 | 肉身本体论、交织 | 《可见的与不可见的》《眼与心》 |
