# 伯特兰·罗素

## 基本信息

- **生卒年**：1872年—1970年
- **国籍/文化背景**：英国哲学家、数学家、逻辑学家、社会活动家
- **学术身份**：分析哲学联合创始人，逻辑主义奠基人
- **学派归属**：分析哲学

## 核心命题

1. **逻辑原子主义（Logical Atomism）**：世界由简单事实构成，语言应通过逻辑分析还原到基本命题（原子命题）。
2. **描述理论（Theory of Descriptions）**：限定描述词（如"法国现任国王"）不是指称词，而是可被逻辑分析的不完全符号。
3. **罗素悖论（Russell's Paradox）**：所有不包含自身之集合的集合是否包含自身？此悖论动摇了朴素集合论。
4. **逻辑主义（Logicism）**：数学可还原为逻辑——《数学原理》试图从逻辑公理推导全部数学。
5. **中立一元论**：心与物不是根本不同的实体，而是同一中立材料的不同排列。

## 思想脉络

### 早期
- 出身英国贵族家庭，剑桥三一学院毕业
- 早期受黑格尔唯心论影响，后转向新实在论
- 1901年发现罗素悖论，引发数学基础危机

### 中期
- 与怀特海合著《数学原理》（1910-1913），试图逻辑化数学
- 发展描述理论，影响维特根斯坦
- 提出逻辑原子主义哲学

### 晚期
- 转向通俗哲学写作，《西方哲学史》广受欢迎
- 积极参与反核、反战社会运动
- 1950年获诺贝尔文学奖

## 核心概念

| 概念 | 说明 |
|------|------|
| 逻辑原子主义 | 语言和世界可分析为逻辑上简单的成分 |
| 描述理论 | 限定描述是不完全符号，需逻辑分析 |
| 罗素悖论 | 挑战朴素集合论，引发第三次数学危机 |
| 类型论 | 解决悖论的分层逻辑系统 |

## 跨领域影响

- **数学**：逻辑主义纲领，数理逻辑发展
- **语言哲学**：描述理论开创分析传统
- **人工智能**：逻辑形式化思想影响早期AI
- **政治**：和平主义运动，核裁军
