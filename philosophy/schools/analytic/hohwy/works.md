## 推荐阅读路径 / Recommended Reading Paths

### 路径一：入门路径 / Path 1: Introductory
1. *The Predictive Mind* (2013) 第1-4章 / Chapters 1-4 — 基本概念框架 / Basic conceptual framework
2. "The Self-Evidencing Brain" (*Noûs*, 2016) — 核心创新论点 / Core innovative thesis
3. *The Predictive Mind* (2013) 第5-12章 / Chapters 5-12 — 深入各个主题 / Deep dive into topics

### 路径二：认识论路径 / Path 2: Epistemological
1. "Evidence, Explanation, and Idealism" (2010) — 认识论基础 / Epistemological foundations
2. *The Predictive Mind* (2013) 第11-12章 — 认识论含义 / Epistemological implications
3. "The Neural Organ Explains the Mind" (2015) — 解释与自然主义 / Explanation and naturalism

### 路径三：意识与现象学路径 / Path 3: Consciousness & Phenomenology
1. "When the Brain is Prepared to Learn" (2012) — 意识与预测 / Consciousness and prediction
2. "Predictive Processing and the Phenomenology of Perception" (2017) — 现象学维度 / Phenomenological dimensions
3. *The Predictive Mind* (2013) 第8-10章 — 意识的预测理论 / Predictive theory of consciousness

### 路径四：与Clark的论争 / Path 4: Debate with Clark
1. *The Predictive Mind* (2013) 第13章 — 对延展心智的批评 / Critique of extended mind
2. Clark, A. (2013) "Whatever next?" (*BBS*) — Clark的回应框架 / Clark's response framework
3. Hohwy (2016) "The Self-Evidencing Brain" — 深化内部分歧 / Deepening internal divergence

### 路径五：东西方比较路径 / Path 5: East-West Comparative
1. *The Predictive Mind* (2013) 第11章 — 间接实在论 / Indirect realism
2. 唯识学《唯识三十颂》/ Vasubandhu's *Triṃśikā* — 三性理论 / Three natures theory
3. 庄子《齐物论》/ Zhuangzi "Equality of Things" — 认知局限性 / Cognitive limitations
