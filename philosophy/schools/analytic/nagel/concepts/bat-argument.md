# 蝙蝠论证 · Bat Argument

## 概念定位 / Conceptual Position
"成为一只蝙蝠是什么感觉？"——即使完全了解蝙蝠的神经科学，我们仍无法知道成为蝙蝠使用回声定位是什么感觉。

"What is it like to be a bat?" — even with complete bat neuroscience, we cannot know what it's like to be a bat using echolocation.

## 核心论述 / Core Argument

### 论证结构 / Argument Structure
1. 意识体验具有主观特征（"what it is like"）
2. 物理描述穷尽的是客观的、第三人称的信息
3. 主观特征不能被客观描述穷尽
4. 因此物理主义对意识的解释是不完备的

### 为什么是蝙蝠？/ Why a Bat?
蝙蝠使用回声定位感知世界——一种与人类完全不同的感知方式。这使得"想象自己成为蝙蝠"不能解决问题——我们需要知道的是成为蝙蝠本身是什么感觉。

Bats use echolocation — a completely different mode of perception. This means "imagining ourselves as bats" cannot solve the problem — we need to know what it's like to be a bat itself.

## 哲学意义 / Philosophical Significance
确立了意识的"硬问题"，直接启发了Chalmers和Levine的解释鸿沟理论。

## 东西方对话 / East-West Dialogue
### 与庄子"子非鱼" / With Zhuangzi's "You Are Not a Fish"
庄子与惠施的"濠梁之辩"——"子非鱼，安知鱼之乐？""子非我，安知我不知鱼之乐？"——与Nagel的蝙蝠论证异曲同工：不同认知主体之间的主观体验是否可以通达？

Zhuangzi's "Hao Bridge" debate — "You are not a fish, how do you know fish happiness?" — parallels Nagel: can subjective experience be accessed across different cognitive subjects?
