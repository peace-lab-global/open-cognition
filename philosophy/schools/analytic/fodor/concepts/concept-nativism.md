# 概念先天论 · Concept Nativism

## 概念定位 / Conceptual Position
所有可思考的概念在某种意义上是先天具有的——学习不能产生真正新的概念，只能"触发"已有的概念。

All thinkable concepts are in some sense innate — learning cannot produce genuinely new concepts, only "trigger" existing ones.

## 核心论述 / Core Argument

### 论证 / The Argument
学习新概念 = 形成假说 → 假说需要已有的概念 → 因此不能学到全新的概念。如同你不能通过假说来学习一个你尚不具备的概念。

Learning a new concept = forming hypotheses → hypotheses require existing concepts → therefore cannot learn genuinely new concepts.

### 争议 / Controversy
这是Fodor最极端的立场，被广泛认为是其理论中最薄弱的环节。许多哲学家和认知科学家反对这种"一切概念先天"的观点。

This is Fodor's most extreme position, widely considered the weakest part of his theory.
