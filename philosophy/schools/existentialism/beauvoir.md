# 西蒙娜·德·波伏瓦

## 基本信息

- **生卒年**：1908年—1986年
- **国籍/文化背景**：法国哲学家、作家、女性主义先驱
- **学术身份**：存在主义女性主义奠基人
- **学派归属**：存在主义 / 女性主义

## 核心命题

1. **"一个人不是生而为女人，而是成为女人"**：性别不是自然本质，而是社会文化建构的结果。
2. **他者（The Other）**：女性在父权社会中被建构为男性的"他者"，丧失主体地位。
3. **模糊性的伦理（Ethics of Ambiguity）**：人的存在是自由与有限性的模糊统一，真正的伦理在于承担这种模糊性。
4. **情境中的自由（Situated Freedom）**：自由不是抽象的，而总是在具体情境中实现，受历史、身体、社会条件制约。
5. **女性处境分析**：系统分析女性在政治、经济、心理、身体各维度的压迫处境。

## 思想脉络

### 早期
- 出生于巴黎，中产阶级家庭
- 巴黎高等师范学院毕业，哲学教师资格考试第二名（第一名是萨特）
- 与萨特建立终身伴侣关系，但拒绝婚姻

### 中期
- 1949年出版《第二性》，震动知识界
- 发展存在主义女性主义框架
- 获龚古尔文学奖（《名士风流》1954）

### 晚期
- 持续写作自传体作品，记录思想历程
- 参与女性解放运动，签署《343宣言》
- 晚年关注老年问题，《老之将至》

## 核心概念

| 概念 | 说明 |
|------|------|
| 他者 | 女性在父权中被建构为男性主体的对立面 |
| 模糊性 | 存在既有自由又有限定，是核心处境 |
| 情境自由 | 自由总在具体历史、身体、社会条件中实现 |
| 性别建构 | 性别身份由社会文化塑造，非天生本质 |

## 跨领域影响

- **女性主义**：第二波女性主义理论基石，影响巴特勒等
- **存在主义**：补充萨特伦理学的空白
- **社会学**：性别社会建构的先驱分析
- **文学**：自传体写作的新范式
