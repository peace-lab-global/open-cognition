# 哲学 · Philosophy

哲学是对真、善、美与可知性的根本追问。本目录按学派组织思想家条目，按概念组织跨学派核心观念。

## 学派树

```
哲学
├── 古希腊 (ancient-greek)
│   ├── 苏格拉底 · 诘问与德性
│   ├── 柏拉图 · 理念论
│   └── 亚里士多德 · 实体与德性伦理
├── 德国唯心论 (german-idealism)
│   ├── 康德 · 批判哲学
│   └── 黑格尔 · 辩证法
├── 存在主义/现象学 (existentialism)
│   ├── 尼采 · 价值重估
│   └── 海德格尔 · 此在与存在
├── 分析哲学 (analytic)
│   └── 维特根斯坦 · 语言游戏
└── 东方哲学 (eastern)
    ├── 老子 · 道与无为
    └── 孔子 · 仁与礼
```
