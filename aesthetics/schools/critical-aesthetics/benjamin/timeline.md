# 本雅明年表 / Benjamin Timeline

## 早年与学术形成 / Early Years

| 年份 | 事件 |
|------|------|
| 1892 | 7月15日出生于德国柏林富裕犹太商人家庭 |
| 1912 | 进入弗莱堡大学学习哲学、艺术史和文学 |
| 1915 | 结识朔勒姆，开始终生思想对话 |
| 1916 | 写作《论语言本身和人的语言》 |
| 1919 | 以《德国浪漫主义的艺术批评概念》获伯尔尼大学博士学位 |

## 批评实践与理论建构 / Critical Practice

| 年份 | 事件 |
|------|------|
| 1921 | 发表《暴力批判》 |
| 1924 | 访问莫斯科，开始接触马克思主义 |
| 1925 | 以《德国悲剧的起源》申请教授资格被拒 |
| 1927 | 首次访问巴黎；开始构思拱廊街计划 |
| 1928 | 出版《德国悲剧的起源》和《单行道》 |
| 1929 | 结识布莱希特 |

## 流亡与最后岁月 / Exile & Final Years

| 年份 | 事件 |
|------|------|
| 1933 | 纳粹上台；流亡巴黎 |
| 1935 | 完成《机械复制时代的艺术作品》第一稿 |
| 1936 | 写作《讲故事的人》 |
| 1939 | 二战爆发；发表《波德莱尔的几个母题》 |
| 1940 | 完成《历史哲学论纲》 |
| 1940 | 9月26日在波尔特沃自杀身亡 |

## 身后影响 / Posthumous Impact

| 年份 | 事件 |
|------|------|
| 1968 | 阿伦特编 *Illuminations* 出版 |
| 1982 | 拱廊街计划德文版首次完整出版 |
| 1999 | 英译《拱廊街计划》出版 |
| 2014 | Eiland & Jennings 权威传记出版 |
