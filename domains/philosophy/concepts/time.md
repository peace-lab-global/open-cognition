# 时间 · Time

## 概述

时间是哲学中最基本也最困难的问题之一。从奥古斯丁的"时间是什么？你不问我我还知道，你一问我我就不知道了"，到胡塞尔的现象学分析和海德格尔的存在论阐释，时间问题贯穿哲学史。

## 核心议题

### 1. 现象学时间（Phenomenological Time）
- 胡塞尔：时间意识三结构——原印象、滞留、前摄
- 内时间意识是意识流的构成性结构
- 时间不是客观容器，而是意识的构造

### 2. 生存时间/此在时间性（Dasein's Temporality）
- 海德格尔：时间是此在存在的意义
- 三重时间性：曾在、当前、将来
- 向死而生赋予时间以有限性和紧迫性
- 常人沉沦遮蔽了本真时间性

### 3. 钟表时间（Clock Time）
- 物理学的客观时间：均匀、可测量、可逆
- 牛顿：绝对时间，与任何外在事物无关
- 爱因斯坦：相对论颠覆绝对时间

### 4. 永恒回归（Eternal Return）
- 尼采：如果生命无限重复，你能否肯定每一刻？
- 不是宇宙论主张，而是伦理测试
- 对虚无主义的回应：肯定此世此在

### 5. 无常（Impermanence）
- 佛教：三法印之一——诸行无常
- 一切有为法如梦幻泡影
- 无常不是悲观，而是解脱的前提

## 哲学传统对比

| 传统 | 时间观 |
|------|--------|
| 奥古斯丁 | 时间是心灵的延展（distentio animi） |
| 康德 | 时间是感性直观的先天形式 |
| 胡塞尔 | 时间是意识流的内在结构 |
| 海德格尔 | 时间是此在存在的意义 |
| 尼采 | 永恒回归——对时间的最高肯定 |
| 佛教 | 无常——时间中无恒常实体 |

## 跨领域关联

- **物理学**：热力学时间箭头，宇宙学时间
- **神经科学**：大脑如何感知时间
- **宗教**：弥赛亚时间，末世论
- **文学**：普鲁斯特的非自主记忆，意识流
- **冥想实践**：活在当下，时间的消解
