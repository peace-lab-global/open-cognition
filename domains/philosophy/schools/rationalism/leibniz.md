# 戈特弗里德·莱布尼茨

## 基本信息

- **生卒年**：1646年—1716年
- **国籍/文化背景**：德国哲学家、数学家、外交家
- **学术身份**：理性主义哲学家，微积分独立发明者
- **学派归属**：大陆理性主义

## 核心命题

1. **单子论（Monadology）**：世界由无数单子（monads）构成，单子是无窗的、不可分的精神实体，每个单子都映射整个宇宙。
2. **前定和谐（Pre-established Harmony）**：单子之间不相互作用，但上帝在创世时已预先安排了各单子之间的和谐协调。
3. **乐观主义**：这是"所有可能世界中最好的一个"——上帝在所有可能世界中选择了最完美者。
4. **充足理由律**：任何事物的存在都有其充足理由，偶然真理也有其理由。
5. **连续性原理**：自然无跳跃，宇宙是连续的等级秩序。

## 思想脉络

### 早期
- 出生于莱比锡，父亲为莱比锡大学道德哲学教授
- 早年涉猎经院哲学、古典文献、数学
- 1666年完成博士论文《论组合术》

### 中期
- 出使巴黎（1672-1676），发展出微积分理论
- 发明二进制，设计机械计算器
- 与斯宾诺莎通信，受其影响但保留不同观点

### 晚期
- 完成《单子论》（1714）和《神义论》（1710）
- 伏尔泰在《老实人》中讽刺其乐观主义
- 与牛顿的微积分优先权之争

## 核心概念

| 概念 | 说明 |
|------|------|
| 单子 | 无窗的、不可分的精神实体，宇宙的基本单位 |
| 前定和谐 | 上帝预先安排的单子间协调，无需相互作用 |
| 可能世界 | 上帝在所有逻辑可能的世界中选择了最优者 |
| 充足理由律 | 任何事实必有其所以如此而非他样的充足理由 |

## 跨领域影响

- **数学**：微积分符号系统沿用至今，二进制成为计算机基础
- **逻辑学**：符号逻辑先驱，预见形式化推理
- **计算机科学**：二进制、组合术对计算理论有先驱意义
- **神学**：神义论试图解决恶的问题
