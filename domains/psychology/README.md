# 心理学 · Psychology

本目录覆盖心理学主要理论流派，按流派组织思想家条目，按概念组织跨流派观念。

## 流派树

```
心理学
├── 精神分析 (psychoanalysis / analytical / individual)
│   ├── 弗洛伊德 · 潜意识与本我自我超我
│   ├── 荣格 · 集体无意识与原型
│   └── 阿德勒 · 目的论与自卑超越
├── 行为主义 (behaviorism)
│   └── 斯金纳 · 操作性条件反射
├── 认知与认知行为 (cognitive)
│   ├── 皮亚杰 · 认知发展
│   ├── 贝克 · 认知行为疗法
│   └── 卡尼曼 · 双系统
├── 人本主义 (humanistic)
│   ├── 马斯洛 · 需求层次
│   └── 罗杰斯 · 来访者中心
└── 积极心理学 (positive)
    └── 米哈里 · 心流
```

## 思想家条目

### 精神分析 psychoanalysis
- [弗洛伊德 Sigmund Freud](./schools/psychoanalysis/freud.md) — 潜意识、性心理发展、本我/自我/超我
- [荣格 Carl Gustav Jung](./schools/psychoanalysis/jung.md) — 集体无意识、原型、个体化
- [阿德勒 Alfred Adler](./schools/psychoanalysis/adler.md) — 自卑情结、目的论、社会兴趣

### 行为主义 behaviorism
- [斯金纳 B.F. Skinner](./schools/behaviorism/skinner.md) — 操作性条件反射

### 认知与认知行为 cognitive
- [皮亚杰 Jean Piaget](./schools/cognitive/piaget.md) — 认知发展四阶段、图式
- [贝克 Aaron T. Beck](./schools/cognitive/beck.md) — 认知三联征、认知扭曲
- [卡尼曼 Daniel Kahneman](./schools/cognitive/kahneman.md) — 系统 1/系统 2

### 人本主义 humanistic
- [马斯洛 Abraham Maslow](./schools/humanistic/maslow.md) — 需求层次、自我实现
- [罗杰斯 Carl Rogers](./schools/humanistic/rogers.md) — 无条件积极关注、来访者中心治疗

### 积极心理学 positive
- [米哈里 Mihaly Csikszentmihalyi](./schools/positive/csikszentmihalyi.md) — 心流
