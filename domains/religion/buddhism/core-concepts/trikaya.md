# 三身 Trikāya

## 一句话定义

三身是佛教对"佛"的多维度理解——**法身**（Dharmakāya，真理之身）、**报身**（Saṃbhogakāya，功德受用之身）、**化身**（Nirmāṇakāya，随缘示现之身），解决了"佛是历史人物还是超越存在"的哲学问题。

## 核心要义

### 三身详解

| 身 | 梵文 | 含义 | 特征 |
|---|---|---|---|
| **法身** | Dharmakāya | 佛与真理合一之身 | 无形无相，遍一切处，永恒不灭 |
| **报身** | Saṃbhogakāya | 佛因无量修行功德所得之身 | 具三十二相，常住净土，为菩萨说法 |
| **化身** | Nirmāṇakāya | 佛随缘示现于世之身 | 如释迦牟尼佛，有生有灭，为凡夫说法 |

### 各宗三身观

| 宗派 | 法身观 | 报身观 | 化身观 |
|------|--------|--------|--------|
| **唯识** | 清净法界 | 受用身（他受用/自受用） | 变化身 |
| **华严** | 毗卢遮那佛 | 卢舍那佛 | 释迦牟尼 |
| **天台** | 三身相即（一身即三身） | | |
| **密教** | 大日如来 | 金刚萨埵等 | 历史诸佛 |

### 三身的哲学意义

三身论解决了佛教的核心悖论：
- 佛已入灭 → 佛还在吗？
- 答：历史性的释迦牟尼是**化身**，已示现灭度；但**法身**（真理）永恒存在，**报身**（净土中）持续为菩萨说法

## 常见误读

- ❌ "三身 = 三个佛" → 三身是同一佛的三个维度，非三个独立存在。
- ❌ "法身 = 上帝" → 法身是真理本身，非人格化的造物主。
