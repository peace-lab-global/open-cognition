# 如来藏 Tathāgatagarbha

## 一句话定义

**如来藏**（Tathāgatagarbha）是大乘佛教的核心命题——一切众生本具**如来**（佛）的**藏**（本质/胚胎），烦恼只是客尘，真如才是主人。如来藏思想与佛性、空性、唯识的关系构成大乘佛教最深刻的本体论论争。

## 核心要义

### 1. 如来藏三义

| 义 | 含义 |
|---|------|
| **法身遍在** | 佛的法身遍一切处 |
| **真如无差别** | 众生与佛的真如无差别 |
| **种性具足** | 众生本具成佛的种子 |

### 2. 如来藏 vs 佛性 vs 空性

| 概念 | 核心 | 代表经论 |
|------|------|---------|
| **如来藏** | 众生本具佛的本质 | 《如来藏经》《胜鬘经》 |
| **佛性** | 众生本具成佛的潜能 | 《涅槃经》 |
| **空性** | 一切法无自性 | 《般若经》《中论》 |

**关键论争**：如来藏是否与空性一致？

| 立场 | 主张 | 代表 |
|------|------|------|
| **如来藏 = 空性** | 如来藏是空性的正面表达 | 中观派 |
| **如来藏 ≠ 空性** | 如来藏是超越空有的真实 | 如来藏系 |
| **如来藏 = 唯识** | 如来藏 = 阿赖耶识的清净面 | 唯识派 |

### 3. 如来藏的九喻

《宝性论》以九个比喻说明如来藏被烦恼遮蔽：

| 喻 | 含义 |
|---|------|
| 莲花在淤泥中 | 佛性在烦恼中 |
| 金在矿中 | 佛性在无明中 |
| 蜜在树中 | 佛性在身心中 |
| 谷在壳中 | 佛性在覆盖中 |
| 贫女怀王 | 不知自己有佛性 |
| 粪中金 | 最污秽处有佛性 |
| 果中种 | 佛性是成佛的种子 |
| 贫人衣中宝 | 不知自己有宝藏 |
| 焦模中金像 | 需要去除遮蔽 |

### 4. 一乘 vs 三乘之争

| 主张 | 内容 | 代表 |
|------|------|------|
| **一乘** | 一切众生终成佛 | 《法华经》《涅槃经》 |
| **三乘** | 五种姓决定不同归宿 | 《解深密经》《瑜伽师地论》 |

### 5. 如来藏与中国佛教

如来藏思想对中国佛教影响最大：
- **天台宗**：性具（一念三千）
- **华严宗**：性起（法界缘起）
- **禅宗**：自性清净（直指人心）
- **净土宗**：是心作佛，是心是佛

## 通俗 vs 学术

| 通俗化版本 | 学术原意 |
|---|---|
| "如来藏 = 灵魂" | 是超越个体的真如，非个体灵魂 |
| "一切众生有佛性 = 都能成佛" | 一乘 vs 三乘有争论 |
| "如来藏 = 梵我" | 佛教内部明确否定梵我 |

## 应用场景

- **哲学**：如来藏 = 本体论的正面表达
- **心理治疗**：本具佛性 = 本具健康潜能
- **教育学**：佛性思想 = 相信每个人都有学习潜能
- **宗教对话**：如来藏 vs 基督教"上帝的形象"

## 常见误读

- ❌ "如来藏 = 灵魂" → 是超越个体的真如，非个体灵魂。
- ❌ "如来藏 = 梵我" → 佛教内部明确否定梵我（ātman）。
- ❌ "如来藏 = 一切都能成佛" → 一乘 vs 三乘有争论。
