# 菩萨道 Bodhisattva Path

## 一句话定义

**菩萨道**是大乘佛教的核心修行路径——发**菩提心**（为利益一切众生愿成佛），修**六度万行**，经**三大阿僧祇劫**利益众生、最终成佛的道途。区别于声闻缘觉的自利解脱。

## 核心要义

### 1. 菩提心 Bodhicitta
- **愿菩提心**：为度众生愿成佛的愿
- **行菩提心**：实际修六度万行

### 2. 四弘誓愿
- 众生无边誓愿度
- 烦恼无尽誓愿断
- 法门无量誓愿学
- 佛道无上誓愿成

### 3. 菩萨十地 Daśabhūmi
欢喜地、离垢地、发光地、焰慧地、难胜地、现前地、远行地、不动地、善慧地、法云地

### 4. 三大阿僧祇劫
成佛需经极长时间修行（但密乘、净土、禅另有快速路径）。

### 5. 菩萨戒
《梵网经》十重四十八轻戒；《瑜伽师地论》三聚净戒。

## 通俗 vs 学术

| 通俗化版本 | 学术原意 |
|---|---|
| "菩萨 = 慈悲的神" | 菩萨是修行中的众生，非神 |
| "度众生 = 拯救" | 是引导，非代赎 |
| "三大阿僧祇劫 = 无限" | 是极长时间，非无限 |

## 应用场景

- **伦理学**：利他主义的典范
- **组织管理**：菩萨领导力
- **社会工作**：菩萨作为社会服务原型
- **心理学**：慈悲与自我超越

## 常见误读

- ❌ "菩萨道比声闻道高级" → 是路径不同，非优劣。
- ❌ "菩萨 = 观音等圣人" → 一切发菩提心者都是菩萨。
- ❌ "度众生是强迫" → 是随缘引导。
