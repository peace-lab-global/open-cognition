# 十二头陀行 Dhutanga

## 一句话定义

头陀行是佛教中十二种自愿苦行法的总称，旨在通过极端简朴的生活方式断除贪著、策励精进。

## 核心要义

### 十二法

| 类别 | 头陀支 | 内容 |
|------|--------|------|
| 衣服 | 粪扫衣支 | 只穿他人丢弃的布片缝制之衣 |
| | 三衣支 | 只持有三衣（僧伽梨/郁多罗僧/安陀会） |
| 饮食 | 常乞食支 | 每日挨户乞食，不挑贫富 |
| | 次第乞食支 | 按顺序逐户乞食，不跳跃 |
| | 一座食支 | 每日只坐一处进食 |
| | 一钵食支 | 所有食物混于一钵中食用 |
| | 时后不食支 | 午后不再进食 |
| 住处 | 阿兰若支 | 住于远离人群的空旷处 |
| | 树下住支 | 住于树下，不住房屋 |
| | 露地住支 | 住于露天，不避风雨 |
| | 冢间住支 | 住于墓地，观不净 |
| 精进 | 常坐不卧支 | 不躺卧，常坐修行（不倒单） |

### 修行原则

- **自愿性**：非戒律强制，而是自愿加行
- **对治性**：针对特定烦恼（贪衣/贪食/贪住/懈怠）
- **中道性**：目的在断贪而非自虐，佛陀反对极端苦行
- **时效性**：可根据根机择修数支，不必全修

### 历史传统

- **大迦叶**：头陀行第一，佛灭后主持结集
- **上座部**：至今保持头陀行传统（泰国森林派：阿姜曼、阿姜查）
- **汉传**：唐代道宣律师《四分律》中亦有头陀行记载

## 通俗 vs 学术

- **通俗**：常被误解为"自虐式苦行"
- **学术**：属于"戒学"范畴的自愿加行法，目的在于减少对外缘的依赖以专注于禅修，与外道苦行有本质区别

## 与相关概念的关系

- [四圣谛](../../traditions/buddhism/four-noble-truths.md) [互补] — 头陀行以苦谛为出发点，助成道谛的修习
- [八正道](../core-concepts/eightfold-path.md) [下位] — 头陀行是正命（正确生活方式）的极端实践
- [三学](../core-concepts/three-trainings.md) [下位] — 头陀行属于戒学的加行
- [中道](../core-concepts/middle-way.md) [张力] — 头陀行看似极端，但以"断贪"为目的而非自虐，仍属中道范畴

## 常见误读

- **误读**：头陀行就是外道苦行
- **正解**：外道苦行以苦为解脱手段；头陀行以简朴生活减少对缘的执著，目的是创造禅修条件，本质是"减缘"而非"受苦"
