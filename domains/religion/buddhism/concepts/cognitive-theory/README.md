## 五层认知框架（完整版）

```
究竟位（佛地）
  四智圆明 → 大圆镜智/平等性智/妙观察智/成所作智
              ↑
修道位（初地~十地）
  后得无分别智 → 断俱生我执
              ↑
见道位（初地）
  根本无分别智 → 实证空性（般若）
              ↑
加行位（暖/顶/忍/世第一）
  相似见到空性
              ↑
资粮位（外凡）
  闻思正法，种下种子
═══════════════════════════════════════════════════
感知层（前五识）→ 思维层（第六识）→ 我执层（第七识）→ 种子层（第八识）
  成所作智          妙观察智            平等性智          大圆镜智
```

## 设计原则

1. **理论完整性**：每个概念都有完整的经典来源和理论脉络
2. **可操作性**：每个Skill都有具体的操作流程和提问范式
3. **系统性**：概念之间有清晰的逻辑关系，产出完整的认知地图
4. **跨学科连接**：每个条目都与现代心理学/哲学建立关联
5. **不传教**：从认知科学角度提取，学术中立
