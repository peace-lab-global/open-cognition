# 净土真宗 Jōdo-Shinshū

## 一句话定义

净土真宗是日本佛教最独特的宗派——由**亲鸾**（1173–1263）创立，以**"恶人正机"**和**"绝对他力"**为核心，将净土信仰推向极致：否定一切自力修行，主张唯有完全信赖阿弥陀佛的本愿力（第十八愿）才能往生净土，是日本净土宗最激进的逻辑发展。

## 历史脉络

- **1173**：亲鸾生于京都，贵族家庭
- **1201**：从法然（净土宗创立者）学法
- **1207**：因"承元法难"被流放越后（新潟），在流放期间深化思想
- **1224**：著《教行信证》，系统阐述净土真宗教义
- **1263**：亲鸾圆寂
- **13–15c**：本愿寺系统兴起，成为日本最大佛教势力
- **15–16c**：一向一揆（净土真宗信众的武装运动）
- **当代**：净土真宗是日本最大佛教宗派之一（约 1000 万信徒）

## 核心要义

### 1. 绝对他力

| 概念 | 含义 |
|------|------|
| **他力** | 完全依赖阿弥陀佛的本愿力——不是自己修行 |
| **绝对** | 不附加任何条件——不需要持戒、不需要禅定、不需要智慧 |
| **信心** | 不是"信"的行为，而是被阿弥陀佛的信心所"摄取" |

### 2. 恶人正机

亲鸾的颠覆性命题：
- **"善人尚能往生，何况恶人乎"**（《叹异抄》）
- 善人依赖自力修行，反而产生"我修得好"的傲慢
- 恶人（无自力修行能力者）才是阿弥陀佛本愿的真正对象
- 这不是鼓励作恶，而是对"自力执着"的彻底破除

### 3. 非僧非俗

亲鸾是日本佛教史上第一位公开结婚的僧人：
- 自称"非僧非俗"——既非出家人也非在家人
- 肉食妻带——净土真宗僧人可以结婚、吃肉
- 这一立场深刻影响了日本佛教的世俗化

### 4. 与净土宗的对比

| 维度 | 净土宗（法然） | 净土真宗（亲鸾） |
|------|--------------|----------------|
| 核心 | 念佛往生 | 信心往生 |
| 修行 | 称名念佛（口称南无阿弥陀佛） | 信心（被本愿力摄取） |
| 立场 | 相对他力（念佛为条件） | 绝对他力（无条件的信心） |
| 戒律 | 保持僧形 | 非僧非俗 |
| 恶人 | 恶人也可念佛往生 | 恶人正机——恶人是首要对象 |

## 通俗 vs 学术

| 通俗化版本 | 学术原意 |
|---|---|
| "净土真宗鼓励作恶" | "恶人正机"是对自力执着的破除，不是道德许可 |
| "亲鸾是第一个结婚的和尚" | 是第一个**公开**主张僧人可婚娶的，之前有私下的实践 |
| "净土真宗 = 基督教式的信仰" | 虽有表面相似性，但亲鸾的"信心"来自他力的摄取，不是主体的信念行为 |

## 常见误读

- ❌ "绝对他力 = 不需要做任何事" → 亲鸾否定的是"自力的执着"，不是生活本身。真宗信众有正常的社会伦理生活。
- ❌ "净土真宗 = 日本净土宗" → 净土宗（法然系）与净土真宗（亲鸾系）是不同宗派。
