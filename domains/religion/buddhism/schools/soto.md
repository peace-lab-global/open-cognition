# 曹洞宗 Sōtō

## 一句话定义

曹洞宗是禅宗两大主流之一——以**默照禅**（只管打坐 / shikantaza）为核心修行方法，由道元从中国传入日本后独立发展，强调**"修证一如"**（修行本身就是证悟，不是证悟的手段），形成了与临济宗截然不同的沉静禅风。

## 历史脉络

- **9c**：洞山良价、曹山本寂创立曹洞宗（中国）
- **13c**：**道元**（1200–1253）入宋学禅，将曹洞宗传入日本
- **13–14c**：道元建永平寺，以严格清规和纯粹禅风立宗
- **14c**：**莹山绍瑾**（1268–1325）将曹洞禅通俗化，使之深入民间
- **17–18c**：面山瑞方等整理道元著作
- **20c**：西方对曹洞禅的关注（与正念运动的对话）

## 核心要义

### 1. 默照禅 / 只管打坐（Shikantaza）

| 要素 | 说明 |
|------|------|
| **只管** | 不求开悟、不求功德、只是打坐 |
| **打坐** | 不是手段，坐本身就是目的 |
| **默照** | 默（无造作）+ 照（清明觉知）——非选择性觉察 |

### 2. 修证一如

道元的根本主张：
- 修行不是证悟的"因"，证悟不是修行的"果"
- **"修证一如"**：坐禅的每一刻都是完全的证悟——不需要等到"开悟"
- 这与临济宗"以公案引发开悟"的路径根本不同

### 3. 有时论（Uji）

道元的时间哲学：
- 时间不是"流过"的——"有时"（being-time）= 每一个瞬间就是一个完整的存在
- "春花秋月"不是时间中的事件，而是时间本身

### 4. 与临济宗的对比

| 维度 | 曹洞宗 | 临济宗 |
|------|--------|--------|
| 方法 | 默照禅（只管打坐） | 看话禅（参公案） |
| 核心 | 修证一如 | 疑情→破参 |
| 开悟观 | 坐即是悟 | 需要突破 |
| 祖师 | 道元→莹山 | 荣西→白隐 |
| 规模 | 日本最大禅宗（约 1.5 万寺） | 约 6000 寺 |

## 通俗 vs 学术

| 通俗化版本 | 学术原意 |
|---|---|
| "曹洞 = 坐着发呆" | 只管打坐是高度清醒的非选择性觉察 |
| "道元反对一切理论" | 道元反对"以理论取代实践"，他自己写了大量深刻的哲学 |
| "修证一如 = 不需要修行" | 恰恰相反——正因为修即证，所以修行更加重要 |

## 常见误读

- ❌ "曹洞宗不讲开悟" → 道元讲的是"修证一如"——开悟不是修行的结果，而是修行本身。
- ❌ "默照禅 = 正念" → 现代正念通常有明确的对象（呼吸/身体），默照禅是"无对象"的非选择性觉察。
