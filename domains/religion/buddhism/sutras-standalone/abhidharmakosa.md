# 俱舍论 · Abhidharmakośa

> "聪明论"——一部让佛教哲学系统化的百科全书，也是世亲从有部转向经量部的思想转折点。

## 一句话定义

《俱舍论》是 5 世纪印度论师**世亲**（Vasubandhu）所造的阿毗达磨（Abhidharma）系统性巨著——它以**说一切有部**（Sarvāstivāda）哲学为骨架，用**经量部**（Sautrāntika）立场做批判性注解，建立了**五位七十五法**的完整法相体系，是理解部派佛教哲学的**标准教科书**。

## 历史脉络

- **2–3 世纪**：说一切有部《大毗婆沙论》完成，体系庞杂。
- **4 世纪**：法胜（Dharmatrāta）《杂阿毗昙心论》、法救（Dharmapāla）等尝试简化有部体系。
- **5 世纪（约 400–430）**：**世亲**造《俱舍论颂》（kārikā，600 颂）+ 自释（bhāṣya），以经量部立场批判有部。
- **563–567**：**真谛**（Paramārtha）首译为汉文。
- **651–654**：**玄奘**重译为《阿毗达磨俱舍论》30 卷，成为标准本。
- **7 世纪**：中国成立**俱舍宗**（→ [俱舍宗](../schools/kosa.md)），与法相唯识宗并行。
- **当代**：Louis de La Vallée Poussin 法译（1923–1931）、Leo Pruden 英译（1988–1990），成为国际佛学标准参考。

## 核心要义

### 1. 五位七十五法：存在的全分类

《俱舍论》将一切存在（法/dharma）分为五大类、75 种：

| 位 | 类别 | 法数 | 核心内容 |
|---|---|:---:|---|
| **色法** | 物质 | 11 | 五根 + 五境 + 无表色 |
| **心法** | 主体认知 | 1 | 心王（一心） |
| **心所法** | 心理伴随 | 46 | 大地法(10) / 大善地法(10) / 大烦恼地法(6) / 大不善地法(2) / 小烦恼地法(10) / 不定地法(8) |
| **心不相应行法** | 非心非物之抽象 | 14 | 得、非得、同分、无想果、灭尽定、命根、生住异灭、名身句身文身 |
| **无为法** | 不生灭 | 3 | 虚空无为、择灭无为、非择灭无为 |
| **合计** | | **75** | |

> 此体系后来被唯识学派扩展为**百法明门**（→ [百法明门](../core-concepts/hundred-dharmas.md)）。

### 2. 三世实有 vs 现在有体

《俱舍论》核心论争之一：

- **有部立场**：三世（过去、现在、未来）法体皆有——"三世实有，法体恒有"。
- **经量部立场**：**只有现在一刹那实有**，过去已灭、未来未生。
- **世亲的态度**：在《颂》中呈有部说，在《释》中多以经量部破有部——这一"以释破颂"的写法本身就是思想史的戏剧。

### 3. 四缘六因

因果关系的精密分析：

**四缘**：
| 缘 | 含义 |
|---|---|
| 因缘 | 直接原因（种子→芽） |
| 等无间缘 | 前念灭为后念生开道 |
| 所缘缘 | 认知对象作为条件 |
| 增上缘 | 不妨碍即为助缘 |

**六因**：能作因、俱有因、同类因、相应因、遍行因、异熟因——对四缘的进一步细化。

### 4. 修道论：从凡夫到圣者

| 阶段 | 名称 | 核心内容 |
|---|---|---|
| 1 | 资粮道 | 积累福德与智慧资粮 |
| 2 | 加行道 | 暖、顶、忍、世第一法 |
| 3 | 见道 | 现观四谛（16 刹那） |
| 4 | 修道 | 断除修所断烦恼 |
| 5 | 无学道 | 阿罗汉果 |

### 5. 无我与补特伽罗

《俱舍论》第 9 品"破我品"（Pudgalaviniścaya）是全书哲学高潮：

- **犊子部**主张"补特伽罗"（pudgala，不可说的我）实有。
- **世亲**以严密论证**破补特伽罗实有**——"若离诸蕴别无我，则我应如虚空等不可见"——但承认世俗假名的"我"。
- 这一论证对后世**无我论**影响深远。

### 6. "聪明论"之名

《俱舍论》被称为"聪明论"——因其**体系严密、论证明晰、以简驭繁**，学完此论即通晓部派佛教全部哲学议题。在中国佛教传统中，它被视为"小乘之极、大乘之阶"。

## 通俗 vs 学术

| 通俗化版本 | 学术原意 |
|---|---|
| "75 法 = 75 种东西" | 法是分析单位，非实体；"法"= 可被心智辨识之最小特征 |
| "世亲反对有部" | 世亲在有部框架内做经量部批判，非简单对立 |
| "俱舍 = 小乘" | 是系统化阿毗达磨，大乘唯识直接继承其方法论 |
| "无为法 = 虚无" | 无为法是"非因缘所生"，择灭 = 涅槃的积极表述 |

## 应用场景

- **佛学院教育**：汉传佛学院必修课，"先学俱舍，再学唯识"
- **佛教哲学研究**：理解部派论争的标准参照
- **比较哲学**：与亚里士多德范畴论、维特根斯坦逻辑原子论对话
- **认知科学**：46 心所法可作为古代"情绪/认知分类学"
- **心理治疗**：心所分析可作为情绪粒度训练的理论基础

## 常见误读

- ❌ "俱舍论 = 小乘 = 低等" → 它是佛教哲学的系统基石，大乘唯识直接继承其方法论。
- ❌ "75 法 = 实体论" → 法是分析单位，非独立实体；"一切法"不等于"一切实体"。
- ❌ "世亲完全站在经量部" → 世亲在有部框架内做批判，很多处仍保留有部说法。
- ❌ "破我品 = 否认人格" → 破的是实有的"补特伽罗"，世俗假名的"我"仍被承认。
