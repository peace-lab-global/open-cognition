# 大乘起信论 · Awakening of Faith in Mahāyāna

> "一心开二门：心真如门、心生灭门。"——大乘起信论核心

## 一句话定义

《大乘起信论》是大乘佛教最具**哲学架构性**的论典——以"**一心二门**"（心真如门/心生灭门）为核心，系统阐述**如来藏缘起**，将如来藏（真如）与生灭现象统一于"一心"之中，提出**信/解/行/证**的修行次第，深刻影响了中国天台、华严、禅宗的思想发展。

## 历史脉络

- **约 5–6 世纪**：传为马鸣（Aśvaghoṣa）所著，真谛（Paramārtha）译。
- **真伪争议**：部分学者认为是中国撰述，但其思想体系与如来藏系完全一致。
- **6 世纪以降**：慧远、智顗、法藏、宗密广泛注释。
- **唐以降**：与《楞严经》《圆觉经》并称"楞严、圆觉、起信"三大家。
- **现代**：铃木大拙英译，进入西方学术主流。

## 核心要义

### 1. 一心二门

| 门 | 含义 | 性质 |
|---|------|------|
| **心真如门** | 心的真如本体——不生不灭、不增不减 | 如来藏 |
| **心生灭门** | 心的生灭现象——念念生灭、因果相续 | 阿赖耶识 |

**关键**：二门不是两个心，而是**同一个心的两面**——如金与器、水与波。

### 2. 如来藏缘起

真如（如来藏）是一切法的本体，无明是烦恼的根源：

```
真如（本体）
    ↓ 不觉（无明）
生灭（现象）
    ↓ 始觉（修行）
真如（回归）
```

真如不是"产生"了生灭，而是**在无明的作用下显现为生灭**。

### 3. 觉与不觉

| 层次 | 含义 |
|------|------|
| **本觉** | 一切众生本具的觉性 |
| **不觉** | 被无明遮蔽的状态 |
| **始觉** | 开始觉悟的过程 |
| **究竟觉** | 完全觉悟 |

本觉 → 不觉 → 始觉 → 究竟觉 = 修行的完整路径。

### 4. 信解行证

| 阶段 | 内容 |
|------|------|
| **信** | 信真如、信佛、信法、信僧 |
| **解** | 理解一心二门 |
| **行** | 止观双运 |
| **证** | 证入真如 |

### 5. 三种熏习

| 熏习 | 含义 |
|------|------|
| **真如熏无明** | 真如的力量使无明逐渐减弱 |
| **无明熏真如** | 无明的力量使真如被遮蔽 |
| **业识熏** | 业力与识的相互作用 |

## 通俗 vs 学术

| 通俗化版本 | 学术原意 |
|---|---|
| "一心二门 = 两个心" | 是同一心的两面，非两个实体 |
| "起信论是伪经" | 真伪有争议，但思想体系完整 |
| "如来藏缘起 = 梵我论" | 佛教内部明确否定梵我 |

## 应用场景

- **哲学**：一心二门 = 本体论与现象论的统一框架
- **心理学**：本觉/不觉 = 心理健康/心理障碍的模型
- **教育学**：信解行证 = 学习过程的完整模型
- **管理学**：真如/生灭 = 愿景/执行的统一

## 常见误读

- ❌ "一心二门 = 两个心" → 是同一心的两面。
- ❌ "起信论是伪经" → 真伪有争议，但思想深度不可否认。
- ❌ "如来藏缘起 = 梵我论" → 佛教内部明确否定梵我。
