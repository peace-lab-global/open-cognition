# 宗教 · Religion

本目录以学术中立视角呈现六大宗教传统的核心思想体系，按传统组织条目，按概念组织跨传统观念。

## 立场说明

本库的宗教条目：
- 从历史、哲学、文化角度呈现教义内涵
- 不作"哪个宗教正确"的判断
- 不进行传教或反宗教论述
- 在涉及神学争议时呈现内部多元立场
