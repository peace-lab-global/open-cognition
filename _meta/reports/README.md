## 命名规范

新报告请使用以下格式：

```
reports/<category>/YYYY-MM-DD-<kebab-case-subject>.md
```

| category | 适用 |
|----------|------|
| `audit/` | 全库覆盖/质量审计 |
| `phase-<N>/` | 特定路线图阶段的计划/执行报告 |
| `cse/` | 认知系统工程专题 |
| `religions/` | 宗教/文学/艺术专题 |
| `buddhism/` | 佛教认知专题（规划中） |
| `misc/` | 其他 |

文件名一律小写 + 短横线，日期前置以便排序。
