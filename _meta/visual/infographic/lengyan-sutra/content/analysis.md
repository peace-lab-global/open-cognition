# Analysis: 楞严经全经脉络信息图

## Topic
楞严经（Śūraṅgama Sūtra）全经十卷核心脉络可视化

## Data Type
宗教/佛经教育类信息图，叙事性结构

## Complexity
高复杂度：
- 7大核心章节/阶段
- 25位圆通圣者
- 50种阴魔
- 需要在一页内完整呈现

## Tone
庄严、典雅、具有东方佛教美学气质

## Audience
佛学研究者、修行者、对佛教经典感兴趣的读者

## Design Instructions
- 风格：storybook-watercolor（绘本水彩风格）
- 布局：需要一页纸全貌 → 高密度模块化布局（dense-modules）结合时间线脉络
- 语言：中文
- 关键要求：信息完整、视觉层次清晰、水彩手绘感
