# Analysis: Hermann Hesse Infographic

## Topic
赫尔曼·黑塞 (Hermann Hesse) — 生平、作品、思想源流与核心主张的全景脉络图

## Data Type
人物传记 + 文学思想 + 跨学科影响
- 时间线数据：9个关键生平节点（1877-1962）
- 作品数据：6部核心代表作
- 阶段数据：4个创作阶段
- 思想数据：6大影响源 + 5大核心思想
- 连接数据：4个跨学科接口

## Complexity
高复杂度 — 多维度信息需要在单页中呈现
- 时间维度：85年生平跨度
- 作品维度：6部核心小说
- 思想维度：东西方哲学融合
- 影响维度：6大思想源流

## Tone
知性、温暖、略带神秘色彩
- 黑塞作品本身具有浪漫主义和东方禅意的气质
- storybook-watercolor 风格（柔和水彩、童话绘本）与黑塞精神世界的梦幻感高度契合

## Audience
文学爱好者、知识工作者、对精神世界探索感兴趣的读者
- 需要一目了然地把握黑塞全貌
- 希望看到清晰脉络而非碎片化信息

## Design Instructions from User
- 风格：storybook-watercolor（柔和水彩、童话绘本）
- 要求：一页纸可以看到全貌
- 核心诉求：讲清楚黑塞的"整个脉络"

## Language
中文为主，保留关键作品/人名原文

## Recommended Layout
`bento-grid` — 多维度模块化布局，适合在单页中展示人物生平、作品、思想源流、核心主张等多个主题。
- 替代方案：`winding-roadmap`（强调精神旅程），但信息密度不如 bento-grid

## Recommended Aspect Ratio
`landscape` (16:9) — 横向布局最适合一页全貌的信息图展示
