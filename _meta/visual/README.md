# 可视化资源目录

本项目所有信息图源文件与 HTML/SVG 可视化文件统一存放于此。

## 目录结构

```
visual/
├── README.md              ← 本文件
├── infographic/           ← Notion 风格信息图源文件与专题
│   ├── hermann-hesse/     ← 赫尔曼·黑塞专题
│   ├── lengyan-sutra/     ← 棱严经专题
│   └── notion-infographic-prompt.md
├── cse/                   ← 认知系统工程架构可视化
├── domain/                ← 各领域专属信息图（9 大领域）
└── general/               ← 综合信息图、Notion 风格大图、时间线
```
