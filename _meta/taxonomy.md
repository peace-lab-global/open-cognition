# 分类体系 · Taxonomy

本文档说明 open-cognition 的三层分类逻辑及其设计理由。

## 三层结构

```
领域 (Domain)
  └─ 学派/传统 (School / Tradition)
        └─ 思想家/核心概念 (Thinker / Core Concept)
```

### 第一层：领域

四个领域并非彼此独立，而是同一组人类根本问题的不同探究路径：

| 领域 | 主要追问 |
|---|---|
| 哲学 | 什么是真、善、美？什么可以被认识？ |
| 宗教 | 何为终极实在？人如何获得救赎/解脱？ |
| 社会学 | 社会如何可能？秩序与变迁的机制是什么？ |
| 心理学 | 心智如何运作？人格如何形成与转化？ |

四个领域共享一组横切主题（见 [TAGS.md 主题维度](../TAGS.md#维度-4主题-theme)），通过主题标签实现跨领域互联。

### 第二层：学派 / 传统

| 领域 | 主要分类轴 |
|---|---|
| 哲学 | 时代+地域（古希腊、德国唯心论、东方哲学等） |
| 宗教 | 宗教传统（佛教、基督教、伊斯兰、犹太、印度、道教） |
| 社会学 | 理论范式（功能主义、冲突论、互动论、批判理论） |
| 心理学 | 理论流派（精神分析、行为主义、认知、人本、积极） |

### 第三层：思想家 / 核心概念

两类原子条目：

- **思想家条目**（thinker）：以个人为中心，记录其生平、思想脉络、关键贡献
- **概念条目**（concept）：以观念为中心，可由多位思想家共同贡献

两类条目互相引用：思想家条目链接其提出的概念，概念条目反向链接所有相关思想家。

## 为什么不按"主题"作顶层

主题（如"自我""权力"）是横切轴，若作顶层会割裂思想家的整体脉络。本库选择"领域→学派→人物/概念"作纵向主轴，"主题"作横向标签，由 [INDEX.md](../INDEX.md) 提供主题视图。

## 跨领域条目归属规则

当一位思想家横跨多个领域时（如弗洛姆同时是精神分析师与社会理论家），按其**最主要贡献**的领域归档，并在另一领域留存交叉链接条目。
