## 研究方法

每个课题研究遵循以下结构：

1. **问题界定** — 明确核心问题与子问题
2. **全球思想扫描** — 按文明/传统/学科系统梳理相关智慧
3. **核心洞见提炼** — 跨传统共性与独特贡献
4. **实践路径整合** — 可操作的方法论综合
5. **进阶阅读地图** — 原典与研究文献指引
