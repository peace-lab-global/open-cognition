# 没用的清单 · Useless Lists

> 一个自媒体频道的内容素材库。每张清单本身就是一个「奇葩盘点」——标题即钩子，10 条为一组，列的是人类文明里那些「看起来没用、却让人挪不开眼」的冷知识。

每条清单独立成文件，按**人物类别**组织为三十一个子目录。表格保留参考价值（人物／时代／成就），但真正的「钩子」是最后那列「没用真相」——具体、真实、离奇，绝不编造。

## 列规范（最佳实践）

每张清单统一采用 **6 列**，顺序固定，便于横向阅读与跨清单跳转：

| 列 | 字段 | 作用 | 写法 |
|----|------|------|------|
| 1 | **#** | 序号 | 01–10，零填充 |
| 2 | **人物（生卒）** | 定位 | 中文名 + 原文名 + 生卒年 |
| 3 | **国／时代** | 背景 | 国籍 + 世纪或流派 |
| 4 | **传世成就** | 参考价值 | 代表作 / 核心贡献（一句话） |
| 5 | **「没用」真相** | 钩子兑现 | 对应标题角度的具体离奇事实（最长一列，必须真实、可查） |
| 6 | **🔗 仓库** | 知识图谱 | 链接到本库已有思想家文件；无则 `—` |

> **铁律**：「没用真相」一列只写**有据可查**的事实，不杜撰、不夸张。荒诞的是历史本身。
