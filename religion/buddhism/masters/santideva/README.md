# 寂天 Santideva

> "所有世间的快乐，来自利益他人；所有世间的痛苦，来自追求自利。"
> —《入菩萨行论》8:129 (Bodhicaryavatara)

> "Whatever joy there is in this world, all comes from desiring others to be happy. Whatever suffering there is in this world, all comes from desiring oneself to be happy."
> — Bodhicaryavatara 8:129

## 基本信息 / Basic Information

**寂天 Santideva**（685-763 CE），印度大乘佛教最具感染力的伦理思想家与哲学诗人，中观学派 Madhyamaka 晚期最重要的代表之一。他以《入菩萨行论》(Bodhicaryavatara) 将大乘菩萨道的伦理实践提升为深刻的哲学诗篇，并以第九品"智慧品"将中观空性哲学融入菩提心修行，实现了伦理实践与形而上学智慧的统一。此论至今仍是藏传佛教最重要的修行读物之一，也被广泛视为世界伦理文学的经典。

- **出生地 / Birthplace**: 西印度梭罗修多罗国 (Saurashtra)，王族出身
- **年代 / Era**: 685-763 CE，笈多王朝后期至波罗王朝初期
- **身份 / Role**: 中观学派哲学家、伦理思想家、那烂陀寺比丘
- **传承 / Lineage**: 继承龙树 Nagarjuna 中观传统，深刻影响藏传佛教各派
- **核心经典 / Core Text**: 《入菩萨行论》Bodhicaryavatara (十品，约 900 颂)

## 核心命题 / Core Propositions

### 1. 自他平等 Paratmasamata / Self-Other Equality

通过严密的逻辑论证"自"与"他"的对称性，打破自我中心的伦理偏见。寂天论证：正如我的痛苦对我来说不可忍受，他人的痛苦对他人而言同样不可忍受——在"不欲苦"这一点上，自他完全平等。因此，只除自苦而不除他苦，在逻辑上没有合理的根据。

**Through rigorous logical reasoning, Santideva demonstrates the fundamental symmetry between self and other: since all beings equally wish to avoid suffering, privileging one's own suffering over others' has no rational justification.**

详见 [自他平等](concepts/self-other-equality.md)。

### 2. 忍辱的哲学 Ksanti-paramita / The Philosophy of Patience

忍辱不是被动的忍受或懦弱的屈服，而是一种深刻的哲学洞察——通过理解嗔恨 (krodha) 的逻辑荒谬来消解嗔恨。寂天的关键论证：嗔恨预设了一个真正的"加害者"和一个真正的"受害者"，但在缘起的世界中，伤害是众多因缘的产物，没有独立自存的加害者可以被嗔恨。

**Patience is not passive endurance but an active philosophical insight — understanding that anger rests on the illusion of an independently existing agent of harm, when in reality all harm arises from a complex web of conditions.**

详见 [忍辱哲学](concepts/patience-philosophy.md)。

### 3. 自他交换 Paratmaparivartana / Self-Other Exchange

第八品的核心修行方法——将"我"的幸福给予他人，将他人的痛苦取为自己。这不是一种字面意义上的痛苦转移，而是一种深刻的心智训练 (bhavana)，通过改变认知模式来瓦解自我中心 (atma-sneha) 的惯性。

**Self-other exchange is a meditative practice that systematically reverses the habitual self-cherishing attitude, training the practitioner to take on others' suffering and give away one's own happiness — a cognitive restructuring that dismantles egocentrism from within.**

详见 [自他交换](concepts/self-other-exchange.md)。

### 4. 智慧与方便的统一 Prajna-upaya-yuganaddha / Unity of Wisdom and Means

第九品"智慧品"是全书的哲学顶峰——寂天在此以中观空性哲学为菩提心修行提供最终的净化。伦理实践 (方便 upaya) 若没有空性智慧 (prajna) 的引导，仍然可能被自我执着所污染；而空性智慧若没有慈悲的实践，则沦为纯智力的游戏。二者必须统一。

**The ninth chapter presents the philosophical summit: ethical practice (means/upaya) without wisdom (prajna) remains contaminated by self-grasping, while wisdom without compassionate practice degenerates into intellectual exercise. Their union is the heart of the bodhisattva path.**

详见 [智慧与方便统一](concepts/wisdom-means-unity.md)。

### 5. 菩提心修 Bodhicitta-bhavana / Cultivation of the Awakening Mind

菩提心 (bodhicitta) 是寂天整个思想体系的核心驱动力——为了一切众生的利益而追求圆满觉悟的誓愿。寂天系统阐述了菩提心的生起、护持、增长和圆满的全过程，将其从抽象的理想转化为可操作的修行次第。

**Bodhicitta — the aspiration to attain full awakening for the benefit of all beings — is the central engine of Santideva's entire philosophical system, systematically developed from initial arising through protection, growth, and ultimate perfection.**

详见 [菩提心修](concepts/bodhicitta-cultivation.md)。

## 思想脉络 / Intellectual Context

### 前驱 / Predecessors

- **龙树 Nagarjuna** (约 150-250): 中观空性哲学的创始人，寂天第九品"智慧品"直接继承其反本质主义立场
- **般若经传统 Prajnaparamita**: 空性智慧与菩萨行的经典源头
- **弥勒 Maitreya / 无著 Asanga**: 菩萨道次第的传统 (《现观庄严论》等)
- **月称 Candrakirti** (约 600-650): 中观应成派集大成者，寂天在哲学立场上与之最为接近

### 同时代 / Contemporaries

- **寂护 Santaraksita** (约 725-788): 瑜伽行中观派综合者，与寂天大致同时代
- 那烂陀寺 Nalanda 的学术环境——当时世界最大的佛教学术中心

### 后继 / Successors

- **Prajnakaramati** (10 世纪): 《入菩萨行论》最重要的印度注释者 (Panjika 注释)
- **阿底峡 Atisa** (982-1054): 将寂天思想系统传入西藏
- **宗喀巴 Tsongkhapa** (1357-1419): 以寂天《入菩萨行论》为菩提道修行的核心文本
- **历代达赖喇嘛**: 尤其是第十四世达赖喇嘛，多次公开讲授此论

## 思想坐标 / Intellectual Positioning

### 哲学定位

- **中观伦理学 Madhyamaka Ethics**: 将龙树的空性哲学转化为具体的伦理实践——在佛教思想史上，寂天是第一个系统完成这一转化的人
- **利他主义哲学 Altruistic Philosophy**: 以严密的逻辑论证利他行为的合理性，可与西方功利主义、关怀伦理学对话
- **情感哲学 Philosophy of Emotions**: 对嗔恨的认知分析——嗔恨源于错误的归因 (将伤害归因于独立的"加害者")
- **修行心理学 Contemplative Psychology**: 菩提心的修习不是道德说教，而是系统的心理训练 (bhavana)

### 与佛教内部的关系

- **vs 龙树**: 寂天继承龙树的中观立场，但将其从纯哲学论证转向伦理实践——如果说龙树确立了"空"的理论，寂天则展示了"空"如何驱动慈悲
- **vs 唯识**: 寂天在第九品中批判唯识学派的"自证分" (svasamvedana) 理论
- **vs 声闻乘**: 寂天批评小乘修行者只求自我解脱的局限性——但这是哲学批评，不是人格贬低

## 当代应用 / Contemporary Applications

### 伦理学 / Ethics

- **功利主义 Utilitarianism**: 寂天的自他平等论证与彼得·辛格 Peter Singer 的有效利他主义 (Effective Altruism) 形成结构性平行——二者都要求超越自利偏见
- **关怀伦理学 Care Ethics**: 寂天对慈悲 (karuna) 的哲学分析为关怀伦理学提供了跨文化视角
- **动物伦理 Animal Ethics**: 寂天的利他论证可延伸至非人众生——"一切有情皆不欲苦"

### 心理学 / Psychology

- **情绪管理 Emotion Regulation**: 忍辱哲学对嗔恨的认知分析，与现代认知行为疗法 (CBT) 对愤怒的认知重构有结构性相似
- **共情训练 Empathy Training**: 自他交换修法与心理学中的观点采择 (perspective-taking) 训练
- **正念 Mindfulness**: 第五品"正知"对心念守护的阐述，与当代正念训练的对话

### 利他主义研究 / Altruism Studies

- **有效利他主义 Effective Altruism**: 寂天的论证为"为何应该利他"提供了哲学基础
- **亲社会行为 Prosocial Behavior**: 自他平等修法的心理学效应研究

### 教育学 / Education

- **品格教育 Character Education**: 《入菩萨行论》作为伦理教育的系统文本
- **冲突解决 Conflict Resolution**: 忍辱哲学在冲突调解中的应用

## 常见误读 / Common Misreadings

### 1. "《入菩萨行论》只是伦理教导"

**误读**: 这部论只是关于如何做好人、如何利他的伦理教导。
**正解**: 第九品"智慧品"是深奥的中观哲学——寂天在此对唯识学派的自证分、对外境的实在论进行了严密的哲学批判。伦理实践必须以空性智慧为引导，否则利他行为本身也可能被自我执着所污染。

### 2. "自他交换是字面意义上的取他苦"

**误读**: 自他交换意味着真的把别人的痛苦取到自己身上。
**正解**: 这是一种心智训练 (bhavana)，通过改变认知模式来打破自我中心的惯性。寂天自己说这是一种修习 (abhyasa)，是通过反复训练来改变心理倾向。

### 3. "忍辱 = 被动忍受一切"

**误读**: 寂天教人无条件忍受一切伤害和不公。
**正解**: 忍辱 (ksanti) 是一种主动的哲学洞察——通过分析伤害的缘起性来消解嗔恨的心理基础。这不是对不公的默许，而是消除嗔恨后以更清醒的心智去回应。

### 4. "寂天否定自我的存在"

**误读**: 寂天在第九品否定了自我的存在，所以一切都是虚幻的。
**正解**: 寂天否定的是自性 (svabhava) 意义上的自我——独立、固定、不变的实体性自我。世俗假名意义上的自我 (conventional self) 在修行过程中仍然被保留和运用。
