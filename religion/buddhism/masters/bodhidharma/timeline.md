# 菩提达摩生平年表 / Bodhidharma Timeline

本年表整合了历史记载、传说与学术考证，区分了可靠史实与传说成分。

This timeline integrates historical records, legends, and scholarly research, distinguishing between reliable historical facts and legendary elements.

---

## 印度时期 / Indian Period

| 时间 / Date | 事件 / Event | 性质 / Nature |
|---|---|---|
| 约 470 CE | 生于南印度，婆罗门种姓，为香至国（Kanchipuram 一带）国王第三子 | 传说 Legend |
| 约 490 | 出家为僧，师从般若多罗（Prajñātāra），受般若多罗指点前往中国传法 | 传说 Legend |
| 约 490–520 | 在印度修行，学习如来藏思想和般若空观 | 推断 Inferred |

---

## 来华初期 / Early Period in China

| 时间 / Date | 事件 / Event | 性质 / Nature |
|---|---|---|
| 约 520 CE | 渡海来华，先到达广州（一说南朝梁国境内） | 半史实 Semi-historical |
| 约 520 | 前往建康（今南京），谒见梁武帝萧衍 | 传说成分较重 Largely legendary |
| 约 520 | 与梁武帝对话："朕造寺度僧，有何功德？""无功德。"——帝不悦 | 传说 Legend |
| 约 520 | 达摩见与梁武帝不契，渡江北上（传说"一苇渡江"） | 传说 Legend |

---

## 嵩山修行时期 / Mount Song Practice Period

| 时间 / Date | 事件 / Event | 性质 / Nature |
|---|---|---|
| 约 520–527 | 到达北魏境内，在嵩山少林寺一带活动 | 半史实 Semi-historical |
| 约 527 | 开始在少林寺面壁修行，"面壁而坐，终日默然" | 半史实 Semi-historical |
| 约 527–536 | 面壁九年，以壁观为核心修行方法 | 传说与史实混合 Mix of legend and history |
| 约 527–536 | 逐渐吸引求法者，形成早期的达摩禅僧团 | 推断 Inferred |

---

## 传法时期 / Dharma Transmission Period

| 时间 / Date | 事件 / Event | 性质 / Nature |
|---|---|---|
| 约 536 | 神光（后改名慧可）来参访，断臂求法（一说自断，一说被贼所伤） | 传说 Legend |
| 约 536 | "安心"公案：慧可请达摩安心，达摩反问"将心来，与汝安"，慧可觅心不可得，达摩说"与汝安心竟" | 传说 Legend |
| 约 536 | 达摩将《楞伽经》四卷和袈裟传给慧可，确立慧可为二祖 | 半史实 Semi-historical |
| 约 536 | 考察四弟子见地：道副得皮、尼总持得肉、道育得骨、慧可得髓 | 传说 Legend |

---

## 晚年与圆寂 / Final Years and Passing

| 时间 / Date | 事件 / Event | 性质 / Nature |
|---|---|---|
| 约 536–543 | 传法后继续在嵩山一带修行和教导 | 推断 Inferred |
| 约 543 | 圆寂（一说被菩提流支毒害，一说自然圆寂） | 半史实 Semi-historical |
| 约 543 | 葬于熊耳山（今河南境内） | 传说 Legend |
| 传说时间 | "只履西归"：宋云从西域归来，在葱岭遇见达摩手提一只鞋向西行去。打开达摩棺椁，只见一只鞋 | 传说 Legend |

---

## 后世发展关键节点 / Key Subsequent Developments

| 时间 / Date | 事件 / Event |
|---|---|
| 约 547 | 杨衒之《洛阳伽蓝记》提到达摩——最早的文献记录之一 |
| 约 645 | 道宣《续高僧传》完成——最早的正式传记 |
| 677 | 慧能（六祖）开始传法——禅宗南宗的兴起 |
| 732 | 神会在滑台大会上确立南宗顿教的正统地位 |
| 1004 | 《景德传灯录》成书——达摩传说故事的集大成 |
| 13 世纪 | 道元将曹洞宗（坐禅传统）传入日本 |
| 1927 | 胡适发表《菩提达摩考》——现代学术研究的开端 |

## 达摩法脉传承 / Bodhidharma's Dharma Lineage

```
般若多罗 Prajñātāra (西天第二十七祖 / 27th Indian Patriarch)
    │
菩提达摩 Bodhidharma (西天第二十八祖 / 东土初祖 / 28th Indian / 1st Chinese Patriarch)
    │
慧可 Huike (二祖 / 2nd Patriarch, c. 487–593)
    │
僧璨 Sengcan (三祖 / 3rd Patriarch, d. 606)
    │
道信 Daoxin (四祖 / 4th Patriarch, 580–651)
    │
弘忍 Hongren (五祖 / 5th Patriarch, 601–674)
    │
    ├── 神秀 Shenxiu (北宗 / Northern School, 606–706)
    │
    └── 慧能 Huineng (六祖 / 6th Patriarch / 南宗 / Southern School, 638–713)
            │
            ├── 南岳怀让 Nanyue Huairang → 马祖道一 Mazu Daoyi → 临济宗 Linji School
            ├── 青原行思 Qingyuan Xingsi → 石头希迁 Shitou Xiqian → 曹洞宗 Caodong School
            ├── 沩仰宗 Weiyang School
            ├── 云门宗 Yunmen School
            └── 法眼宗 Fayan School
```
