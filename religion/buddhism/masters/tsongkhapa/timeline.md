# 宗喀巴年表 / Timeline of Tsongkhapa

## 生平年表

| 年代 | 事件 |
|------|------|
| 1357 | 生于青海宗喀 (今湟中县) |
| ~1363 | 从噶玛巴若贝多杰出家 |
| ~1373 | 入藏，遍学萨迦、噶举、宁玛诸派 |
| ~1385 | 从仁达瓦学习中观 |
| ~1390 | 见西藏佛教戒律败坏，发心改革 |
| 1397 | 证悟空性的关键体验 |
| 1402 | 完成《菩提道次第广论》 |
| 1406 | 完成《密宗道次第广论》 |
| 1409 | 建甘丹寺，创立格鲁派；主持拉萨祈愿大法会 |
| 1415 | 弟子根敦主巴 (一世达赖) 出生 |
| 1419 | 圆寂于甘丹寺 |
