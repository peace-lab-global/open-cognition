# 二谛 Two Truths

## 一句话定义

**二谛**（Satyadvaya）是佛教各宗派共享的**诠释框架**——**世俗谛**（现象层面的真实）与**胜义谛**（终极层面的真实），不同宗派对二谛的理解不同，形成了佛教思想史上最丰富的**二谛论争**。

## 核心要义

### 1. 基本定义

| 谛 | 梵文 | 含义 |
|---|------|------|
| **世俗谛** | Saṃvṛti-satya | 世间共许的真实（名言/现象） |
| **胜义谛** | Paramārtha-satya | 终极真实（空性/真如） |

### 2. 各宗二谛观对比

| 宗派 | 二谛性质 | 核心主张 |
|------|---------|---------|
| **说一切有部** | 本体二谛 | 世俗 = 假名有；胜义 = 实有（法体） |
| **中观（印度）** | 本体二谛 | 世俗 = 缘起有；胜义 = 自性空 |
| **三论宗** | 言教二谛 | 二谛是教化的两种策略，非本体两层 |
| **唯识** | 识境二谛 | 世俗 = 遍计所执；胜义 = 圆成实 |
| **天台** | 三谛 | 空谛/假谛/中谛（超越二谛） |
| **华严** | 理事二谛 | 事法界/理法界 → 事事无碍 |

### 3. 吉藏的四重二谛

三论宗吉藏提出**四重二谛**——逐层深化：

| 重 | 世俗谛 | 胜义谛 |
|---|--------|--------|
| 第一重 | 有 | 空 |
| 第二重 | 有+空 | 非有非空 |
| 第三重 | 二（有/空）与不二 | 非二非不二 |
| 第四重 | 以上三重 | 言亡虑绝 |

每一重的"胜义"在下一重成为"世俗"——**不断超越**。

### 4. 二谛与中道

龙树《中论》："诸佛依二谛为众生说法"——

- 不说世俗谛，众生无法理解
- 不说胜义谛，众生无法解脱
- **中道 = 二谛的平衡运用**

### 5. 现代意义

二谛框架在现代仍有生命力：
- **科学哲学**：现象描述 vs 理论解释
- **语言学**：字面义 vs 隐喻义
- **心理治疗**：表层叙述 vs 深层真实

## 通俗 vs 学术

| 通俗化版本 | 学术原意 |
|---|---|
| "二谛 = 两种真实" | 是诠释框架，非本体论承诺 |
| "胜义谛更高" | 是教化策略，非等级 |
| "二谛过时了" | 其精神 = 多层次理解 |

- **哲学**：二谛 = 多层次真理观
- **语言学**：字面义 vs 隐喻义
- **心理治疗**：表层 vs 深层真实
- **科学哲学**：现象 vs 理论

## 常见误读

- ❌ "二谛 = 两种真实" → 是诠释框架，非本体论。
- ❌ "胜义谛 = 空的代名词" → 各宗对胜义谛理解不同。
- ❌ "二谛是固定的" → 吉藏的四重二谛显示是动态的。
