# 止观 Śamatha-Vipaśyanā

## 一句话定义

**止观**（Śamatha-Vipaśyanā）是佛教修行的两大支柱——**止**（奢摩他，寂静定）与**观**（毗钵舍那，智慧观察）的配合运用。止是"让心安定"，观是"如实观察"，二者如鸟之双翼，缺一不可，是三学中"定"与"慧"的实践形态。

## 核心要义

### 1. 止（Śamatha）

| 要素 | 内容 |
|------|------|
| **定义** | 心一境性——让心专注于一个对象 |
| **方法** | 数息、观像、念佛、持咒等 |
| **目标** | 九住心（从粗心到极细心） |
| **效果** | 身心轻安、烦恼暂时伏住 |
| **局限** | 只伏烦恼，不断根本 |

### 2. 观（Vipaśyanā）

| 要素 | 内容 |
|------|------|
| **定义** | 以智慧观察诸法的真实相 |
| **方法** | 观无常、观无我、观空、观缘起 |
| **目标** | 断除烦恼根本（无明） |
| **效果** | 智慧开显、烦恼根断 |
| **前提** | 需要止的基础 |

### 3. 止观双运

| 模式 | 说明 |
|------|------|
| **先止后观** | 先修止得定，再修观发慧 |
| **先观后止** | 先修观明理，再修止安心 |
| **止观同时** | 定中有慧，慧中有定 |

### 4. 天台止观体系

智顗建立最完整的止观体系：

| 止观类型 | 内容 |
|---------|------|
| **渐次止观** | 从持戒到禅定到智慧，次第修行 |
| **不定止观** | 不拘次第，随机应变 |
| **圆顿止观** | 一念心中具足止观 |

**六妙门**：数息→随息→止→观→还→净

### 5. 各宗止观对比

| 宗派 | 止的核心 | 观的核心 |
|------|---------|---------|
| **上座部** | 四十业处 | 十六观智 |
| **天台** | 三止（体真/方便随缘/息二边） | 三观（空/假/中） |
| **禅宗** | 默照 | 参话头 |
| **密宗** | 本尊观 | 生起/圆满次第 |
| **净土** | 念佛 | 观想极乐 |

## 通俗 vs 学术

| 通俗化版本 | 学术原意 |
|---|---|
| "止 = 冥想" | 止是心一境性，冥想只是入门 |
| "观 = 思考" | 观是超越概念的直观智慧 |
| "先止后观" | 三种模式皆可，非固定次第 |

## 应用场景

- **心理治疗**：止 = 正念冥想，观 = 认知觉察
- **教育学**：止观 = 专注力 + 批判思维的训练
- **体育**：止 = 心理定，观 = 战术观察
- **管理学**：止观 = 冷静决策 + 系统分析

## 常见误读

- ❌ "止 = 冥想" → 止是心一境性，冥想只是入门。
- ❌ "观 = 思考" → 观是超越概念的直观智慧。
- ❌ "止观只能先止后观" → 三种模式皆可。
