# 十二处 Dvādaśa Āyatana

## 一句话定义

十二处（āyatana）是佛教对认识结构的分析框架——由**六内处**（六根）与**六外处**（六境）组成，是认识发生的基本条件。与五蕴、十八界并列为佛教分析存在的三大框架，侧重于**认识发生论**的视角。

## 核心要义

### 十二处结构

| 六内处（根） | 六外处（境） | 产生的认识 |
|-------------|-------------|-----------|
| 眼处 (cakṣus) | 色处 (rūpa) | 眼识 |
| 耳处 (śrotra) | 声处 (śabda) | 耳识 |
| 鼻处 (ghrāṇa) | 香处 (gandha) | 鼻识 |
| 舌处 (jihvā) | 味处 (rasa) | 舌识 |
| 身处 (kāya) | 触处 (spraṣṭavya) | 身识 |
| 意处 (manas) | 法处 (dharma) | 意识 |

### 五蕴 vs 十二处 vs 十八界

| 框架 | 侧重点 | 包含 |
|------|--------|------|
| **五蕴** | "我"的组件分析 | 色/受/想/行/识 |
| **十二处** | 认识发生的条件 | 六根+六境 |
| **十八界** | 认识结构的完整展开 | 六根+六境+六识 |

> 十二处 = 十二处 + 六识 = 十八界。十二处侧重"根境相触"的发生学视角。

### 意处与法处的特殊性

- **意处**（manas）：不是感官器官，而是"意根"——前五识灭后的"无间灭意"（俱舍论）或末那识（唯识）
- **法处**（dharma）：最广泛的境——包含一切意识对象（概念/抽象/心理对象），含"无表色"等非感官对象

## 常见误读

- ❌ "六根 = 六种感官器官" → 六根不仅是物质器官，更是**认识能力**（如"意根"不是大脑，而是认知功能）。
- ❌ "十二处 = 唯物主义" → 十二处是分析框架，不是本体论主张——佛教不讨论"物质是否独立存在"。
