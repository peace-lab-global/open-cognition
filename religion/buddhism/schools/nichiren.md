# 日莲宗 Nichiren

## 一句话定义

日莲宗是日本佛教中最具**入世精神**与**批判性**的宗派——由日莲（1222–1282）创立，以《法华经》为唯一正法，以**唱题**（"南无妙法莲华经"）为修行核心，以"**五纲判教**"批判其他一切宗派，强调佛法应直接介入社会与政治。

## 历史脉络

- **1222**：日莲生于安房国（今千叶县）
- **1253**：在清澄山高唱"南无妙法莲华经"，宣布开宗
- **1260**：著《立正安国论》呈镰仓幕府，预言灾难——被流放伊豆
- **1271**：龙口法难（差点被处死），流放佐渡
- **1274**：赦免，退隐身延山
- **1282**：圆寂
- **后世**：分裂为多个派别（日莲正宗、创价学会、立正佼成会等）

## 核心要义

### 1. 五纲判教

日莲以五个标准判定一切佛法的高低正邪：

| 纲 | 内容 |
|---|---|
| **教** | 经典的高低 |
| **机** | 众生的根机 |
| **时** | 时代（正法/像法/末法） |
| **国** | 国土的因缘 |
| **教法流布先后** | 佛法传播的顺序 |

结论：**末法时代**只有《法华经》适合日本众生。

### 2. 唱题行

"**南无妙法莲华经**"（Namu Myōhō Renge Kyō）——日莲宗的核心修行：
- 不依赖复杂的禅定或学问
- 任何人都可以通过唱题与法华经的功德相应
- 唱题 = 身口意三业的统一

### 3. 四大格言

| 格言 | 含义 |
|------|------|
| "念佛无间" | 念佛是堕无间地狱之因 |
| "禅天魔" | 禅宗是天魔所创 |
| "真言亡国" | 密教真言宗会导致亡国 |
| "律国贼" | 律宗是国贼 |

> 这些激烈的批判语言反映了日莲"法华至上"的绝对立场。

### 4. 三大秘法

| 秘法 | 内容 |
|------|------|
| **本门本尊** | 大曼荼罗（十界互具的图绘） |
| **本门戒坛** | 唱题修行的场所即戒坛 |
| **本门题目** | "南无妙法莲华经" |

## 通俗 vs 学术

| 通俗化版本 | 学术原意 |
|---|---|
| "日莲宗 = 极端排他" | 是法华经至上的绝对主义，但内在逻辑是自洽的 |
| "唱题 = 念咒" | 唱题是对法华经全体的归命，非单纯咒语 |
| "日莲是民族主义者" | 日莲的"国"观是基于佛法的理想国，非近代民族国家 |

## 常见误读

- ❌ "日莲宗只是日本民族宗教" → 创价学会等现代日莲系运动已全球化，信众超过 1200 万。
- ❌ "四大格言 = 宗教仇恨" → 是日莲对"末法时代何者最有效"的判教结论，非个人仇恨。
