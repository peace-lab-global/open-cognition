# 大智度论 Mahāprajñāpāramitā-śāstra

## 一句话定义

《大智度论》是**龙树**（Nāgārjuna，约 2–3 世纪）所造、**鸠摩罗什**（402–405）汉译的**般若经释论**——全书 100 卷，是对《大品般若经》（Pañcaviṃśatisāhasrikā Prajñāpāramitā）的系统注释，内容涵盖般若思想、六度修行、佛果功德、因明论辩与佛教百科全书，是汉传佛教最重要的论典之一。

## 历史脉络

- **约 150–250**：龙树造《大智度论》（梵文原本已佚，仅存龙树释与鸠摩罗什合译本）
- **402–405**：鸠摩罗什于长安译出《大智度论》100 卷（含经与论）
- **5–6 世纪**：慧远作《大智度论抄序》，开启中国学者的研究传统
- **唐代**：吉藏、窥基等引用《大智度论》构建各自学派
- **至今**：为研究初期大乘佛教（般若思想、菩萨道、佛教百科全书）的第一手资料

## 核心要义

### 1. 百科全书式内容

《大智度论》被誉为"佛教百科全书"，内容远超对《大品般若经》的字面注释：

| 类别 | 内容举例 |
|------|---------|
| 般若哲学 | 十八空、三解脱门、一切智/道种智/一切种智 |
| 六度修行 | 布施/持戒/忍辱/精进/禅定/智慧各度的详释 |
| 佛果功德 | 十力、四无所畏、十八不共法 |
| 因明论辩 | 对外道与小乘的系统驳难 |
| 佛教制度 | 僧团规则、居士修行、寺院生活 |
| 宇宙论 | 三千大千世界、六道轮回 |
| 故事与譬喻 | 大量本生故事与法喻 |

### 2. 般若思想的核心

- **十八空**：内空、外空、内外空、空空、大空、第一义空、有为空、无为空……层层递进破除一切执著
- **三解脱门**：空、无相、无愿——三种通向解脱的智慧门
- **二谛**：世俗谛中说因果善恶，胜义谛中一切法空
- **空与慈悲的统一**：般若为母，方便为父——空性智慧与度众悲心不可偏废

### 3. 六度思想的系统展开

- **布施度**：财施/法施/无畏施三轮体空
- **般若度**：六度之眼，前五度若无般若则非波罗蜜
- **核心命题**：六度不是线性递进，而是以般若贯穿一切

## 重要概念

- [空性](../concepts/foundational/emptiness.md) [核心] — 大智度论是"空"的最详尽展开（十八空）
- [二谛](../concepts/foundational/two-truths.md) [继承] — 以二谛架构统摄世俗与胜义
- [菩萨道](../concepts/foundational/bodhisattva-path.md) [展开] — 六度波罗蜜的系统论述
- [四无所畏](../concepts/foundational/four-fearlessnesses.md) [来源] — 佛果功德"十力/四无畏/十八不共法"的主要出处
- [三乘](../concepts/foundational/three-vehicles.md) [讨论] — 对三乘与一乘关系的广泛讨论

## 与其他论典的关系

- [中论](mulamadhyamakakarika.md) [同源] — 同为龙树所造，中论侧重哲学论证，大智度论侧重般若经的系统释义
- [入中论](madhyamakavatara.md) [平行] — 皆以般若思想为核心，但大智度论更具百科全书性质
- [入菩萨行论](bodhicaryavatara.md) [互补] — 大智度论侧重理论展开，入菩萨行论侧重修行实践

## 常见误读

- **误读**：大智度论只是般若经的注释，没有独立思想价值
- **正解**：大智度论包含大量不见于般若经的独立论述（因明论辩、佛教制度、宇宙论等），其"十八空"体系也是对般若经思想的创造性系统化
