# 唯识百法分析 · Hundred Dharmas Analysis

## 一句话功能

对任一心理/认知/存在现象，以唯识百法五位（心法/心所法/色法/心不相应行法/无为法）逐项归类诊断——揭示现象背后的**识的结构**，将日常经验还原为唯识的认知模型。

## 何时使用

- 分析某种情绪、认知偏误或心理状态时，需要精确的佛教心理学分类
- 对比佛教百法体系与现代心理学概念（如认知偏误、情绪调节）
- 佛学院唯识学课程的实修训练——将抽象法相应用于具体经验
- 诊断某一烦恼的根源：是根本烦恼还是随烦恼？与哪一识相应？
- 写跨学科论文时需要佛教心理学的精细分类框架

## 何时不使用

- 只需粗略了解心理状态 → 用五蕴（色受想行识）即可
- 分析社会/文化/制度现象 → 百法是个体心理分析框架，不适合宏观分析
- 禅修实操指导 → 百法是分析框架，不是禅修方法（应用止观 Skill）
- 用户对唯识学基本架构完全陌生（应先了解八识/三性）

## 理论基础

- **百法五位**：天亲（世亲）《百法明门论》将一切法分为五大类 100 种 → [百法明门](../../../domains/religion/buddhism/core-concepts/hundred-dharmas.md)
- **八识体系**：前五识 + 第六意识 + 第七末那识 + 第八阿赖耶识 → [成唯识论](../../../domains/religion/buddhism/sutras-standalone/cheng-weishi-lun.md)
- **三性**：遍计所执 / 依他起 / 圆成实——判断现象的认知层次
- **二障**：烦恼障（人我执）/ 所知障（法我执）→ [二障](../../../domains/religion/buddhism/core-concepts/two-obstacles.md)
- **心所与心相应**：51 心所与八识的相应关系是诊断的关键

## 操作流程

### Step 1：锁定目标现象

明确要分析的现象，用日常语言描述：

- "我在社交媒体上刷到别人成功时，感到一种混合了羡慕、不安和自我贬低的复杂情绪"
- "开会时我确信自己的方案最好，拒绝听取他人意见"
- "冥想时出现的深度宁静感"

**提问范式**：
- "这个现象的核心体验是什么？"
- "它涉及情绪、认知还是纯粹感知？"

### Step 2：五位初步归类

将现象归入百法五位的第一层分类：

| 位 | 包含 | 识别标准 |
|---|------|---------|
| **心法**（8） | 八识本身 | 是认知的主体/功能 |
| **心所法**（51） | 心理活动的附属因素 | 与心相应、伴随心法运作 |
| **色法**（11） | 物质/感官对象 | 六根+六境（不含识） |
| **心不相应行法**（24） | 抽象概念/关系 | 非心非色，依心色假立 |
| **无为法**（6） | 超越因果的实相 | 非因缘造作 |

> 绝大多数日常心理现象落在**心所法**范围内。

### Step 3：心所精细诊断

对落入心所法的现象，在 51 心所中精确定位：

| 组 | 心所 | 特征 |
|---|------|------|
| **遍行**（5） | 触/作意/受/想/思 | 一切心理活动的基本成分 |
| **别境**（5） | 欲/胜解/念/定/慧 | 特定对象才引发 |
| **善**（11） | 信/惭/愧/无贪/无嗔/无痴/勤/轻安/不放逸/行舍/不害 | 导向解脱的因素 |
| **根本烦恼**（6） | 贪/嗔/慢/无明/疑/恶见 | 轮回的根本驱力 |
| **随烦恼**（20） | 小随（10）/中随（2）/大随（8） | 根本烦恼的衍生 |
| **不定**（4） | 悔/眠/寻/伺 | 可善可恶 |

**提问范式**：
- "这个情绪中有哪些根本烦恼在驱动？"
- "随烦恼中哪些是直接表现？（忿/恨/覆/恼/嫉/悭/诳/谄/害/憍）"
- "善心所中哪些可以作为对治？"

### Step 4：识的相应分析

判断此现象与哪一识相应：

| 识 | 相应心所 | 典型现象 |
|---|---------|---------|
| 前五识 | 遍行 + 部分善/烦恼 | 纯粹感官体验（痛、冷热） |
| 第六意识 | 全部 51 心所 | 思维、想象、计划、判断 |
| 第七末那 | 遍行 + 四根本烦恼（贪/痴/慢/恶见） | 深层自我执着 |
| 第八阿赖耶 | 仅遍行（5） | 种子储存，不直接体验 |

**关键**：如果现象涉及深层自我感（"我的""我比别人…"），一定涉及第七末那识。

### Step 5：三性判定与二障诊断

| 层次 | 分析 |
|------|------|
| **遍计所执** | 现象中哪些是概念虚妄建构？（如"我是失败者"） |
| **依他起** | 现象的因缘和合实际是什么？（如贪嗔等心所的生灭） |
| **圆成实** | 此现象的真实本性是什么？（空性/真如） |

同时诊断：
- **烦恼障**：此现象涉及哪种人我执？（贪嗔慢等）
- **所知障**：此现象涉及哪种法我执？（对概念/理论的执着）

### Step 6：输出百法分析报告

```
【百法分析报告】
├── 目标现象：___
├── 五位归类：
│   ├── 主要落入：心所法（___组）
│   └── 涉及色法：___
├── 心所精确定位：
│   ├── 遍行成分：触/作意/受/想/思 中的___
│   ├── 根本烦恼：___
│   ├── 随烦恼：___
│   └── 可对治善法：___
├── 识的相应：
│   ├── 主要与第___识相应
│   └── 是否涉及末那（深层自我）：是/否
├── 三性判定：
│   ├── 遍计所执层面：___
│   ├── 依他起层面：___
│   └── 圆成实层面：___
├── 二障诊断：烦恼障___/ 所知障___
└── 对治建议：___
```

## 完整示例

**输入场景**：
> 用户："分析'开会时我确信自己的方案最好，拒绝听取他人意见'这个现象。"

**Skill 应用过程**：

1. **锁定现象**：确信自己正确 + 拒绝他人 → 认知+情绪复合体
2. **五位归类**：主要落入**心所法**，涉及心法（第六意识）
3. **心所精确定位**：
   - 遍行：想（概念把握"我的方案"）、受（乐受——自我确认的满足感）
   - 根本烦恼：**慢**（我慢——"我的最好"）、**无明**（不明自身方案的局限）
   - 随烦恼：**憍**（自大）、**覆**（掩盖自身不足）
   - 对治善法：**惭**（反思自身不足）、**无痴**（如实认知）、**行舍**（平等心）
4. **识的相应**：主要与**第六意识**相应（概念判断），深层涉及**第七末那**（"我的"——人我执）
5. **三性判定**：
   - 遍计所执："我的方案=最好的"是虚妄分别
   - 依他起：慢+无明+憍等心所的因缘生灭
   - 圆成实：方案本身无自性，"好/坏"依缘而立
6. **二障**：烦恼障——我慢（人我执）；所知障——对"正确方案"的法我执

**输出**：
> 此现象以**我慢**（根本烦恼）为驱动，伴以**无明**（不明自身局限）和**憍/覆**（随烦恼）。深层根植于第七末那识的"人我执"——将"我的方案"等同于"我"本身。对治：以**惭**反省自身局限，以**无痴**如实评估各方方案，以**行舍**放下对"我的"的执着。

## 反例（误用）

**误用场景**：
> 用户试图用百法分析"公司组织架构的权力关系"。

**问题**：百法是对**个体心理**的精细分析框架，不适合分析制度、权力、社会结构。社会层面的分析需要用社会学/政治学框架。

**正确做法**：百法适用于个体内心的认知-情绪现象。制度性分析应使用社会理论工具。
