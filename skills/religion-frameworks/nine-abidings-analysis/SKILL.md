# 止观九住心分析 · Nine Abidings Analysis

## 一句话功能

对禅定状态做九住心阶段的定位诊断，识别当前障碍并提供对治方法。

## 何时使用

- 用户描述自己的禅修体验，需要判断处于哪个阶段
- 用户在止禅修习中遇到障碍，需要对治建议
- 教学中需要对禅定阶段做系统分析
- 比较不同禅修者的进度与方法

## 何时不使用

- 用户讨论的是观禅（vipassanā）而非止禅（śamatha）
- 用户询问的是密教生起/圆满次第
- 用户只是想了解九住心理论而非做实践分析

## 理论基础

九住心（navākārā cittasthiti）是弥勒《瑜伽师地论》提出的止禅九阶段：

1. **内住**（sthāpayat）— 初次将心从外境收摄至所缘
2. **续住**（samsthāpayat）— 心能短暂持续在所缘上
3. **安住**（avasthāpayat）— 心散乱时能自觉并带回
4. **近住**（upasthāpayat）— 心几乎不散乱，持续贴近所缘
5. **调伏**（damayat）— 心对所缘自然生起悦意
6. **寂静**（śamayati）— 烦恼不起，内心寂然
7. **最极寂静**（vyupasthāpayati）— 微细烦恼亦能速断
8. **专住**（ekotīkaroti）— 无功用行，心自然安住
9. **等持**（samādadhāti）— 平等持心，不偏不倚，成就奢摩他

六力对应：听闻力→思惟力→念力→正知力→精进力→串习力

## 操作流程

### 第一步：采集禅修描述

- 引导用户描述：修习时长、所缘对象（呼吸/佛像/名号等）、心的状态
- 关键问题：心散乱频率？是否自觉散乱？散乱后多久能回？有无昏沉？

### 第二步：阶段定位

根据描述匹配九住心阶段：

| 特征 | 阶段 |
|------|------|
| 心频繁散乱，不自觉 | 第1-2住 |
| 散乱时能自觉带回 | 第3-4住 |
| 心较稳定，偶有微细散乱 | 第5-6住 |
| 几乎无散乱，微细烦恼偶现 | 第7-8住 |
| 无功用，自然安住 | 第9住 |

### 第三步：障碍诊断

- **掉举**（auddhatya）：心过于活跃，向外攀缘
- **昏沉**（laya）：心过于低沉，模糊不清
- **微细掉举/昏沉**：仅以极微细的方式干扰
- **懈怠**（kausīdya）：缺乏修习动力

### 第四步：对治建议

| 障碍 | 对治 |
|------|------|
| 粗掉举 | 放松所缘、降低目标、观不净/无常 |
| 粗昏沉 | 收紧所缘、提升光明想、经行/洗脸 |
| 微细掉举 | 正知觉察，不加干涉 |
| 微细昏沉 | 提起精进，略振作 |
| 懈怠 | 思惟禅定利益（信/欲/勤/轻安四断行） |

### 第五步：进阶指导

- 说明从当前阶段进入下一阶段的要点
- 提醒不可急进（"松紧适度"原则）
- 建议配合观禅（止观双运）的时机

## 完整示例

**用户描述**："我坐禅时能安住约5分钟，之后会突然发现自己已经在想别的事情。发现后能很快把心带回来，但过一会儿又会走神。偶尔会有微细的模糊感。"

**分析**：
1. **阶段定位**：第3-4住之间。能自觉散乱（第3住特征），但持续力不够（尚未稳固在第4住）
2. **障碍诊断**：粗掉举（突然走神）+ 微细昏沉（模糊感）
3. **对治建议**：
   - 对粗掉举：缩短单次安住目标（3分钟→逐步延长），散乱时温和带回
   - 对微细昏沉：略提升所缘的清晰度，或做几次深呼吸
4. **进阶要点**：当能在10-15分钟内稳定不散，即进入第5住"调伏"

## 反例

- **错误**：将所有安静状态都判定为高阶段——安静不等于等持
- **错误**：忽视微细昏沉——"感觉平静但模糊"常被误认为好状态
- **错误**：跳过止禅直接修观——九住心是止禅基础，未达第9住不宜深入观禅
