# 亚里士多德德性检验 · Aristotle's Virtue Test

## 一句话功能

从亚里士多德伦理学出发，通过中庸检验、功能检验与实践智慧三个维度，判断某行为或品质是否构成德性。

## 何时使用

- 判断某种行为（慷慨、勇敢、诚实等）是否在特定情境中适度
- 分析为何某人在某情境下的"中庸"行为实际上并不道德
- 评估某人格特质是否可被称为德性
- 需要将抽象德性概念转化为具体行为判断

## 何时不使用

- 涉及绝对伦理规则的场景（不可撒谎、不可杀人）——亚里士多德承认极端情况下德性可能表现为"违背一般规则"
- 需要权利论证的情境——德性伦理不擅长处理"人有权做X"的问题
- 纯粹的结果评估——德性伦理关注行为者与行为动机，而非仅关注结果
- 缺乏具体情境信息时——中庸之道必须在具体情境中判断

## 理论基础

- 来源思想家：[亚里士多德 Aristotle](../domains/ethics-politics/schools/virtue-ethics/aristotle.md)
- 来源概念：[德性 Virtue](../domains/ethics-politics/concepts/virtue.md)
- 关键文献：Aristotle, *Nicomachean Ethics* II–VI

## 操作流程

### Step 1：确定要检验的行为/品质

将模糊的道德问题转化为具体行为判断："X在Y情境中做了Z"
**提问范式**：
- "我们要检验的是哪个具体行为？"
- "行为者是谁，在什么情境下，做了什么？"

### Step 2：定位两个极端

确定该行为所属德性领域的两个极端——过度与不足。
**提问范式**：
- "这个领域的两个极端是什么？"（如勇敢 → 鲁莽 / 怯懦）
- "行为是偏向过度、不足，还是接近中道？"

### Step 3：中庸检验

判断该行为是否在两个极端之间适度。
**打分标准**：
- 接近过度：-1
- 接近不足：-1
- 适度（因情境而异）：+1

**注意**：适度不等于"中等"，而是该情境中正确反应。需要实践智慧（phronesis）来判断。

### Step 4：功能检验

判断该行为是否有助于人实现其特有功能——理性灵魂的优秀活动。
**提问范式**：
- "这个行为是理性灵魂的展现吗？"
- "它促进还是阻碍了行为者作为人的整体优秀？"

### Step 5：综合结论

| 检验结果 | 综合判断 |
|---|---|
| 中道 + 发挥功能 | 德性（真正的慷慨/勇敢/诚实） |
| 中道但未发挥功能 | 偶然符合德性，非真正德性 |
| 偏向任一极端 | 德性缺陷（恶或过度/不足的恶） |

## 完整示例

**输入场景**：
> 某人在朋友患重病时，被医生私下告知朋友只剩三个月生命。朋友问"我还能活多久？"这人说谎说"没事，可能是误诊，再检查一下。"

**Skill 应用过程**：
1. **确定行为**：对患绝症朋友隐瞒病情，谎称可能是误诊
2. **定位极端**：
   - 诚实领域：过度（无情揭露全部真相）← 诚实 → 不足（无原则撒谎）
   - 仁慈领域：过度（溺爱式隐瞒）← 仁慈 → 不足（冷漠不关心）
3. **中庸检验**：
   - 诚实维度：说谎偏向不足一侧（-1）
   - 仁慈维度：隐瞒出于仁慈关心，接近中道（+1）
4. **功能检验**：理性灵魂的优秀活动需要关怀与诚实的平衡——纯粹说谎不符合理性对话，但完全不关心朋友感受也不是真正的理性
5. **综合**：这不是真正的德性行为，而是德性缺陷（仁慈掩盖了诚实的不足）

**输出**：
> 亚里士多德检验结论：此行为在仁慈维度接近中道，但诚实维度存在缺陷。真正的德性需要在关怀与诚实之间寻找情境中的适度——可能的方式是部分告知（不揭露全部残酷真相，但也不绝对说谎）或陪伴朋友自己发现真相。

## 反例（误用）

**误用场景**：
> 将亚里士多德德性检验用于评估制度正义问题——如"种族隔离政策是否正义？"这个问题需要的是权利论证（谁有权享受什么）和分配分析，而非个人品格评估。

**正确做法**：涉及结构性不正义的问题，应使用[无知之幕分析](./veil-of-ignorance-analysis/SKILL.md)或[正义原则检验](./justice-principle-test/SKILL.md)。
