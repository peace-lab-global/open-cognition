# 麦金生平年表 · Colin McGinn Biography Timeline

## 早年与学术训练（1950–1976）

| 年份 | 事件 |
|------|------|
| 1950.3.10 | 生于英国兰开夏郡（Lancashire），工人阶级家庭 |
| c. 1968 | 进入牛津大学（University of Oxford）学习哲学 |
| 1972 | 获牛津大学哲学学士学位（BA in Philosophy），一等荣誉 |
| 1972–1976 | 在华威大学（University of Warwick）攻读博士学位 |
| 1976 | 获华威大学博士学位（PhD），论文论主观视角与指称理论 |
| 1976 | 出版首部专著 *The Subjective View* |

## 早期学术生涯（1976–1989）

| 年份 | 事件 |
|------|------|
| 1976–1980 | 在伦敦大学学院（UCL）任讲师 |
| 1980 | 出版 *The Character of Mind*（第一版），系统概述心灵哲学 |
| 1982 | 出版 *Wittgenstein on Meaning*，对维特根斯坦意义理论的阐释 |
| 1983–1988 | 任牛津大学讲师（Fellow of New College） |
| 1986 | 在 *Mind* 发表重要论文 "Rigidity and Essentialism" |
| 1989 | 出版 *Mental Content*，讨论心理内容的外部论 |

## 意识转向与新神秘主义（1989–1999）

| 年份 | 事件 |
|------|------|
| 1989 | 在 *Mind* 发表开创性论文 "Can We Solve the Mind-Body Problem?"——认知闭合论题首次系统阐述 |
| 1988 | 赴美国罗格斯大学（Rutgers University）任哲学教授 |
| 1991 | 出版 *The Problem of Consciousness*——新神秘主义（New Mysterianism）奠基之作 |
| 1993 | 出版 *Problems in Philosophy: The Limits of Inquiry*——将认知闭合扩展到哲学核心问题 |
| 1995 | 出版 *The Problem of Philosophy*——面向初学者的哲学导论 |
| 1996 | 出版 *Minds and Bodies: Philosophers and Their Ideas*——对当代心灵哲学家的评论 |
| 1997 | 出版 *Ethics, Evil, and Fiction*——伦理学与文学哲学 |
| 1999 | 出版 *The Mysterious Flame: Conscious Minds in a Material World*——面向大众的新神秘主义通俗阐述 |
| 1999 | 出版 *Knowledge and Reality: Selected Essays*——重要论文集 |

## 深化与多领域拓展（2000–至今）

| 年份 | 事件 |
|------|------|
| 2001 | 出版 *Logical Properties*——逻辑哲学 |
| 2004 | 出版 *Consciousness and Its Objects*——意识与意向性的关系 |
| 2005 | 出版 *The Power of Movies: How Screen and Mind Interact*——电影哲学 |
| 2006 | 出版 *The Making of a Philosopher*——学术自传 |
| 2008 | 出版 *Shakespeare's Philosophy*——文学哲学 |
| 2011 | 出版 *The Character of Mind*（第二版） |
| 2011 | 转至迈阿密大学（University of Miami）任哲学教授 |
| 2012 | 出版 *Mindfucking: A Critique of Mental Manipulation*——对心理操纵的哲学分析 |
| 2015 | 出版 *Philosophical Provocations: 55 Short Essays*（MIT Press） |
| 2016 | 因性骚扰指控离开迈阿密大学（学术争议） |
| 2020s | 继续独立从事哲学写作与评论 |

## 思想分期总结

| 阶段 | 年代 | 关键词 |
|------|------|--------|
| 学术训练 | 1950–1976 | 牛津、华威、主观视角 |
| 早期生涯 | 1976–1989 | 语言哲学、维特根斯坦、心理内容 |
| 新神秘主义 | 1989–1999 | 认知闭合、《意识问题》、《神秘的火焰》 |
| 多领域拓展 | 2000–至今 | 电影哲学、莎士比亚、自传 |
