# 证伪主义 · Falsificationism

## 概念定位 / Conceptual Position

证伪主义（Falsificationism）是波普尔科学哲学的核心学说：科学与非科学的划界标准不是"可证实性"（verifiability），而是"可证伪性"（falsifiability）——一个理论是科学的，当且仅当存在某种可能的经验观察能够反驳它。这一理论彻底改变了20世纪科学哲学，至今仍是科学划界的核心参考标准。

Falsificationism is the core doctrine of Popper's philosophy of science: the demarcation criterion between science and non-science is not "verifiability" but "falsifiability" — a theory is scientific if and only if there exists some possible empirical observation that could refute it.

## 核心论述 / Core Argument

### 归纳问题 / The Problem of Induction

休谟证明了归纳推理在逻辑上不成立——无论观察到多少只白天鹅，都不能逻辑地推导出"所有天鹅都是白的"。波普尔接受了休谟的结论，但提出了不同的解决方案：科学不依赖归纳——它依赖的是**演绎性的反驳**。

Hume proved induction is logically invalid. Popper accepted Hume's conclusion but proposed a different solution: science does not depend on induction — it depends on **deductive refutation**.

### 可证伪性的标准 / The Criterion of Falsifiability

一个理论是可证伪的，当且仅当：存在至少一个逻辑上可能的观察陈述（"基础陈述"），如果该陈述为真，则理论为假。

A theory is falsifiable if and only if: there exists at least one logically possible observation statement ("basic statement") that, if true, would make the theory false.

**可证伪的 / Falsifiable**: "所有金属导电" → 找到一种不导电的金属即可反驳
**不可证伪的 / Unfalsifiable**: "命运不可违" → 任何事件都可以被解释为"命运"

### 猜想与反驳 / Conjectures and Refutations

科学进步的模式：
1. 遇到问题 → 2. 提出大胆猜想 → 3. 设计严格检验 → 4. 反驳失败的猜想 → 5. 新问题出现

Scientific progress pattern: 1. Problem → 2. Bold conjecture → 3. Rigorous test → 4. Refute failed conjectures → 5. New problems emerge

关键洞见：一千次"证实"不如一次反驳有价值——因为"证实"不能排除未来的反例，而反驳是决定性的。

Key insight: a thousand "verifications" are less valuable than one refutation — because "verification" cannot exclude future counterexamples, while refutation is decisive.

## 哲学意义 / Philosophical Significance

1. **科学划界 / Scientific Demarcation**: 至今仍是区分科学与伪科学的核心标准
2. **方法论规范 / Methodological Norms**: 科学理论的"大胆性"和"可检验性"标准
3. **被挑战 / Challenged**: 库恩（科学革命）、拉卡托斯（研究纲领）、Feyerabend（方法论无政府主义）

## 与东方思想的对话 / Dialogue with Eastern Thought

### 与佛教"四依四不依" / With Buddhism's "Four Reliances"
佛教"四依四不依"之一是"依法不依人"——依靠可检验的法义而非权威人物。这与证伪主义的精神相似：理论的权威不来自提出者的声望，而来自其经受反驳的能力。

One of Buddhism's "Four Reliances" is "rely on the teaching, not the person" — relying on testable doctrine rather than authoritative figures. This resembles falsificationism's spirit: a theory's authority comes not from the proposer's prestige but from its ability to withstand refutation.
