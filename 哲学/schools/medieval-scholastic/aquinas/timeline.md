# 阿奎那生平年表 · Thomas Aquinas Biography Timeline

## 早年与修会抉择（1225–1244）

| 年份 | 事件 |
|------|------|
| c. 1225 | 生于意大利南部罗卡塞卡（Roccasecca）城堡，阿基诺伯爵家族 |
| c. 1230 | 5岁被送入卡西诺山（Monte Cassino）本笃会修道院接受启蒙教育 |
| 1239 | 那不勒斯大学（Frederick II创建）学习文科与哲学 |
| 1244 | 不顾家族反对加入多明我会（Dominican Order） |
| 1244–1245 | 被兄弟囚禁于罗卡塞卡城堡约一年，期间研读圣经与彼得·隆巴德《教父箴言集》 |

## 巴黎求学（1245–1256）

| 年份 | 事件 |
|------|------|
| 1245 | 获释后赴巴黎大学，师从大阿尔伯特（Albertus Magnus） |
| 1245–1248 | 在巴黎跟随大阿尔伯特学习亚里士多德哲学 |
| 1248–1252 | 随大阿尔伯特赴科隆继续学业 |
| 1252 | 返回巴黎大学开始神学学士课程 |
| 1252–1256 | 撰写《教父箴言集评注》（*Scriptum super Sententias*） |
| c. 1256 | 撰写《论存在与本质》（*De Ente et Essentia*） |
| 1256 | 获巴黎大学神学硕士学位（Magister in Theologia） |

## 第一次巴黎教学（1256–1259）

| 年份 | 事件 |
|------|------|
| 1256 | 开始在巴黎大学任神学教授 |
| 1256–1259 | 撰写《论真理》（*Quaestiones Disputatae de Veritate*） |
| 1258–1259 | 讲授波爱修斯的《论三位一体》 |

## 意大利时期（1259–1268）

| 年份 | 事件 |
|------|------|
| 1259 | 返回意大利，在那不勒斯、罗马、奥尔维耶托（Orvieto）等地修道院任教 |
| 1259–1265 | 撰写《反异教大全》（*Summa Contra Gentiles*），4卷 |
| 1263 | 参加罗马的多明我会总会会议 |
| 1265 | 在罗马开设多明我会学习中心（studium），开始撰写《神学大全》 |
| 1265–1268 | 完成《神学大全》第一部分和第一部分之二 |

## 第二次巴黎教学（1268–1272）

| 年份 | 事件 |
|------|------|
| 1268 | 被多明我会派遣返回巴黎，应对阿维罗伊主义争议 |
| 1268–1272 | 撰写大量亚里士多德评注：《形而上学》《伦理学》《政治学》等 |
| 1270 | 撰写《论理智的统一性》反驳阿维罗伊主义 |
| 1270–1272 | 完成《神学大全》第二部分之二 |
| 1271 | 撰写《论灵魂》（*Quaestiones de Anima*） |
| 1272 | 开始撰写《神学大全》第三部分（论道成肉身） |

## 晚年与逝世（1272–1274）

| 年份 | 事件 |
|------|------|
| 1272 | 返回意大利那不勒斯，重组多明我会学习中心 |
| 1273 | 据传在一次弥撒中获得神秘体验，此后停止写作："我所写的一切与我所见到的相比如同草芥" |
| 1274.1 | 启程赴里昂参加教皇格列高利十世召集的教会会议 |
| 1274.3.7 | 在弗萨诺瓦（Fossanova）修道院病逝，年仅49岁 |

## 身后影响

| 年份 | 事件 |
|------|------|
| 1323 | 教皇约翰二十二世封圣（canonization） |
| 1368 | 遗骸迁至图卢兹雅各宾教堂 |
| 1567 | 教皇庇护五世宣布阿奎那为"教会博士"（Doctor of the Church） |
| 1879 | 教皇利奥十三世发布《永恒之父》（*Aeterni Patris*）通谕，将托马斯主义定为天主教官方哲学 |
| 1880 | 被宣布为天主教学校和大学的守护圣人 |
| 20世纪 | 新托马斯主义运动：马里坦（Maritain）、吉尔松（Gilson）等复兴阿奎那哲学 |

## 思想分期总结

| 阶段 | 年代 | 关键词 |
|------|------|--------|
| 早年与修会 | 1225–1244 | 那不勒斯、多明我会 |
| 巴黎求学 | 1245–1256 | 大阿尔伯特、亚里士多德 |
| 第一次教学 | 1256–1259 | 《论真理》、早期著作 |
| 意大利时期 | 1259–1268 | 《反异教大全》、《神学大全》开始 |
| 第二次教学 | 1268–1272 | 亚里士多德评注、反阿维罗伊主义 |
| 晚年 | 1272–1274 | 神秘体验、去世 |
