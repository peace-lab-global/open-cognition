# 怀疑 · Skepticism

## 一句话定义

怀疑（skepticism）是对知识主张的根据提出系统质疑的哲学立场；从皮浪的悬搁判断（epochē）到笛卡尔的方法怀疑，再到阿格里帕三难困境（Agrippa's trilemma），怀疑论既是认识论的试金石，也是佛教修行中需要超越的"疑"（vicikitsā）。

## 历史脉络

古希腊怀疑论分为两派：皮浪主义（Pyrrhonism）主张对一切命题悬搁判断以达到心灵宁静（ataraxia）；学园怀疑论（Academic Skepticism，阿尔克西拉乌斯、卡尔内阿德斯）则主张"知识不可能"这一否定性教条。塞克斯都·恩披里柯（Sextus Empiricus）系统记录了皮浪主义的十式与五式论证。

近代，笛卡尔将怀疑转化为方法工具：通过普遍怀疑（恶魔假设、梦境论证）找到不可怀疑的基点——"我思故我在"。当代认识论中，阿格里帕三难困境指出任何证成都面临无穷后退、循环论证或武断终止的困境。佛教将"疑"（vicikitsā）列为五盖之一，指对三宝、因果、修行道路的犹豫不决——它不是哲学反思，而是障碍修行的心理缠缚。

## 核心要义

1. **皮浪主义（Pyrrhonism）**：不做任何肯定判断，悬搁一切信念（epochē），由此获得心灵宁静。
2. **学园怀疑论（Academic Skepticism）**：肯定"知识不可能"，只承认"似真"（pithanon）作为行动指南。
3. **笛卡尔方法怀疑（Method Doubt）**：怀疑不是目的而是工具，通过普遍怀疑寻找确定基础。
4. **阿格里帕三难（Agrippa's Trilemma）**：一切证成要么无穷后退、要么循环、要么武断终止——基础主义与融贯论的困境。
5. **佛教疑（Vicikitsā）**：五盖之一，对真理与修行的犹豫障碍，需以正见和实证超越，非以怀疑为终极立场。

## 通俗 vs 学术

| 通俗用法 | 学术原义 |
|----------|----------|
| "我怀疑这件事"＝不确信 | 哲学怀疑论是对知识可能性本身的系统质疑 |
| "怀疑一切就是虚无主义" | 皮浪怀疑论追求宁静，笛卡尔怀疑追求确定基础 |
| "怀疑论不可反驳" | 怀疑论自身面临自指困难（"你确定你什么都不确定？"） |

## 代表思想家

| 思想家 | 贡献 |
|--------|------|
| 皮浪（Pyrrho） | 悬搁判断与心灵宁静的理想 |
| 塞克斯都·恩披里柯（Sextus Empiricus） | 系统记录怀疑论论证模式 |
| 笛卡尔（Descartes） | 方法怀疑：从普遍怀疑到确定基础 |
| 休谟（Hume） | 因果怀疑论与归纳问题 |
| 龙树（Nāgārjuna） | 以归谬法破一切自性见，但非虚无主义 |

## 应用场景

- **科学方法论**：可证伪性原则与怀疑精神的制度化
- **批判性思维教育**：区分健康怀疑与阴谋论式怀疑
- **法律**：合理怀疑（reasonable doubt）标准
- **冥想修行**：识别并超越"疑盖"对修行的障碍

## 常见误读

1. ❌ "怀疑论者什么都不相信" → ✅ 皮浪主义者依现象和习俗行动，只是不做独断的形而上学判断。
2. ❌ "笛卡尔是怀疑论者" → ✅ 笛卡尔使用怀疑作为方法，目的是建立确定知识，非停留于怀疑。
