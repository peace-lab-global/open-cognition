# 维特根斯坦著作导读 / Wittgenstein Works Guide

## 生前出版 / Published During Lifetime

### 《逻辑哲学论》*Tractatus Logico-Philosophicus* (1921)
- **结构 / Structure**: 7个主命题，编号递进式展开（1, 1.1, 1.11...）/ 7 main propositions, hierarchically numbered
- **核心主题 / Core Themes**: 语言界限、图像论、可说与不可说 / Limits of language, picture theory, sayable vs. showable
- **阅读建议 / Reading Advice**: 先读序言和最后几节（6.4–7），再回头读中间 / Read Preface and §§6.4–7 first, then return to the middle sections
- **重要英译 / Key Translations**: Pears & McGuinness (1961); Ogden (1922, 罗素序言版)

### "关于逻辑形式的一些Remarks" (1929)
- 维特根斯坦重返哲学后的首篇论文，承认《逻辑哲学论》中颜色排斥问题对图像论的挑战。
- First paper after return to philosophy; acknowledges the color-exclusion problem's challenge to the picture theory.

## 身后出版主要著作 / Major Posthumous Works

### 《哲学研究》*Philosophische Untersuchungen* (1953)
- **结构 / Structure**: 第一部分693节，第二部分（现称"第二部分"）14个章节 / Part I: 693 sections; Part II: 14 sections
- **核心主题 / Core Themes**: 语言游戏、私人语言、家族相似、遵守规则、看见为 / Language-games, private language, family resemblance, rule-following, seeing-as
- **阅读建议 / Reading Advice**:  §§1–88（语言游戏入门）→ §§243–315（私人语言论证）→ §§65–77（家族相似）/ Read §§1–88, then §§243–315, then §§65–77
- **重要英译 / Key Translations**: Anscombe, Hacker & Schulte (4th ed., 2009)
- **中文译本 / Chinese Translation**: 陈嘉映译（上海人民出版社）

### 《论确定性》*Über Gewissheit / On Certainty* (1969)
- **背景 / Background**: 1950–1951年最后的手稿，回应摩尔的"常识"论文
- **核心主题 / Core Themes**: 铰链命题（hinge propositions）、知识与确定性的关系、怀疑论的界限
- **阅读建议 / Reading Advice**: 关注"铰链命题"概念——它连接了认识论与实践哲学

### 《蓝皮书与棕皮书》*The Blue and Brown Books* (1958)
- **性质 / Nature**: 1933–1935年讲课笔记 / Lecture notes from 1933–1935
- **价值 / Value**: 展示从《逻辑哲学论》到《哲学研究》的过渡阶段
- **核心主题 / Core Themes**: 意义与使用、语言游戏概念的早期形态

### 《文化与价值》*Culture and Value* (1977, expanded 1998)
- **性质 / Nature**: 1914–1951年间格言式笔记的选编
- **价值 / Value**: 展示维特根斯坦的文化观、宗教反思和美学见解
- **名言 / Famous Quote**: "哲学的目的是思想的逻辑澄清"

### 《哲学评论》*Philosophical Remarks* (1964/1975)
- **时期 / Period**: 1929–1930年手稿 / Manuscript from 1929–1930
- **核心主题 / Core Themes**: 现象学语言的可能性、感觉语言的分析

### 《数学基础评论》*Remarks on the Foundations of Mathematics* (1956/1978)
- **核心主题 / Core Themes**: 对数学形式主义和逻辑主义的批评、规则遵守的数学维度
- **争议 / Controversy**: 部分数学评论被认为不够精确，但其哲学洞见仍有价值

### 《哲学语法》*Philosophical Grammar* (1974)
- **时期 / Period**: 1931–1934年间手稿 / Manuscripts from 1931–1934
- **核心主题 / Core Themes**: 意义即使用的早期阐述、命题与期望的语法

### 《最后著作》*Last Writings on the Philosophy of Psychology* (1982/1992)
- **核心主题 / Core Themes**: "看见为"（seeing-as）、鸭兔图的深入分析、心理学概念的哲学分析

## 推荐入门路径 / Recommended Entry Paths

1. **初学者 / Beginners**: 《哲学研究》§§1–64 → 《蓝皮书》→ 《哲学研究》§§243–315
2. **逻辑与语言 / Logic & Language**: 《逻辑哲学论》→ "关于逻辑形式" → 《哲学语法》
3. **心灵哲学 / Philosophy of Mind**: 《哲学研究》第二部分 → 《最后著作》→ 《论确定性》
4. **文化与价值 / Culture & Values**: 《文化与价值》→ 传记（Ray Monk）→ 《哲学研究》
