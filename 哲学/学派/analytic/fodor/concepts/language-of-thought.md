# 思维语言 · Language of Thought (LOT)

## 概念定位 / Conceptual Position
思维不在自然语言中进行，而在内在符号系统（mentalese）中进行。自然语言是mentalese的"翻译"。

Thinking occurs in mentalese, not natural language. Natural language is a "translation" of mentalese.

## 核心论述 / Core Argument

### Mentalese的特征 / Features of Mentalese
- **组合性 / Compositionality**: 复杂思想由简单符号组合而成
- **生产性 / Productivity**: 可表达无限多的思想
- **系统性 / Systematicity**: 能想P的人也能想Q（结构相关）

### 论证 / Arguments for LOT
1. 思维具有生产性 → 需要组合性符号系统
2. 思维具有系统性 → 需要结构化表征
3. 动物和前语言婴儿也能思考 → 思维不依赖自然语言

## 哲学意义 / Philosophical Significance
为认知科学和AI提供了"思维是什么"的精确理论框架。

## 东西方对话 / East-West Dialogue
### 与唯识学"名言种子" / With Yogacara's "Name-Seeds"
唯识学的"名言种子"是一种深层符号系统，从中产生表层认知。与mentalese类似，但唯识学的种子是业力因果机制，Fodor的是计算符号操作。

Yogacara's "name-seeds" are a deep symbol system producing surface cognition, similar to mentalese but karma-driven rather than computational.
