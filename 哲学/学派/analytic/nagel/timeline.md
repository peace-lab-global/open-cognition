# 内格尔生平年表 · Thomas Nagel Biography Timeline

## 早年与学术训练（1937–1966）

| 年份 | 事件 |
|------|------|
| 1937.7.4 | 生于南斯拉夫贝尔格莱德（父亲为美国外交官），在华盛顿特区长大 |
| 1954–1958 | 在康奈尔大学（Cornell University）学习哲学，获学士学位 |
| 1958–1960 | 获马歇尔奖学金赴牛津大学学习，获学士学位 |
| 1960–1963 | 在哈佛大学攻读博士，导师：约翰·罗尔斯（John Rawls） |
| 1963 | 获哈佛大学博士学位，论文论利他主义 |
| 1963–1966 | 在加利福尼亚大学伯克利分校任教 |

## 普林斯顿时期（1966–1980）

| 年份 | 事件 |
|------|------|
| 1966 | 转至普林斯顿大学任哲学教授 |
| 1970 | 出版 *The Possibility of Altruism*——首部专著 |
| 1970 | 发表 "Death"——死亡的坏处分析 |
| 1971 | 发表 "Brain Bisection and the Unity of Consciousness" |
| 1974 | 发表 "What Is It Like to Be a Bat?"——20世纪被引用最多的哲学论文之一 |
| 1976 | 发表 "Moral Luck"——道德运气理论 |
| 1979 | 出版 *Mortal Questions*——论文集 |

## 纽约大学时期（1980–2016）

| 年份 | 事件 |
|------|------|
| 1980 | 转至纽约大学（NYU）任大学哲学教授 |
| 1986 | 出版 *The View from Nowhere*——主观-客观张力的全面分析 |
| 1991 | 出版 *Equality and Partiality*——政治哲学 |
| 1995 | 出版 *Other Minds: Critical Essays 1969-1994* |
| 1997 | 出版 *The Last Word*——捍卫理性的客观性 |
| 2002 | 出版 *Concealment and Exposure* |
| 2010 | 出版 *Secular Philosophy and the Religious Temperament* |
| 2012 | 出版 *Mind and Cosmos*——论证物理自然主义不足，引发巨大争议 |
| 2016 | 从纽约大学荣休 |

## 退休后（2016–至今）

| 年份 | 事件 |
|------|------|
| 2016– | 荣休后继续写作与参加学术活动 |
| 2017 | 发表关于意识和价值客观性的论文 |
| 2020s | 持续参与关于 *Mind and Cosmos* 的学术讨论 |

## 思想分期总结

| 阶段 | 年代 | 关键词 |
|------|------|--------|
| 学术训练 | 1937–1966 | 康奈尔、牛津、哈佛、罗尔斯 |
| 普林斯顿 | 1966–1980 | 蝙蝠论文、道德运气 |
| 纽约大学 | 1980–2016 | 《来自无处的视角》、《心灵与宇宙》 |
| 退休后 | 2016– | 继续写作 |
