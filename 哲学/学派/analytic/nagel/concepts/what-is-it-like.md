# 成为X是什么感觉 · What It Is Like

## 概念定位 / Conceptual Position
意识的核心特征是主观性——存在一个"成为"某种有意识存在的"感觉"（phenomenal character），不能被客观描述穷尽。

The core feature of consciousness is subjectivity — a "what it is like" (phenomenal character) that cannot be exhausted by objective description.

## 核心论述 / Core Argument

### 概念的内涵 / The Concept's Content
"成为X是什么感觉"不是问"X像什么"（比较），而是问：从X的第一人称视角体验世界是什么感受？这个"what it is like"是意识体验的本质特征。

"What it is like to be X" is not asking "what X is like" (comparison) but: what is the experience from X's first-person perspective? This "what it is like" is the essential feature of conscious experience.

### 与物理主义的冲突 / Conflict with Physicalism
即使我们拥有蝙蝠大脑的完整物理描述（神经元放电、化学信号等），我们仍然缺乏"成为蝙蝠"的主观信息。这说明物理描述在原则上是不完备的。

Even with complete physical descriptions of bat brains (neural firing, chemical signals), we still lack subjective information about "being a bat." This suggests physical description is in principle incomplete.

## 哲学意义 / Philosophical Significance
成为Chalmers"硬问题"和Levine"解释鸿沟"的直接来源。
