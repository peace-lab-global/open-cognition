# 令人厌恶的结论 · Repugnant Conclusion

## 概念定位 / Conceptual Position
人口伦理学的核心难题：对于任何具有高质量生活的人口A，都存在一个更大的人口Z，其中每个人的生活质量仅勉强值得过下去，但总体价值大于A。

Central puzzle of population ethics: for any population A with very high quality of life, there exists a larger population Z with barely-worth-living lives yet greater total value.

## 核心论述 / Core Argument

### 推导过程 / The Derivation
从100亿极度幸福的人开始，逐步增加人口并降低每个人的幸福度——每一步都被视为"改善"（因为总价值增加），最终到达Z：巨大的、勉强值得活下去的人口。

Start with 10 billion extremely happy people. Gradually increase population while reducing each person's happiness — each step seems an "improvement" (total value increases) — until reaching Z: an enormous population with lives barely worth living.

### 逃避尝试 / Attempts to Avoid
- 平均功利主义：但面临"精英淘汰"问题
- 临界值理论：但难以确定临界值的合理位置
- Parfit自己提出的"理论X"：他一直在寻找但未能完成

## 哲学意义 / Philosophical Significance
揭示了目的论伦理学（特别是功利主义）在处理人口问题时的根本困难。至今没有令人满意的解决方案。

## 东西方对话 / East-West Dialogue
### 与佛教"众生无边" / With Buddhism's "Countless Beings"
佛教发愿"众生无边誓愿度"——追求的是所有众生的解脱，而非最大化幸福总量。这提供了一种不同于功利主义的人口伦理视角。

Buddhism vows to "liberate countless beings" — pursuing liberation for all beings, not maximizing total happiness. This offers an alternative to utilitarian population ethics.
