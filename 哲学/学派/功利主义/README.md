# 功利主义 · Utilitarianism

## 一句话定义

功利主义是 18 世纪末至 19 世纪兴起于英国的规范伦理学与政治哲学传统，主张行为的道德正当性取决于其对“最大多数人的最大幸福”的促进，并以后果而非动机或义务作为道德判断的核心标准。

## 历史脉络

**前驱与思想渊源（17—18 世纪）**
功利主义的思想先驱包括伊壁鸠鲁的快乐主义、洛克与休谟的道德情感论，以及法国启蒙思想家爱尔维修、霍尔巴赫对社会功利与教育改革的讨论。休谟在《道德原则研究》中已使用“功利”（utility）一词说明道德赞许的基础。

**边沁的奠基（18 世纪末）**
杰里米·边沁（Jeremy Bentham，1748—1832）在《道德与立法原理导论》（1789）中首次系统提出功利原则，将苦乐视为道德与立法的最终标准，并发展出“快乐计算”（felicific calculus）以量化行为后果。

**密尔的修正与发展（19 世纪中）**
约翰·斯图尔特·密尔（John Stuart Mill，1806—1873）在《功利主义》（1861）中区分高级快乐与低级快乐，将质的重要性引入功利计算；在《论自由》（1859）中将伤害原则与个性发展纳入自由主义框架。

**当代争论（20 世纪至今）**
20 世纪西季威克（Henry Sidgwick）在《伦理学方法》（1874）中 refined 功利主义；摩尔、罗尔斯、威廉姆斯等人从不同角度批评其忽略权利、个人完整性与分配正义。当代有效利他主义（Effective Altruism）可视为功利主义在实践伦理领域的延伸。

## 核心主张

1. **后果论（Consequentialism）**
   行为的道德价值取决于其实际或预期后果；正确的行为是能够在相关范围内产生最佳总体后果的行为。

2. **功利原则（Principle of Utility）**
   快乐（或幸福、福祉）是唯一内在善，痛苦是唯一内在恶；道德与制度的目标是最大化幸福总量、最小化痛苦总量。

3. **最大化要求**
   行动者或制度应当选择能带来最大净功利（总快乐减去总痛苦）的选项。这一要求既适用于个体行为，也适用于法律与公共政策。

4. ** impartiality（不偏不倚）**
   每个人的幸福都计入同一总量，无人被赋予内在优先性。边沁的名言是“每个人只算一个，没人超过一个”。

5. **行动功利主义与规则功利主义**
   行动功利主义直接评估每一具体行为；规则功利主义则评估行为所遵循的规则，认为遵守某些规则（如“不撒谎”）从长远看能最大化功利。

## 关键概念

- **功利（Utility）**：通常指幸福、快乐或福祉的总量。
- **快乐计算（Felicific Calculus）**：边沁提出的七个维度：强度、持续时间、确定性、接近性、繁殖性、纯粹性、范围。
- **高级快乐与低级快乐**：密尔区分智力、审美、道德快乐与肉体快乐，主张有经验者更偏好前者。
- **伤害原则（Harm Principle）**：密尔提出，限制个人自由的唯一正当理由是防止对他人造成伤害。
- **平均功利 vs 总体功利**：当代争论焦点之一，涉及人口伦理学中的“令人厌恶的结论”。

## 常见误读

**误读一：功利主义只关心金钱或物质利益。**
纠正：功利主义将“功利”理解为幸福、快乐或福祉，而非货币收益。它涵盖痛苦、知识、友谊、自主等一切影响福祉的因素。

**误读二：功利主义允许为整体利益牺牲无辜者。**
纠正：经典功利主义在逻辑上可能推出此类结论，这正是规则功利主义、权利约束功利主义与当代学者（如密尔、西季威克）试图修正的问题。但不能将某一极端推论等同于整个传统。

**误读三：功利主义忽视正义与分配。**
纠正：功利主义内部有对分配问题的持续讨论，包括平等主义功利主义、优先论等。但它确实以总体福祉而非应得为首要标准，这是其与义务论、德性伦理的关键分歧。
