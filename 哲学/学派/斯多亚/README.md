# 斯多亚学派 · Stoicism

## 一句话定义

斯多亚学派是公元前 4 世纪末兴起于雅典、延续至罗马帝国的古希腊哲学流派，主张德性即幸福，强调通过理性区分可控之事与不可控之事，以内心的自足与平和（ataraxia）为人生目标。

## 历史脉络

**奠基期（公元前 4 世纪末—前 3 世纪）**
基提翁的芝诺（Zeno of Citium，约公元前 334—前 262）于约公元前 300 年在雅典的彩绘长廊（Stoa Poikile）讲学，学派因此得名。克里安西斯（Cleanthes）与克里西波斯（Chrysippus，约公元前 279—前 206）将斯多亚哲学系统化，建立起涵盖逻辑学、物理学与伦理学的完整体系。

**罗马时期（公元前 1 世纪—公元 2 世纪）**
斯多亚思想在罗马时期转向实践伦理。塞涅卡（Seneca，约公元前 4—公元 65）、爱比克泰德（Epictetus，约 50—135）与罗马皇帝马可·奥勒留（Marcus Aurelius，121—180）以书信、讲演录与自省笔记的形式，将斯多亚伦理学发展为面向日常生活的修身传统。

**衰落与复兴（公元 3 世纪后）**
斯多亚学派随着古代哲学的整体衰落而式微，但其思想通过西塞罗、普鲁塔克等人的引介进入拉丁传统，并在 20 世纪末以“现代斯多亚主义”（Modern Stoicism）的形式重新受到关注。

## 核心主张

1. **德性即唯一之善**
   唯有品格与德性（aretē）是真正的善；财富、健康、名誉等外在事物本身既非善亦非恶，而是“无所谓之物”（adiaphora）。幸福在于德性的实践，而非外在境遇。

2. **顺应自然（homologoumenos tē phusei）而活**
   理性是人的本质，按照理性亦即按照宇宙的自然法则（logos）生活，是人应当追求的目标。斯多亚的“自然”兼具规范性与宇宙论含义。

3. **控制二分法**
   事物分为“在我们能力范围内者”（判断、冲动、欲望）与“不在我们能力范围内者”（身体、财产、名誉、命运）。对前者负全责，对后者全然接纳，是内心平静的根源。

4. **宇宙城邦与普遍理性**
   所有人共享同一理性与宇宙法则，因此都是宇宙城邦（kosmopolis）的公民。斯多亚伦理学蕴含早期自然法与世界主义思想。

5. **激情是错误判断**
   愤怒、恐惧、欲望等激情（pathē）被理解为错误的价值判断；通过理性审查，激情可被转化为适度的情感（eupatheia），如愉悦、愿望与谨慎。

## 关键概念

- **德性（Aretē）**：理性而完整的人格品质，包括智慧、勇敢、节制、正义四主德。
- **自然法（Natural Law）**：贯穿宇宙的理性法则，构成道德判断的客观依据。
- **自足（Autarkeia）**：不依赖外在善的心灵独立状态。
- **不动心（Apatheia）**：不被激情支配，而非情感的彻底缺失。
- **宿命论与相容论**：万物依因果必然性发生，但人的道德责任仍可通过内在同意而保留。

## 常见误读

**误读一：斯多亚主义是压抑情感、麻木不仁。**
纠正：斯多亚追求的是不被错误判断驱动的激情所奴役，而非消除一切情感。它区分 pathē（病态激情）与 eupatheia（适度情感），后者包括对善的真实喜悦。

**误读二：斯多亚主义是消极宿命论，教人放弃行动。**
纠正：斯多亚强调“命运引导愿意者，拖拽不情愿者”。人虽不能控制结果，却须尽其所能地行动，并对行动本身负责。接纳不可控之事与积极行动并不矛盾。

**误读三：斯多亚主义只关注个人内心，不关心社会。**
纠正：斯多亚具有强烈的宇宙城邦与世界主义维度，塞涅卡、奥勒留均将伦理责任延伸至政治与公共生活。德性不仅是个体修养，也是社会角色的正当履行。
