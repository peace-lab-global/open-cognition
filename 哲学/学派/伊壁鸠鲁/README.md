# 伊壁鸠鲁主义 · Epicureanism

## 一句话定义

伊壁鸠鲁主义是公元前 4 世纪由伊壁鸠鲁创立的希腊化哲学流派，主张快乐（ataraxia，宁静）是人生的最高善，并通过原子论物理学与感觉论认识论为伦理生活提供自然主义基础。

## 历史脉络

**创立与花园学派（公元前 4—前 3 世纪）**
伊壁鸠鲁（Epicurus，公元前 341—前 270）于公元前 306 年在雅典建立“花园”（Kepos），接纳包括奴隶与女性在内的各类学生，这在古代哲学团体中较为罕见。他撰写约 300 部著作，大多散佚，核心思想保存于三封书信与《要道篇》（Kyriai Doxai）。

**罗马传播（公元前 1 世纪—公元 2 世纪）**
罗马诗人卢克莱修（Titus Lucretius Carus，约公元前 99—前 55）以长诗《物性论》（*De Rerum Natura*）系统阐释伊壁鸠鲁的原子论与伦理学，成为该学派最重要的拉丁文献。西塞罗、塞涅卡等人虽持批判态度，但其记述亦保存了大量伊壁鸠鲁思想。

**衰落与近代影响（公元 3 世纪后）**
伊壁鸠鲁主义在古代晚期因基督教批判而衰落。文艺复兴时期《物性论》的重新发现影响了近代机械论与自然主义，17—18 世纪功利主义亦受其快乐主义伦理影响。

## 核心主张

1. **快乐是最高善**
   “快乐是幸福生活的开端与归宿。”但伊壁鸠鲁所理解的快乐并非纵欲，而是“身体的无痛苦与心灵的无纷扰”（ataraxia）。最大快乐在于欲望的简约与满足的持久。

2. **欲望三分法**
   欲望分为自然的且必要的（如基本温饱）、自然的但非必要的（如奢华饮食）、既不自然也不必要的（如权力与名望）。伦理智慧在于满足第一类，审慎对待第二类，摒弃第三类。

3. **原子论物理学**
   宇宙由不可分割的原子在虚空中运动构成；灵魂亦是精细原子，随身体死亡而消散。否认来世、神罚与目的论，为“无惧死亡”提供本体论依据。

4. **感觉论认识论**
   感觉是所有知识的最终标准。错误不在于感觉本身，而在于对感觉所作的判断与意见。理性应服务于感觉的澄清与欲望的节制。

5. **友谊与隐退生活**
   最可靠的快乐来源是亲密的友谊（philía）与远离政治纷扰的简朴生活。伊壁鸠鲁将友谊视为最高层面的安全与快乐。

## 关键概念

- **Ataraxia（宁静）**：身体无痛苦、心灵无恐惧的安稳状态，是真正的快乐。
- **Aponia（无痛苦）**：身体层面的无痛苦状态，是 ataraxia 的 bodily 维度。
- **原子与虚空（Atoms and Void）**：宇宙的本原；运动与变化由此产生。
- **神的中立性**：神存在，但不干预人间事务；对神的恐惧源自错误观念。
- **死亡与我们无关**：灵魂随身体死亡而消散，死亡不是生者所经历之事，故无需恐惧。

## 常见误读

**误读一：伊壁鸠鲁主义是纵欲享乐主义。**
纠正：伊壁鸠鲁明确区分动态快乐（满足欲望的过程）与静态快乐（无痛苦、无纷扰的状态），并将后者视为更高。他警告纵欲带来的欲望膨胀与痛苦远大于快乐。

**误读二：伊壁鸠鲁主义是逃避现实的隐遁哲学。**
纠正：伊壁鸠鲁建议远离政治是为了避免无谓的焦虑与依附，而非拒绝一切社会责任。花园学派本身就是一种共同体生活，友谊与互助是其核心实践。

**误读三：伊壁鸠鲁主义否定神与宗教。**
纠正：伊壁鸠鲁承认神的存在，但主张神是不受世界事务所扰的完美存在。他批判的是大众宗教中的神罚观念与对来世的恐惧，而非宗教本身。
