# 与海德格尔关系 · Forhold til Heidegger · Dialogue with Heidegger

## 一句话定位

海德格尔（Martin Heidegger）是克尔凯郭尔最重要的哲学继承者之一——他将克尔凯郭尔的焦虑、个体性和存在分析**去基督教化**并纳入基础存在论（Fundamentalontologie），但这一"继承"同时也是一种**深刻的改写和遮蔽**。

---

## 思想传承的结构

### 一、焦虑（Angst / Angest）的传承与变形

海德格尔在《存在与时间》（Sein und Zeit, 1927）中的焦虑（Angst）分析直接源于克尔凯郭尔：

| 维度 | 克尔凯郭尔的焦虑（Angest） | 海德格尔的焦虑（Angst） |
|------|---------------------------|------------------------|
| 面对的对象 | 自由的可能性 / 虚无 | 存在本身 / 在世存在的整体 |
| 神学维度 | 有——焦虑与原罪关联 | 无——去基督教化 |
| 出路 | 信仰的跳跃 | 本真存在的决断（Entschlossenheit） |
| 文本 | 《焦虑的概念》(1844) | 《存在与时间》§40 (1927) |

海德格尔自己承认了这一传承——《存在与时间》脚注中引用了克尔凯郭尔，尽管引用次数远少于克尔凯郭尔对他的实际影响所应有的。

### 二、"向死存在"（Sein-zum-Tode）vs "面对上帝"

克尔凯郭尔的个体在信仰中面对上帝——他的"终极处境"是与绝对者的关系。海德格尔将这一"终极处境"**世俗化**为"向死存在"：

- 克尔凯郭尔：个体面对上帝时的不可替代性和孤独
- 海德格尔：此在面对死亡时的不可替代性和孤独
- 两者都强调：在面对最根本的存在问题时，"常人"（das Man / Mængden）的帮助完全无效
- 但克尔凯郭尔的"上帝"被海德格尔的"死亡"取代——**终极关切从超越者转向了此在自身的有限性**

### 三、"常人"（das Man）vs "群众"（Mængden）

海德格尔的"常人"（das Man / the They）概念直接继承了克尔凯郭尔的"群众"批判：

| 克尔凯郭尔 | 海德格尔 |
|------------|----------|
| 群众即非真理 | 常人的非本真存在 |
| "拉平"（Nivellering） | "平均状态"（Durchschnittlichkeit） |
| 公众（Publikum）的抽象暴力 | 闲言（Gerede）的匿名权威 |
| 宗教维度——群众基督教 | 存在论维度——日常性的沉沦 |

### 四、"本真"（Eigentlichkeit）vs "内在性"（Inderlighed）

| 克尔凯郭尔的内在性 | 海德格尔的本真 |
|-------------------|---------------|
| 个体与上帝的内在关系 | 此在对自身存在的本真关系 |
| 主观真理——存在地把握真理 | 决断（Entschlossenheit）——面对自身可能性的决断 |
| 宗教阶段的激情 | 先行的决断（vorlaufende Entschlossenheit） |
| 有上帝作为参照 | 无超越者——此在自身的有限性 |

### 五、海德格尔对克尔凯郭尔的"遮蔽"

海德格尔对克尔凯郭尔的继承伴随着系统的**去基督教化**和**存在论化**：

1. 克尔凯郭尔的焦虑有**神学维度**（与原罪关联）——海德格尔取消了这一维度
2. 克尔凯郭尔的个体概念预设了**与上帝的关系**——海德格尔的此在没有超越者
3. 克尔凯郭尔的信仰概念包含**荒谬**和**激情**——海德格尔的"本真"是冷静的决断
4. 海德格尔将克尔凯郭尔的**存在分析**转化为**基础存在论**——从生存描述变为哲学体系

这一遮蔽的效果：20世纪大部分哲学读者是通过海德格尔而非克尔凯郭尔原本来理解"焦虑"和"本真"概念的——克尔凯郭尔的原始力量被海德格尔的学院化语言所稀释。

### 六、"存在"概念的根本分歧

尽管海德格尔继承了克尔凯郭尔的许多分析工具，但两者对"存在"的理解有根本分歧：

- 克尔凯郭尔的"存在"（Tilværelse / Existence）是**个体的生存方式**——审美/伦理/宗教
- 海德格尔的"存在"（Sein / Being）是**存在论的基本问题**——"存在的意义"
- 克尔凯郭尔问的是"**我**如何存在？"——存在总是**我的**存在
- 海德格尔问的是"存在的**意义**是什么？"——存在是一个普遍的问题

---

## 下游影响

克尔凯郭尔-海德格尔的传承链塑造了20世纪存在哲学的核心面貌：

- **萨特**通过海德格尔阅读克尔凯郭尔——将"焦虑"和"本真"推向无神论存在主义
- **梅洛-庞蒂**通过海德格尔的身体化此在概念——间接继承了克尔凯郭尔的身体-精神二元性
- **伽达默尔**的海德格尔诠释学——间接传承了克尔凯郭尔的"内在性"概念
- **德里达**对海德格尔的解构——在某种意义上恢复了克尔凯郭尔对体系的不信任
