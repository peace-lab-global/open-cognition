# 雅克·艾吕尔 · Jacques Ellul

雅克·艾吕尔（Jacques Ellul，1912–1994）是法国社会学家、法学家与神学家。他以“技术社会”批判著称，指出“技术”（la technique）已成为自主的、追求效率至上的总体系统，深刻塑造了现代生活的各个领域。

## 核心命题 / Core Theses

### 1. 技术作为自主系统

技术不是中性的工具集合，而是一种自我增长的逻辑：凡事追求效率、标准化与可计算。这一逻辑逐渐渗透经济、政治、法律与伦理。

### 2. 手段对目的的倒置

在技术社会中，手段不再服务于明确的目的；相反，效率本身成为最高目的，人则被改造为适应技术系统的环节。

### 3. 宣传的技术基础

现代宣传之所以有效，不仅因为意识形态，更因为它借助技术手段（媒体、数据、心理学）将个体嵌入一个看似自然的社会环境。

## 关键著作 / Key Works
