# 与海德格尔关系 · Hegel und Heidegger

## 关系定位

海德格尔（Martin Heidegger, 1889–1976）与黑格尔的关系是他整个思想生涯中最持久、最深层的对话之一。海德格尔既将黑格尔视为**形而上学的完成者**——西方存在-神-逻辑学（Onto-theo-logie）传统的最高峰——又将黑格尔视为必须被**解构**（Destruktion）的思想对象，以便重新提出"存在的意义"问题。

## 继承与转化

### 一、现象学方法的改造

海德格尔在《存在与时间》（*Sein und Zeit*, 1927）中采用了"现象学"的名称，但赋予了与黑格尔不同的含义：

| 黑格尔的现象学 | 海德格尔的现象学 |
|---------------|-----------------|
| 意识的经验之路——从感性确定性到绝对知识 | 存在者的存在方式之揭示——从日常性到本真性 |
| 辩证的、上升的 | 描述的、诠释学的 |
| 最终达到"绝对知识" | 最终开放"存在的意义"问题 |
| 精神（Geist）的现象学 | 此在（Dasein）的分析论 |

### 二、时间性（Zeitlichkeit）

海德格尔对黑格尔时间概念的批判性分析：

- 黑格尔将时间理解为**空间的否定**——时间是"存在的非存在"
- 海德格尔认为这种时间概念是**流俗的**（vulgär）——它从"现在"的序列出发，遮蔽了时间的**生存论**结构
- 海德格尔的**时间性**（Zeitlichkeit）：此在的存在方式——将来（Zukunft）、过去（Gewesenheit）、当下（Gegenwart）的三维统一

### 三、对黑格尔"精神"概念的批判

海德格尔在《存在与时间》第82节专门讨论了黑格尔的精神概念：

- 黑格尔的"精神"（Geist）被海德格尔解读为一种**存在-神-逻辑学**的构造——它将上帝（绝对者）与人的精神合为一体
- 海德格尔主张：黑格尔的"精神"预设了一种**现成存在**（Vorhandenheit）——精神是"已经在那里的"，而非"在时间中展开的"
- 真正的"精神"问题必须回到**此在**的存在论层次——不是"精神是什么"，而是"此在如何存在"

## 批判

### 四、形而上学的完成

海德格尔将西方形而上学史描述为"存在的遗忘"（Seinsvergessenheit）的历史——从柏拉图到黑格尔，存在者被思考，但**存在本身**（Sein）被遗忘。

在这个框架中，黑格尔的位置是独特的：

- 黑格尔是形而上学的**完成者**（Vollender）——他将存在的总体化推到了极限
- 绝对精神 = 自我认识的存在者总体 = 存在-神-逻辑学的最高形式
- 在黑格尔之后，形而上学不再能前进——它只能被**克服**（Verwindung）

### 五、存在-神-逻辑学（Onto-theo-logie）

海德格尔将形而上学的结构描述为"存在-神-逻辑学"：

- **存在论**（Ontologie）：对存在者的普遍规定（逻辑学）
- **神学**（Theologie）：对最高存在者的规定（上帝/绝对）
- **逻辑学**（Logie）：以逻各斯/理性来把握存在

黑格尔完美地体现了这个三重结构：
- 他的《逻辑学》是存在论
- 他的绝对精神是神学（上帝的哲学概念化）
- 他的辩证法是逻辑学

### 六、否定性的不同理解

| 黑格尔的否定性 | 海德格尔的否定性 |
|---------------|-----------------|
| 概念的内在矛盾——推动辩证运动 | 存在的虚无（Nichts）——存在者的根基不在自身 |
| 否定性被扬弃为肯定性 | 虚无不能被扬弃——它是此在的基本境况 |
| "否定之否定"达到更高的肯定 | "否定之否定"遮蔽了虚无的原始力量 |

海德格尔认为：黑格尔的"否定之否定"是形而上学将虚无**驯化**的方式——通过辩证法将虚无转化为更高肯定的环节，从而**遮蔽**了虚无的原始力量。

## 共同遗产

1. **历史性**（Geschichtlichkeit）：两者都认为人的存在本质上是**历史性的**——不是超越时间的永恒存在
2. **整体性关怀**：两者都追求思想的"全体"——虽然海德格尔的"全体"不是概念的总体，而是存在的"四重体"（Geviert）
3. **对技术的反思**：两者都对现代性的"合理化"有所警惕——虽然方式不同
4. **希腊遗产**：两者都回到前苏格拉底哲学寻找思想资源

## 后世对话

| 思想家 | 关系 |
|--------|------|
| 伽达默尔 | 在黑格尔的辩证法与海德格尔的诠释学之间建立"哲学诠释学" |
| 德里达 | 以"解构"同时批判黑格尔的扬弃和海德格尔的"存在" |
| 马尔库塞 | 在《黑格尔本体论与历史性理论的基础》中综合两者 |
| 让-吕克·南希 | 以"意义的分享"超越黑格尔的总体和海德格尔的存在 |
