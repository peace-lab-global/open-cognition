# 永久和平 · Zum ewigen Frieden

## 一句话定义

《永久和平论》（*Zum ewigen Frieden*, 1795）是康德政治哲学的核心文本——提出了一套以**共和制**、**国际联盟**和**世界公民权**为三大支柱的永久和平方案，深刻影响了现代国际法和全球治理的制度设计。

## 核心论证

### 一、文本背景与结构

《永久和平论》发表于法国大革命（1789）和巴塞尔和约（1795）之间的历史节点。康德以模仿条约的形式撰写此文，包含六条"预备条款"（Präliminarartikel）、三条"确定条款"（Definitivartikel）、两篇"附论"和一篇"秘密条款"。

标题中的"永久和平"（ewiger Friede）也是一个墓碑铭文的双关——暗示只有死者才能获得"永久和平"，而活人必须为和平而不断工作。

### 二、六条预备条款

预备条款规定了缔结和平条约的最低条件：

1. **不得保留未来战争的隐患**：任何暗含未来战争意图的和约都不是真正的和约
2. **不得吞并独立国家**：任何独立国家不得被另一国通过继承、交换、购买或赠与获得
3. **逐步废除常备军**：常备军（miles perpetuus）本身就是战争威胁
4. **不得为战争目的举借国债**：将金融作为战争工具是对和平的威胁
5. **不得武力干涉他国内政**：尊重各国的自主权
6. **不得在战争中使用使未来互信不可能的手段**：如暗杀、投毒、煽动叛变

### 三、三条确定条款

**第一条：每个国家的公民宪政应当是共和制的**

共和制（Republik）的三个原则：
- **自由**：所有社会成员作为人的自由
- **唯一的共同立法**：所有人作为臣民服从同一法律
- **法律上的平等**：所有人作为公民的平等

康德的"共和制"不等于现代的"民主制"——他强调的是**代议制**和**权力分立**，反对直接民主（他认为这是专制——"所有人对一个人做决定"）。

**第二条：国际法应当建立在自由国家的联盟之上**

康德论证：正如个人在自然状态中通过社会契约建立国家，各国也应通过**自由联盟**（freier Bund / foedus pacificum）走出国际自然状态。但康德明确反对建立"世界国家"（Weltstaat）：

> "这样的联盟……不是一个国际国家（Völkerstaat）——这会是一个矛盾。"

康德选择"联盟"而非"联邦"的理由：世界国家要么走向暴政，要么因疆域过大而瓦解。自由联盟保留了各国的主权，同时提供和平解决争端的机制。

**第三条：世界公民权应当限于普遍友善的条件**

世界公民权（Weltbürgerrecht / ius cosmopoliticum）是康德最具前瞻性的构想：
- 每个人都有权访问（Besuchsrecht）他国，但不等于定居权（Gastrecht）
- 任何国家不得拒绝外来者，如果这会导致其灭亡
- 这一权利的基础是**地球表面的共同所有权**——地球是圆的，人类不可能无限分散，最终必须忍受彼此的邻近

康德特别批判了欧洲殖民主义：

> "文明国家对待与我们接触的土地和人民的不正义……它们把访问视为征服。"

### 四、附论：自然保证与道德政治

**自然的保证**：康德论证自然机制（战争迫使人类扩散到全球各地、商业精神与战争不相容）倾向于和平——这是一种目的论论证，不是道德论证。

**道德政治**：康德区分了"政治道德家"（将道德服务于政治目的）和"道德政治家"（将政治服从于道德原则）：

> "一切政治都必须在道德面前屈膝。"

### 秘密条款与历史语境

《永久和平论》包含一条"秘密条款"：各国应邀请哲学家就战争与和平的条件发表意见——这不是赋予哲学家以政治权力，而是让理性的公开运用影响政治决策。这一条款体现了康德对"哲学家-国王"理想（柏拉图）的拒绝：哲学家不应统治国家，但国家应倾听哲学家的声音。

此文发表于1795年巴塞尔和约签订之际——普鲁士与法国革命政府缔结和约。康德敏锐地意识到，这一和约不过是暂时的休战，而非真正的和平。他的永久和平方案是对一切"权宜之计"式的和平的哲学批判。

### 五、对后世的影响

康德的永久和平方案直接影响了：

| 影响领域 | 具体表现 |
|----------|----------|
| **国际联盟**（1919） | 康德的国际联盟构想的第一次制度化尝试 |
| **联合国**（1945） | 安理会、大会和秘书处的结构呼应了康德的制度设计 |
| **民主和平论** | 政治学实证研究证实：民主国家之间极少发生战争 |
| **欧盟** | 超国家联盟的实践——尽管超越了康德的"自由联盟"概念 |
| **罗尔斯《万民法》** | 直接继承康德的和平论，发展为"人民社会"的构想 |
| **哈贝马斯"后民族格局"** | 将康德的世界公民权发展为"宪法爱国主义"和全球公共领域理论 |

### 六、永久和平的当代挑战

康德的永久和平方案在21世纪面临新的挑战：

**民主和平论的局限**：政治学的实证研究证实民主国家之间极少发生战争（Doyle, 1986），但民主国家与非民主国家之间的战争仍然频繁。康德的方案预设了共和制的全球扩散——但威权资本主义的兴起挑战了这一预设。

**国际组织的困境**：联合国在结构上继承了康德的国际联盟构想，但安理会常任理事国的否决权使其经常陷入瘫痪。康德设想的"自由联盟"缺乏强制执行力——这既是其优点（保留主权）也是其弱点。

**全球不平等的挑战**：康德的世界公民权概念在全球不平等加剧的背景下显得尤为紧迫——"访问权"（Besuchsrecht）对于因贫困和战争而被迫迁移的数百万难民来说是一个生死攸关的问题。

**数字主权与网络战争**：康德的理论没有预见网络战争和信息战——当"战争"不再需要军队跨越国境时，国际法的框架需要根本性的更新。

## 与相关概念的比较

| 概念 | 关系 | 差异 |
|------|------|------|
| **卢梭的《圣皮埃尔神父的永久和平方案》** | 直接来源 | 康德将卢梭的方案从政治层面提升到道德和法哲学层面 |
| **黑格尔的国家观** | 批判 | 黑格尔认为国家是"地上的精神"，国际法只是"应然"——战争是历史的必然 |
| **罗尔斯的万民法** | 当代发展 | 罗尔斯将康德的构想扩展为"合宜人民"和"合理人民"的国际秩序 |
| **施米特的敌友区分** | 激进批判 | 施米特认为政治的本质是敌友区分，永久和平取消了政治本身 |

## 东西方对话

### 永久和平与儒家的"大同"理想

《礼记·礼运》中的"大同"理想——"大道之行也，天下为公"——在目标上与康德的永久和平类似：都追求人类的普遍和平与正义。但路径截然不同：

- **康德**：通过**制度设计**（共和制、联盟、世界公民权）保障和平
- **儒家**：通过**道德教化**（修身齐家治国平天下）实现太平

儒家的和平理想以德性为基础，康德的和平方案以法权为基础——这反映了中西政治哲学传统的根本差异。

### 永久和平与佛教的"不杀生"

佛教的第一戒——不杀生（ahiṃsā）——在个体层面体现了和平的绝对命令。但佛教的和平观是**出世间的**——追求的是从轮回中解脱的终极和平（涅槃），而康德的和平观是**世间的**——追求的是国际政治秩序中的制度性和平。
