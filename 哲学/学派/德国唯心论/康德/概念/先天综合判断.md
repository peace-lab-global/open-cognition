# 先天综合判断 · Synthetische Urteile a priori

## 一句话定义

先天综合判断是康德认识论的核心概念——指**既扩展知识(综合)又具有普遍必然性(先天)**的判断,如数学命题和自然科学基本原理;其可能性条件是批判哲学必须解答的核心问题。

## 核心论证

### 一、两个区分的交叉

康德引入了两组独立的区分,形成一个四格矩阵:

**第一组区分: 分析 vs 综合**

- **分析判断** (analytisches Urteil): 谓词已包含在主词概念中(如"一切物体都有广延")——不扩展知识,但具有必然性
- **综合判断** (synthetisches Urteil): 谓词不包含在主词概念中(如"一切物体都有重量")——扩展知识,但在经验中缺乏必然性

**第二组区分: 先天 vs 后天**

- **先天** (a priori): 独立于一切经验,具有严格普遍性和必然性
- **后天** (a posteriori): 来自经验,缺乏严格普遍性

### 二、四格矩阵

|  | 先天 (a priori) | 后天 (a posteriori) |
|---|---|---|
| **分析** | 逻辑和定义(如"单身汉是未婚男子") | (罕见/无意义) |
| **综合** | **数学、自然科学原理、形而上学?** | 经验观察(如"这朵花是红色的") |

关键格子是**先天综合判断**:如果存在这种判断,它必须既扩展知识(综合),又具有普遍必然性(先天)。

### 三、先天综合判断的实例

康德认为以下领域包含先天综合判断:

**1. 数学**
- "7 + 5 = 12": 分析"7""5"和"+"的概念,无法从中得出"12"——必须借助直观(如手指计数)才能完成综合,但这个判断又是普遍必然的
- "两点之间直线最短": "直"的概念不包含"短"——这是综合的,但又是必然的

**2. 自然科学**
- "一切变化都有原因": "变化"的概念不包含"有原因"——这是综合的,但我们不把它当作经验概括,而是当作必然法则
- "物质在一切变化中保持量不变": 这是综合的,但被认为是普遍有效的

**3. 形而上学(是否可能?)**
- "每一个事件都有原因"
- "灵魂是不灭的实体"
- "上帝存在"
这些**声称**是先天综合判断——但康德的任务是审查它们是否**真的**是合法的先天综合判断

### 四、先天综合判断如何可能?

康德的回答取决于判断所属的认知能力:

**1. 数学先天综合判断如何可能?**
-- 因为数学建立在纯粹直观(时空)之上。算术基于时间的纯粹直观(相继),几何基于空间的纯粹直观(并列)。我们在纯粹直观中"建构"(konstruieren)数学概念,这就是数学既能扩展知识(通过建构)又具有必然性(因为建构在先天直观中进行)的原因。

**2. 自然科学先天综合判断如何可能?**
-- 因为知性范畴(尤其是因果性范畴)是使经验成为可能的先天条件。"一切变化都有原因"不是经验概括,而是我们组织经验的规则——没有因果性范畴,我们根本无法形成关于"变化"的经验。

**3. 形而上学先天综合判断是否可能?**
-- 康德的答案是:**不可能**。当理性试图以范畴(如因果性、实体性)去把握超越经验范围的对象(灵魂、宇宙整体、上帝)时,就陷入了**先验幻相**(transzendentaler Schein)。这些判断看似先天综合,实则缺乏客观有效性,因为它们没有对应的经验直观。

### 五、图式论: 范畴如何应用于直观?

范畴是纯粹的知性概念,直观是感性的杂多——二者如何连接?康德引入**图式**(Schema)作为中介:

- 图式是**想象力的先验产物**,是范畴在时间中的"翻译"
- 例如:因果性范畴的图式是"时间中的规则性相继"
- 实体性范畴的图式是"时间中的持久性"

图式论解决了"纯粹概念如何应用于感性对象"的问题——这是先验分析论中最困难也最具原创性的部分。

## 与相关概念的比较

| 概念 | 关系 | 差异 |
|---|---|---|
| **休谟的观念关系/事实** | 休谟的二分法 | 休谟没有"先天综合"这一格——这正是康德要追问的 |
| **莱布尼茨的先天观念** | 理性论前驱 | 莱布尼茨认为先天观念是"天赋"的;康德认为是认知结构的必然形式 |
| **弗雷格的分析性** | 下游重定义 | 弗雷格以逻辑主义重新定义分析性,将算术判断归为分析的 |
| **奎因的两个教条** | 激进批判 | 奎因否认分析/综合的截然区分,从根本上挑战康德的框架 |

## 东西方对话

### 先天综合判断与量论

陈那(Dignāga)的量论将有效认知(pramāṇa)分为现量(直接知觉)和比量(推理)。问题在于:是否存在一种既非纯知觉又非纯推理、却具有普遍有效性的认知方式?

康德的答案是"先天综合判断"——通过范畴在纯粹直观中的建构。唯识学的回答可能更接近"瑜伽现量"(yogipratyakṣa)——修行者在定中获得的直接但超越日常知觉的认知。但瑜伽现量的普遍有效性依赖于修行境界,而康德的先天综合判断依赖于一切理性认知者共享的认知结构。

### 先天综合判断与庄子

庄子的"齐物论"从根本上质疑普遍必然性的判断:如果"彼亦一是非,此亦一是非",那么声称某个判断具有"普遍必然性"本身就是一种独断。庄子可能会说:先天综合判断之所以"看起来"普遍必然,只是因为我们都共享同一个"成心"(固化的认知框架)——打破这个框架("莫若以明"),这些判断的必然性就消失了。
