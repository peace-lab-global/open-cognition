# 吉尔·德勒兹 · Gilles Deleuze

吉尔·德勒兹（Gilles Deleuze，1925–1995）是法国哲学家，后结构主义关键人物。他与加塔利提出“块茎”“生成”“欲望机器”等概念，重新思考差异、情感与权力。

## 核心命题 / Core Theses

### 1. 差异先于同一

传统哲学将差异视为同一的附属；德勒兹则认为，存在就是差异的生成，同一性是事后建构。

### 2. 情感作为身体之力

情动（affect）不是主观的情绪，而是身体在遭遇他者时力量增强或减弱的动态过程，先于被命名的情感。

### 3. 块茎思维

知识不应被还原为树状的等级起源，而应像块茎一样横向连接、多元生成、无中心。

## 关键著作 / Key Works

- *Difference and Repetition* (1968)
- *A Thousand Plateaus* (1980, with Félix Guattari)
- *What is Philosophy?* (1991, with Félix Guattari)

## 跨学科关联 / Interdisciplinary Links

- 与 [巴鲁赫·斯宾诺莎](../理性论/巴鲁赫.md) 在情动理论上遥相呼应，德勒兹将斯宾诺莎的“情感”发展为后结构主义的“情动”。
- 与 [雅克·德里雅](雅克.md) 共同属于法国后结构主义潮流，但德勒兹更强调力量、生成与身体。
