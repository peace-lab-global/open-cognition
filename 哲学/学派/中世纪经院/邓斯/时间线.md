# 司各脱生平年表 · Duns Scotus Biography Timeline

## 早年与学术训练（1266–1291）

| 年份 | 事件 |
|------|------|
| c. 1266 | 生于苏格兰邓斯镇（Duns），靠近英格兰边境 |
| c. 1278 | 进入方济各会（Order of Friars Minor），在苏格兰达姆弗里斯（Dumfries）修道院开始基础学习 |
| c. 1280 | 前往牛津大学攻读文科（arts），接受亚里士多德逻辑学与自然哲学的训练 |
| 1288 | 获文学士学位（BA） |
| 1290 | 开始在牛津大学攻读神学 |
| 1291 | 受神职（ordination），获文学硕士学位 |

## 牛津教学时期（1291–1301）

| 年份 | 事件 |
|------|------|
| c. 1291 | 开始在牛津讲授《教父箴言集》（Sentences of Peter Lombard） |
| c. 1295–1297 | 撰写《亚里士多德形而上学问答》（*Quaestiones super Metaphysicam*），发展单义性学说 |
| c. 1297–1299 | 完成《牛津评注》（*Ordinatio*）核心部分：单义性、此性、形式区分 |
| 1300 | 完成神学学士学位（BTh）的讲座要求 |
| 1301 | 在牛津大学获神学硕士学位（Master of Theology） |

## 巴黎教学时期（1302–1307）

| 年份 | 事件 |
|------|------|
| 1302 | 赴巴黎大学讲授《教父箴言集》，开始形成《巴黎报告》（*Reportatio Parisiensis*） |
| 1303 | 因拒绝签署法王腓力四世（Philip the Fair）反对教皇博尼法斯八世的文件，被驱逐出法国 |
| 1303–1304 | 短暂流亡，可能返回英国 |
| 1304 | 法王态度缓和后返回巴黎，继续教学 |
| 1305 | 获神学博士学位（Doctor of Theology），巴黎大学正式授予"博士"称号 |
| 1305–1306 | 在巴黎完成《辩论问题集》（*Quodlibetal Questions*）最后阶段 |

## 科隆与早逝（1307–1308）

| 年份 | 事件 |
|------|------|
| 1307 | 转至科隆方济各会学校任教（可能由方济各会总会派遣） |
| 1308.11.8 | 在科隆骤然去世，年仅42岁（死因不明，有中风、中毒或疾病诸说） |
| 1308 | 安葬于科隆方济各会教堂（Minoritenkirche） |

## 身后影响

| 年份 | 事件 |
|------|------|
| 1319 | 方济各会正式尊司各脱为学派领袖，"司各脱主义"（Scotism）成为方济各会官方哲学 |
| c. 14世纪 | 司各脱学派与托马斯主义、奥卡姆主义并列为中世纪三大学派 |
| 1474 | 教皇西斯都四世封司各脱为真福品（beatification process initiated） |
| 1639 | Wadding版《司各脱全集》（*Opera Omnia*）在里昂出版 |
| 1854 | 教皇庇护九世宣布圣母无原罪始胎教义——该教义最初由司各脱系统论证 |
| 1993 | 教皇约翰·保罗二世正式确认司各脱的真福品（beatification） |

## 思想分期总结

| 阶段 | 年代 | 关键词 |
|------|------|--------|
| 学术训练 | 1266–1291 | 牛津、方济各会、亚里士多德 |
| 牛津教学 | 1291–1301 | 《牛津评注》、单义性、此性 |
| 巴黎教学 | 1302–1307 | 《巴黎报告》、神学博士 |
| 科隆与早逝 | 1307–1308 | 最后教学、骤然离世 |
