# 邓斯·司各脱 · John Duns Scotus

## 基本信息

- **生卒年**：约 1265/66–1308
- **国籍/文化背景**：苏格兰，方济各会士
- **学术身份**：中世纪经院哲学"精微博士"（Doctor Subtilis），牛津与巴黎大学执教
- **学派归属**：经院哲学·方济各会传统·司各脱主义

## 核心命题

1. **直觉认知 vs 抽象认知（cognitio intuitiva / abstractiva）**：奠定西方"直接认知"的独立概念模态——直觉认知把握对象**作为现存的、在场的东西**，抽象认知抽象出可独立于对象存在的本质。这是西方思想首次把"直接性"作为认知的**独立模态**加以理论化。
2. **此性（haecceitas / "thisness"）**：个别性是存在的终极样式，每个个别物拥有不可还原的"此性"。
3. **存在的单义性（univocatio entis）**：被造之物与上帝的"存在"概念是单义的——这是中世纪形上学最技术化的贡献之一。
4. **意志优于理智**（voluntarism）：意志不是被动接受理智判断的"欲望"，而是自由的自发能力。

## 思想脉络

### 早期
- 在牛津与剑桥接受方济各会教育，受哈勒斯的亚历山大与根特的亨利影响。

### 中期（牛津）
- 撰写《牛津评注》（*Ordinatio*），建立直觉/抽象认知区分、此性学说、存在单义性。

### 晚期（巴黎）
- 转赴巴黎大学，与多明我会学派激烈论战；其"意志主义"与方济各会立场强化。
- 病逝于科隆，年仅约 43 岁。

## 关键著作

- 《牛津评注》（*Ordinatio*，原名 *Opus Oxoniense*）——其系统神学-哲学代表作
- 《巴黎评注》（*Reportatio parisiensis*）
- 各类《问题集》（*Quaestiones*，论灵魂、论第一原理）

## 重要概念

| 概念 | 说明 |
|---|---|
| 直觉认知（cognitio intuitiva） | 对对象**作为现存/在场**的直接认知，把握"它在" |
| 抽象认知（cognitio abstractiva） | 抽象出无需对象在场亦可设想的内容 |
| 此性（haecceitas） | 个别物的终极不可还原性 |
| 存在单义性（univocatio entis） | "存在"在被造与神圣间是同一概念 |
| 意志主义 | 意志是自由自发的，优于理智 |

## 思想坐标

- **承接**：哈勒斯的亚历山大、根特的亨利的方济各会传统；亚里士多德形上学。
- **批判**：阿奎那的理智主义（理智高于意志、共相在殊相中的温和实在论）——司各脱以意志主义与此性学说纠正之。
- **影响**：奥康的威廉（继承并激进直觉认知学说，用奥康剃刀去除"可对不存在对象直觉认知"）；近代"清楚明白观念"与现象学"直观"的共同远祖。

## 直接认知：司各脱的核心贡献

司各脱是西方思想中**第一个明确把"直接认知"作为独立逻辑模态**加以理论化的哲学家，使其成为本研究（[02-直接认知](../../../研究/02-直接认知/README.md)）的关键人物：

**直觉认知不是感知的模糊整体，而是有清晰逻辑位置的概念**——它把握的是"对象的存在性 / 在场性"，而非其可抽象的本质。这一区分经奥康传到近代，构成笛卡尔"清楚明白观念"和胡塞尔"直观"（Anschauung）的共同概念骨架。

**争议点**：司各脱承认上帝可赋予对一个**不存在对象**的直觉认知（这是奥康后来用奥康剃刀砍掉的部分）。这一让步削弱了"直接认知 ⟹ 对象实在"的推论，却更精确地划定了直接性的逻辑结构。

## 当代应用

- **认识论**：直觉/抽象认知之分预示了当代"直接指称"（克里普克）与"摹状指称"（罗素）之争。
- **形上学**：此性学说被当代个体化（individuation）研究重新激活。
- **意志主义**：影响叔本华、尼采的意志哲学，以及当代行动哲学。

## 常见误读

- **误读 1**："经院哲学 = 千篇一律的实在论"——司各脱与阿奎那在意志主义、个别性、直觉认知上立场尖锐对立。
- **误读 2**："直觉认知 = 神秘直觉"——它是一个**逻辑上精密的认知模态**概念，与神秘体验无关。
