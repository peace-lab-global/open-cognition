# 詹姆斯·洛夫洛克 · James Lovelock

詹姆斯·洛夫洛克（James Lovelock，1919–2022）是英国独立科学家与发明家，以“盖亚假说”闻名。他认为地球是一个自我调节的生命系统，生物圈、大气、海洋与地质过程相互作用，维持适合生命的环境。

## 核心命题 / Core Theses

### 1. 盖亚假说

地球可被视为一个自我调节的复杂系统：生物体与无机环境共同演化，维持大气成分、温度与化学状态的相对稳定。

### 2. 行星层面的整体观

生命不能仅从局部有机体理解，而应看作行星尺度的过程。地球本身展现出类似生命的特征。

### 3. 气候危机的反馈

人类活动打破盖亚系统的自我调节，可能触发不可逆的临界点。洛夫洛克晚年对气候未来持悲观但行动主义的立场。
