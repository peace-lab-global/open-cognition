# 阿恩·内斯 · Arne Næss

阿恩·内斯（Arne Næss，1912–2009）是挪威哲学家，深层生态学（deep ecology）的创始人。他区分“浅层生态学”（资源管理）与“深层生态学”（生态中心主义与自我实现）。

## 核心命题 / Core Theses

### 1. 深层与浅层生态学

浅层生态学只关心污染与资源对人类的影响；深层生态学追问人类中心主义本身，主张生态系统的内在价值。

### 2. 生态自我

真正的自我实现不是个人成功，而是“大我”的扩展：将自我认同从个体扩展到社区、物种与生态系统。

### 3. 多样性与共生

自然的丰富性与复杂性本身具有价值。人类应减少人口与消费，给其他生命留下空间。

## 关键著作 / Key Works

- *Ecology, Community and Lifestyle* (1989)

## 跨学科关联 / Interdisciplinary Links

- 与 [奥尔多·利奥波德](利奥波德.md) 的土地伦理一脉相承，但内斯将共同体边界扩展到整个生态系统。
- 与 [默里·布克钦](布克钦.md) 的社会生态学存在论争：内斯强调个体意识转变，布克钦强调社会结构变革。
