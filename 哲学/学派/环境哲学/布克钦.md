# 默里·布克钦 · Murray Bookchin

默里·布克钦（Murray Bookchin，1921–2006）是美国社会理论家与无政府主义者，社会生态学的创始人。他认为生态危机根源于社会等级制与统治关系，解决生态问题必须同时改造社会结构。

## 核心命题 / Core Theses

### 1. 社会生态学

生态问题不是抽象的人口或技术问题，而是社会等级制、阶级统治与商品化的结果。人类对自然的支配复制了人对人的支配。

### 2. 自由市镇主义

直接民主的地方自治共同体是替代国家与资本主义的政治形式，也是生态社会的制度基础。

### 3. 差异中的统一

自然不是铁板一块的“整体”，而是差异与自发性的辩证统一。生态思维应反对把自然浪漫化或神秘化。

## 关键著作 / Key Works

- *Post-Scarcity Anarchism* (1971)
- *The Ecology of Freedom* (1982)

## 跨学科关联 / Interdisciplinary Links

- 与 [卡尔·马克思](../../../社会学/学派/古典社会学/马克思/README.md) 共享对社会结构变革的关注，但布克钦更强调去中心化的直接民主。
- 与 [阿恩·内斯](内斯.md) 的深层生态学形成著名论争：布克钦批评其神秘主义倾向。
