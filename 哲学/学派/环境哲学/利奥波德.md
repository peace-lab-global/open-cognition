# 奥尔多·利奥波德 · Aldo Leopold

奥尔多·利奥波德（Aldo Leopold，1887–1948）是美国生态学家、林务员与作家。他在《沙乡年鉴》中提出“土地伦理”，主张将道德共同体的边界从人类扩展到土壤、水、植物与动物。

## 核心命题 / Core Theses

### 1. 土地伦理

伦理演化的历史是共同体边界的不断扩展：从自我到家庭、部落、国家，最终应包括整个生物共同体。

### 2. 生物共同体的整体性

土地不是仅供开发的资源，而是包括土壤、水、植物与动物的复杂生命系统。健康是土地完整、稳定与美丽的同义词。

### 3. 生态良知

保护不能仅靠法律与经济激励，更需要一种深植于文化的生态良知，让人类视自己为土地的普通成员与公民。
