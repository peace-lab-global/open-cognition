# 叔本华速读 · 一篇文章读透叔本华

> 本文是叔本华哲学体系的全景认知地图。一篇读完，建立完整框架。

---

## 一句话定位

**叔本华是第一个说"世界的本质不是理性，而是盲目欲求"的西方哲学家。他把这个欲求叫做"意志"，并由此推导出一整套关于痛苦、美学、伦理和解脱的思想体系——同时成为东西方哲学对话的开创者。**

---

## 叔本华是谁？——理解哲学的人格入口

| 事实 | 为什么重要 |
|------|-----------|
| 1788年生于但泽（今波兰格但斯克），富裕商人家庭 | 一生经济独立，不需要迎合学术体制 |
| 父亲16岁时疑似自杀身亡 | "人生本质是痛苦"不是书斋推论——是切肤之痛 |
| 母亲约翰娜是当时的畅销小说家，与歌德交好 | 少年时代就接触歌德的色彩理论，获得感知理论早期启发 |
| 师从费希特，但以批判康德为起点 | 不走费希特的"绝对自我"路线，也不走黑格尔的"绝对精神"路线 |
| 1818年完成《作为意志和表象的世界》，30岁 | 首版500册大部分被当废纸处理——被忽视长达20余年 |
| 柏林大学讲师期间故意把课排在黑格尔同一时间 | 只有寥寥数人来听——与黑格尔理性主义的根本对立贯穿一生 |
| 1813年左右接触拉丁文《奥义书》，称之为"我一生的慰藉" | 第一位认真对待印度哲学的西方主要思想家 |
| 1851年《附录与补遗》出版，终于获得迟到的声誉 | 63岁才成名——"我等待了三十年，人们终于开始听我说话" |
| 晚年在法兰克福与卷毛狗 Atma（世界灵魂）为伴 | 孤独、规律、对动物的深情——哲学的活体现 |
| 1860年因肺炎去世 | 遗嘱将全部财产留给动物保护事业 |

**一句话概括这个人**：一个被时代忽视了三十年的天才，用一辈子证明"世界不是理性的——而是欲求的"。

## 核心逻辑链：六个命题如何环环相扣

```
① 世界是我的表象（认识论起点）
      ↓
② 表象背后的本体是意志（形而上学突破）
      ↓
③ 意志的本质是无尽欲求→必然导致痛苦（悲观主义）
      ↓
④ 审美沉思提供暂时解脱（美学）
      ↓
⑤ 同情穿透个体化的幻觉（伦理学）
      ↓
⑥ 意志的自我否定提供终极解脱（解脱论）
```

**这六步不是并列的，而是层层递进的。** 每一步都从上一个命题必然推出。

## 命题一：世界是我的表象

> "世界是我的表象"——这是第一个也是最根本的真理。
> "The world is my representation" — this is the first and most fundamental truth.

**逻辑**：我们所经验的一切——颜色、声音、物体、他人——都不是"自在之物"，而是经过我们的认知形式（时间、空间、因果性）加工后的**表象**（Vorstellung）。

**类比**：你戴着蓝色眼镜看世界，世界呈现为蓝色。蓝色不是世界本身的属性，而是眼镜的属性。时空因果性就是人类认知的"眼镜"。

**叔本华对康德的关键改造**：

康德说认知形式有12个范畴，叔本华说只需要**一个根本法则的四种形式**——充足理由律的四重根：

| 形式 | 领域 | 例子 |
|------|------|------|
| 生成律（因果性） | 物理世界 | 球撞击→球运动 |
| 认识律（逻辑） | 概念推理 | 前提→结论 |
| 存在律（数学） | 时空关系 | 三角形的内角和=180° |
| 行动律（动机） | 人类行为 | 动机→行为 |

**核心洞见**：充足理由律只管辖表象世界。表象之外，还有更深的东西。

## 命题二：表象背后的本体是意志

> "意志是一切事物的内在本质。"
> "Will is the inner essence of everything."

**逻辑**：康德说"物自体"不可知。叔本华说：有一条从**内部**通达物自体的路径。

**关键论证——身体的双重性**：

当我举起手臂，这同时是两件事：
1. **物理事件**（表象层面）：肌肉收缩、神经传导、服从因果律
2. **意志行为**（本体层面）：我"意愿"举起手臂

这两个描述**不是因果关系**（不是"意志导致运动"），而是**同一事件的内、外两面**。

**大胆的类比推广**：如果我的身体的内在本质是意志，那么万物的内在本质也是意志——引力、磁力、生长本能，都是意志在不同等级的**客体化**（Objektivation）。

**意志的三个根本特征**：

| 特征 | 含义 | 反对谁 |
|------|------|--------|
| 非理性 | 先于认识，不服从理性法则 | 黑格尔（理性=世界本质） |
| 无目的 | 欲求，但不"为了"什么而欲求 | 亚里士多德（目的因）/ 基督教（神意） |
| 统一不可分 | 意志是一，个体多样性是时空造成的幻觉 | 常识（"你"和"我"是两个独立存在） |

**核心格言**：*"意志是主人，理性是仆人。"*

## 这本书怎么读？——《作为意志和表象的世界》四卷地图

叔本华的核心著作 *Die Welt als Wille und Vorstellung*（1818/1844）按四个书卷精心组织，每卷各有分工：

```
┌─────────────────────────────────────────────────────┐
│ 第一卷：世界作为表象（认识论）                          │
│   → 充足理由律四重根、时空因果性、主体-客体结构            │
│   → 对应命题①                                       │
├─────────────────────────────────────────────────────┤
│ 第二卷：世界作为意志（形而上学）                         │
│   → 身体双重性、意志的客体化等级、非理性/无目的/统一       │
│   → 对应命题②                                       │
├─────────────────────────────────────────────────────┤
│ 第三卷：世界作为表象——再论（美学）                       │
│   → 理念、无意志的认识主体、艺术等级、音乐的崇高地位       │
│   → 对应命题④                                       │
├─────────────────────────────────────────────────────┤
│ 第四卷：世界作为意志——再论（伦理学）                     │
│   → 痛苦与无聊的钟摆、同情伦理、禁欲主义、意志的否定       │
│   → 对应命题③⑤⑥                                    │
└─────────────────────────────────────────────────────┘
```

**阅读建议**：第一卷最难（认识论），第三卷最美（美学），第四卷最重要（伦理+解脱）。如果时间有限，可以先读第四卷再回头。

## 命题三：意志=无尽欲求=必然痛苦

> "人生如钟摆，在痛苦与无聊之间来回摆荡。"
> "Life swings like a pendulum to and fro between pain and boredom."

**逻辑**：意志的本质是无尽的欲求→欲求受阻=痛苦→欲求满足=无聊→新欲求产生→痛苦……循环永不终结。

**痛苦与快乐的不对称分析**（这是叔本华悲观主义最精妙的论证）：

| | 痛苦 | 快乐 |
|---|---|---|
| 性质 | **积极的**（positive）——直接、实在的感受 | **消极的**（negative）——仅仅是痛苦的暂时消除 |
| 类比 | 生病时你**感觉到**某个器官 | 健康时你**感觉不到**身体 |
| 结论 | 痛苦是"有"——是实在的存在 | 快乐是"无"——是匮乏的缺席 |

**痛苦的三个层面**：

1. **生物学**：一切有机体处于永恒匮乏——需要不断进食、避危、繁衍
2. **心理学**：时间意识使我们不仅感受当下痛苦，还回忆过去、预见未来
3. **社会学**：人与人之间的竞争、嫉妒、虚荣使社会生活成为痛苦的另一个来源

**性欲作为意志的核心显现**：

> 性欲是"所有人类努力和追求的目的"。浪漫爱情的表面（诗意、灵魂契合）掩盖了意志的真实目的（生殖、种族延续）。

**爱情的四重幻觉**：

| 你以为的 | 意志真正在做的 |
|---------|-------------|
| 追求个人幸福 | 追求种族的最优后代 |
| 灵魂契合/精神共鸣 | 基因互补性的直觉判断 |
| 自由选择伴侣 | 被种族意志无意识地操纵 |
| 爱情是目的 | 爱情只是手段——一旦生殖完成，幻觉即消散 |

**伴侣选择的"逻辑"**：男性偏好年轻（生育力）、健康；女性偏好资源、地位（养育能力）。双方的互补性（高个子找矮个子、胖找瘦）是为了后代的最优体质。这一切都在意识层面之下运行。

**性欲与痛苦的深层关联**：性欲是意志最强烈的表达→满足极其短暂→新的欲求立即产生→因此性欲是"痛苦永不终结"的最有力证据。禁欲主义的核心就是贞洁——切断意志的最强通道。

**对后世的影响**：
- **弗洛伊德**：叔本华的"意志=无意识的性驱力"→弗洛伊德的"力比多"（libido）；意志压抑→弗洛伊德的"压抑机制"；性欲作为核心驱力→精神分析的核心假设
- **进化心理学**：巴斯（Buss）的配偶选择理论、特里弗斯（Trivers）的亲代投资理论——几乎是叔本华伴侣选择逻辑的实证版

**对乐观主义的批判**：

> 让一个人在充分了解了世界的全部痛苦之后，决定是否愿意再次经历同样的一生。任何理性的存在者都会选择"不再存在"。

**悲观主义的伦理功能**：悲观不是绝望——认识痛苦是同情的**前提**，也是意志否定的**动机**。

## 命题四：审美沉思——暂时的解脱

> "在审美沉思中，我们暂时从意志的奴役中获得了自由。"
> "In aesthetic contemplation we are temporarily freed from the servitude of the will."

**逻辑**：如果意志=痛苦，那么暂时悬搁意志=暂时脱离痛苦。审美沉思就是这种悬搁。

**审美状态中发生了什么**：

| 维度 | 日常状态 | 审美状态 |
|------|---------|---------|
| 主体 | 有意志的个体（"我要什么"） | **纯粹的、无意志的认识主体** |
| 客体 | 个别事物（受因果律制约） | **理念**（Idee）——意志的直接客体化 |
| 认知方式 | "这有什么用？"（功利性） | **"这就是它本身"**（纯粹直观） |

**理念**（Idee）：柏拉图式的永恒原型，是意志在各等级上的充分显现，尚未被时空分化为无数个别事物。审美沉思的对象不是"这朵花"，而是"花"的理念。

**艺术的等级体系**：

```
建筑（最低——展现引力/刚性的冲突）
  → 雕塑（人体形态）
    → 绘画（自然与人类生活）
      → 诗歌（人类行动与性格）
        → 悲剧（生命意志的内在冲突）
          → 音乐（最高——意志本身的直接写照）
```

**音乐的崇高地位**：其他一切艺术都是理念的摹本，唯独**音乐是意志本身的直接写照**。低音部对应无生命的自然，高音部对应人类意识的反思活动。

**美与崇高**：

- **美**：对象和谐地邀请你进入纯粹直观→意志**自然地**退场→宁静愉悦
- **崇高**：对象的巨大/危险**威胁**着意志→你有意识地**超越**恐惧→仍然保持纯粹认识

**天才**（Genie）：能暂时悬搁意志、成为纯粹认识主体、直观理念并将其传达的人。天才不是"更聪明"，而是"看得更纯粹"。

**审美沉思的限度**：它是**暂时的**。一旦审美状态消退，主体重新被意志捕获。真正的解脱需要命题六。

## 命题五：同情——穿透个体化的幻觉

> "一切真正的善行，其源泉不是理性认识，而是直观的同情认识。"
> "The source of all genuine virtue is not rational knowledge but intuitive knowledge through compassion."

**逻辑**：如果一切个体在本体层面都是同一个意志，那么他者的痛苦就是我的痛苦——同情（Mitleid，字面义"共同受苦"）就是对这一本体论真理的**直观认识**。

**对康德伦理学的三重批判**：

1. 绝对命令预设理性自主→但理性只是意志的工具
2. 绝对命令是空洞的形式主义→不能指导具体道德困境
3. "义务"概念是隐蔽的神学遗产→没有上帝却想保留上帝的"律令"

**同情的三动机分类**：

| 动机 | 德语 | 道德评价 |
|------|------|---------|
| 利己主义 | Egoismus | 道德中性（绝大多数行为的动机） |
| 恶意 | Bosheit | 道德邪恶 |
| **同情** | **Mitleid** | **道德善良** |

**两种基本美德从同情推导**：

- **正义**（消极美德）：同情阻止我伤害他者→"不伤害任何人"（neminem laede）
- **仁爱**（积极美德）：同情驱使我减轻他者痛苦→"尽力帮助一切人"（omnes, quantum potes, juva）

**个体化原理的穿透**：

> "汝即彼"（tat tvam asi）——每一个有意识的存在者都可以对另一个存在者说出这句话。在最深层面，"你"和"我"是同一个实在的不同显现。

**同情天然扩展到动物**：动物也拥有意志，其痛苦在本体论上与人类痛苦同一。对动物残忍=道德低劣。

## 命题六：意志的否定——终极解脱

> "意志自行转向否定的那一刻，一切痛苦都结束了。"
> "The moment the Will turns of itself to denial, all suffering comes to an end."

**逻辑**：审美沉思=暂时解脱，意志否定=永久解脱。通过深刻的苦难体验或智识洞察，意志"看到"自身的本质而**自行熄灭**。

**禁欲主义的三个阶段**：

1. **自愿贫穷**：放弃财产——削弱意志的外在支撑
2. **自我克制与斋戒**：直接抑制身体中的生命意志——战略性地"饿死"意志
3. **完全的内在弃绝**（Gelassenheit）：对一切事物完全无动于衷的内在宁静

**圣人与自杀者的关键区分**（最常被误解的部分）：

```
圣人：  痛苦 → 洞察 → 意志自行熄灭 → 宁静地活着
自杀者：痛苦 → 愤怒 → 以暴力消灭身体 → 意志仍然完好
```

- **圣人**：不再**欲求**（但身体仍活着）——是从意志中**解脱**
- **自杀者**：强烈欲求生命的美好但无法忍受当前痛苦——是意志的**肯定**而非否定

**恩赐（Gnade）问题**：意志否定不是通过理性努力可以获得的——它更像是一种"恩赐"，意志本身的自发性转向。禁欲实践可以**准备**条件，但不能**导致**意志否定。

**意志否定后的状态**：不是"无"，也不是"有"——超越了存在/虚无的二元对立。对于意志仍然活跃的人来说，它看起来像"无"；对于已经否定意志的人，"这个如此充满的世界……就是——无"。这一描述与佛教中观学派（Mādhyamaka）对涅槃的描述深度呼应：涅槃不是某个"地方"或"状态"，而是对一切执着的彻底超越。

**意志否定的自我指涉难题**：如果一切都是意志，谁来否定意志？叔本华的回答：否定意志的是**意志本身**——意志通过人类意识的自我认识能力，可能自发地否定自身。这是"意志否定意志"——一种自我指涉的辩证超越。

## 吠檀多的关键分歧：意志 vs 梵——解脱是"消灭"还是"实现"？

叔本华与印度吠檀多不二论（Advaita Vedānta，尤其是商羯罗 Śaṅkara）表面高度相似，但在本体论上根本对立：

| 维度 | 叔本华 | 吠檀多 |
|------|--------|--------|
| 本体 | **意志**（盲目欲求，痛苦的根源） | **梵**（Brahman，纯粹意识+至福 ānanda） |
| 世界 | 意志的必然客体化 | 梵的幻力（māyā）显现 |
| 解脱目标 | **消灭**欲求——从痛苦中释放（消极的） | **实现**梵我一如——证悟至福（积极的） |
| 解脱路径 | 恩赐（不可控制） | 知识（jñāna）——"汝即梵"（可通过修行获得） |
| 解脱后状态 | 空/无（Nichts） | 存在-意识-至福（sat-cit-ānanda） |

**为什么这个差异重要**：如果本体是"痛苦"（意志），解脱只能是"消灭"；如果本体是"至福"（梵），解脱就是"重新发现"。同一个否定意志的动作，在叔本华体系中是**终结**，在吠檀多体系中是**回归**。

## 死亡：意志不灭

> 死亡不伤害我们的真实本质——因为我们的真实本质是意志，而意志超越时间。

**大海与波浪的类比**：波浪不断生灭，但大海始终如一。个体是意志的"波浪"，死亡是波浪的消散——但大海（意志）不受影响。

- **本体不朽**：意志作为物自体，超越时间，永不消灭
- **物种不朽**：通过生殖将生命传递给后代
- **个体灵魂不朽**：叔本华拒绝——个体性在死亡时终结

**讽刺**：否定意志的人比肯定意志的人更不怕死——他已经在精神上"死于"意志。

## 性格与自由意志

> "一个人可以做他所意愿的，但不能意愿他所意愿的。"
> "A person can do what he wills, but he cannot will what he wills."

**性格三重结构**：

| 类型 | 含义 | 可变性 |
|------|------|--------|
| 悟知性格 | 意志在个体中的本体层面规定 | 不可改变 |
| 验知性格 | 悟知性格在经验中的显现 | 不可改变 |
| 获得的性格 | 认识自己的性格并据此安排生活 | 可以获得 |

**自由意志的解答**：

| 层面 | 自由？ | 原因 |
|------|--------|------|
| "能做所愿"（行为层面） | 是 | 身体服从意志的命令 |
| "能愿所愿"（意志层面） | 否 | 意志由性格和动机决定 |

**行为公式**：`行为 = f（性格 × 动机）`——同一个人面对不同动机做出不同行为，但性格在所有行为中保持不变。

## 人生智慧

> "健康的乞丐比有病的国王更幸福。"

决定幸福的三个因素（按重要性排序）：

1. **你是什么**（人格：健康、气质、智识能力）→ 最重要
2. **你有什么**（财产：够用即可，超出后效用递减）→ 中等
3. **你在他人眼中是什么**（名誉：最不可控也最不可靠）→ 最不重要

**核心教导**：降低期望比增加享受更有效。幸福不是"得到更多"，而是"期望更少"。

## 三大批评：后人如何挑战叔本华

### 尼采：从信徒到叛徒

尼采年轻时是叔本华的虔诚信徒，后来彻底反转：

| 维度 | 叔本华 | 尼采 |
|------|--------|------|
| 意志 | 生存意志（Wille zum Leben） | 权力意志（Wille zur Macht） |
| 对痛苦的态度 | 否定→逃避 | **肯定→拥抱** |
| 悲观主义 | 真理 | 颓废（décadence）的症状 |
| 禁欲理想 | 解脱之路 | 生命的自我否定 |
| 解脱 | 意志的自我熄灭 | 永恒轮回中的生命肯定 |

尼采的诊断：叔本华不是"看透了世界"，而是**生命力衰退**的表现。

### 女性主义：体系的性别盲点

叔本华在《论女人》中将女性定义为"第二性"——理智、审美、道德皆低于男性。女性主义批评指出：

- 这些"经验"论断在科学上站不住脚
- 天才概念被隐性编码为男性
- 同情伦理理论上应性别中立，但实践中排除了女性

但也有可提取的资源：同情伦理与关怀伦理学有结构性亲和力；对理性至上主义的批判可被女性主义利用。

### 后殖民：东方主义挪用

叔本华是第一个认真对待印度哲学的西方思想家——但他的挪用方式有问题：

- **确认性引用**：不是在"学习"印度哲学，而是在用它确认自己的结论
- **选择性阅读**：只取吻合的概念（摩耶、涅槃），忽略不符合的（行动哲学、社会伦理）
- **去语境化**：将印度概念从原有语境中抽离，重新整合进自己的体系

## 四个常见误读——读叔本华前必须知道的事

### 误读一："叔本华主张自杀"

**真相**：恰恰相反。自杀是意志的**肯定**（因无法忍受痛苦而暴力消灭身体），否定意志才是解脱（通过洞察使意志自行熄灭）。圣人"不再欲求"但宁静地活着；自杀者"强烈欲求"但以暴力对抗生命。

### 误读二："叔本华只是悲观的，没有积极贡献"

**真相**：叔本华提供了独立于康德和功利主义的第三条伦理路径（同情伦理），开创了比较哲学（东西方对话），预示了整个深度心理学传统，对美学（瓦格纳、普鲁斯特）、动物伦理（辛格）、生态伦理都有原创贡献。

### 误读三："叔本华与尼采基本一致"

**真相**：尼采早期是信徒，成熟期彻底反转——用"权力意志"（Wille zur Macht）取代"生存意志"，用"生命肯定"取代"生命否定"，将叔本华的"否定"诊断为颓废（décadence）。两者的关系是**最根本的对立**。

### 误读四："叔本华的'意志'是心理学概念"

**真相**：意志是**形而上学的本体论概念**，不是心理状态。它先于一切意识，是引力、磁力、化学力等一切自然力的内在本质。将"意志"还原为"人类欲望"，恰恰违背了叔本华的核心主张。

## 东西方对话：叔本华 × 佛教 × 唯识学

### 四圣谛与叔本华的结构平行

| 佛教四圣谛 | 叔本华 |

| 道谛：八正道 | 禁欲主义+审美沉思 |

| 唯识学 | 叔本华 |
|--------|--------|
| 一切法唯识（vijñapti-mātra） | 世界是我的表象 |
| 阿赖耶识（ālaya-vijñāna） | 意志（作为本体） |
| 末那识的我执 | 个体化原理 |
| 转依（āśraya-parāvṛtti） | 意志的自我否定 |

### 三个关键差异

| 维度 | 叔本华 | 佛教/唯识 |
|------|--------|----------|
| 本体 | 意志（盲目力量） | 空性（śūnyatā）/ 识 |
| 解脱路径 | 恩赐（不可控制） | 系统修行（八正道/五道） |
| 解脱后状态 | 空/无（Nichts） | 涅槃/大圆镜智（积极的） |

### 对MOCICI冥想的四个启示

1. **审美悬搁→冥想准备**：训练从意志驱动的注意力中退出
2. **身体直观→直接认知**：从内部直接体验身体，不经概念中介
3. **同情认知→穿透我执**：训练穿透个体化的直观能力
4. **恩赐问题→训练框架**：将不可控的"恩赐"转化为概率性的系统训练

## 为什么今天还要读他？——当代影响谱系

叔本华的哲学像地下水一样渗透了整个现代文化。你可能不知道他的名字，但你已经在用他的思想：

| 领域 | 受影响的代表 | 核心继承 |
|------|------------|---------|
| **精神分析** | 弗洛伊德 | 无意识驱力→本我（Id）；意志压抑→压抑机制；性欲作为核心驱力→力比多 |
| **神经科学** | 达马西奥（Damasio） | 躯体标记理论：理性决策依赖情绪/身体过程→"理性是意志的仆人" |
| **存在主义** | 尼采→海德格尔→萨特 | 人生荒谬性/焦虑的问题意识，虽然方向相反（肯定vs否定） |
| **生命哲学** | 柏格森（Bergson） | "生命冲动"（élan vital）继承意志概念，但赋予积极创造性含义 |
| **动物伦理** | 彼得·辛格（Singer） | 同情伦理扩展到动物——动物意志与人类意志本体同一 |
| **生态伦理** | 深层生态学 | 万物意志同一性→对自然的破坏就是对自身意志的伤害 |
| **文学** | 瓦格纳、普鲁斯特、托马斯·曼、托尔斯泰、哈代、博尔赫斯 | 欲望的不可满足性、审美超越、时间中的虚无 |
| **视觉艺术** | 勃克林《死之岛》、克林格、康定斯基 | 象征主义（呈现理念而非模仿自然）、表现主义（内在意志的外化） |
| **电影** | 塔可夫斯基、伯格曼、伍迪·艾伦 | 崇高美学、痛苦-无聊钟摆、面对死亡的恐惧与超越 |
| **正念/心理治疗** | MBSR（正念减压） | 审美沉思与冥想的结构平行——从意志驱使中暂时解脱 |

### 与谢林的未完成对话

谢林（Schelling）是叔本华的同时代人，两人共享根源（康德遗产、反黑格尔、自然哲学兴趣），但在核心问题上根本对立：

| 维度 | 谢林 | 叔本华 |
|------|------|--------|
| 本体 | **绝对**（das Absolute）——超越主体/客体的无差别同一 | **意志**（Wille）——前理性的盲目欲求 |
| 自然 | 自然是绝对的精神化表达 | 自然是意志的客体化 |
| 方法论 | **智识直观**（超越感性和理性的直接认识） | **身体经验**（从内部直接体验意志行为） |

叔本华嘲笑谢林的"智识直观"为"哲学魔术"。但20世纪哲学史重新评价认为，两人比表面上更相似——都试图超越康德二元论，都强调非理性力量的核心地位。

### 视觉艺术与电影中的叔本华

**造型艺术**：叔本华的艺术理论——艺术不是模仿自然，而是呈现理念——直接催生了象征主义运动。勃克林的《死之岛》呈现"死亡"的理念而非具体场景；康定斯基的抽象画绕过个别事物直接呈现普遍性——这正是叔本华的理念概念。

**电影**：塔可夫斯基的《潜行者》中"区"=意志的客体化领域（充满危险但许诺满足）；伯格曼的《第七封印》=死亡哲学的视觉化（骑士与死神下棋）；伍迪·艾伦的《安妮·霍尔》=欲望不可满足性的喜剧版。实验电影对线性叙事的解构，与MOCICI解构认知叙事（"我"的故事）高度一致。
