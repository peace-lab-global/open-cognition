# 怀特海生平年表 · A.N. Whitehead Biography Timeline

## 早年与剑桥时期（1861–1910）

| 年份 | 事件 |
|------|------|
| 1861.2.15 | 生于英国肯特郡兰斯盖特（Ramsgate），牧师家庭 |
| 1875 | 进入谢尔本学校（Sherborne School）学习 |
| 1880 | 进入剑桥大学三一学院（Trinity College）学习数学 |
| 1884 | 获数学学士学位，当选三一学院研究员（Fellow） |
| 1884–1910 | 在剑桥大学任数学讲师，长期教授应用数学 |
| 1891 | 与 Evelyn Willoughby Wade 结婚 |
| 1898 | 出版 *A Treatise on Universal Algebra* |
| 1901 | 罗素成为怀特海的学生，二人开始合作 |
| 1903 | 出版 *The Axioms of Projective Geometry* |
| 1907 | 出版 *The Axioms of Descriptive Geometry* |

## 《数学原理》时期（1910–1913）

| 年份 | 事件 |
|------|------|
| 1910 | 离开剑桥（因妻子被怀疑与罗素有暧昧关系，学术关系紧张） |
| 1910 | *Principia Mathematica* 第一卷出版 |
| 1911 | 第二卷出版 |
| 1911 | 出版 *An Introduction to Mathematics* |
| 1913 | 第三卷出版——《数学原理》全书完成 |

## 伦敦帝国学院时期（1911–1924）

| 年份 | 事件 |
|------|------|
| 1911–1914 | 任伦敦大学学院（UCL）应用数学讲师 |
| 1914–1924 | 任伦敦帝国学院（Imperial College）应用数学教授 |
| 1917 | 出版 *The Organisation of Thought* |
| 1919 | 出版 *An Enquiry Concerning the Principles of Natural Knowledge* |
| 1920 | 出版 *The Concept of Nature*——自然哲学核心著作 |
| 1922 | 出版 *The Principle of Relativity* |

## 哈佛哲学时期（1924–1937）

| 年份 | 事件 |
|------|------|
| 1924 | 63岁时被哈佛大学聘为哲学教授——这一意外邀请使他完全转向形而上学 |
| 1925 | 出版 *Science and the Modern World*（洛威尔讲演） |
| 1926 | 出版 *Religion in the Making* |
| 1927 | 出版 *Symbolism: Its Meaning and Effect* |
| 1929 | 出版 *Process and Reality*——过程形而上学巨著 |
| 1929 | 出版 *The Function of Reason* 和 *The Aims of Education* |
| 1933 | 出版 *Adventures of Ideas* |
| 1934 | 出版 *Nature and Life* |
| 1937 | 从哈佛退休 |

## 晚年与逝世（1937–1947）

| 年份 | 事件 |
|------|------|
| 1938 | 出版 *Modes of Thought*——最后的方法论反思 |
| 1937–1947 | 退休后继续在剑桥写作与演讲 |
| 1945 | 获英国功绩勋章（Order of Merit） |
| 1947.12.30 | 在美国马萨诸塞州剑桥去世，享年86岁 |

## 身后影响

| 年份 | 事件 |
|------|------|
| 1950s–1960s | 哈茨霍恩（Hartshorne）和科布（Cobb）发展过程神学 |
| 1978 | *Process and Reality* 校正版纳（Griffin & Sherburne eds.）出版 |
| 1990s–2000s | 怀特海研究在生态哲学、复杂性科学中复兴 |
| 2010s | "过程思想"在当代中国哲学界获得广泛关注 |

## 思想分期总结

| 阶段 | 年代 | 关键词 |
|------|------|--------|
| 剑桥数学家 | 1861–1910 | 代数、几何、罗素 |
| 《数学原理》 | 1910–1913 | 数理逻辑 |
| 伦敦科学哲学 | 1911–1924 | 自然哲学、相对论 |
| 哈佛形而上学 | 1924–1937 | 《过程与实在》、过程哲学 |
| 晚年 | 1937–1947 | 思维模式、教育 |
