# 帕菲特生平年表 · Derek Parfit Biography Timeline

## 早年（1942–1967）

| 年份 | 事件 |
|------|------|
| 1942.12.11 | 生于中国成都（父母为英国传教士），在中国生活至1945年 |
| 1945 | 随父母返回英国 |
| c. 1955 | 进入伊顿公学（Eton College）学习 |
| 1961 | 进入牛津大学巴利奥尔学院（Balliol College） |
| 1964 | 获牛津大学哲学、政治学和经济学学士学位（PPE），一等荣誉 |
| 1964–1965 | 获哈佛-耶鲁奖学金赴美游学 |
| 1966 | 返回牛津，当选万灵学院（All Souls College）研究员 |
| 1967 | 开始在万灵学院从事哲学研究（此后终身） |

## 个人同一性与早期工作（1967–1984）

| 年份 | 事件 |
|------|------|
| 1967–1970 | 研究个人同一性问题，发展还原论观点 |
| 1971 | 发表开创性论文 "Personal Identity" (*Philosophical Review*) |
| 1973 | 发表 "Later Selves and Moral Principles" |
| 1970s | 在万灵学院潜心写作 *Reasons and Persons*，极少发表 |
| 1982 | 发表关于未来世代和非同一性问题的重要论文 |

## 《理由与人格》与学术声誉（1984–2010）

| 年份 | 事件 |
|------|------|
| 1984 | 出版 *Reasons and Persons*——彻底改变伦理学和个人同一性研究 |
| 1984–1990 | *Reasons and Persons* 引发大量学术讨论和批评 |
| 1986 | 发表对罗尔斯正义理论的评论 |
| 1990s | 专注于人口伦理学和"令人反感的结论"（Repugnant Conclusion） |
| 1997 | 发表 "Reasons and Motivation"——客观理由论 |
| 1999 | 当选英国国家学术院院士（FBA） |
| 2000s | 专注于 *On What Matters* 的写作——这一工作持续二十余年 |

## 《论重要之事》与晚年（2010–2017）

| 年份 | 事件 |
|------|------|
| 2011 | 出版 *On What Matters* Vols. 1-2——论证伦理理论的趋同 |
| 2011–2016 | 回应来自批评者的反对意见，修订第三卷 |
| 2014 | 获罗尔夫·肖克奖（Rolf Schock Prize）——逻辑与哲学 |
| 2017.1.1 | *On What Matters* Vol. 3 出版 |
| 2017.1.1 | 在牛津去世，享年74岁 |

## 身后影响

| 年份 | 事件 |
|------|------|
| 2017 | 全球哲学界悼念——被誉为"过去50年最重要的道德哲学家" |
| 2018– | 大量纪念论文集和帕菲特遗产整理 |
| 2020s | 帕菲特的个人同一性理论在AI和认知科学中获得新的关注 |

|------|------|--------|
