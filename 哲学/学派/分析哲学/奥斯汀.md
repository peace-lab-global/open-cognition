# 约翰·奥斯汀 · J. L. Austin

约翰·兰肖·奥斯汀（J. L. Austin，1911–1960）是英国语言哲学家，日常语言学派代表。他提出“言语行为理论”，指出说话不只是描述世界，也是在施行行动。

## 核心命题 / Core Theses

### 1. 言语即行为

说出一句合适的话本身就是一种行为。例如“我道歉”“我命名这艘船”不只是陈述，而是在完成道歉或命名这一行为。

### 2. 言内、言外与言后行为

言内行为是说出有意义的话语；言外行为是话语所执行的特定功能（承诺、命令、请求）；言后行为是话语对听者产生的实际效果。

### 3. 记述话语与施行话语

并非所有语句都是真值可判断的记述。许多语句的价值在于是否“合适地”完成某种社会行为，而非是否符合事实。

## 关键著作 / Key Works

- *How to Do Things with Words* (1962)
- *Sense and Sensibilia* (1962)

## 跨学科关联 / Interdisciplinary Links

- 与 [路德维希·维特根斯坦](维特根斯坦.md) 共同奠定日常语言哲学，但奥斯汀更注重语言的细微分类。
- 与 [约翰·塞尔](../分析哲学/约翰.md) 形成直接的师生与批评关系，塞尔发展了奥斯汀的言语行为理论。
