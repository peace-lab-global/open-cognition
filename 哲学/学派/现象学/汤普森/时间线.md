# 汤普森生平年表 · Thompson Biography Timeline

## 早年与学术训练

| 年份 | 事件 |
|------|------|
| 1962 | 生于加拿大 |
| 1980s | 本科阶段学习哲学与认知科学 |
| 1980s 后期 | 与 Francisco Varela 建立学术合作关系 |
| 1987 | 开始在认知科学与现象学的交叉领域进行研究 |

## 生成认知的奠基

| 年份 | 事件 |
|------|------|
| 1991 | 与 Varela、Rosch 合著 *The Embodied Mind*——生成认知范式的奠基之作 |
| 1990s | 从事博士后研究，深化生成认知理论 |
| 1996 | 发表 "Sensorimotor Subjectivity and the Intersubjectivity of Experience" |
| 2000 | 开始在多伦多大学哲学系任教 |
| 2001 | 编著 *Between Ourselves: Second-Person Issues in the Study of Consciousness* |

## 生命-心智连续性的系统论证

| 年份 | 事件 |
|------|------|
| 2001–2007 | 系统研究自组织理论、现象学与生物学的整合 |
| 2004 | 发表 "Autopoiesis, Adaptivity, Teleology, Agency"——自组织理论的哲学分析 |
| 2007 | 出版 *Mind in Life: Biology, Phenomenology, and the Sciences of Mind*——生命-心智连续性的代表作 |
| 2008 | 转至不列颠哥伦比亚大学（UBC）哲学系任教授 |

## 冥想哲学与东西方对话

| 年份 | 事件 |
|------|------|
| 2010 | 开始深入研究印度哲学与佛教冥想传统 |
| 2011 | 与 Siderits、Zahavi 合编 *Self, No Self?*——佛教无我论与现象学的核心对话 |
| 2014 | 发表关于梦中自我结构的重要论文 |
| 2015 | 出版 *Waking, Dreaming, Being*——意识三态与东西方哲学对话 |
| 2016 | 与 Varela、Rosch 出版 *The Embodied Mind* 修订版，更新导言与新反思 |

## 反思与批评期

| 年份 | 事件 |
|------|------|
| 2018 | 持续在不列颠哥伦比亚大学从事教学与研究 |
| 2020 | 出版 *Why I Am Not a Buddhist*——对佛教现代主义的批评性反思 |
| 2020s | 持续参与4E认知、冥想研究、东西方哲学对话的国际讨论 |

## 学术荣誉与影响

- 不列颠哥伦比亚大学（UBC）哲学教授
- 生成认知范式的联合创始人
- *The Embodied Mind* 成为4E认知运动的奠基文本
- *Mind in Life* 被誉为当代现象学与认知科学对话的里程碑
- 在神经现象学、冥想研究、比较哲学领域具有广泛影响力

## 思想分期总结

| 阶段 | 年代 | 关键词 |
|------|------|--------|
| 生成认知的奠基 | 1980s–1991 | Varela、Embodied Mind、enaction |
| 生命-心智连续性 | 2000s–2007 | autopoiesis、Mind in Life |
| 冥想哲学 | 2014–2015 | 意识三态、东西方对话 |
| 反思与批评 | 2020– | 佛教现代主义批评 |
