# 伊曼努尔·列维纳斯 · Emmanuel Levinas

伊曼努尔·列维纳斯（Emmanuel Levinas，1906–1995）是立陶宛裔法国哲学家，现象学与犹太思想传统的重要代表。他将伦理学置于第一哲学，强调“他者之脸”对自我的无限责任。

## 核心命题 / Core Theses

### 1. 他者之脸

面对面相遇时，他者的脸既向我呈现脆弱，又发出“不可杀人”的伦理命令。它阻止我将他者还原为同一。

### 2. 伦理学作为第一哲学

不是存在论或知识论优先，而是我对他者的无限责任优先。主体性首先是一种为他者的主体性。

### 3. 享受与脆弱

生活首先是享受（enjoyment），但享受的他者性与身体的依赖性也构成脆弱的伦理根源。

## 关键著作 / Key Works

- *Totality and Infinity* (1961)
- *Otherwise than Being or Beyond Essence* (1974)

## 跨学科关联 / Interdisciplinary Links

- 与 [马丁·海德格尔](../存在主义/海德格尔/README.md) 在现象学传统中关系密切，但列维纳斯批评其存在论忽视伦理优先性。
- 与 [伊曼努尔·康德](../德国唯心论/康德/README.md) 的道德律令形成对话，但列维纳斯的伦理源于面对面的不对称责任。
