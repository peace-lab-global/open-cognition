# 阿伦特生平年表 · Arendt Biography Timeline

## 早年与学术训练（1906–1933）

| 年份 | 事件 |
|------|------|
| 1906.10.14 | 生于德国汉诺威，世俗犹太家庭 |
| 1913 | 父亲去世 |
| 1924 | 在马堡大学师从海德格尔 |
| 1925 | 与海德格尔开始恋情 |
| 1926 | 转至海德堡大学师从雅斯贝尔斯 |
| 1929 | 以奥古斯丁的爱的概念获海德堡大学博士学位 |
| 1929 | 与 Günther Stern（第一任丈夫）结婚 |
| 1930 | 在德国开始研究反犹主义 |
| 1933 | 纳粹上台；阿伦特被盖世太保短暂拘留 |
| 1933 | 流亡巴黎 |

## 巴黎流亡（1933–1941）

| 年份 | 事件 |
|------|------|
| 1933–1941 | 在巴黎从事犹太难民救助工作 |
| 1936 | 与 Heinrich Blücher 相识 |
| 1938 | 与 Stern 离婚 |
| 1940 | 与 Blücher 结婚 |
| 1940 | 被法国维希政府拘押于 Gurs 集中营 |
| 1941 | 逃离集中营，经里斯本移居美国纽约 |

## 美国时期：极权主义分析（1941–1958）

| 年份 | 事件 |
|------|------|
| 1941 | 抵达纽约，开始英文写作 |
| 1944–1946 | 任犹太文化重建委员会研究主任 |
| 1951 | 出版 *The Origins of Totalitarianism*——极权主义分析经典 |
| 1951 | 获美国公民身份 |
| 1953 | 开始在普林斯顿大学、芝加哥大学讲学 |
| 1958 | 出版 *The Human Condition*——劳动/制作/行动理论 |

## 恶的平庸性与革命理论（1961–1970）

| 年份 | 事件 |
|------|------|
| 1959 | 成为普林斯顿大学首位女性正教授 |
| 1961 | 为《纽约客》报道艾希曼审判 |
| 1961 | 出版 *Between Past and Future* |
| 1963 | 出版 *Eichmann in Jerusalem*——引发巨大争议 |
| 1963 | 出版 *On Revolution*——美国革命与法国革命比较 |
| 1967 | 在纽约社会研究新学院任教 |

## 晚年（1970–1975）

| 年份 | 事件 |
|------|------|
| 1970 | Blücher 去世 |
| 1972 | 出版 *Crises of the Republic* |
| 1973 | 开始撰写《心智生活》 |
| 1975 | 获丹麦松宁奖（Sonning Prize） |
| 1975.12.4 | 在纽约去世，享年69岁 |
| 1978 | *The Life of the Mind* 遗著出版 |

## 思想分期总结

| 阶段 | 年代 | 关键词 |
|------|------|--------|
| 学术训练与流亡 | 1906–1941 | 海德格尔、雅斯贝尔斯、流亡 |
| 极权主义分析 | 1941–1958 | *Origins*、*Human Condition* |
| 恶的平庸性 | 1961–1970 | 艾希曼、革命理论 |
| 心智生活 | 1970–1975 | 思维、意志、判断 |
