# 亚里士多德著作年表 · Aristotle Works Chronology

亚里士多德的著作分为两类：论著（treatises, pragmateiai）——学园内部的讲义和笔记，是现存的主要文本；以及外传著作（exoteric works）——面向公众的对话录，大部分已散佚。

Aristotle's works fall into two categories: treatises (*pragmateiai*)—internal lecture notes and essays, comprising the main surviving texts; and exoteric works—public-facing dialogues, mostly lost.

---

## 逻辑学：《工具论》Logic: The Organon

| 著作 Work | 要点 Key Points |
|-----------|-----------------|
| 《范畴篇》*Categories* | 十范畴；第一实体与第二实体 / Ten categories; primary and secondary substance |
| 《解释篇》*On Interpretation* | 命题逻辑；矛盾律 / Propositional logic; law of contradiction |
| 《前分析篇》*Prior Analytics* | 三段论理论 / Syllogistic theory |
| 《后分析篇》*Posterior Analytics* | 科学知识的结构；演绎与归纳 / Structure of scientific knowledge; deduction and induction |
| 《论题篇》*Topics* | 辩证推理 / Dialectical reasoning |
| 《辩谬篇》*Sophistical Refutations* | 逻辑谬误分类 / Classification of logical fallacies |

## 理论哲学 Theoretical Philosophy

### 自然哲学 Natural Philosophy

| 著作 Work | 要点 Key Points |
|-----------|-----------------|
| 《物理学》*Physics* (8 卷) | 运动、变化、时间、空间、因果、无限 / Motion, change, time, space, causation, infinity |
| 《论天》*On the Heavens* | 天体运动；四元素 / Celestial motion; four elements |
| 《论生成与消灭》*On Generation and Corruption* | 元素的相互转化 / Inter-transformation of elements |
| 《气象学》*Meteorology* | 天气现象 / Meteorological phenomena |

### 生物学 Biology

| 著作 Work | 要点 Key Points |
|-----------|-----------------|
| 《动物志》*History of Animals* | 500+ 物种的描述和分类 / Description and classification of 500+ species |
| 《动物部分论》*On the Parts of Animals* | 器官的功能与目的 / Function and purpose of organs |
| 《动物运动论》*On the Motion of Animals* | 动物运动的机制 / Mechanism of animal motion |
| 《动物生成论》*On the Generation of Animals* | 繁殖与遗传 / Reproduction and heredity |

### 心理学 Psychology

| 著作 Work | 要点 Key Points |
|-----------|-----------------|
| 《论灵魂》*De Anima* | 灵魂作为身体的形式；五感与理智 / Soul as form of body; senses and intellect |
| 《自然短论》*Parva Naturalia* | 感觉、记忆、睡眠、梦、寿命 / Sensation, memory, sleep, dreams, longevity |

### 形而上学 Metaphysics

| 著作 Work | 要点 Key Points |
|-----------|-----------------|
| 《形而上学》*Metaphysics* (14 卷) | 作为存在的存在；实体论；四因说；不动的动者 / Being qua being; substance; four causes; Unmoved Mover |
