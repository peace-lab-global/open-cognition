# 老子年表 / Timeline of Laozi

## 老子其人

关于老子的历史身份，自司马迁以来即有争议。以下列出传统记载与学术考证两条线索。

### 传统记载（司马迁《史记》）

| 时间 | 事件 |
|------|------|
| 约公元前6世纪 | 生于楚国苦县厉乡曲仁里（今河南鹿邑），名李耳，字聃 |
| 春秋末期 | 任周守藏室之史（国家图书馆馆长） |
| 约公元前518年 | 孔子问礼于老子（传说） |
| 约公元前5世纪 | 见周之衰，西出函谷关 |
| — | 关令尹喜请老子著书，乃著《道德经》五千言 |
| — | 出关后不知所终 |

司马迁另列两位候选人：老莱子（楚人，与孔子同时）、太史儋（周太史，孔子后百余年）。

### 学术考证

| 观点 | 代表学者 | 论据 |
|------|----------|------|
| 春秋末期人，与孔子同时或略早 | 传统说法 | 《史记》记载、先秦文献引用 |
| 战国中期人 | 梁启超、冯友兰 | 文本中的战国用语、反对儒家仁义的历史背景 |
| 文本渐成说（非一人一时之作） | 当代主流 | 郭店楚简证明文本是逐渐编纂成书的 |

## 《老子》文本年代

| 年代 | 事件 |
|------|------|
| 约公元前4世纪 | 郭店楚简（最早文本，约2000字） |
| 约公元前3世纪 | 文本继续扩充 |
| 约公元前200年 | 马王堆帛书（接近今本，德经在前道经在后） |
| 约公元前2世纪 | 河上公注本（最早的完整注疏之一） |
| 约249年 | 王弼注本（玄学化阐释，影响最大的通行本基础） |
| 1973年 | 马王堆帛书出土 |
| 1993年 | 郭店楚简出土 |

## 老子影响史

| 时代 | 事件/发展 |
|------|-----------|
| 战国中期 | 庄子继承和发展老子思想 |
| 战国末期 | 《吕氏春秋》吸收老子思想 |
| 西汉初 | 黄老之学成为治国理念（文景之治） |
| 东汉 | 道教兴起，老子被神化为太上老君 |
| 魏晋 | 王弼、何晏以老子为核心建立玄学 |
| 唐代 | 李唐皇室尊老子为祖先，道教鼎盛 |
| 宋明 | 理学吸收老子本体论思维；道教内丹学发展 |
| 近现代 | 出土文献革命性地改变了老子研究 |
| 当代 | 老子成为世界哲学的重要组成部分 |
