# 东方哲学 · Eastern Philosophy

## 一句话定义

东方哲学是发源于亚洲（尤其中国、印度、日本）的哲学传统群，以儒家、道家、佛教为核心，强调关系性自我、实践智慧、身心修养与超越性体验，与西方分析—形而上学传统形成对话与互补。

## 历史脉络

**先秦奠基（公元前 6—3 世纪）**
孔子以仁、礼、义重构伦理—政治秩序；老子以道、无为、自然提出对文明人为性的批判；墨子倡兼爱、非攻；庄子以齐物、逍遥深化道家精神。这一时期奠定了中国哲学的问题域与表达方式。

**印度哲学与佛教（公元前 6 世纪—公元后）**
印度六派哲学、佛教四圣谛与缘起论、中观与瑜伽行派，发展出精细的认识论、逻辑与解脱论。佛教东传后与中国本土思想交融，形成汉传佛教、禅宗等形态。

**宋明理学与东亚展开（10—17 世纪）**
朱熹、王阳明等重新诠释儒家经典，回应佛教挑战，发展出理气论、心学等体系。儒学同时影响朝鲜、日本与越南，形成各具特色的东亚儒学。

**近现代转型（19 世纪以后）**
在西方冲击与现代化进程中，东方哲学经历保守、批判与重构：新儒家（牟宗三、唐君毅）尝试会通中西；比较哲学与佛学研究在全球学术体制中重新定位东方思想。

## 核心特征

1. **实践取向**
   哲学不仅是理论反思，更是修身、治家、治国与解脱的实践指南。

2. **关系性自我**
   人并非孤立原子，而是在五伦、缘起、共同体与自然之中构成的关系性存在。

3. **整体主义思维**
   强调天—地—人、身—心、知—行的连续与互动，而非笛卡尔式主客二分。

4. **境界与修养**
   许多传统以成圣、成仙、成佛、成君子为最高目标，重视工夫、冥想与伦理训练。

## 主要传统

- **儒家**：仁、义、礼、智、信；修身齐家治国平天下。
- **道家**：道、无为、自然、齐物、逍遥。
- **佛教**：缘起、空性、慈悲、解脱、中道。
- **墨家**：兼爱、非攻、尚贤、天志。
- **法家**：法、术、势；富国强兵与政治控制。

## 与其他传统关系

- **西方哲学**：东方哲学常被用来批判主体性、理性主义与原子化个人观，也为现象学、过程哲学、生态思想提供资源。
- **宗教研究**：东方哲学与宗教界限模糊，许多传统同时是哲学体系与修行传统。
- **当代应用**：正念、领导力、生态伦理、关系伦理与人工智能伦理中，东方概念持续被重新激活。

## 现当代影响

东方哲学在全球人文社科、心理学（正念、禅宗治疗）、管理学（儒家领导力）、生态运动（深层生态学与道家）以及比较哲学研究中保持活力，是跨文化对话的重要思想资源。
