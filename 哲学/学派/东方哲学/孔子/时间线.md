# 孔子生平年表 · Confucius Biography Timeline

## 早年（公元前 551–前 522）

| 年份 | 事件 |
|------|------|
| 前551 | 生于鲁国陬邑（今山东曲阜），名丘，字仲尼。父叔梁纥，母颜征在 |
| 前548 | 三岁丧父，随母迁居阙里，家境贫寒 |
| 前535 | 约十五岁，"吾十有五而志于学"——立志求学 |
| 前532 | 约二十岁，为季氏委吏（管仓库），后为乘田（管畜牧） |
| 前531 | 娶宋国亓官氏为妻 |
| 前530 | 生子孔鲤（字伯鱼） |

## 求学与教学（前 522–前 501）

| 年份 | 事件 |
|------|------|
| 前522 | 约三十岁，"三十而立"——开始收徒讲学 |
| 前518 | 赴周都洛邑问礼于老子（传统说法，有争议） |
| 前517 | 鲁国内乱（季氏与昭公冲突），孔子适齐 |
| 前515 | 在齐闻韶乐，"三月不知肉味" |
| 前515 | 齐景公问政，孔子答以"君君、臣臣、父父、子子" |
| 前514 | 返鲁，继续教学，弟子日众 |

## 从政（前 501–前 497）

| 年份 | 事件 |
|------|------|
| 前501 | 约五十岁，"五十而知天命"——任中都宰（中都地方官） |
| 前500 | 升任司空（工程官），后任大司寇（司法官） |
| 前500 | 夹谷之会：孔子以礼折冲齐国，维护鲁国尊严 |
| 前498 | 推行"堕三都"——拆除季孙、叔孙、孟孙三家的城防（部分成功） |
| 前497 | 齐人馈女乐，鲁君受之而不理朝政；孔子失望离去 |

## 周游列国（前 497–前 484）

| 年份 | 事件 |
|------|------|
| 前497 | 率弟子离开鲁国，开始十四年的流亡生涯 |
| 前497 | 至卫国，卫灵公礼遇但未予重任 |
| 前495 | 经匡地被围（"子畏于匡"），弟子以为孔子已被杀 |
| 前493 | 过宋国，遭司马桓魋追杀（"天生德于予，桓魋其如予何"） |
| 前492 | 至陈国，后又至蔡国 |
| 前489 | 陈蔡之间绝粮七日（"陈蔡之厄"），弟子饥病，孔子弦歌不辍 |
| 前489 | 楚昭王欲聘孔子，被令尹子西劝阻 |
| 前488 | 返卫国 |
| 前484 | 卫国大夫孔文子欲攻太叔疾，孔子劝阻后离开卫国 |

## 晚年（前 484–前 479）

| 年份 | 事件 |
|------|------|
| 前484 | 约六十八岁，"六十而耳顺，七十而从心所欲不逾矩" |
| 前484 | 应鲁国季康子之邀返鲁，不再从政 |
| 前483 | 专心整理文献、教授弟子 |
| 前482 | 子孔鲤去世 |
| 前481 | 弟子颜回去世，孔子悲痛欲绝（"天丧予！天丧予！"） |
| 前480 | 弟子子路在卫国内乱中殉难 |
| 前479 | 鲁哀公十六年四月己丑，孔子去世，享年约七十三岁 |

## 死后影响

| 时间 | 事件 |
|------|------|
| 前479 之后 | 弟子守墓三年（子贡守六年），后辑录言行成《论语》 |
| 战国 | 孟子（约前372–前289）发展仁政与性善论；荀子（约前313–前238）发展礼法与性恶论 |
| 汉代 | 董仲舒"罢黜百家，独尊儒术"（前136），儒学成为官方意识形态 |
| 宋代 | 朱熹（1130–1200）理学化诠释，《四书章句集注》成为科举标准 |
| 明代 | 王阳明（1472–1529）心学转向 |
| 近现代 | 新儒家运动（牟宗三、唐君毅、杜维明）；"批林批孔"运动（1974） |
| 当代 | 孔子学院全球扩展；儒学复兴与东亚现代化讨论 |

## 思想分期总结

| 阶段 | 年代 | 关键词 |
|------|------|--------|
| 求学立志 | 前551–前522 | 志于学、贫寒自学 |
| 教学与从政 | 前522–前497 | 三十而立、鲁国司寇 |
| 周游列国 | 前497–前484 | 天命、颠沛流离 |
| 删订教学 | 前484–前479 | 七十从心所欲、整理文献 |
