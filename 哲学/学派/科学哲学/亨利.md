# 亨利·庞加莱 · Henri Poincaré

亨利·庞加莱（Henri Poincaré，1854–1912）是法国数学家、物理学家、天文学家与科学哲学家，被誉为“最后一位通才”。他在天体力学中研究三体问题时发现，即使确定性方程也可能产生对初始条件极端敏感的轨道——这一洞见被后来的混沌理论继承。

## 核心命题 / Core Theses

### 1. 敏感依赖与不可预测性

三体问题的研究表明，极小的初始误差会在演化中被指数放大，使长期预测在实践上不可能。这是“确定性混沌”的早期数学预兆。

### 2. 约定主义

科学定律并非纯粹经验归纳，也包含人类为便利而选择约定；但约定并非任意，需受经验约束。这影响了后来的科学哲学与相对论解释。

### 3. 数学直觉

数学发现依赖非逻辑的直觉与审美选择，而非纯机械演绎；这一观点影响了后来的数学哲学与认知研究。

## 关键著作 / Key Works

- *Science and Hypothesis* (1902)
- *The Value of Science* (1905)
- *Science and Method* (1908)
- *New Methods of Celestial Mechanics* (1892–1899)

## 跨学科关联 / Interdisciplinary Links

- 其敏感依赖思想在 [爱德华·洛伦兹](../../../认知系统/学派/复杂系统/爱德华.md) 的气象模型中得到具体化，催生了“蝴蝶效应”。
- 与希尔伯特的公理化纲领、罗素的逻辑主义形成数学基础论争。
- 其哲学影响了托马斯·库恩的科学范式理论及后来的科学实践研究。
