# 认知系统工程视角下的 Open Cognition 知识库：结构性缺口分析

> 分析日期：2026-05-23
> 分析视角：认知系统工程（Cognitive Systems Engineering, CSE）
> 状态：已批准执行

---

## 一、总判断

Open Cognition 在**个体认知**（心理学各流派）和**文明智慧**（哲学/宗教/社会学）方面已建设得相当扎实——约 221 文件、80+ 思想家、90 Skills、跨学科互链密度高。但从**认知系统工程（CSE）** 的角度看，存在一个深层的**结构性盲区**：

> **整个知识库隐含的认知模型是"认知 = 发生在个体颅腔内的心理过程"，而 CSE 的核心主张是"认知 = 人-技术-环境构成的联合系统中的涌现过程"。**

这不是缺少几个条目就能补上的——而是缺少一整条认知观的轴线。

---

## 二、缺少的认知观轴线：从"颅内认知"到"系统认知"

当前知识库覆盖了两极：

| 已有 | 代表 |
|---|---|
| **个体内部认知**（心理学） | 皮亚杰图式、卡尼曼双系统、贝克CBT、弗洛伊德潜意识 |
| **意识本体论**（佛教认知理论） | 八识体系、心物一元、转识成智 |

但**缺失了中间层**——把认知视为**分布在人、工具、环境之间的系统工程问题**的整条思想谱系：

```
颅内认知 ←→ [缺失的系统层] ←→ 意识本体论
(Piaget, Kahneman)    (?)          (唯识宗)
```

这个系统层包括：**控制论、生态心理学、分布式认知、活动理论、具身认知、延展认知、自然决策、韧性工程**。

| **恩格斯托姆 Yrjö Engeström** | 活动理论（Activity Theory） | 分析"人-工具-目标-规则-共同体"的完整活动系统 |

## 四、缺失的核心概念（约 10 个）

| 概念 | 定义 | 与已有概念的关联 |
|---|---|---|
| **有限理性 Bounded Rationality** | 人在信息、时间、计算能力受限时用启发式而非优化来决策 | → 卡尼曼双系统 |
| **可供性 Affordance** | 环境直接提供的行动可能性 | → 颠覆"刺激-反应"和"认知加工"两种传统 |
| **认知负荷 Cognitive Load** | 工作记忆的内在/外在/关联负荷 | → 对教学设计、界面设计的革命性影响 |
| **态势感知 Situation Awareness** | 对环境中元素的感知、理解和对未来的预测 | → 从飞行员到医疗到AI辅助决策 |
| **分布式认知 Distributed Cognition** | 认知过程分布在人、工具、环境之间 | → 超越个体主义认知观 |
| **意义建构 Sensemaking** | 在模糊情境中主动建构理解的过程 | → 不同于"信息加工" |
| **心智模型 Mental Models** | 人对系统如何运作的内部心理表征 | → 连接认知心理学与系统设计 |
| **活动理论 Activity Theory** | 分析人-工具-目标-规则-共同体-分工的完整活动系统 | → 维果茨基传统的系统化 |
| **韧性工程 Resilience Engineering** | 系统如何在意外和压力下保持功能 | → 不同于"安全=防止错误" |
| **联合认知系统 Joint Cognitive System** | 人与技术构成的认知整体 | → CSE 的核心分析单元 |
| **预测加工 Predictive Processing** | 大脑是预测机器，主动生成对世界的假设 | → 当代认知科学最大范式转换之一 |
| **生成认知 Enaction** | 认知不是表征世界，而是在与环境的耦合中生成世界 | → 连接现象学与认知科学 |

## 六、深层结构问题

### 6.1 认知观的隐含偏差

当前知识库的**隐含认知观**是**方法论个体主义**的——认知被理解为"发生在个体内部的心理/意识过程"。

CSE 要求的是**方法论关系主义/系统论**——认知不是"在人里面"的，而是"在人与世界的耦合中涌现的"。

### 6.2 缺失的"工程"维度

知识库收录的都是**理解性**知识（understanding）。但 CSE 的核心是**设计性**知识（designing）——如何**设计**人-技术系统以优化认知表现和人类福祉。

### 6.3 AI 时代的认知系统问题

- **人机协作认知系统**：LLM 如何改变人类的认知过程？
- **自动化偏见**：人过度信任 AI 输出的认知偏差
- **认知卸载**：当 AI 替代了记忆、推理、决策，人类的认知能力会怎样？
- **分布式认知的新形态**：人+LLM 构成了一种全新的联合认知系统
