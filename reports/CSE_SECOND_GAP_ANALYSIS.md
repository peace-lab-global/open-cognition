# 认知系统工程（CSE）二次缺口分析报告

> 分析日期：2026-05-23
> 分析视角：认知系统工程（Cognitive Systems Engineering, CSE）全局视角
> 基础：v0.4 已建成 — 5 学派 / 15 思想家 / 12 概念 / 10 Skills
> 状态：已批准执行

---

## 一、总判断

v0.4 的 CSE 建立了很好的骨架——控制论、生态心理学、分布式认知、认知工程、自然决策五大学派。但从**全球 CSE 知识体系**的角度看，仍有 **6 个结构性缺口**需要补充。

核心缺失：安全科学（CSE 的另一半核心）、自动化与人-AI 协作（AI 时代最紧迫议题）、社会技术系统理论（CSE 的根之一）。

---

## 二、缺口详情

### 缺口 1：安全科学与事故理论（最严重遗漏）

CSE 核心关切之一是"复杂系统为何失败"，但当前几乎无安全科学覆盖。

| 思想家 | 为何必须补 |
|---|---|
| **Erik Hollnagel** | Safety-II 创始人、ETTO 原则、与 Woods/Rasmussen 并称 CSE 三巨头。最严重遗漏 |
| **James Reason** | 瑞士奶酪模型发明者。Skill 里引用了他但没有思想家条目 |
| **Sidney Dekker** | "漂移进入失败"、Just Culture、安全科学最活跃的当代思想家 |
| **Richard Cook** | "复杂系统如何失败"经典论文、系统韧性实践者 |
| **Nancy Leveson** | STAMP/STPA 方法论、系统论事故模型 |

新增概念：Safety-I vs Safety-II、漂移进入失败、正常事故、安全文化与公正文化

新增学派：`schools/safety-science/`

### 缺口 2：自动化与人-AI 协作认知系统（AI 时代最紧迫）

| 思想家 | 为何重要 |
|---|---|
| **Lisanne Bainbridge** | "自动化的讽刺"(1983) — 被引用数万次的经典 |
| **Raja Parasuraman** | 自动化四阶段模型、人机信任研究 |
| **Neville Stanton** | 人机交互设计、MABA-MABA 框架 |

新增概念：自动化偏见、脱离回路、人-自动化信任校准、自动化讽刺、团队认知

新增 Skills：自动化级别评估、人-AI 团队设计

### 缺口 3：社会技术系统理论

| 思想家 | 为何重要 |
|---|---|
| **Kim Vicente** | 认知工作分析当代代表、《Cognitive Work Analysis》、生态界面设计 |
| **Fred Emery / Eric Trist** | Tavistock 学派创始人、社会技术系统理论奠基人 |

新增概念：社会技术系统、生态界面设计、认知任务分析

### 缺口 4：宏观认知与专家研究

| 缺失概念 | 定义 |
|---|---|
| **宏观认知 Macrocognition** | 真实复杂环境中专家使用的高级认知过程 |
| **专业知识与刻意练习** | 专家能力的本质——模式库、情境化知识 |
| **情境化认知 Situated Cognition** | 认知总是嵌入在具体情境中 |
| **分布式态势感知** | SA 不是个人的，而是团队/系统层面的 |

### 缺口 5：认知系统工效学方法论

| 缺失 Skill | 功能 |
|---|---|
| 认知任务分析（CTA） | 提取专家的隐性知识和决策策略 |
| 关键决策方法（CDM） | Klein 团队标准化 NDM 数据收集 |
| 系统理论事故模型（STPA） | Leveson 基于控制论的事故分析 |
| Safety-II 学习分析 | Hollnagel 的工作即学习方法 |

### 缺口 6：与已有领域的深层对话

| 已有条目 | 缺失的 CSE 对话 |
|---|---|
| 贝克·风险社会 | 正常事故 + 漂移失败 |
| 福柯·规训 | 自动化与权力 |
| 哈贝马斯·交往理性 | 人机交往理性条件 |
| 阿伦特·公共领域 | AI 时代的认知公共品 |
| 拉图尔·行动者网络 | AI Agent 作为非人行动者 |
