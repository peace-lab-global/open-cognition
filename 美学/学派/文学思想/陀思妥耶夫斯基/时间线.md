# 陀思妥耶夫斯基美学思想年表

> 完整传记年表见 [文学领域](../../../../文学/学派/小说家/费奥多尔/时间线.md)。

## 美学思想发展

| 年代 | 事件 | 美学意义 |
|---|---|---|
| 1845 | 《穷人》发表 | 心理现实主义的萌芽 |
| 1849 | 死刑未遂与流放 | 灵魂转变的决定性经历 |
| 1850–1854 | 西伯利亚苦役 | 苦难美学的个人体验 |
| 1864 | 《地下室手记》 | 非理性自由的发现 |
| 1866 | 《罪与罚》 | 超人理论的心理实验 |
| 1869 | 《白痴》 | 绝对善良的悲剧 |
| 1872 | 《群魔》 | 意识形态批判 |
| 1880 | 《卡拉马佐夫兄弟》 | 复调美学的巅峰 |
