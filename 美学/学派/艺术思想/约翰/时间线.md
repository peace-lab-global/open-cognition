# 凯奇年表 / Cage Timeline

## 早期 / Early Years

| 年份 | 事件 |
|------|------|
| 1912 | 9月5日出生于洛杉矶 |
| 1930s | 师从阿诺德·勋伯格（短暂）学习作曲 |
| 1938 | 创作首件预制钢琴作品 *Bacchanale* |
| 1939 | 在西雅图康沃尔学院任教 |

## 中期 / Middle Period

| 年份 | 事件 |
|------|------|
| 1946-48 | 完成《预制钢琴奏鸣曲与间奏曲》 |
| 1950 | 获得《易经》，开始偶然性创作 |
| 1951 | 创作《音乐的变化》(*Music of Changes*) |
| 1952 | 首演《4分33秒》；在黑山学院参与"偶发事件" |
| 1955-56 | 在铃木大拙的纽约讲座学习禅宗 |
| 1961 | 出版《沉默》文集 |

## 晚期 / Late Period

| 年份 | 事件 |
|------|------|
| 1967 | 出版《一年从星期一开始》 |
| 1970s | 发展mesostic诗歌和视觉艺术 |
| 1987 | 开始创作"数字作品"（Number Pieces）系列 |
| 1989-90 | 哈佛诺顿讲座 |
| 1992 | 8月12日在纽约去世，享年79岁 |
