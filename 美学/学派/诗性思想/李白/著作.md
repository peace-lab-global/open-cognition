# 李白作品总览 · Li Bai Works

## 总览

李白一生创作诗歌约一千余首，现存约990首（据清人王琦注《李太白全集》）。其诗歌体裁多样，包括乐府、歌行、律诗、绝句等，但以乐府和歌行最为出色，最能体现其"自然天成"的美学理念。其创作以742年入长安为分水岭，分为早期漫游豪放阶段和后期理想幻灭与超越阶段。

## 一、乐府诗（旧题乐府）

李白最擅长以旧题乐府写新意，赋予古题以全新的生命力。

| 诗名 | 题源 | 特色 |
|---|---|---|
| 《将进酒》 | 汉乐府旧题 | 酒与时间虚无，"天生我材必有用，千金散尽还复来" |
| 《行路难》三首 | 鲍照旧题 | 政治理想幻灭的悲壮，"长风破浪会有时，直挂云帆济沧海" |
| 《蜀道难》 | 古题 | 语言的极限体验，"噫吁嚱！危乎高哉！蜀道之难，难于上青天" |
| 《战城南》 | 汉乐府旧题 | 反战主题，"乃知兵者是凶器，圣人不得已而用之" |
| 《关山月》 | 古题 | 边塞与乡愁，"明月出天山，苍茫云海间" |
| 《长干行》 | 古题 | 商妇的爱情与等待，"郎骑竹马来，绕床弄青梅" |
| 《子夜吴歌》四首 | 南朝乐府 | 四季的爱情与思念 |
| 《侠客行》 | 古题 | 游侠精神，"十步杀一人，千里不留行" |

## 二、歌行体

李白的歌行体最能体现其自由奔放、不受格律约束的风格。

| 诗名 | 特色 |
|---|---|
| 《梦游天姥吟留别》 | 梦境、自由与对权贵的拒绝，"安能摧眉折腰事权贵，使我不得开心颜" |
| 《庐山谣寄卢侍御虚舟》 | 山水与仙游，"我本楚狂人，凤歌笑孔丘" |
| 《宣城见杜鹃花》 | 乡愁与花意象 |
| 《把酒问月》 | 月的哲学，"今人不见古时月，今月曾经照古人" |
| 《白云歌送刘十六归山》 | 自由与隐逸 |
| 《江上吟》 | "兴酣落笔摇五岳，诗成笑傲凌沧洲" |

## 三、绝句

李白绝句被誉为"唐代第一"，以自然天成、不事雕琢著称。

### 七绝

| 诗名 | 特色 |
|---|---|
| 《早发白帝城》 | "朝辞白帝彩云间，千里江陵一日还"——劫后重生的轻快 |
| 《望庐山瀑布》 | "飞流直下三千尺，疑是银河落九天"——自然崇高 |
| 《黄鹤楼送孟浩然之广陵》 | "孤帆远影碧空尽，唯见长江天际流"——送别与无限 |
| 《赠汪伦》 | "桃花潭水深千尺，不及汪伦送我情"——友情 |
| 《闻王昌龄左迁龙标遥有此寄》 | "我寄愁心与明月，随君直到夜郎西" |

### 五绝

| 诗名 | 特色 |
|---|---|
| 《静夜思》 | "床前明月光，疑是地上霜。举头望明月，低头思故乡"——中国最著名的诗 |
| 《独坐敬亭山》 | "相看两不厌，只有敬亭山"——人与自然的非工具性关系 |
| 《夜宿山寺》 | "危楼高百尺，手可摘星辰"——夸张与天真 |
| 《怨情》 | "美人卷珠帘，深坐蹙蛾眉"——含蓄的闺怨 |

## 四、律诗

虽然李白以乐府和歌行见长，但其律诗亦有佳作。

| 诗名 | 特色 |
|---|---|
| 《渡荆门送别》 | "山随平野尽，江入大荒流"——出蜀的壮阔 |
| 《送友人》 | "青山横北郭，白水绕东城"——送别诗的经典 |
| 《登金陵凤凰台》 | 与崔颢《黄鹤楼》齐名的登临之作 |
| 《秋登宣城谢朓北楼》 | "人烟寒橘柚，秋色老梧桐"——秋思 |

## 十二、阅读路径

### 入门
1. 《静夜思》— 中国最著名的诗（20字）
2. 《将进酒》— 酒与自由的宣言（约100字）
3. 《望庐山瀑布》— 自然崇高（28字）
4. 《早发白帝城》— 劫后重生的轻快（28字）

### 进阶
5. 《梦游天姥吟留别》— 梦境、自由与对权贵的拒绝
6. 《行路难》三首— 政治理想幻灭的悲壮
7. 《蜀道难》— 语言的极限体验
8. 《月下独酌》四首— 孤独美学

### 深潜
9. 《古风》五十九首— 咏史、感怀、政治批评
10. 葛晓音《天际悬明月——李白》
11. 李长之《道教徒的诗人李白及其痛苦》
