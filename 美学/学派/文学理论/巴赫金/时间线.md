# 巴赫金年表 / Bakhtin Timeline

## 早年与学术形成 / Early Years & Formation

| 年份 | 事件 |
|------|------|
| 1895 | 11月17日出生于俄国奥廖尔，父亲为银行职员 |
| 1914 | 进入彼得堡大学历史-哲学系 |
| 1918 | 大学毕业，返回维捷布斯克 |
| 1919-24 | 在维捷布斯克和涅韦尔组织哲学小组，与沃洛希诺夫、梅德韦杰夫等密切交流 |
| 1924 | 移居列宁格勒（彼得格勒），继续学术活动 |

## 创作与迫害 / Creative Output & Persecution

| 年份 | 事件 |
|------|------|
| 1929 | 出版《陀思妥耶夫斯基创作问题》——首次提出复调概念 |
| 1929 | 因参加"非官方"学术活动被捕，判处流放 |
| 1930-36 | 流放至库斯塔奈（哈萨克斯坦），在地方中学教书 |
| 1935 | 撰写《小说中的话语》 |
| 1937-38 | 撰写《小说中的时空体形式》 |
| 1937 | 流放结束，但被禁止居住在大城市 |
| 1941 | 撰写《史诗与小说》 |
| 1942 | 完成《拉伯雷与民间文化》初稿（学位论文） |
| 1945-49 | 在萨兰斯克师范学院任教 |

## 晚期与"重新发现" / Late Period & Rediscovery

| 年份 | 事件 |
|------|------|
| 1961 | 莫斯科的年轻学者"发现"了巴赫金，开始整理其遗稿 |
| 1963 | 《陀思妥耶夫斯基创作问题》修订再版，引起广泛关注 |
| 1965 | 出版《弗朗索瓦·拉伯雷的创作》——狂欢化理论 |
| 1969 | 获准迁居莫斯科 |
| 1970 | 发表《答〈新世界〉编辑部问》 |
| 1975 | 3月7日在莫斯科去世，享年79岁 |

## 身后影响 / Posthumous Impact

| 年份 | 事件 |
|------|------|
| 1979 | Julia Kristeva 将巴赫金引入法国理论界 |
| 1981 | *The Dialogic Imagination* 英文版出版，巴赫金在英语世界获得广泛声誉 |
| 1984 | *Problems of Dostoevsky's Poetics* 英文版出版 |
| 1986 | *Speech Genres and Other Late Essays* 英文版出版 |
| 1980s-90s | 巴赫金研究成为全球性学术产业 |
| 1990s | 巴赫金全集中文版（六卷）出版 |
| 2000s | 对话主义被引入教育学、传播学、组织研究等领域 |
