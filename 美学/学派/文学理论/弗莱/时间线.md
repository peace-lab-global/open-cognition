# 弗莱年表 / Frye Timeline

## 早年与学术形成 / Early Years & Formation

| 年份 | 事件 |
|------|------|
| 1912 | 7月14日出生于加拿大新不伦瑞克省舍波第 |
| 1929 | 进入多伦多维多利亚学院 |
| 1933 | 获学士学位 |
| 1936 | 在多伦多大学以神学毕业，受按立为联合教会牧师 |
| 1936-39 | 赴牛津大学莫顿学院深造 |
| 1939 | 返回维多利亚学院任教 |

## 学术巅峰 / Academic Ascendancy

| 年份 | 事件 |
|------|------|
| 1947 | 出版《威严的对称》——布莱克研究的里程碑 |
| 1948 | 获加拿大皇家学会奖章 |
| 1957 | 出版《批评的剖析》——20世纪文学批评最重要的著作之一 |
| 1959 | 被任命为维多利亚学院院长 |
| 1963 | 出版《身份的寓言》 |
| 1964 | 出版《受过教育的想象力》——CBC 广播讲座 |
| 1971 | 出版《批评之路》 |
| 1976 | 出版《世俗经文》——罗曼司叙事研究 |

## 晚期 / Late Period

| 年份 | 事件 |
|------|------|
| 1982 | 出版《伟大代码》——圣经与文学关系研究 |
| 1986 | 出版《论莎士比亚》 |
| 1990 | 出版《有力度的话语》——《伟大代码》续篇 |
| 1991 | 1月23日在多伦多去世，享年78岁 |

## 身后影响 / Posthumous Impact

| 年份 | 事件 |
|------|------|
| 1996 | 《诺思罗普·弗莱全集》开始出版 |
| 2000s | 弗莱日记出版（两卷），揭示其私人思考 |
| 2012 | 百年诞辰，多伦多大学举办国际会议 |
| 2010s | 弗莱研究在全球范围内的持续讨论 |
