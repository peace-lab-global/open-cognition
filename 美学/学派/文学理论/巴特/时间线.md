# 巴特年表 / Barthes Timeline

## 早年与学术形成 / Early Years & Formation

| 年份 | 事件 |
|------|------|
| 1915 | 11月12日出生于法国瑟堡，父亲在第一次世界大战中阵亡 |
| 1924 | 随母亲和祖母在巴黎和巴约纳生活 |
| 1935 | 进入巴黎大学学习古典文学 |
| 1939 | 获古典文学学位 |
| 1942-45 | 因肺结核在比利牛斯山疗养，期间大量阅读 |
| 1947-50 | 在巴黎索邦大学和罗马尼亚布加勒斯特大学任教 |

## 结构主义时期 / Structuralist Period

| 年份 | 事件 |
|------|------|
| 1953 | 出版《写作的零度》——首部专著 |
| 1954 | 在法国国家科学研究中心任职 |
| 1957 | 出版《神话学》——大众文化符号学的经典 |
| 1960 | 进入高等研究实践学院 (EPHE) |
| 1964 | 出版《符号学原理》 |
| 1966 | 发表《叙事作品结构分析导论》 |
| 1967 | 发表《作者之死》——后结构主义的宣言之一 |
| 1968 | 发表《真实效果》 |

## 后结构主义时期 / Post-Structuralist Period

| 年份 | 事件 |
|------|------|
| 1970 | 出版《S/Z》和《符号帝国》——从结构主义向后结构主义转向 |
| 1971 | 被任命为法兰西学院"文学符号学"教授 |
| 1971 | 发表《从作品到文本》 |
| 1973 | 出版《文本的愉悦》 |
| 1975 | 出版《罗兰·巴特自述》 |
| 1977 | 出版《恋人絮语》——大畅销书 |

## 最后岁月 / Final Years

| 年份 | 事件 |
|------|------|
| 1978 | 母亲去世，深刻影响巴特晚期写作 |
| 1980 | 出版《明室》——摄影现象学 |
| 1980 | 2月25日在巴黎被洗衣车撞倒 |
| 1980 | 3月26日因伤不治去世，享年64岁 |

## 身后影响 / Posthumous Impact

| 年份 | 事件 |
|------|------|
| 1982 | *A Lover's Discourse* 英文版出版，英语世界巴特热潮 |
| 1987-2002 | 巴特全集 (Œuvres complètes) 五卷本出版 |
| 1995 | 巴特诞辰80周年，全球学术纪念活动 |
| 2002 | 新版全集出版 |
| 2010s | 巴特日记、课程笔记等新材料陆续出版 |
| 2015 | 百年诞辰，国际会议和出版物 |
