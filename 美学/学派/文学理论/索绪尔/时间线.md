# 索绪尔年表 / Saussure Timeline

## 早年与学术形成 / Early Years & Formation

| 年份 | 事件 |
|------|------|
| 1857 | 11月26日出生于日内瓦，出身学术世家 |
| 1876 | 进入莱比锡大学学习比较语言学 |
| 1878 | 转至柏林大学 |
| 1879 | 发表《论印欧语元音的原始系统》——喉音理论，21岁即展示非凡才华 |
| 1880 | 以梵语论文获莱比锡大学博士学位 |
| 1881 | 赴巴黎，在高等研究实践学院任教 |

## 巴黎时期 / Paris Period

| 年份 | 事件 |
|------|------|
| 1881-91 | 在巴黎任教十年，培养了梅耶、格拉蒙等著名语言学家 |
| 1891 | 返回日内瓦大学，任印欧比较语言学教授 |

## 日内瓦时期 / Geneva Period

| 年份 | 事件 |
|------|------|
| 1891-1906 | 在日内瓦大学任教，但很少发表研究成果 |
| 1906-07 | 第一次讲授普通语言学课程 |
| 1908-09 | 第二次讲授普通语言学课程 |
| 1910-11 | 第三次讲授普通语言学课程 |
| 1913 | 2月22日在日内瓦附近的维弗瓦城堡去世，享年55岁 |

## 身后影响 / Posthumous Impact

| 年份 | 事件 |
|------|------|
| 1916 | Bally 和 Sechehaye 编辑出版《普通语言学教程》 |
| 1940s-50s | 列维-斯特劳斯将索绪尔结构方法引入人类学 |
| 1950s-60s | 法国结构主义运动——巴特、拉康、福柯等受索绪尔影响 |
| 1966 | 拉康"无意识具有语言的结构"——精神分析的符号学化 |
| 1967 | 德里达《论文字学》——对索绪尔的解构性重读 |
| 1993 | 《第三次普通语言学教程》出版 |
| 2002 | 索绪尔原始手稿出版，引发对《教程》编辑过程的反思 |
| 2010s | 索绪尔研究的新材料和新解读持续涌现 |
