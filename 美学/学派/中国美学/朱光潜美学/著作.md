# 朱光潜作品总览 · Zhu Guangqian Works

## 总览

朱光潜的学术生涯横跨六十余年（1920s–1980s），著有美学专著、文学评论、翻译作品数十种。其创作可分为三个时期：早期引介西方美学（1930s）、中期中西融合与美学论争（1940s–1950s）、晚期史学建构与回归传统（1960s–1980s）。

## 一、美学专著

| 书名 | 年份 | 特色 |
|---|---|---|
| 《谈美》 | 1932 | 通俗美学入门，书信体，15封致青年的信 |
| 《悲剧心理学》 | 1933 | 博士论文（法文），悲剧快感的心理学分析 |
| 《文艺心理学》 | 1936 | 系统引介克罗齐、立普斯、布洛等西方美学理论 |
| 《诗论》 | 1942 | 中西诗歌美学的比较研究 |
| 《诗论》增订版 | 1948 | 扩充中国诗学部分，增加"陶渊明"等章节 |
| 《谈文学》 | 1946 | 文学创作与鉴赏的通俗指导 |
| 《西方美学史》（上卷） | 1963 | 从古希腊到19世纪初 |
| 《西方美学史》（下卷） | 1963 | 从19世纪到20世纪（1979年修订） |
| 《美学拾穗集》 | 1980 | 晚年美学思考的结集 |
| 《谈美书简》 | 1982 | 《谈美》的续篇，回应新时代美学问题 |

## 五、阅读路径

### 入门
1. 《谈美》— 美学启蒙的经典（约150页）
2. 《给青年的十二封信》— 文化启蒙（约100页）
3. 《谈修养》— 生活美学（约120页）

### 进阶
4. 《文艺心理学》— 西方美学引介
5. 《诗论》— 中西诗歌比较
6. 《西方美学史》选读

### 深潜
7. 《西方美学史》全书
8. 《悲剧心理学》
9. 夏中义《朱光潜美学论纲》
