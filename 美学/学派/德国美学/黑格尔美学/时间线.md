# 黑格尔美学年表 / Hegel Aesthetics Timeline

## 早年与学术形成 / Early Years & Formation

| 年份 | 事件 |
|------|------|
| 1770 | 8月27日出生于斯图加特，父亲为公务员 |
| 1788 | 进入图宾根大学神学院，与荷尔德林、谢林同窗 |
| 1793 | 毕业，先后在伯尔尼和法兰克福担任家庭教师 |
| 1801 | 任耶拿大学私人讲师 |
| 1805 | 被任命为耶拿大学副教授 |
| 1807 | 出版《精神现象学》——法国军队进入耶拿当日完成 |

## 体系建构 / System Building

| 年份 | 事件 |
|------|------|
| 1808-16 | 任纽伦堡中学校长 |
| 1812-16 | 出版《逻辑学》两卷 |
| 1816 | 任海德堡大学教授 |
| 1817 | 出版《哲学百科全书》——首次将美学纳入完整体系 |
| 1818 | 被任命为柏林大学教授，接替费希特的教席 |

## 柏林时期与美学讲演 / Berlin Period & Aesthetics Lectures

| 年份 | 事件 |
|------|------|
| 1818 | 开始柏林大学生涯，成为普鲁士最具影响力的哲学家 |
| 1820-21 | 首次讲授美学 |
| 1821 | 出版《法哲学原理》 |
| 1823 | 第二次讲授美学（柏林） |
| 1826 | 第三次讲授美学 |
| 1827 | 创办《科学评论年鉴》，成为黑格尔学派的核心刊物 |
| 1828-29 | 最后一次讲授美学 |
| 1830 | 被选为柏林大学校长 |
| 1831 | 11月14日因霍乱在柏林去世，享年61岁 |

## 身后影响 / Posthumous Impact

| 年份 | 事件 |
|------|------|
| 1835 | Hotho 整理出版《美学讲演录》三卷 |
| 1830s-40s | 黑格尔学派分裂为左派（青年黑格尔派）和右派 |
| 1843-44 | 马克思《1844年经济学哲学手稿》——唯物辩证法 |
| 1840s | 克尔凯郭尔、尼采对黑格尔总体性的批判 |
| 1910-20 | 意大利新黑格尔主义（Croce）的美学阐释 |
| 1930s-40s | 法国新黑格尔主义（Kojève, Hyppolite） |
| 1964 | Danto 提出"艺术界"概念，继承黑格尔的艺术终结论 |
| 1970s-80s | Gadamer 对黑格尔美学的阐释学解读 |
| 1990s-2000s | 黑格尔美学在全球艺术哲学中的持续讨论 |
