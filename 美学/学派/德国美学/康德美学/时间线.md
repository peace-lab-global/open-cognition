# 康德美学年表 / Kant Aesthetics Timeline

## 早年与学术形成 / Early Years & Academic Formation

| 年份 | 事件 |
|------|------|
| 1724 | 4月22日出生于东普鲁士柯尼斯堡，父亲为马具匠 |
| 1740 | 进入柯尼斯堡大学，受 Martin Knutzen 影响接触牛顿物理学和沃尔夫哲学 |
| 1746 | 因父亲去世辍学，担任家庭教师近十年 |
| 1755 | 以《论火》获博士学位，开始在柯尼斯堡大学任私人讲师 |
| 1764 | 发表《对美感和崇高感的观察》——最早的美学著述 |
| 1770 | 被任命为柯尼斯堡大学逻辑学和形而上学正教授 |

## 批判哲学时期 / Critical Period

| 年份 | 事件 |
|------|------|
| 1781 | 出版《纯粹理性批判》——"哥白尼式转向" |
| 1783 | 出版《未来形而上学导论》 |
| 1785 | 出版《道德形而上学原理》 |
| 1787 | 出版《纯粹理性批判》第二版（大幅修改） |
| 1788 | 出版《实践理性批判》 |
| 1790 | 出版《判断力批判》——美学与目的论的批判完成 |

## 晚期 / Late Period

| 年份 | 事件 |
|------|------|
| 1793 | 出版《单纯理性限度内的宗教》 |
| 1795 | 发表《永久和平论》 |
| 1797 | 出版《道德形而上学》 |
| 1798 | 出版《实用人类学》；退出大学教职 |
| 1804 | 2月12日在柯尼斯堡去世，享年79岁 |

## 美学思想影响史 / Aesthetic Influence History

| 年份 | 事件 |
|------|------|
| 1795 | 席勒《审美教育书简》——对康德美学的创造性发展 |
| 1798-1800 | 谢林《先验唯心论体系》——将美学置于哲学最高点 |
| 1818 | 叔本华《作为意志和表象的世界》——接受并改造康德美学 |
| 1835 | 黑格尔《美学讲演录》出版——对康德形式主义的批判性回应 |
| 1871 | 尼采《悲剧的诞生》——以酒神/日神二元论超越康德 |
| 1892 | Cohen《康德美学》——新康德主义的美学阐释 |
| 1954 | Marcuse《审美维度》——马克思主义对康德美学的继承 |
| 1960s-70s | 分析美学（Dickie, Danto）——重新审视康德美学的基本概念 |
| 1987 | Derrida《绘画中的真理》——解构主义解读康德 |
| 1990s-2000s | 环境美学（Berleant, Carlson）——康德自然美学的当代延伸 |
