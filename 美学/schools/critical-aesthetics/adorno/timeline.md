# 阿多诺年表 / Adorno Timeline

## 早年与学术奠基 / Early Years & Academic Formation

| 年份 | 事件 |
|------|------|
| 1903 | 9月11日出生于德国法兰克福，父亲为葡萄酒商，母亲为歌唱家 |
| 1921 | 进入法兰克福大学，师从 Hans Cornelius 学习哲学 |
| 1924 | 以论文《胡塞尔现象学中先验事物的理论》获博士学位 |
| 1925 | 前往维也纳，跟随 Alban Berg 学习作曲，结识 Alban Berg 和 Anton Webern |
| 1927 | 返回法兰克福，在大学担任哲学助教 |
| 1931 | 以《克尔凯郭尔：审美阶段的建构》获得授课资格 (Habilitation) |
| 1932 | 开始在《社会研究杂志》发表文章，与霍克海默密切合作 |

## 流亡岁月 / Exile Years

| 年份 | 事件 |
|------|------|
| 1933 | 纳粹上台，因其犹太血统被剥夺教职 |
| 1934 | 流亡英国，在牛津大学莫顿学院学习 |
| 1938 | 移居美国纽约，加入社会研究所 |
| 1938-41 | 参与普林斯顿广播研究项目，研究广播对听众的影响 |
| 1941 | 迁往洛杉矶，与霍克海默合作撰写《启蒙辩证法》 |
| 1944 | 《启蒙辩证法》完成（正式出版于1947年） |
| 1947 | 与 Else Frenkel-Brunswik 等合著《权威人格》 |
| 1949 | 出版《新音乐哲学》 |

## 回归与成熟 / Return & Maturation

| 年份 | 事件 |
|------|------|
| 1950 | 返回法兰克福，任法兰克福大学社会学与哲学教授 |
| 1951 | 出版《最低限度的道德》，格言体哲学杰作 |
| 1952 | 出版《论瓦格纳》 |
| 1955 | 出版《棱镜》（含"文化批评与社会"一文） |
| 1958 | 与 Horkheimer 共同担任社会研究所主任 |
| 1962 | 出版《音乐社会学导论》 |
| 1963 | 被选为德国社会学学会主席 |

## 晚年与遗产 / Final Years & Legacy

| 年份 | 事件 |
|------|------|
| 1966 | 出版《否定辩证法》——其最重要的独立哲学著作 |
| 1966 | 发表广播演讲《奥斯维辛之后的教育》 |
| 1967 | 发表演讲《论艺术的承诺性》，引发与萨特主义者的论争 |
| 1968 | 五月风暴——与学生领袖发生冲突，拒绝将理论直接转化为街头行动 |
| 1968 | 学生占领法兰克福大学社会研究所，阿多诺报警驱散 |
| 1969 | 完成《美学理论》手稿 |
| 1969 | 8月6日在瑞士瓦莱州的 Visp 因心脏病突发去世，享年65岁 |

## 身后影响 / Posthumous Impact

| 年份 | 事件 |
|------|------|
| 1970 | 《美学理论》出版（Gretel Adorno 和 Rolf Tiedemann 编辑） |
| 1970s | 全集 *Gesammelte Schriften*（20卷）陆续出版 |
| 1980s-90s | 阿多诺研究英语世界复兴，出现大量研究专著 |
| 1993 | 法兰克福成立阿多诺研究所 |
| 2003 | 百年诞辰纪念，全球学术活动 |
