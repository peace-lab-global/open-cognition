# 艺术哲学 / Aesthetics

## 学科定位

艺术哲学（Aesthetics）是哲学的分支学科，研究美、艺术、审美经验及其本质。
该学科追问：什么是美？艺术的本质是什么？审美判断如何可能？审美经验在人类生活中占据何种地位？
