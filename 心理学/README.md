# 心理学 · Psychology

本目录覆盖心理学主要理论流派，按流派组织思想家条目，按概念组织跨流派观念。

## 流派树

```
心理学
├── 精神分析 (psychoanalysis / analytical / individual)
│   ├── 弗洛伊德 · 潜意识与本我自我超我
│   ├── 荣格 · 集体无意识与原型
│   └── 阿德勒 · 目的论与自卑超越
├── 行为主义 (behaviorism)
│   └── 斯金纳 · 操作性条件反射
├── 认知与认知行为 (cognitive)
│   ├── 皮亚杰 · 认知发展
│   ├── 贝克 · 认知行为疗法
│   └── 卡尼曼 · 双系统
├── 人本主义 (humanistic)
│   ├── 马斯洛 · 需求层次
│   └── 罗杰斯 · 来访者中心
└── 积极心理学 (positive)
    └── 米哈里 · 心流
```
