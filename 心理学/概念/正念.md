# 正念 / Mindfulness

## 一句话定义

正念是有意识地、不带评判地把注意力集中于当下经验——一种源于佛教禅修、经当代心理学操作化定义的觉察状态，既是冥想技术，也是心理品质。

## 历史脉络

```
佛教念处（satipaṭṭhāna）：四念处的"如实观照"
    ↓
禅宗/内观：见性 / 洞察身心无常
    ↓
卡巴金（1979）：剥离宗教语境，创立正念减压疗法（MBSR）
    ↓
当代：MBSR / MBCT / ACT，正念进入主流临床心理学与神经科学
```

## 核心要义

### 1. 觉察的三个要素
- **有意**（intentional）：主动转向当下，而非被习惯牵引
- **当下**（present-moment）：锚定于此时此地的身体感受、情绪、念头
- **不评判**（non-judgmental）：对经验保持观察而非反应，不急于贴好/坏标签

### 2. 操作化定义（Kabat-Zinn）
- "正念是以特定方式专注：有目的的、当下的、不评判的"
- 可测量、可训练——由此进入实验心理学与神经科学

### 3. 临床应用
- **MBSR**（正念减压）：慢性疼痛、压力、焦虑
- **MBCT**（正念认知疗法）：预防抑郁复发
- **ACT / DBT**：接纳承诺疗法、辩证行为疗法的核心组件

### 4. 注意力机制
- 正念训练增强注意控制的执行功能
- 默认模式网络（DMN）活动降低——减少反刍与自我参照
- 与[卡尼曼](../schools/behavioral-economics/kahneman.md)系统二的"慢思考"形成对照：正念是把自动反应切换为有意觉察

## 通俗 vs 学术

| 层次 | 含义 |
|------|------|
| 日常 | "活在当下""专注做事"的笼统说法 |
| 临床 | 有结构的8周训练程序（MBSR），有剂量、有疗效证据 |
| 佛学 | 八正道之"正念"（sammā-sati），与[八识体系](../../宗教/佛教/概念/cognitive-theory/八识体系.md)的第六识"意识"直接相关 |

## 跨学科关联

- **宗教传统**：源出[佛教冥想与默观](../../宗教/概念/冥想与默观.md)；[禅宗](../../宗教/传统/佛教/禅宗.md)的"只管打坐"是其东亚形态
- **神经科学**：[西格尔](../schools/interpersonal-neurobiology/siegel.md)的"正念大脑"整合研究——前额叶增厚、杏仁核反应性降低
- **积极心理学**：与[契克森米哈伊](../schools/positive/csikszentmihalyi.md)的"心流"共享"完全沉浸于当下"的结构，但正念强调"觉察"而非"任务投入"
- **认知科学**：正念作为研究注意与元认知的实验范式

## 常见误读

1. ❌ "正念就是放松" → ✅ 正念是觉察训练，可能伴随不适感，目标不是放松而是看清
2. ❌ "正念是佛教的" → ✅ 操作化定义已剥离宗教，但理解其佛学源流能深化实践
3. ❌ "正念万能" → ✅ 疗效有边界，对严重创伤、急性精神病有风险
