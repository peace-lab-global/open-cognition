# 记忆 · Memory

## 一句话定义

记忆（memory）是信息编码（encoding）、存储（storage）与提取（retrieval）的动态过程；它不是对过去的忠实复制，而是每次提取时的主动建构（constructive reconstruction），佛教以"念"（sati）指称对当下经验的清醒觉知——与西方记忆概念形成互补。

## 历史脉络

艾宾浩斯（Ebbinghaus, 1885）以无意义音节实验开创记忆的科学研究，发现遗忘曲线。巴特莱特（Bartlett, 1932）在《记忆》中证明记忆是"图式"（schema）驱动的建构过程，而非被动回放。阿特金森和希夫林（Atkinson-Shiffrin, 1968）提出多重存储模型：感觉记忆→短时记忆→长时记忆。

巴德利（Baddeley, 1986）以工作记忆（working memory）模型取代被动的"短时存储"概念，包含中央执行器、语音环路、视空间画板和情景缓冲器。图尔文（Tulving）区分情景记忆（episodic）与语义记忆（semantic）。洛夫特斯（Loftus）的研究证明记忆极易被暗示和误导信息扭曲。佛教的"念"（sati/mindfulness）不是回忆过去，而是对当下身心现象的持续觉知——四念处（身受心法）是系统训练。

## 核心要义

1. **编码-存储-提取（Encoding-Storage-Retrieval）**：记忆三阶段模型；编码深度影响存储质量（加工水平理论）。
2. **工作记忆（Working Memory）**：主动操作信息的有限容量系统；巴德利模型含中央执行器与两个从属系统。
3. **情景记忆与语义记忆（Episodic vs Semantic）**：情景记忆是"何时何地"的个人经历；语义记忆是脱离情境的知识。
4. **建构性记忆（Constructive Memory）**：记忆是每次提取时的重新建构，受图式、情绪和暗示影响（巴特莱特、洛夫特斯）。
5. **佛教念（Sati）**：四念处修行中的"念"是对当下身、受、心、法的持续觉知，非回忆而是清醒注意。

## 通俗 vs 学术

| 通俗用法 | 学术原义 |
|----------|----------|
| "记忆力好"＝像录像机一样回放 | 记忆是建构过程，每次回忆都是重新组装 |
| "催眠能恢复真实记忆" | 催眠增加暗示感受性，可能制造虚假记忆 |
| "正念就是记住当下" | sati 更接近"清醒觉知"而非"记忆"的存储功能 |

## 应用场景

- **教育**：间隔重复（spaced repetition）与检索练习（retrieval practice）
- **司法**：目击证人记忆的不可靠性与访谈技术改进
- **心理治疗**：创伤记忆的再加工（EMDR）
- **老年医学**：记忆衰退的早期识别与认知训练

## 常见误读

1. ❌ "记忆像硬盘，存进去就不变了" → ✅ 记忆每次提取都会被重新建构和修改（再巩固理论）。
2. ❌ "多重复就能记住一切" → ✅ 间隔效应、测试效应和精细加工比简单重复更有效。
