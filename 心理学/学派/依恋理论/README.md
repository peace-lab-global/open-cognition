# 依恋理论 · Attachment Theory

## 一句话定义

依恋理论是 20 世纪中后期由约翰·鲍尔比创立、玛丽·安斯沃思实证发展的发展心理学理论，研究人类与特定他人形成情感纽带的进化基础、个体差异及其对终身关系模式的影响。

## 历史脉络

**临床起源（1930—1950 年代）**
鲍尔比作为精神分析师，在临床工作中观察到早期母婴分离与儿童反社会行为、情感冷漠之间的关联。1944 年发表的《四十四名少年小偷》首次系统呈现早期分离与“无情型人格”的统计关系。

**多学科整合（1950—1969）**
面对精神分析界对“母性剥夺”假说的批评，鲍尔比转向行为学（洛伦茨的印记、廷伯根的本能行为）、控制论（反馈与调节）、认知心理学（内部表征）和精神分析客体关系理论，提出依恋是一个独立的、进化而来的行为系统。

**理论确立与实证验证（1969—1990）**
1969 年《依恋与丧失》第一卷出版，系统阐述依恋行为系统、安全基地与内部工作模型。安斯沃思通过“陌生情境”实验识别出安全型、回避型、焦虑—矛盾型三种依恋类型；玛丽·梅因（Mary Main）后来补充混乱型。鲍尔比晚年将理论扩展至成人心理治疗（《安全基地》，1988）。

**当代发展（1990 年代至今）**
依恋理论进入成人浪漫关系、神经科学、心理治疗（情绪聚焦治疗、心智化治疗）、组织心理学与数字时代研究。成人依恋访谈（AAI）与多种自评量表成为标准研究工具。

## 核心主张

1. **依恋是基本行为系统**
   依恋行为与觅食、性行为同等重要，是人类进化而来的生存机制，其功能是与照顾者保持亲近以获得保护。

2. **安全基地与探索**
   可靠的照顾者提供“安全基地”，使儿童能够自信地探索环境；遭遇威胁时返回寻求安慰。安全基地是健康心理发展的必要条件。

3. **内部工作模型**
   个体基于早期依恋经验形成关于自我、他人与关系的心理表征。这些模型在无意识层面运作，影响个体对后续关系的预期与反应。

4. **依恋类型具有相对稳定性但可改变**
   依恋模式在婴儿期后具有一定稳定性，但并非固定不变。新的关系经验、心理治疗与反思功能的提升可以修正早期不安全模式。

5. **依恋贯穿生命周期**
   依恋需要不仅限于婴儿期，而是贯穿儿童、青少年、成人亲密关系与老年阶段的普遍人类需要。

## 关键概念

- **依恋行为系统（Attachment Behavioural System）**：以维持与依恋对象亲近为目标的行为组织系统。
- **安全基地（Secure Base）**：照顾者提供的心理与物理基地，支持探索行为。
- **内部工作模型（Internal Working Models）**：关于自我是否值得被爱、他人是否可依赖的无意识表征。
- **依恋类型**：安全型、回避型、焦虑—矛盾型、混乱型。
- **分离反应三阶段**：抗议（protest）、绝望（despair）、脱离（detachment）。
- **敏感回应（Sensitivity）**：照顾者准确、及时地回应婴儿信号，是安全依恋形成的关键因素。

## 常见误读

**误读一：依恋理论要求母亲必须全职照顾孩子。**
纠正：鲍尔比强调的是照顾的敏感性、连续性与可及性，而非照顾者的性别或是否全职。父亲、祖父母或稳定的专业照顾者均可成为安全基地。

**误读二：依恋类型在婴儿期固定后不可改变。**
纠正：依恋模式虽具有稳定性，但新的关系经验（如稳定的伴侣关系、心理治疗）可以产生“习得的安全感”（earned security）。

**误读三：依恋理论只适用于母婴关系。**
纠正：依恋理论已扩展至友谊、浪漫关系、治疗关系，甚至人与宠物、人与地方的联结，并贯穿整个生命周期。
