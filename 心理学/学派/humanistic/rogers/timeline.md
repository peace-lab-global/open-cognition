# 卡尔·罗杰斯 · Carl Rogers — 生平年表 / Life Timeline

## 早年与教育 / Early Life and Education

| 年份 | 事件 / Event | 详情 / Details |
|------|------------|---------------|
| 1902 | 出生 / Birth | 1月8日出生于美国伊利诺伊州奥克帕克（Oak Park, Illinois），在六个孩子中排行第四。父亲为土木工程师，母亲为虔诚的基督教徒。 |
| 1912 | 迁居农场 / Move to Farm | 全家迁至威斯康星州农村，罗杰斯在农场度过了青少年时期，培养了独立性和对科学的兴趣。 |
| 1919 | 入学威斯康星大学 / University of Wisconsin | 进入威斯康星大学麦迪逊分校，最初学习农业。 |
| 1921 | 宗教觉醒 / Religious Awakening | 参加基督教青年会（YMCA）学生会议，经历深刻的宗教体验，从农业转向历史学。 |
| 1922 | 中国之行 / Trip to China | 参加国际基督教学生会议，在中国旅行六个月。这段跨文化经历深刻动摇了他的宗教基要主义信仰，培养了对文化多样性的尊重。 |
| 1924 | 大学毕业 / Graduation | 获威斯康星大学历史学学士学位。与海伦·艾略特（Helen Elliott）结婚。 |
| 1924 | 进入纽约协和神学院 / Union Theological Seminary | 进入纽约协和神学院学习神学，但逐渐对教条化的宗教教育产生怀疑。 |
| 1926 | 转入哥伦比亚大学 / Columbia University | 转入哥伦比亚大学师范学院，师从 Goodwin Watson 和 William H. Kilpatrick，学习临床心理学和教育心理学。 |
| 1928 | 博士学位 / Doctoral Studies | 开始在哥伦比亚大学攻读博士学位（1931年获教育学博士 Ed.D.）。 |

## 罗切斯特时期 / The Rochester Period (1928–1940)

| 年份 | 事件 / Event | 详情 / Details |
|------|------------|---------------|
| 1928 | 罗切斯特儿童指导诊所 / Rochester Child Guidance Clinic | 在纽约州罗切斯特市儿童指导诊所开始职业生涯，从事受虐儿童和青少年的评估和治疗工作。 |
| 1930 | 奥托·兰克的影响 / Influence of Otto Rank | 接触奥托·兰克（Otto Rank）的关系治疗思想，兰克强调治疗关系本身的重要性，对罗杰斯产生深远影响。 |
| 1931 | 获博士学位 / Ed.D. Awarded | 获哥伦比亚大学教育学博士学位。 |
| 1939 | 首部著作出版 / First Book Published | 《问题儿童的临床治疗》(*The Clinical Treatment of the Problem Child*) 出版。 |
| 1940 | 学术转型 / Academic Transition | 在罗切斯特的十二年临床经验使他逐渐质疑精神分析的权威模式，开始发展自己的治疗方法。发表"治疗的新概念"演讲，引起广泛关注。 |

## 学术生涯与范式建立 / Academic Career and Paradigm Building (1940–1963)

| 年份 | 事件 / Event | 详情 / Details |
|------|------------|---------------|
| 1940 | 俄亥俄州立大学 / Ohio State University | 受聘为俄亥俄州立大学心理学教授，开始系统阐述自己的治疗理论。 |
| 1942 | 《咨询与心理治疗》/ *Counseling and Psychotherapy* | 出版第二部著作，首次提出"非指导性治疗"理念，标志着与精神分析的分道扬镳。 |
| 1945 | 芝加哥大学 / University of Chicago | 受聘为芝加哥大学心理学教授，建立咨询中心（Counseling Center）。 |
| 1949 | 系统录制治疗会话 / Recording Therapy Sessions | 在芝加哥大学开始系统录制和分析治疗会话，开创心理治疗过程的实证研究。 |
| 1951 | 《来访者中心疗法》/ *Client-Centered Therapy* | 出版集大成之作，完整阐述来访者中心范式的理论、方法和研究基础。 |
| 1956 | 杰出科学贡献奖 / Distinguished Scientific Contribution Award | 获美国心理学会（APA）首个杰出科学贡献奖，表彰其在心理治疗实证研究方面的开创性贡献。 |
| 1957 | 经典论文 / Seminal Paper | 发表《治疗性人格改变的充分必要条件》(*The Necessary and Sufficient Conditions of Therapeutic Personality Change*)，提出治疗关系的六个充分必要条件。 |
| 1957 | 威斯康星大学 / University of Wisconsin | 受聘为威斯康星大学麦迪逊分校心理学和精神病学教授。 |
| 1959 | 系统理论 / Comprehensive Theory | 发表最系统的理论论文《治疗、人格和人际关系的理论》，提出19条命题。 |
| 1961 | 《论人的成长》/ *On Becoming a Person* | 出版最具影响力的著作，将治疗思想拓展为关于人的成长的哲学。 |
| 1962 | 与斯金纳辩论 / Debate with Skinner | 与行为主义者B.F.斯金纳就自由意志和决定论进行公开辩论。 |

## 学术传承 / Academic Legacy

罗杰斯去世后，个人中心取向持续发展：

After Rogers' death, the person-centered approach continued to develop:

- **世界个人中心协会 (World Association for Person-Centered and Experiential Psychotherapy and Counseling, WAPCEPC)** 持续推动学术交流和培训
- **动机性访谈 (Motivational Interviewing)**（Miller & Rollnick, 1991）直接继承罗杰斯精神，成为全球应用最广的咨询方法之一
- **情绪聚焦治疗 (Emotion-Focused Therapy)**（Greenberg, 2002）整合罗杰斯和格式塔传统
- **心理治疗共同因素研究 (Common Factors Research)** 持续验证罗杰斯关于治疗关系核心性的洞见
