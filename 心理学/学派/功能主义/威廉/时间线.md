# 威廉·詹姆斯生平年表 · William James Life Timeline

---

## 早年与教育 / Early Life and Education

| 年份 / Year | 事件 / Event | 备注 / Notes |
|---|---|---|
| 1842 | 1月11日出生于纽约市 | 父亲Henry James Sr.为宗教哲学家；弟弟Henry James后成为著名小说家 |
| 1855–1858 | 在欧洲接受教育 | 先后在日内瓦、波恩、巴黎等地学习 |
| 1858 | 在纽波特（Newport）学习绘画 | 师从William Morris Hunt，艺术志向 |
| 1861 | 进入哈佛大学劳伦斯科学学院 | 学习化学 |
| 1861 | 南北战争爆发 | 两个弟弟参加联邦军，詹姆斯因健康原因未能参军 |
| 1864 | 转入哈佛医学院 | 从化学转向医学 |
| 1865–1866 | 参加阿加西斯（Agassiz）的亚马逊考察队 | 赴巴西进行自然科学考察，期间患天花 |
| 1867–1868 | 在德国柏林大学学习 | 跟随赫尔姆霍茨（Helmholtz）、杜布瓦-雷蒙（Du Bois-Reymond）学习生理学 |
| 1869 | 获得哈佛大学医学博士学位 | 但从未行医 |

---

## 心理危机与转向 / Psychological Crisis and Turning Point

| 年份 / Year | 事件 / Event | 备注 / Notes |
|---|---|---|
| 1870 | 经历严重的心理危机与抑郁 | 受自由意志问题和存在焦虑困扰 |
| 1870 | 阅读雷诺维叶（Renouvier）关于自由意志的论述 | "我的第一个自由意志行为将是相信自由意志"——这一决定成为"信仰意志"的个人起源 |
| 1872 | 心理状态逐渐恢复 | 开始参加查尔斯·皮尔士等人的"形而上学俱乐部" |
| 1873 | 心理危机基本结束 | 此后思想逐渐走向乐观和行动主义 |

---

## 学术生涯 / Academic Career

| 年份 / Year | 事件 / Event | 备注 / Notes |
|---|---|---|
| 1872 | 开始在哈佛大学教授比较解剖学和生理学 | 学术生涯的起点 |
| 1875 | 开设美国第一个心理学课程 | "生理学与心理学的关系"（The Relations between Physiology and Psychology） |
| 1875 | 建立非正式心理学实验室 | 略晚于冯特在莱比锡的实验室（1879） |
| 1876 | 成为哈佛大学生理学副教授 | |
| 1878 | 与Alice Howe Gibbens结婚 | 婚后心理状态趋于稳定，进入高产期 |
| 1880 | 成为哈佛大学哲学副教授 | 从生理学转向哲学 |
| 1885 | 成为哈佛大学哲学正教授 | |
| 1889 | 成为哈佛大学心理学正教授 | |
| 1890 | 出版《心理学原理》（两卷本） | 耗时12年完成，成为心理学史里程碑 |
| 1892 | 出版《心理学简编》 | 教学用缩写本 |
| 1897 | 出版《信仰意志及其他论文》 | 从心理学转向哲学和宗教研究 |
| 1898 | 在加州大学演讲"哲学概念与实际结果" | 首次公开使用"实用主义"一词 |
| 1899 | 出版《与教师谈心理学》 | 教育心理学著作 |

---

## 晚年与哲学 / Later Years and Philosophy

| 年份 / Year | 事件 / Event | 备注 / Notes |
|---|---|---|
| 1898 | 健康状况恶化 | 心脏病和失眠困扰 |
| 1901–1902 | 在爱丁堡大学发表吉福德讲座（Gifford Lectures） | 题为"自然宗教的各个方面" |
| 1902 | 出版《宗教经验之种种》 | 宗教心理学经典 |
| 1903 | 参与反对美国帝国主义的争论 | 反对美菲战争 |
| 1904 | 发表"意识存在吗？"论文 | 挑战"意识"作为独立实体的概念 |
| 1905 | 在斯坦福大学讲学 | 旧金山大地震期间在校园 |
| 1906 | 在斯坦福大学和哈佛大学发表洛威尔讲座 | |
| 1907 | 出版《实用主义》 | 实用主义哲学宣言 |
| 1907 | 从哈佛大学退休 | 结束34年的教学生涯 |
| 1909 | 出版《多元的宇宙》 | 在牛津大学的希伯特讲座 |
| 1910 | 出版《实用主义与多元的宇宙》合订本 | |
| 1910 | 8月26日在新罕布什尔州乔科鲁阿去世 | 享年68岁 |
