# 认知心理学 · Cognitive Psychology

## 一句话定义

认知心理学是 20 世纪中叶兴起的心理学分支，以信息加工范式研究注意、知觉、记忆、语言、推理与问题解决等心智过程，将人脑比作接收、编码、存储与提取信息的复杂系统。

## 历史脉络

**认知革命（1950—1960 年代）**
在行为主义主导数十年后，乔姆斯基、米勒、奈瑟等人重新将内部心理表征与计算过程纳入合法研究对象。奈瑟《认知心理学》（1967）标志着该学科正式命名。

**信息加工模型（1960—1980 年代）**
阿特金森—谢夫林记忆模型、短时记忆容量研究、组块与编码策略成为核心议题。认知心理学与计算机科学、人工智能、语言学形成跨学科联盟。

**联结主义与具身转向（1980 年代以后）**
并行分布式加工（PDP）模型挑战符号计算假设，强调神经网络与分布式表征。随后具身认知、生态心理学与预测加工理论进一步拓展认知的边界。

## 核心主张

1. **心理表征**
   认知过程依赖对信息的内部表征，如命题、图像、图式与心理模型。

2. **信息加工阶段**
   刺激经过感觉登记、注意选择、短时加工与长时存储等阶段，形成可提取的知识。

3. **资源限制**
   注意与工作记忆容量有限，认知系统通过策略、自动化与组块来克服瓶颈。

4. **计算—表征取向**
   经典认知心理学将认知视为对符号表征的计算操作，后来模型则引入概率推断与神经实现。

## 主要研究领域

- **记忆**：编码、存储、提取、遗忘与记忆错觉。
- **注意**：选择性注意、分配性注意与执行控制。
- **语言**：词汇识别、句子加工与语用推理。
- **推理与决策**：启发式、偏差与双系统加工。
- **问题解决**：表征、策略与专家—新手差异。

## 与其他传统关系

- **行为主义**：认知心理学反对行为主义对内部状态的排斥，重新确立心智的可研究性。
- **认知神经科学**：将认知模型与脑成像、电生理数据结合，追问心理过程的神经基础。
- **人工智能**：彼此借用模型与隐喻，认知心理学为 AI 提供人类认知约束，AI 为心理学提供计算工具。

## 现当代影响

认知心理学的概念与方法论已渗透教育、临床、人机交互、用户体验、法律与公共政策，成为理解人类思维与决策失误（如认知偏差、记忆不可靠）的核心框架。
